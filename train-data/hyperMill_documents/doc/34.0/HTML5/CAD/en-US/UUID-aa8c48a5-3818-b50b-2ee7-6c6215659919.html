HC_ADDITIONAL_HCCONFIG

Read in user-defined presets for user-defined commands *.xml.

Set up the Windows environment variable. Enter the HC_ADDITIONAL_HCCONFIG environment variable under Variable name and enter the path, e.g. W:\Companyfolder\HCCONFIG under Variable value. The files located in the designated folder are evaluated in alphabetical order.

Configure the availability of commands

Generate an additional toolbar or menu with command calls and specify in which software based on the hyperMILL® CAD platform these entries should be available. This is a licence-independent procedure.

  1. Adapt HC_ADDITIONAL_HCCONFIG

    To do this, use the HC_ADDITIONAL_HCCONFIG Windows environment variable. Select an existing folder or generate a new folder. Enter this folder in the environment variable, for example, X:\server\cad\myconfig. The folder should not be read-only.

  2. Adapt HCCONFIG file

    Copy an existing *.hcconfig.xml file to this folder and adapt it:

    The ProgramName attribute can be used to include or exclude a software program. To exclude, a ! must be placed in front. A combination of include and exclude is not possible.

    Example 33. 

    ProgramName=“hyperCAD-S” includes the display of the toolbar in hyperMILL®. The toolbar will not be available in all other software programs.

    ProgramName=“!hyperCAD-S” excludes the display of the toolbar in hyperMILL®. The toolbar will be available in all other software programs.



    The attribute can be used in <Command> and <Configuration>.

    Write the software program names, for example, as follows: “hyperCAD-S”, “CAD Viewer”, “hyperMILL SHOP Viewer”.

    Once inserted, it may then look like this:

    Example 34. Menu item for opening an Excel file
    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
<Configuration>
	<Commands FormatVersion="1">
		<Command Name="Start_excel01"
		         ProgramName="hyperCAD-S"
		         Menu="Special"
		         MenuText="Start excel file1..."
		         Tooltip="Start excel file..."
		         Statustip="Start excel file..."
		         Enabled="IsDocumentActive"
		         Icon="%MYDIR%/files/images/excel-in.png"
		         Run="RunScript">
                         Execute('C:/Program Files (x86)
                                    /Microsoft Office
                                    /root/Office16
                                    /EXCEL.EXE',
                                 'c:/temp/test.xlsx');
                </Command>
	</Commands>
</Configuration>


    Example 35. Associated toolbar for opening an Excel file
    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
<Configuration>
	<Toolbars>
		<Toolbar Name="Excel toolbar">
			<Command Name="Start_excel01"
			         ProgramName="hyperCAD-S"/>
		</Toolbar>
	</Toolbars>
</Configuration>
