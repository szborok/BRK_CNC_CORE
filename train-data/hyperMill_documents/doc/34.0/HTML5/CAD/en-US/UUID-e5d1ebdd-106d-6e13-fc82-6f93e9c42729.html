1.4. Other software products

This software documentation also describes the commands contained in the hyperMILL® SHOP Viewer software and in the CAD Viewer software.

hyperMILL® SHOP Viewer

The software is used for shop floor-oriented display, analysis and documentation of hyperMILL® CAD / CAM data.

Note

This software documentation explains the complete range of CAD commands. Commands explained in this software documentation may not exist in the hyperMILL® SHOP Viewer software or may not be available to the described extent. Help information for CAM commands can be found in the CAM help.
