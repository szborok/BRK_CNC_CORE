5.3. Sketcher options

Options for the Sketch and V-sketch commands

File → Options → Sketcher options

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 21. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Reset all settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

Copy the text or value of the selected area to the clipboard.



All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The settings are stored in the documentSketcherProps.xml file.

Sketcher options

Graphic properties

Enable constraints: The position of a next point can be selected using constraints. Enable the use of constraints. The constraints can also be enabled and disabled using a keyboard shortcut. Set a shortcut under File → Options → Customize shortcuts → More commands.

Max. number of cached entities: Constrain the number of possible directions for preview lines of previously specified entities.

Constraints

Select constraints for the next position of a point.

V-sketch options

Angular tolerance: Specify up to which deviation of an angle of a curve such curve is still – for example, viewed as horizontal or vertical – adjusted automatically and the constraint is still automatically assigned.

Max. diagonal length of chamfers: To identify chamfers, enter a value for the maximum diagonal length of a chamfer.

Assign curves to V-sketch layer: All entities added to a V-sketch are moved to the current active layer, the V-sketch layer.

Strict chirality: If the option is not selected, the point-arc constraint (or the point-line constraint) can slide beyond the imaginary extension of the entity (line and circle). If the option is selected, it does not slide, but remains limited to the existing part of the arc or line.

Show modified curves: Display which curves of the turning contour have changed as a result of entering a tolerance (using the Modify linear tolerance command). A message is displayed in the Info tab. Select the message to highlight the changed entities. This makes it possible to detect unwanted changes and prevent these by adding further constraints.
