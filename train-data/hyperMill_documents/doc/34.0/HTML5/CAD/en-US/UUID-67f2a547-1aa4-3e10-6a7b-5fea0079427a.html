AutoCAD file formats

Import or save document in the AutoCAD *.dxf, *.dwg file format.

2D geometries, mesh entities, layers, colors, line types, text and dimensions are imported from the paper or model space as vectorial graphics.

3D-data is saved as mesh data and 2D data.

Various options are available to display the text imported from *.dxf format with the correct font:

Access AutoCAD font files with user variable:

In order to correctly display *.shx fonts, the software needs to know where the AutoCAD font files are located.

  • The font files must be stored in a folder with user access rights.

  • Enter the folder in a new Windows system variable or ACAD user variable.

This is the best approach for correctly representing content.

Convert font file to another existing font :

Another option is editing the file: ...\hyperCAD-S\<Version>\files\converters\dxfloadfontsmapping.xml.

The edited file can be stored in the folder ...\Users\<username>\AppData\Roaming\OPEN MIND\hyperCAD-S\<Version>\Converter in order to prevent it being overwritten during an update.

This file can be used to convert any *.shx font file into an existing *.ttf font file.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 41. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Reset all settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

Copy the text or value of the selected area to the clipboard.



All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

The settings are stored in the dxfloadfilesettingsvalues.xml file.
