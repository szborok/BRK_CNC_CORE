19.2. Python options

Select Python options.

File → Options → Python options

PYTHONToolkit

2024 | Update 2

Visual Studio Code folder: Select the folder in which the Python source code editor is installed, for example, this could be in C:\Users\[User]\AppData\Local\Programs\Microsoft VS Code.

User workspace folder: Select a folder to use as the root folder for the user workspace, for example, in C:\Users\Public\Documents\OPEN MIND\python (the default setting). If no folder is specified, the Roaming folder is used.

Library workspace folder: Select a folder to use as the root folder for the user workspace, for example, in C:\Users\Public\Documents\OPEN MIND\pythonlibrary (the default setting).

User template folder Enter a folder for your own *.py Python script templates, e.g., C:\Users\Public\Documents\OPEN MIND\pythonTemplates.

TCP port for debug: Select a port to connect hyperMILL® to the Python source code editor. If problems occur, this can be adjusted by an administrator.

Clean output at run: Clean output when script is executed.

Lazy graphic update: The graphics area and the model tree are not rebuilt after every action.
