Further file formats

The following file formats can also be imported:

  • *.e3 with job list and entity selection

  • *.e2

  • *.gkd

All attributes such as color, layer and number of isoparametric curves are supported. The edge IDs of solids are retained. The stored view is transferred for the direction, zoom section and render mode. Component information with hierarchy and names can be preserved. The prerequisite is that you have a working version of hyperCAD® installed in parallel and a license.

Bookmarks are converted to Visibility filters wherever possible. All layer settings within the bookmarks are taken over. None of the other settings in the bookmarks are taken over.

An *.e3 file must be saved with a hyperMILL® version of 2009.2 or higher and an active hyperMILL® session to ensure that all selections can be reproduced in hyperMILL®.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 30. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Reset all settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

Copy the text or value of the selected area to the clipboard.



All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

The settings are stored in the loadtdfilesettingsvalues.xml file.
