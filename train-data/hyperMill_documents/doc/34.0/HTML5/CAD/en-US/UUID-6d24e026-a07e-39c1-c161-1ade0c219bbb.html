Offset

Create a curve with an offset to an initial curve.

Curves → Offset

Create a curve with an offset to an initial curve.

Select

Curves: Select the initial curves for the offset . The number of selected elements is displayed.

Offset: Enter an offset value .

Invert: Reverse the direction of the offset for individual curves. For this purpose, select the curve preview.

All: Reverses the alignment of all curves that are displayed in the preview .

Transitions

None: No transitions are created between multiple curves .

Sharp: Transitions between multiple curves are created with sharp edges .

Rounded: Creates rounded transitions between multiple curves .

If Sharp and Rounded have been selected, the offset curves will be automatically trimmed if they intersect .

Multiple offset

Generate curves repeatedly at the distance selected with Offset. Enter the number of repetitions .

Symmetric: Generate curves in both alignments, ‘inside’ as well as ‘outside’ .

Reference plane

Selecting On current workplane creates the curve on the current workplane . Alternatively, select an Entity as the reference plane for the position . This entity must be planar.

More options

Keep original: The initial curves are not removed.

Keep attributes: The offset curves adopt the layers and the colors of their respective initial curves.
