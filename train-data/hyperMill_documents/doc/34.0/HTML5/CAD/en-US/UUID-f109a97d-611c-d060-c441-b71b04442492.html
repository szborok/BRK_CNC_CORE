Analyze toolpath

Analyze toolpaths.

CAM → Analyze toolpath

Display coordinate values of toolpath points in relation to a reference coordinate system. Display vectors of the tool position at any toolpath point. Display face normal vectors in any contact point of the tool. Display feedrate, speed, and sequence number for each toolpath point. Display the job name for a toolpath. Display the direction of movement at the starting point within the toolpath.

Feedrate or speed of the toolpaths are displayed in different colors.

Select the toolpath to be analyzed in the hyperMILL® browser, for example, by enabling/disabling using the icon.

The tooltip can be used to display properties of a toolpath point by resting the cursor on the point. Configure this behavior in File → Options → Tooltip content while the Analyse toolpath command is active. There, drag the tags for the toolpath analysis to the right column to activate them. Also see Tooltip content.

Visibility

Enable the visualization of toolpath properties in the graphics area:

Jobinfo: Display the name of the toolpath from the job information in the graphics area.

Directions: Display the direction of the machining in the graphics area – an arrow is shown on the toolpath point.

Points: Show the toolpath points.

Tool vector: Display the tool vectors for 5-axis positions in the toolpath points in the graphics area.

Tool contact vector: Display the face normal vectors at the contact point of the tool with the surface for 3D radius compensation. The 3D radius compensation option must be selected in the job under Settings → Tolerance parameters. The ballmill tool type is supported.

For checkboxes with three switching states, the settings from the job settings are applied if the “undefined” state is set. In other words, the display that is visible even when no toolpath analysis is being performed.

Color

Feedrate: Display the feed values of the toolpaths in different colors . Select the settings in the Color map tab.

Spindle speed: Display the speed values of the toolpaths in different colors . Select the settings in the Color map tab. The cutting speed is displayed during turning operations.

Toolpath reference

Select the tool reference point to which the display of the toolpath with ball mill or lollipop cutter should refer.

The globally set color for toolpaths NC path GO and NC path G1 from the hyperMILL settings is used to display additional toolpath profiles.

  • From Job

  • Tip only

  • Center only

  • Both variants

Compensated path: Display the compensated path of a path correction if it has been calculated for the selected toolpath.

Limit toolpath

Limit the analysis of toolpaths to a smaller toolpath area. The slide controls can be moved to the left or to the right. To do this, hold the left mouse button down on the slide control. The analysis area that has been limited in this way can be moved further along the toolpath. To do this, hold the left mouse button down between the two slide controls. If the slide controls overlap, move the mouse to the middle and keep it held down there.

Select the beginning of an area to be analyzed. To do this, click a toolpath point.

Select the end of an area to be analyzed. To do this, click a toolpath point.

Limiting the analysis area does not affect the Internal simulation in hyperMILL®. The simulation also takes place outside of the limited analysis area.

View segments: Display segments of a toolpath (e.g., 3D optimized roughing, additive machining) in order to analyze them individually. If no segments are available, the option is grayed out. The segments are numbered consecutively. To select a segment, enter a value in the “Spinner” UI entity next to the option, or select it by clicking the up or down arrow. Click to increase or decrease the value. Only one toolpath may be displayed. The slider setting only affects the individual segment. When deactivated, everything is displayed again. The segment of a selected toolpath point is displayed in the analysis result under Segment information.

Values

Select

Analyze the toolpath points or the movement between toolpath points. The analysis can be carried out for different parts within one or between two different toolpaths. Measure distances between toolpath points, toolpath movements, points, curves and faces.

Click on a toolpath point to select between two such points for a toolpath movement.

Info: Select a part (toolpath point or movement). Use the arrow buttons to go to the next or previous toolpath part.

Measure: Optionally select a second part or an entity.

Reference coordinate system

Specify a coordinate system for the analysis. Select from NCS, Frame and WP.

Analysis result.

The analysis results are listed according to the selected parts and entities.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

Color map

Color toolpaths differently based on assigned feedrates, speeds, or cutting speeds.

Limit data

Narrow down the scope of analyzed data.

Min.: Manually change the smallest value to be analyzed for the color assignment of intervals. To do this, activate the option and enter a value. The smallest value found in the selected toolpaths is displayed at the end of the line.

Max.: Manually change the largest value to be analyzed for the color assignment of intervals. To do this, activate the option and enter a value. The largest value found in the selected toolpaths is displayed at the end of the line.

Clearance feedrate as max.: Include the clearance feedrate in the analysis. The feedrate value is shown at the end of the line.

Number of colors: Change the number of intervals manually. To do this, activate the option and enter a value. The admissible range of intervals is between 3 and 100. If a different setting is made, the number of intervals is determined automatically based on the available data. The number of colors currently available for values or intervals is displayed at the end of the line.

The color assignment automatically switches to interval display if there are more than 100 individual feedrates, speeds, or cutting speeds in the toolpaths to be analyzed.

Color map

A table indicating how colors are assigned to intervals is displayed. The colors are assigned automatically and cannot be changed. The table is sorted from the largest to the smallest interval. Values outside the user-defined limits are displayed in white (larger) and black (smaller).

Configuration

Toolpath settings → Size

Change display settings temporarily, only while the command is being used:

Point size factor: Change the size of the toolpath points.

Line thickness factor: Change the thickness of the toolpaths.

Vector length factor: Change the length of the toolpath vectors.

The color gradient of the background in the graphics area is fixed and cannot be changed.

Toolpath settings → Data filter

Show clipped areas: Display the colliding parts of a toolpath (clipped) if they have been calculated for the selected toolpath. The option is available if clipped areas exist.

Temporary graphic

Tessellation tolerance and Minimum radius can be used to generally influence the display of the temporary graphic for CAM entities (such as for feedbacks) in the CAD graphics area.
