Intersect

Create a curve along intersecting entities.

Curves → Intersect

Create a curve along intersecting entities. In addition to intersections of faces, it is also possible to calculate intersections of mesh pairs.

Select

Select the first set of entities with Shapes A and the second set of entities with Shapes B.

You can select multiple faces in a face set , but only one mesh may be selected. The number of curves found is displayed in the dialog under Total curves. The curves at the intersection of the entities are displayed as a preview.

Switch automatic selection: In case only one entity A1 intersects with several faces B1 to Bn, enable option . After clicking A1 (Shapes A active), Shape B is automatically preselected. Click the entities B1 to Bn.

Without the option enabled, first click all entities from A1 to An, then select Shapes B (or double-click with the left mouse button) and click all entities from B1 to Bn.

More options

Self-intersection: Create curve where faces intersect themselves . The Shapes B option cannot be selected.
