Filling

Create a filling face.

Shapes → Filling

Create a filling face from selected curves or face boundaries. When curves are used as boundaries, the continuity is automatically restricted to G0. The boundary does not need to be closed and can consist of more than four parts. If the curves you have chosen have gaps, the command automatically closes the gaps.

The command also works with open contours. You can also apply the command to two curves that face each other.

It can contain inner boundaries. If you select a boundary in the form of a closed contour, the filling face will be created within an individual face with the aid of its base face.

Distance shows the maximum distance between the filling face and the boundaries. If you select Optimize, the filling face will be iterated based on the settings.

Select

Boundaries: Select external boundaries (curves, face boundaries) . The boundaries do not need to be closed. The number of selected entities is displayed.

Continuity

Specify the continuity for the external boundaries using Position for a positional transition and Tangency for a tangential transition.

Internal points / curves

Select: Select curves and points within the boundaries if you want to specifically influence the curvature of the filling face . The filling face adapts itself by means of the selected entities. The number of selected entities is displayed.

Existing reference

Face aligns the isoparametric curves of the new face on the basis of the existing boundaries. It must be possible to project the boundaries onto the reference face or the face is larger than the projected curves. The result is a filling face shape that is based on the shape of the reference face.

Double curvature reference: In the case of a double-curved face, create the required result in the shape of a spherical face and with a tangentially constant transition (G1) on the basis of an automatically calculated internal reference face.

Settings

Positionality (Pos.), tangency (Tang.) and curvature continuity (Curv.) are target settings for the continuity under Optimize. If the result does not meet your requirements, modify the settings.

Tension is the target setting for the internal tension of the filling face.

Use No. of points to specify the number of points placed on the boundaries that are to be used in the calculation during the optimization. The default value for the number of points is 15. If the values entered for the tolerance do not result in the required level of precision, you can increase the number of points. This will improve the precision at the boundaries.

Iterations specifies the number of calculation loops for the optimization. If the result is not as precise as required, you can increase the number of loops. Max. degree is the maximum degree of the filing face. If the result does not meet requirements, the degree can be decreased. Max. spans specifies the maximum number of NURBS spans of the filling face.

Example 12. Fill in missing face in free-form geometry

Experience has shown that it is advisable to start with tried-and-tested settings when handling similar situations.

Select the Position continuity and set the value 1 for the Tension. This creates a face without any high internal tension.

Example 13. Fill in missing face in the course of a fillet

Experience has shown that it is advisable to start with tried-and-tested settings when handling similar situations.

Select the Tangential continuity and set the value 2 for the Tension. This ensures a continuous transition to the boundary faces and therefore a continuous course of the fillet.
