7.2. Hide and unhide

Commands to hide and unhide entities in the View menu.

Apply visibility filter

View → Apply visibility filter

Check visibility.

View → Check visibility

Only show entities of an entity type.

View → Hide / unhide → ...

Hide entities.

View → Hide / unhide → Hide

Unhide entities.

View → Hide / unhide → Unhide

Unhide group entities and child entities.

View → Hide / unhide → Unhide with all members

Unhide group entities and parent entities.

View → Hide / unhide → Unhide with all parents

Only show the selected entities.

View → Hide / unhide → Hide all except this
