Extend / shorten curve

Use different modes to extend 2D and 3D curves.

Modify → Extend / shorten curve

Select

Curves: Select the Curves to be extended. The number of selected entities is displayed.

Modes

Parameter extends or shortens the curve based on the curve parameterization .

Natural extends or shortens the curve in accordance with the curve parameterization .

Linear extends the curve linearly with a tangential transition .

Point: The curve is extended to a destination point .

Parameter

Start and End: Modify the start and end parameters for Parameter mode .

Length

Enter the values for extending the curve in the Natural and Linear modes .

Select

Select: Select a single point or the snap point of an entity as the target point for extending the curve in the Point mode (start, end, middle, on curve point).

Continuity: Set the continuity of the transitions from curve to curve .

1 = Positional transition (G0)

2 = Tangential transition (G1)

3 = Curvature constant transition (G2)
