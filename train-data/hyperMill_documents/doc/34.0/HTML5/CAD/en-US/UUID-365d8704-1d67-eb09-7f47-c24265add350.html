3.7. Color and display

Entities are displayed in the graphics area based on assigned color attributes and light settings.

Faces within a solid can have different colors. If the Visibility of a color is deactivated, only the edges are still displayed for all views with edges. A solid is not displayed at all if the visibility for all colors within a solid is deactivated.
