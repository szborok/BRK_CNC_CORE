Visibility

The visibility filter controls the visibility of entities in the graphics area using different selectable attributes.

The tab is divided into the Entities, Layer, Color, Tags, User defined tags and Stored filters sections. The size of each section can be adjusted by dragging the edge of the section with the left mouse button. The layout is saved automatically. You can rearrange columns (drag the table header with the left mouse button) and sort columns (click the table header with the left mouse button).

The sections are operated similarly. Each section of the visibility filter represents a part of the whole. If all the properties in a section are deactivated, nothing is visible.

Click the icon with the left mouse button to control the action of the condition, that is, the visibility of the entities.

Some attributes of the branch are activated and others are not; activation also affects all child attributes.

‘Entity open’ allows you to control the visibility of enclosed entities, such as the visibility of faces within a shape. Make sure to also activate the required entity type.

‘Entity closed’ - an entity such as a shape can only be displayed or hidden in its entirety.

The ‘Entity open’ and ‘Entity closed’ options affect visibility in the Layer section. The ‘Entity open’ option is always worked with automatically in the Color section.

The number of combined entities (such as shape solids) is displayed in round brackets, irrespective of the visibility, even if ‘Entity open’ has been selected.

The visibility filter has priority over the Hide and Unhide commands. It is applied immediately to new entities. A warning is output on the Info tab. Filter settings also apply to hidden entities.
