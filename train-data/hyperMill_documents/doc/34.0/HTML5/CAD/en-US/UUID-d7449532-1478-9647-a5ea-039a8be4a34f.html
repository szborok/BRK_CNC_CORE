On face

Create a workplane on a face with the Z direction in the face normal direction.

Workplane → On face

Create a workplane on a face, so that the Z direction of the workplane corresponds to the face normal of this face at the workplane origin. The origin of the workplane is first placed at the middle point of the untrimmed face and can then be modified.

For plane trimmed faces, the valid projection zone is extended to the domain of the face. Otherwise, points projected outside the outer face boundary will be discarded.

Select

Face: Select the face .

On face position

Origin: Specify a point on the face to be used as the origin of the workplane .

Position the workplane using the U and V parameters of the face or use the two slide controls. The parameter combination (0, 0) represents a corner of the untrimmed face and the parameter (1, 1) stands for the opposite corner.

XY axis direction

Automatic: Align the X and Y axes parallel to the U and V isoparametric curves of the face.

X: Align the X axis to an entity .

X: Align the Y axis to an entity .

Entity: Align the X or Y axis to an entity, e.g. a point parallel to a direction (line). Select an entity to do so .

If the selected point or direction (line) is not already on the default XY plane, the point or line will be projected onto the XY plane. The result is used for alignment of the X or Y axis.

Z axis

Invert: Invert the positive Z axis direction in the opposite direction . The X direction remains unchanged, and the Y direction is adjusted.

Offset: Move the workplane in the Z axis direction. Enter a value .

More options

Associative: The workplane is linked with the entities selected for creation. Modifications to the entities are carried over to the linked workplane.

The workplane is marked with an associative lock symbol. The size of the associative lock symbol changes relative to the workplane display size.

The workplane is automatically assigned with a name. The automatically generated workplane name also references the ID of the linked entity.

An associative workplane is marked with a lock symbol in the WP tab.

A tooltip lists the linked entities. Rest the mouse pointer on the symbol to show the linked entities. For modifications, select the appropriate workplane in the WP workplane tab and run the command.

Save as is used to save the workplane in the document. Click the line with the left mouse button and enter a name. You can also select an existing workplane to assign the new position to this workplane and thus overwrite its existing position. To do this, click the right edge of the line with the left mouse button to display the list of stored workplanes. Then click the workplane you require with the left mouse button.
