User interface

Single document behavior: Click Yes to use only one instance of the software. To open a document, a document that is already open must be closed first. Click No to open each document in a separate instance of the software.

Language: Select a different language for the user interface. The language codes are displayed. To work with the language of the operating system, select default. If a language is not available, English will be used for the user interface.

Max. recent commands: Number of recently used commands displayed on the context menu.

Number of columns in swiping grid tab: Change the order of the icons in the columns on the Swiping grid tab. Enter the number of columns.

Save reminder: Specify a time interval between 1 and 120 minutes for the recurring reminder to save the document. Disable the command by entering 0.

The settings are stored in the applicationGraphicUserInterface.xml file.
