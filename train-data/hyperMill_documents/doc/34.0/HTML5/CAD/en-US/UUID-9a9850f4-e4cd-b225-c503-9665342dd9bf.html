Extension faces

Create extension faces - for milling as well.

Shapes → Extension faces

2022.1

Use additional faces to create extensions that do not belong to the model geometry and can be used for purposes such as milling face extensions and stop faces for CAM programming.

Select

Faces: Select the face area to be extended . The number of selected entities is displayed.

Boundaries: Select the boundaries or edges where an extension is to be made. If several boundaries are selected, filling faces are created between the faces. Mixing the selection of faces and boundaries is not possible. The Milling mode cannot be used. The number of selected boundaries is displayed.

Extend faces

Create extension faces directly on selected faces and boundaries.

The extension consists of additional, new tangential faces .

Two line segments are calculated for each selected boundary/edge (one at each end). Both lines project tangentially out of the face. The face extension is generated by moving the first line along the boundary/edge until the second line is reached. Here, a tangential transition is maintained to the existing face along the boundary or edge.

Corner extensions are calculated with G1 continuity (tangency).

Extension width: Enter the extension width .

Normal direction: An extension is automatically calculated from the direction of the adjacent boundary of the given faces. In a solid, this is the common edge with the adjacent solid face. As this direction can be oblique, the width of the extension is often less than the length entered. Enable the Normal direction option to calculate an extension normal to the selected edge or boundary .

Wider blending: If a chain of boundaries / edges has been selected and there are usable face extensions before and after a gap, the Fill algorithm will attempt to close the gap. If this filling face is also unusable, further adjacent extensions faces are removed and approximated by the Fill algorithm. The warning ‘Some faces have been replaced by an approximation’ appears. Click the warning to highlight the approximated faces in color. To remove the highlighting, click the empty area below the warnings.

In some cases, this approach can produce poor results. Then, switch off the Wider blending option to fill the gap manually.

Tolerance: The tolerance is required to find the free (open) edges and to orientate between selected and adjacent faces.

Milling mode

Use the Milling mode option to prepare the extensions for processing in hyperMILL® jobs. Stop faces can be created. Allow filter definition based on these CAD features by assigning specific colors and layers.

Add stop faces: Additional stop faces can be added around the extension. Use the stop faces to define an additional area where no machining takes place . The tool should not touch a stop surface.

Width: Enter the width of the stop surfaces .

Check for undercuts

Use the Check undercuts option to determine whether an undercut in the faces to be created results from a tool approach direction due to the orientation of the surfaces to be processed.

Use face normals: If this option is disabled (default mode), the existing milling face normals are not evaluated. It is generally assumed that they are all oriented ‘upwards’. If undercuts occur in the faces to be created, an error message is displayed.

If this option is enabled, it is assumed that the milling faces are already oriented as intended. The orientation of the created faces results accordingly. If they point ‘downwards’, they are considered to be correct.

Set the planned static tool position by selecting the Direction in Milling frame Z axis:

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Adjacencies

Check the relationships of the adjacency at the tangential transitions to other faces.

Check adjacent faces: Without checking, a milling extension is created around all selected faces, as well as within the model . With the check, a milling extension is only created on edges and fillets .

Beyond fillets: It is possible to handle fillets differently when checking the relationships of the adjacency depending on the maximum fillet radius Max. fillet radius. No face extensions are created if the radius of the fillet is smaller than the entered value .

Properties

Layer and color are automatically assigned to the entities to be created for both extensions. The current layer, the preset color for the extension and the first color in the color library for the stop faces are used.

Assign: Enable this option to set a Layer for both the extension and stop areas and an entity Color for both. Allow filter definition based on these CAD features by assigning specific colors and layers. To do this, enter a layer name and select a color for each from the drop-down menu. If the layer does not exist, it will be created.
