Fillet

Create a fillet face.

Shapes → Fillet

Create a fillet face with constant or variable radius between faces or with an 'oversize' radius along a face flank and a curve contour.

Select

Face A and Face B: Select two faces as the flanks of the fillet face. The number of selected entities is displayed .

Curve: Select curves or face boundaries for the Face - Curve option as reference for the radius .

Invert: The position of the fillet face is the result of the orientation of the flank faces. To position the fillet in another (chamfer) course, reverse the orientation of the flank faces by clicking with the left mouse button . Click the button first.

Constant

Create a fillet with a constant radius. Enter a Radius .

Variable

Create a fillet between two faces with a variable radius. If the Tangential faces chain option is used, the center radius is not taken into account. In the case of solids, only the fillet is created. The faces within the solid are not trimmed.

Mode: In Linear mode, the radius between the start radius and end radius changes continuously . In Parabolic mode, the change can be influenced by a radius value in the center of the fillet course . Invert swaps the start and end of the fillet.

Start: Enter a radius . The value 0 is permitted.

Middle: Enter a radius . The value 0 has no influence on the radius between the start and end value.

End: Enter a radius . The value 0 is permitted.

Face - Curve

Create a fillet with a radius larger than the available space. The fillet is inserted starting from the selected curve. The fillet is 'held' to this curve and runs tangentially into the face selected as flank. Use to exclude areas when milling with auxiliary faces, for example.

Radius: Enter a radius .

Tri-tangent

Fillet with tangential transition between 2 side faces and a central face separating these faces – such as a rib or the inflow of an impeller, etc. This ensures seamless transitions free of curvature between the three faces. The radius of the fillet is not constant, but adapts to the angles between the faces to create tangency.

Select the three faces. The face-normal directions are determined automatically.

Mid-face: Select mid-face .

Delete mid-face: The mid-face is removed from the result .

More options

Tangential faces chain: Automatically continues the fillet, in case a tangential extension can be found automatically .

Extend: Automatically adapts the start and end edges of the fillet face to the face area of the respective flank faces from Face A and Face B. The Tangential faces chain and Extend options cannot be used at the same time.

Automatic trim: Trim the flank faces contacted by the fillet at the edge of the fillet face .
