Controlling visibility via the context menu

The visibility filter can be controlled using the Visibility submenu of the context menu. First select an entity with the required target attributes. Right-click to open the context menu and then select one of the following commands:

Table 13. Visibility filter commands on the context menu

Command

Explanation

Just this layer and color

Only displays the entities that are on the layer and have the color attribute of the selected entity.

Except this layer and color

Displays all entities that are not on the layer of the selected entity and do not have the color attribute of the selected entity.

Just this layer

Only the entities on the layer of the selected entity are displayed.

Except this layer

Displays all entities that are not on the layer of the selected entity.

Just this color

Only the entities with the color attribute of the selected entity are displayed.

Except this color

Displays all entities that do not have the color attribute of the selected entity.
