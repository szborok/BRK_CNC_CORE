Fitting

Approach an existing curve or a series of points with a curve.

Curves → Fitting

Approach an existing curve or a series of points within the permitted tolerances and settings with a curve. You use Tolerance to define the permissible deviation. Max. distance displays the deviation .

Select

Curves: Select one or several curves .

Points: Select points.

The number of selected entities is displayed.

Constraints

If you select Position, the positions of the start and end point will be retained. If you select Tangency, the tangents in these points will also be retained. There is no reference to other entities.

Settings

Min. degree defines the minimum permissible degree of the approached entity and Max. degree defines the maximum permissible degree. Cont. defines the continuity of the approached entity.

Same parameterization: The parameterization of the initial curve is taken over into the result.

Keep original: The initial curve is retained .
