Entity properties

Lists the attributes of one or more entities.

Analysis → Entity properties

Lists the attributes of one or more entities. The information that is available adapts itself to the type composition of the selected entities. When one entity is selected, all the properties are made available. When multiple entities are selected, the intersecting set of useful parameters of the simultaneously selected entities is calculated, such as the size in the X, Y and Z direction. If no information can be determined in such a case, this is indicated by the text ‘Varying ...’. Entities can be optionally added to or removed from the selection.

To select or deselect an entity, click the entity with the left mouse button while holding down the CTRL key.

All changeable properties are highlighted by a different background color.

Entity properties

Show entity properties.

Preview

Show orientation: Shows the orientation of a curve with the aid of an arrow or the U and V direction of the isoparametric curves of a face.

Show bounding box: A best-fit outline contour is displayed around the selected entities. The edges of the curve align themselves with the current workplane.

Show control points: Displays the NURBS control points of an entity.

Show arc joints: Displays the NURBS arc joints within an entity.

Show normal: Show the normals of the face.

Geometric parameters

Show geometric parameters, such as:

Entity precision: Existing geometric deviation of the entity.

Radius: The radius of circular arcs can be changed. If you want to modify several curves at the same time, you should only select circular arcs.

Center of mass: Display the coordinates of the center of mass of a solid.

Select the Create point command on the context menu to generate a point at a determined coordinate position.

Graphic properties

Various graphic attributes such as Color and Layer can be modified.

System parameters

The Creation time and the Modification time can be viewed. Information in text form can be entered for the Creator and Creator ID.

Drafting properties

Modify the attributes of a dimension, text, text line or label.

Use the Flip text sight direction option to change the direction if the text is displayed the wrong way round from the required viewing direction. Do this by selecting the entity beforehand.

More options

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.
