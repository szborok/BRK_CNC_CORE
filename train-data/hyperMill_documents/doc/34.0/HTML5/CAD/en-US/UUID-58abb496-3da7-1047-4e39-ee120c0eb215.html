Shape spherical

Analyze the radii of faces and meshes in relation to a sphere diameter.

Analysis → Shape spherical

Analyze the radii of faces and meshes (for example, in fillets) in relation to a sphere diameter (to facilitate a sphere milling tool).

The tooltip displays the values at the cursor position. A sphere visualizes the geometry values determined at the position of the mouse .

Swap orientation: Invert the normal direction of the entities for analysis . The shape could have a normal direction opposite to the required direction of analysis so that this can be used to invert the results. Perform the analysis for either entities that are curved ‘outwards’ or entities that are curved ‘inwards’.

Radius / Diameter: Switch analysis between radius and diameter.

The command is based on the tessellation. The lower the values of Tessellation and Edge length, the more accurate the result, but the longer the calculation takes. Reduce the selected range if a precise calculation is needed and therefore the entire model cannot be included. Use the Tessellation view to assess the change in the result based on the selected values.

Select

Shapes: Select the entities . The number of selected entities is displayed.

Color map

Analyze all radii of curved entities . Planar entities and simple convex entities are automatically excluded from the analysis. They are displayed in dark pink.

The X axis shows a color map and assigns a color to each value in the analysis area. A color symbolizes the value assigned to an analyzed entity. The Y axis provides qualitative information about the scope of the respective value in relation to all determined values. A thin, vertical line shows the analyzed position.

Min. and Max. show at the start of the analysis the determined minimum and maximum values of the selected geometry for the selected orientation .

Min.: Enter a value for the sphere size (for example, a minimum radius of an available milling tool) for which to perform the analysis. Areas that are no longer accessible are displayed in dark grey .

Max.: Select a maximum value to limit the analysis area of the geometry for large radii, outside of chamfer courses. If the determined sphere size is larger than the Max. value to be analyzed, the sphere will be displayed in red. Areas that are no longer analyzed are displayed in light grey .

Use Reset to set a restricted analysis area to its original size. After modifying the orientation, reconfigure the analysis area.

The analyzes in Sphere size and Color map are separate. The entries do not influence each other.

Sphere size

Size: Enter a radius or diameter for the analysis .

Two areas can be determined as the analysis result.

Greater than: The area can be reached using a sphere, in other words, it can potentiality be machined using a tool with this diameter. Select the color for marking the entities.

Equal to or less than: The area cannot be reached using a sphere, in other words, it could not be completely machined using a tool with this diameter . Select the color for marking the entities.

Extract curves: Delineate the areas determined using the Equal to or less than option with entities of the curve polyline type .

Check collisions: A collision check of the faces involved with a spherical shape in order to identify the areas for the correct tool diameter. Conversely, the collision check also makes it possible to detect sharp edges and corners, for whose production an electrode may be necessary. Without collision checking, each individual face is analyzed for its inherent minimum curvature/radius. With collision checking, all adjacent faces are additionally taken into account for the diameter/radius.

The analyzes in Sphere size and Color map are separate. The entries do not influence each other.
