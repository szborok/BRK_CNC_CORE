Compare and merge

Compare processing states and optionally transfer modified geometric entities.

File → Compare and merge

Compare entities of a model in different CAD processing states. Merge the changes in the currently open document. Only entities which have changed are transferred. After running the command, select the other version of the document in the file dialog.

Faces that are used in milling areas are protected from being modified by the user. This protection can be disabled under CAM → Setup → Settings → Document → Lock so that it is possible to edit these faces.

Comparing model

Show the document selected for the comparison.

Current model: different entities

Select: Select entities to be included in the comparison. The number of selected entities is displayed. Use the Entire model option to include all entities of the selected types.

Show: Show modified entities from the currently open document.

Delete: Delete modified entities from the currently open document. Invisible and hidden entities are also deleted. Faces in solids are not deleted. Avoid unintentionally deleting entities using the Lock entities command.

Add tag: Add a tag to modified entities in the currently open document. Enter a tag name.

Comparing model: different entities

Show: Show modified entities from the comparing model.

Merge: Merge modified entities from the comparing model into the currently open model.

Add tag: Label inserted entities with a tag. Enter a tag name.

Precision

Tolerance: Two entities to be compared are considered to be different if the check positions calculated by the software differ so much that they are no longer located within a still permissible distance from each other. Enter a value for this maximum permissible deviation.

Skip small entities: During the import, discard faces that have an area that is equal to or less than the value specified. Whereby the tolerance specified here is squared: tolerance² e.g. (0.01 mm)² = 0.0001 mm².

Distance between NURBS entities

The comparison of NURBS faces can take place based on the actual existing tessellation of the faces or based on a new (more precise) tessellation to be calculated over all NURBS faces. In this way, consolidate the check for NURBS faces.

Tessellation tolerance: Enter a tessellation tolerance for a finer tessellation of the NURBS faces.

Simplify

Use simplified rotational faces: For conical hole ends and, for example, hole cylinders consisting of multiple entities, the entities are first simplified to one entity and then compared. Ensure that geometrically identical data are not recognized as different. Switching the option on and off changes the comparison result.

Types

Curves: Compare curves.

Edges: Compare solid edges.

Polylines: Compare polylines.

Faces: Compare faces.

Meshes: Compare meshes.

PMI: Compare Product and Manufacturing Information (PMI).

Points: Compare points.
