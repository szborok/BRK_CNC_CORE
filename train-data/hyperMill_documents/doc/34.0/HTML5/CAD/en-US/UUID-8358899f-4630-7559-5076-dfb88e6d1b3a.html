Wrapping

Wrap planar curves and text to a rotated face.

Curves → Wrapping

Unwrap planar curves and text to a rotated face (sweep face, cylinder face, rotational face). The length is kept at the defined origin. The direction, scaling and mirroring can be modified.

Text with a font in TrueType format (TTF) is converted automatically to NURBS curves during the calculation.

Keep attributes: Color attributes are retained.

Select

Entities: Select the planar entities to be deformed . The number of selected entities is displayed. For coplanar entities, the area to be wrapped is highlighted and can be used to snap positions. A green arrow shows the initial direction of the selected entities . Use the Automatic computation option to immediately generate a preview for a modification.

Keep original: Specify whether the selected entities should be retained .

Tolerance: The tolerance influences

  • the conversion of NURBS cylinder faces and NURBS rotational faces into analytical data.

  • keeping the length of the base entities on the target face (e.g. if the target face is a rotational face or a linear sweep with poorly parametrized surface line).

  • creating the wrapped NURBS entity.

  • creating the highlight of the area to be wrapped, if the selected entities are coplanar.

Initial

Mode: Select whether the entities Linear or Radial are to be unwrapped. In the case of Radial entities, for example, text is unwrapped in the radial direction starting from the reference point true to the original length.

Select: Using the Origin option, select a position that can be related to a position on the target face . Then use Direction to specify the initial orientation of the selected entities . The direction is indicated by a green arrow. These specifications are optional.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Target

Select: Select a Face . The target face can be a cylinder face, a rotational face or a linear sweep. A NURBS cylinder face or a NURBS rotational face can be selected if they can be converted into an analytical cylinder face or rotational face within the input tolerance. Position the selected origin point at the target location using Origin. The U and V face parameters are shown for the position .

Orientation: Use the handle to smoothly change the orientation of the unwrapping or use the Rotate option to move in 90° steps . Use the Mirror option to mirror the unwrapping .

Fit: Scale the unwrapping so it fits the target face .
