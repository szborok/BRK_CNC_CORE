Selection

The selection filter controls which entities should be included in or excluded from a selection, for example, using Window. The entities are therefore filtered for the selection process.

Different filter options are available. You can filter by Entities, Attributes and possible combinations of different attributes using Advanced. Entities allows you to include entity types in the selection or exclude them. Attributes allows you to examine layers or colors across different entities. To select according to further criteria, you can construct an extended Advanced selection filter.

You adjust the size by dragging the edge with the left mouse button. NURBS combines all NURBS geometries.

Deactivate the selection filter for curves and activate the selection filter for faces in the Wireframe view to select a face.

Open and closed entities

Some entity types may be composed of other entity types. For example, a solid of the ‘shape solid’ type is made up of faces. It may be necessary to optionally select the entire solid or just some of the faces of the solid.

Open and closed entities therefore exist. Clicking the button with the right mouse button opens a menu in which the ‘Entity open’ or ‘Entity closed’ option can be set for group, shape and feature entities.

Set the ‘Entity open’ or ‘Entity closed’ option for V-sketch entities.

The set combination is displayed with the following icons:

Entity open - allows you to select entities that are enclosed in another entity. To do this, also activate the required entity type of the enclosed entity.

Entity closed - you can only select the complete entity.

The settings are displayed as a button with changing icons:

‘Feature closed’ - you can only select a complete feature.

‘Group closed’ - you can only select complete entities within a group.

‘Solid closed’ - you can only select a complete solid.

‘Group closed’ and ‘Feature closed’ are combined, solids are open.

‘Solid closed’ and ‘Feature closed’ are combined, groups are open.

‘Solid closed’ and ‘Group closed’ are combined, features are open.

A combination has been set - use ‘All’ to open or close everything.

The setting cannot be changed.

Attributes – filtering by layer and color

You can filter by layer and color attributes in order to make a selection. The layer and the color are defined by formulating a condition or by incorporating the attributes of an entity:

You formulate conditions in the condition list. A condition comprises a property, an operator and a value or value range.

Use + to add a further condition and - to remove a condition. When you use the OR operator, multiple conditions act independently of each other. If you want multiple conditions to apply, they must be linked using the AND operator (for advanced attributes only).

Transfer the attributes (color, layer) of the entities directly from the graphics area. Select the command and click the entity with the left mouse button.

Advanced:

To select according to further criteria, an advanced selection filter can be set up. These criteria include system information and the geometric attributes of entities or their representation, among other things. For example, you can filter for all red cylinder faces with a diameter between 9.5 and 10 mm.

To do this, formulate the conditions in the condition list.

Then proceed as usual with the entity selection options, for example, make a selection using Select window. Entities will then be included in or excluded from the selection in accordance with the selected conditions. After the selection had been made, the conditions are automatically discarded.
