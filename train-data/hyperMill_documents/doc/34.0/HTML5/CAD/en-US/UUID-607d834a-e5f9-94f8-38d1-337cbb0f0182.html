17.2. AUTOMATION Center

Open the AUTOMATION Center to create a script for an automated sequence.

CAM → Automation → AUTOMATION Center

The CAM programming commands themselves already provide powerful options for automation through base commands for feature and macro programming and the Customized Process Feature technology.

This can be further expanded with the AUTOMATION Center. The objective is automation that significantly reduces interactive programming effort. In some cases, full automation is possible.

A complete automation solution can simply be run as a script. The AUTOMATION Center functions as an execution platform.

Note

Help information is integrated into AUTOMATION Center and is available in PDF format.
