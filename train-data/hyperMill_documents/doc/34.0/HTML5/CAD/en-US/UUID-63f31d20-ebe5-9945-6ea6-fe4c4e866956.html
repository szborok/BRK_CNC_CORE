Window visible entities

Select all entities that are visible from the viewing direction within a rectangular selection area.

Select → Window visible entities

Select all entities that are visible from the viewing direction within a rectangular selection area. Click in the graphics area with the left mouse button, drag the mouse and click the left mouse button again. When an area is drawn from ‘right to left’, all entities that are visible from the viewing direction will be selected, together with the visible entities that intersect the edge of the selection area. If drawing takes place from ‘left to right’, only those entities that are visible from the viewing direction and fit into the area completely will be selected. The filter for the selection of the entities is included.
