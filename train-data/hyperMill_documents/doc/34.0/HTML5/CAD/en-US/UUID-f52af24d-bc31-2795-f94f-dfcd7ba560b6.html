Chain

Select a contour or a boundary.

Select → Chain

Select a chain, also known as a chain of curves or a contour, to select it for another command.

Click and hold the required contour with the left mouse button. All sections of the chain that can be captured are highlighted . Gaps in the contour are identified. Click with the left mouse button to apply the selection.

Change the linear tolerance to the size of the maximum permitted gap if you want to select a complete contour line despite its gaps . No gaps are closed. This needs to be taken into account in the actions that follow.

The chain can be built from the face boundaries of several faces.

Total entities: The number of selected entities is displayed.

Modes

Influences the path of the chain.

Stop at cross: The chosen contour is continuously selected until the contour line branches .

The option can be accessed via the context menu Select → Chain. Assign a shortcut in Shortcuts in More commands. Incorporate the command into a toolbar in Toolbars and tabs under Commands.

Minimal angle: The chosen contour is continuously selected . Selecting the contour at a bifurcation ignores the minimal angle between the sections for the remainder of its course. If the angles at the bifurcation between the sections are equal, automatic selection takes place based on the entity ID order.

Tangent: Select a contour whose sections come together as being tangential to each other . Use the Angular tolerance option to specify an angle for a permissible deviation from the tangency .

Shortest way: The chosen contour is continuously selected. At a bifurcation, the further course along solid edges is calculated so that the shortest route is selected.

Select

User driven: Automatically select a chain from a start to an end entity and influence the path . The Modes influence the path of the chain.

Steps:

  1. Click the Curves button. Curve selection is greyed out if User driven is not selected.

  2. Click the start entity with the left mouse button . The click position affects the initial search direction. The arrow always points towards the other end of the selected entity . The entity remains selected.

  3. Change the direction of the arrow by double-clicking, if necessary.

  4. Click the end entity with the left mouse button . The entity remains selected.

  5. The recognized chain is highlighted. The start and end are marked with a temporary point.

  6. It is possible to adjust the course of the chain between the start and end entities . To do so, go to the exact position where the unwanted branch of the chain occurs, and click the desired branch with the left mouse button. Holding down the CTRL key at the same time allows you to remove a selected entity from the chain.

  7. It is possible to save the selected chain using the Save selection command on the Curves context menu.

For a closed chain, click an entity directly next to the start entity to make it the end entity. In a closed chain, the temporary points are no longer displayed.

When selecting a boundary, the User driven option automatically uses all identical curves and edges from the location. This means it does not matter whether the chain is composed of curves, edges, boundaries or a combination.

Only closed: If a chain is selected that extends beyond the graphic area of the screen, it is not possible to see whether the chain is open or closed. This information is important, for example, for the delimitation of a processing area. When using this option, only closed chains are marked . Other chains are discarded.
