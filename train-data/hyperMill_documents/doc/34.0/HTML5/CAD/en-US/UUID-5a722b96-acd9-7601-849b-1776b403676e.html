Using the calculation result

Calculation results can be reused in commands. To do this, click in an option in a command dialog . Enter the name of the variable . The calculated result is displayed and used . If you make a change to a calculation in the parameter list, values in the command dialog will not be adapted. Exceptions are the Text, Mating and Mating clamping device commands.

In the commands Text, Mating and Mating clamping device, a list of variables, commands and constants can be selected from the parameter list for inputting values using the context menu.

Note

If a mating value defined in the Text, Mating and Mating clamping device commands changes as a result of a recalculation of the variables in the parameter list, the affected options in the command dialog and the entities in the graphics area are updated.
