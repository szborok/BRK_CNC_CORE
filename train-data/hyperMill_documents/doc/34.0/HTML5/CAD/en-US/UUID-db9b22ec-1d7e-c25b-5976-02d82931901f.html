8.1. Select entities

Entities can be selected before calling a command and during a command sequence. The commands for selecting faces within a solid can be combined.

Selection commands, primarily in the Select menu:

Select all entities.

Select → All

Select a contour or a boundary.

Select → Chain

Select entities within an area delineated by a polygonal line.

Select → Lasso

Select entities within a rectangular selection area.

Select → Window

Select entities within a circular selection area.

Select → Circular

Select entities arranged in an arc.

Select → Circular sector

Select entities within a closed chain of curves.

Select → From curves loop

Select all entities that are visible from the viewing direction within an area delineated by a polygonal line.

Select → Lasso visible entities

Select all entities that are visible from the viewing direction within a rectangular selection area.

Select → Window visible entities

Select all entities that are visible from the viewing direction within a circular selection area.

Select → Circular visible entities

Select all entities arranged in an arc that are visible from the viewing direction.

Select → Circular sector visible entities

Select visible entities within a closed chain of curves.

Select → Visible entities from curves loop

Select entities for hyperMILL® job.

Available from inside hyperMILL® commands

Select the solid to which the child entity belongs.

Context menu → Select solid

Select the group to which the child entity belongs.

Context menu → Select group

In a solid or in a face model, select all faces within adjacent faces.

Select → Limited faces

Select faces in the course of a constant fillet within a solid or a face model.

Select → Fillet faces

Select faces in the course of a constant chamfer within a solid or a face model.

Select → Chamfer faces

Select faces that are ‘linked’ with a tangential transition within a solid or a face model.

Select → Tangent faces

Select coplanar faces.

Select → Coplanar faces

Select coaxial faces within a solid.

Select → Coaxial faces

Invert the selection of entities.

Select → Invert

Set a filter for the selection of entities.

Select → Filter entities → ...

Reset the selection filter to the previous setting.

Select → Reset selection filter
