16.10. Modify EDM parameters

Modify EDM parameters.

Electrodes → Modify EDM parameters

Electrode

Change electrode properties for document, eroding process and eroding machine. Change information, which is not assigned to an electrode.

Mode

Electrode: Select an electrode. The electrode name is shown.

Document: Change information for the title block of the summary, which is not assigned to an electrode. To do so, enable the option.

This information is automatically added to a report and when printing electrodes.

Electrode description

Change the description for the electrode.

Predefined electrode descriptions can be selected from the drop-down menu. Texts can be entered as defaults in .../hyperCAD-S/files/electrode/electrode_descriptions.xml. For example:

<?xml version="1.0"?>
<ElectrodeDescriptions>
  <Description item="Core electrode"/>
  <Description item="Cavity electrode"/>
  <Description item="Insert electrode"/>
  <Description item="Side electrode"/>
  <Description item="Master electrode"/>
  <Description item="Virtual electrode"/>
  <Description item="Injection electrode"/>
  <Description item="Rotational electrode"/>
  <Description item="User defined electrode"/>
</ElectrodeDescriptions>

Raw material size

X, Y or Diameter and Z, and EDM erosion area: When changing the raw material size of the solid of an electrode (by means of direct modeling, for instance), the changed values are highlighted with a new background color, and by using Apply, the values in the electrode TAGs are updated.

Note

These changes must always be applied using Modify EDM parameters, otherwise they are no longer included in the process.

Example 21. Changing a raw material that is assigned to an electrode

The designation of a solid as a raw material is carried out using the name Raw material of the solid. The raw material must be assigned to the electrode and correctly colored. The shape of the raw material (cuboid or cylinder) must remain the same. The size must be chosen in such a way that the raw material or the resulting block still fits the electrode holder.

In case a raw material needs to be slightly adjusted in size:

  1. Select the solid with the name Raw material in the model tree inside the electrode and Unhide it.

  2. Adjust the shape of the raw material via direct modeling. Make sure that the group and the solid are open for the selection of the desired face.

  3. Select this solid and Hide it.

  4. Check that the Raw material size in Modify EDM parameters has been updated correctly. Update the values in the electrode TAGs by using Apply. A message in the Info tab confirms the successful update.

In case a raw material has been unintentionally erased:

  1. Set the layer of the electrode as the current construction layer by using Set current layer.

  2. Activate the color assigned in File → Options → Electrode options under Colors → Raw material - when using the presets, this is the color ‘Electrode 08 raw material’.

  3. Construct the new raw material as a solid.

  4. Move this solid to the electrode group by using Edit → Group / Ungroup.

  5. Select this solid and enter the name Raw material under Context menu → Edit → Assign name.

  6. Select this solid and Hide it.

  7. Check that the Raw material size in Modify EDM parameters has been updated correctly. Update the values in the electrode TAGs by using Apply. A message in the Info tab confirms the successful update.



Attributes

Depending on the selected mode, change attributes of the selected electrode or document.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

For Creation date and Modification date, variables can be entered as wildcard characters for the date and time.

Variables for date and time

The following variables can be entered as wildcard characters.

%date%

Enter the current date automatically based on the date of the local operating system and write in short format, for example 10/04/2008.

%shortdate%

The date is formatted in the same way for %date%.

%longdate%

Enter the current date automatically based on the date of the local operating system and write in long format, for example Thursday, April 10, 2008.

%time%

Enter the current time automatically based on the time of the local operating system and write in short format, for example 06:30.

%shorttime%

The time is formatted in the same way for %time%.

%longtime%

Enter the current time automatically based on the time of the local operating system and write in long format, for example 06:30:00.
