Shape curvature

Analyze the curvature and radii of faces and meshes.

Analysis → Shape curvature

Analyze the shape and scope of curvatures and radii in faces and meshes using a color-coded representation of the curvature or radius values.

The tooltip displays the values at the cursor position.

Select

Shapes: Select the entities . The number of selected elements is displayed.

Curvature type

Select a curvature type for the analysis .

The curvature is the inverse of the radius. If a radius remains constant, the curvature is also constant. If the curvature converges to 0, the radius is ‘infinite’. If the radius converges to 0, a very large curvature exists. The two principal curvatures are the minimum and the maximum curvature values from all curvatures at the analyzed point.

Mean: Displays the ‘mean curvature’ .

The mean curvature is the average of the two principal curvatures (k 1 and k 2) for a point analyzed on a face.

Gaussian: Displays the ‘Gaussian curvature’ .

The Gaussian curvature (K) is the product of the two principal curvatures (k 1 and k 2) for a point analyzed on a face.

  • K = 0 shows where the face is planar or single-curved. The face is planar, cylindrical or conical at the analyzed point.

  • K > 0 shows where the principal curvatures have the same sign. The face is bowl-shaped at the analyzed point.

  • K < 0 shows where the principal curvatures have opposite signs. The face is saddle-shaped at the analyzed point.

Min. radius/Min. diameter: Find areas with the smallest radius/diameter . The values are signed. In an analysis area that is not restricted, the smallest values tend to be in the middle of the analysis area.

Abs. min. radius / Abs. min. diameter: Find the absolute values of the smallest radii/diameters . The smallest values are located at the left edge of the analysis area.

Radius / Diameter: Switch display between radius and diameter.

Skip planes: Omit very flat entities from the radius analysis when assigning colors and mark them with a separate color . Select the color.

Color map

Min. and Max. display the determined minimum and maximum values for the selected curvature type.

The X axis shows a color map and assigns a color to each value in the analysis area. A color symbolizes the value assigned to an analyzed entity. The Y axis provides qualitative information about the scope of the respective value in relation to all determined values. A thin, vertical line shows the analyzed position .

Use Limits to limit the analysis area and find a color map that is suitable for the analysis. Select a minimum Min. value and a maximum Max. value. Values below Min. are shown in dark grey . Values above Max. are displayed in light grey . Use Reset to set a restricted analysis area to its original size. After modifying the curvature type and using the Diameter / radius option, reconfigure the analysis area.

For example, reduce a maximum radius value until all the component’s radii that cannot be fillets are outside the analysis range. Planes and very flat entities flow into the analysis with a very small radius due to tessellation. Exclude these areas using the Skip planes option or select a minimum radius value within the tolerance of 0 different so that all planar entities are outside the analysis area.

When analyzing concave or convex areas of the geometry using Min. radius / Min. diameter, configure the boundaries:

  • convex: [- value, 0]

  • concave: [0, value]

The analyzes in Extract curve and Color map are separate. The entries do not influence each other.

Extract curve

Automatically generate a curve (polyline) that delineates two areas from one another.

Higher values: Select the color for areas above the target value.

Target value: Select analysis value.

Lower values: Select the color for areas below the target value.

The analyzes in Extract curve and Color map are separate; the entries do not influence each other.

For example, enter the diameter of a ball milling tool for an analysis. If an area cannot be reached, that is, cannot be milled by the tool with the analyzed target value, two different colored areas are output. Otherwise, only one area is output with just one color. The area is restricted by curves.
