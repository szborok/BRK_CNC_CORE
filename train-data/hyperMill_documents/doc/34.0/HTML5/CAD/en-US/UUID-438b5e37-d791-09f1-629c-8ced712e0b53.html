12.2. Curves

Commands to create spatial curves in the Curves menu:

Create or modify a cubic spline.

Curves → Spline

Approach an existing curve or a series of points with a curve.

Curves → Fitting

Create curves at face boundaries and mesh edges.

Curves → Boundaries

Project curves onto shapes or onto a plane.

Curves → Project

Create a spiral curve.

Curves → Spiral

Create a curve along intersecting entities.

Curves → Intersect

Create one or more curves from an isoparametric curve in the U or V direction on a face.

Curves → Isoparametric

Generate an isoparametric curve at 0.5 between two curved curves in space.

Curves → Average

Compensate for manufacturing deviations using a scanned curve.

Curves → Curves → Compensate

Create a new spline curve between two curves.

Curves → Blending

Create one curve from multiple connected curves (chain).

Curves → Join

Create a curve with an offset to an initial curve.

Curves → Offset

Create offset curves on faces.

Curves → Offset on faces

Create the contour of a shape as a curve.

Curves → Shape contour

Create curves and points through entities with the help of section planes.

Curves → Section

Create a curve in the center of suitable faces.

Curves → Center

Wrap planar curves and text to a rotated face.

Curves → Wrapping
