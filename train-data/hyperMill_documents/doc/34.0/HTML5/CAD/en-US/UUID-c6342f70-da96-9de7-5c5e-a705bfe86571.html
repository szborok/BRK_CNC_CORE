Align Best-Fit

Fit entities in the best possible way by simultaneously moving and rotating.

Edit → Align Best-Fit

Deformation

Move and rotate entities, such as meshes created from imported scan data, at the same time to fit them into a particular geometric situation in the best possible way.

Reference systems can also be fitted. They are marked with <Selected_Ref_Sys_Name>_MOVED.

Select

Entities: Select all entities to be fitted . The number of selected entities is displayed.

Keep attributes: Color attributes are retained.

Keep original: Specify whether the selected entities should be kept.

Assign

Select points (pairs) where positions should be matched as far as possible. The number of target entities must be equal to the number of base entities. An already selected target point cannot be selected as a start point and vice versa.

A number is displayed next to the two points of a point pair in order to be able to better identify the points of a pair.

Matching points

Initial

Points: Select the base entities . The number of selected entities is displayed.

Target

Points: Select the target entities . The number of selected entities is displayed.

Constraints

Reference system

WP: Select a workplane that is essential for locking the translation and fixing rotation axes . If no selection is made, the world workplane will be used.

Locked translation

X, Y and Z: Select a proportion of the translation that should remain constant when fitting.

Fixed rotation axis

X, Y and Z: Select a proportion of the rotation that is to remain constant when fitting.

To prevent any rotation, it is sufficient to lock two rotation axes, because if two axes cannot change their direction, the direction of the third axis is also fixed.

Weight

Point: Select a start point and enter a numerical value to determine the ‘intensity’ with which to match the points of the pair of points. Select further points (pairs) one after the other and enter a weight. Those with the lowest weight are matched less than those with a higher weight.

The default value is 1. The weights are ‘relative to each other’. For example, if all points were weighted with 1000, it would be as if they were all weighted with 1.
