Workplane

The entry of constructive values such as coordinates relates to the current workplane. A new document is created automatically with a world workplane. Workplanes can be saved together with transformation information and selected directly as references for moving and copying, for example.

Determine the current workplane and control the visibility of the axes of the workplanes created in the document:

Activated

The X, Y and Z axes are displayed at the coordinate origin of the workplane.

Deactivated

The X, Y and Z axes are not displayed at the coordinate origin of the workplane.

New workplane

Newly created workplane: Commands for modifying the workplane directly affect the new workplane. When a name is entered, the workplane is stored permanently in the document.

The current workplane is highlighted in bold. To change the current plane, select the entry with the left mouse button, right-click and then select Set current on the context menu.

To rename a workplane, click the name of the workplane with the left mouse button, change the text and then apply the new name by pressing the ENTER key or by clicking with the left mouse button outside the text input area.

To delete a workplane, use the Delete command on the context menu. Select the workplane with the left mouse button and right-click to open the context menu. You cannot delete the current workplane.

If a workplane cannot be deleted, check the following possible causes:

  • The workplane is being used, for example, as a reference system in the print box or for dimensions.

  • The workplane is being used in an Undo / Redo action. Release the workplane by saving and opening the document.

Active, currently set workplane (displayed in the graphics area)

Inactive, visible workplane (displayed in graphics area)

Associative: The workplane is linked with the entities selected for creation. Modifications to the entities are carried over to the linked workplane.

The workplane is marked with an associative lock symbol. The size of the associative lock symbol changes relative to the workplane display size.

The workplane is automatically assigned with a name. The automatically generated workplane name also references the ID of the linked entity.

An associative workplane is marked with a lock symbol in the WP tab.

A tooltip lists the linked entities. Rest the mouse pointer on the symbol to show the linked entities. For modifications, select the appropriate workplane in the WP workplane tab and run the command.

The translation and rotation are displayed for each workplane. Rotations are listed in the order Z, Y, X. This corresponds to the transformation sequence. A workplane that has been selected is highlighted as selected in the graphics area. The current workplane may be hidden!

The world workplane can be enabled or disabled when creating a new document. Under File → Options → Options / properties → Model → Workplane, select the Show with new document option.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.
