11.1. Edit

Commands to edit and manage entities in the Edit menu:

Cut to clipboard.

Edit → Cut

Copy to clipboard.

Edit → Copy

Paste from clipboard.

Edit → Paste

Undo an action.

Edit → Undo

Redo an action.

Edit → Redo

Manage and calculate variables.

Edit → Parametric → Spreadsheet

Show graphic properties.

Edit → Graphic properties

Specify a layer of an entity as current.

Edit → Set layer as current

Specify a color of an entity as the current color.

Edit → Set color as current

Apply the graphic properties of an entity as the current properties.

Edit → Set graphic properties as current

Modify entities by transferring the attributes of an entity.

Edit → Copy attributes to entities

Change entities by transferring properties of the last selected entity.

Edit → Copy attributes from entity

Delete entities.

Edit → Delete

Delete entity with the highest ID number.

Edit → Dismantle model

Break down entities into their subentities.

Edit → Break

Unlink features from pattern or mirror

Edit → Unlink

Group entities into a group within the model structure or separate them from a group.

Edit → Group / ungroup

Give an entity a name.

Edit → Assign name

Move text, text label and dimension entities.

Edit → Move text / dimension

Move, rotate and copy individual or multiple entities.

Edit → Move / copy

Edit → Align Best-Fit

Fit entities in the best possible way by simultaneously moving and rotating.

Scale, stretch or shrink entities.

Edit → Scale

Mirror entities via axes and planes.

Edit → Mirror

Stretch two-dimensional contour curves.

Edit → Stretch 2D

Mate faces, groups and solids and define the relationship between them.

Edit → Mating

Mating clamping device.

Edit → Mating clamping device

Assign material.

Edit → Assign material

Lock entities to ensure that the entities cannot be modified.

Edit → Lock entities

Unlock entities that have been locked.

Edit → Unlock entities
