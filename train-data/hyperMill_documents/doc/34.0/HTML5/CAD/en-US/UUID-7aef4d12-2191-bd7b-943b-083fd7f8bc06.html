Selection

Input mode: Set the default setting for Project snap points.

Mesh highlight: Define the highlight for displaying meshes. If Default is selected, the mesh will be highlighted in color if the mouse is hovered over it. If Bounding box is selected, a box is displayed around the mesh. Use No highlight to avoid highlighting a mesh when hovering over it with the mouse.

Automatic preselection: After exiting a command, the results of the command are selected so that they can be used for the next action without having to be re-selected.

Keep preselection: If entities have been selected before you call a command, these preselected entities remain selected if the command is then cancelled. For this to work, Automatic preselection must be enabled.

Toolpath highlight: If dynamic entity selection is no longer feasible or is slowed down due to the size of toolpaths, it can be switched off separately for toolpaths. To do so, select No dynamic highlight. Alternatively, a bounding box can be displayed around the selected toolpath. In this case, select Bounding box. If Default is selected, the dynamic entity selection of all entities is used.

Pick through transparent color: Make entities that are hidden from view selectable through transparent colored entities.

Mesh edge selection: Switch the selection of mesh edges on or off.

Mesh face selection: Switch the selection of mesh faces on or off.

The settings are stored in the applicationSelectionProperties.xml file.
