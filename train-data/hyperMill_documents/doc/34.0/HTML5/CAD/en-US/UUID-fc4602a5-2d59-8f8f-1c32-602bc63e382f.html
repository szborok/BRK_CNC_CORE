Swarf cutting

Create faces to guide the tool during swarf milling.

Shapes → Swarf cutting

Prepare a stripe of faces in such a way that the approach information from the isoparametric lines of the faces can be used for a milling tool intended to roll along faces.

Use the Show inconsistencies option to show whether the tessellation tolerance of a face is too large for the selected tolerance. Create split faces at the tangential face transitions with the Split stripe option .

Base face

Faces: Select entities. The number of selected entities is displayed .

Offset: Enter a distance between the center of the resulting swarf faces and the base face .

Stripe height: Enter the width of the swarf faces.

Side faces

Faces: Select entities. The number of selected entities is displayed .

Tolerance: Enter the permitted size of the gap between the faces.

Diameter

Enter a diameter. A fillet is inserted in the face stripe for inner corners .

A fillet is not only inserted in the face stripe for inner corners; it is also inserted for inner fillets of the geometry that are smaller than the specified diameter.

Change side: Change the side of the side faces on which the swarf faces are generated.

Layer

A layer can be assigned to new entities. Select the Assign option. To do this, enter a layer name. If the layer does not exist, it will be created.
