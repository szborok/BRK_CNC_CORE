Markups

Exchange information with other users.

Select the data and mark it up with text information to be considered by other users in the model, without modifying the original data . To do so, create positions, curves, text or dimensions for the data at the locations in question and select . These entities are then converted into a markup and are only available within the markup . The display properties and active view are saved with the markup.

Faces can also be selected for a markup. Up to 256 entities can be selected.

Save the current workplane in a markup. Activating the markup makes the saved workplane visible and active.

Related topics

Markup options

Manage markup

Commands to create, manage and delete a markup:

Create new markup.

Drafting → Markup

Tab → Context menu → New markup

Zoom and display markup in the graphics area.

Tab → Context menu → Apply

Modify markup, e.g. processing status.

Tab → Context menu → Modify

Delete markup. The markup can only be deleted by its creator.

Tab → Context menu → Delete

Markup is removed. Modified entities are indicated by a change in display.

Tab → Context menu → Break

Choose from the following options in the Markup command when creating markups or the Modify command when editing:

Select: Select Entities . Text, dimensions, positions and curves can be selected. The number of selected entities is displayed.

Naming: Accept the suggestion or enter a new name.

Status: Control the markup status. The color or the markup entities is controlled by the status.

Description: Describe the data. The description is mandatory.

There are three areas in the Markups tab:

  • In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

    Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

  • All markups are displayed in a structure tree. When the cursor hovers over a markup, a thumbnail preview is displayed. Double-click with the left mouse button to zoom in on the markup in the graphics area. Click the column header with the left mouse button to sort Naming, Status, Author and Last modification in ascending or descending order.

    Markup is shown in the graphics area. Click with the left mouse button to hide the markup.

    Markup is hidden in the graphic area. Click with the left mouse button to show the markup.

  • History: Information on the history of the selected markup.

Forward markup

Commands to populate a markup in the document.

Compare the displayed markups with the markups currently saved in the document container. The markups are reloaded.

File → Markups → Reload

Import markups from a *.hmup document. Markups saved by other users with the Export my own command will be imported.

File → Markups → Merge from other user

Transfer the currently displayed markups into the document container.

File → Markups → Save

Save your markups in a *.hmup document. Only the markups will be saved, not the model geometry.

File → Markups → Export my own

Updating transfers the information. The information is saved in another part of the *.hmc document, and the original dataset remains unchanged. In addition, the file is also simultaneously open on two computers. A message appears to inform the user of a change. In this case, compare the information using Reload.

Markups can be exported to a *.hmup document and then sent.

Markups can also be edited and saved in the hyperMILL® SHOP Viewer. This is even possible if the document is write-protected.
