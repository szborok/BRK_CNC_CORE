Spline

Create or modify a cubic spline.

Curves → Spline

Create or modify a 2D / 3D free-form curve with degree = 3 and continuity = 2 (cubic spline).

If you select the Closed curve option , the end point is set automatically to the start point of the curve, creating a closed curve with a tangential transition. To move points in the Z axial direction, Use the manipulator. You can use the handles of the manipulator to move points in the Z direction and in all other directions.

To modify an existing spline curve with a degree of 3 and continuity of 2, select the curve and then call the Spline command.

Select the Change parameters option on the context menu and modify the spline by dragging the handles. The Modify control points or Convert to NURBS command can be used to attempt to adjust the degree and continuity.

If the selected spline is planar, it remains on its plane even if the points are moved. A 3D spline is always modified parallel to the current view. Press the CTRL key to select multiple spline points at the same time.

To insert an additional interpolation point, click once with the left mouse button on or near the required spline position.

To delete a control point, select the control point and press SHIFT+DEL or just DEL.

Select

Select the Points that are needed to create the spline curve. The number of selected entities is displayed.
