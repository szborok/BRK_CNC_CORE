Rectangle

Create a rectangle.

Drafting → Rectangle

Create a rectangle as a closed contour or as four individual curves or modify an entity of the ‘curve rectangle’ type.

Entity type

As rectangle: The result is a closed contour of the curve rectangle entity type .

As lines: Results in four individual curves .

The center point of the “curve rectangle” entity type can be snapped. Split an entity of the “curve rectangle” type into four individual curves using the Break command.

Modes

Diagonal: Define the rectangle by selecting two corner points .

Center and size: Define the size using X and Y and place the middle point of the rectangle in the graphics area . The size can be varied by dragging the handles.

Rotation

Angle: Align the rectangle by entering an angle or by dragging the handles .

Direction: Align the rectangle to an entity . Select the entity.

Reference plane

Selecting On current workplane creates the rectangle on the current workplane . Alternatively, select an Entity as the reference plane of the rectangle . This entity must be planar.
