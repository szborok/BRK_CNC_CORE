Sketch

Create one or several lines.

Drafting → Sketch

Create one or several lines. The individual line entity is limited by the start point and end point. When you select lines, face boundaries and edges before calling the command, the preview lines are also calculated for these entities.

Coordinates

Cartesian: A start and end point of a line is selected by entering coordinates, snapping or by clicking with the left mouse button in the graphics area (Z remains 0).

Polar: After you have selected a start point, define the end point of the line by entering the Length and Angle . The angle depends on the current workplane.

Sequence

Mono creates a single line. Selecting Multi creates a polyline. The end point is also the start point for the next entity. This next point can be selected using snap lines, preview lines and constraints.

Construction

Line: Create straight lines depending on the Mono or Multi setting.

T. Arc: Create a tangential arc with a tangential transition from the previous entity, if available. Enter an amplitude and a radius for the arc.

2P. Arc: Fit an arc into the selected entities in accordance with the selected conditions .

All types of curves (including NURBS curves, offset curves and ellipses) can be selected to create an arc.

Constraints

A grid-based entry is possible. Set Angular step and Linear step in File → Options → Options / properties. A 'Grid' is preferred over settings for snapping entities.

None: Freely select start and end points for lines.

Tangent: A line can only run tangentially into an entity to be selected.

Normal: A line can only run at the shortest distance normal to an entity to be selected.

Reference plane

Selecting On current workplane creates the line on the current workplane. Alternatively, select an Entity as the reference plane of the line. This entity must be planar.

Reference height: Change the position of the reference plane in the Z direction.

Show reference plane: Show the reference plane.

Context menu

Discontinue: Cancels the current line input.

Continue: Continue entering (at another position).

Copy start point: Copy the start point coordinates of the segment to the clipboard.

Copy end point: Copy the end point coordinates of the segment to the clipboard.
