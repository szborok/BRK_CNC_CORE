5.6. Mating options

Configure options for mating.

File → Options → Mating options

Configure options for mating.

Tolerance

Linear tolerance: The linear tolerance is used for distancing, for example, for the distance of a plane that must be parallel to another plane. For the distance of a rotation axis that should lie on the target axis for coaxial mating, or for the maximum distance during coincident mating. Enter a value for the permissible deviation.

Angular tolerance: The angular tolerance is used for alignment, for example, for the parallelism of normals when mating planar faces, parallelism of axes for coaxial mating or for perpendicularity during perpendicular mating. Enter a value for the permissible deviation.

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

To save and restore settings, right-click to open a context menu containing the following commands:
