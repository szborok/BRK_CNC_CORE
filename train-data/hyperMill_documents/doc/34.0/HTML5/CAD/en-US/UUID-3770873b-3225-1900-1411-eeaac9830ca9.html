Untrim faces

Restore the domain as an initial face or selected areas to the state they were in before the face was trimmed.

Modify → Untrim faces

You can reset all the trims for all faces in one go or you can just undo the trims for the edges or boundaries that have been selected .

Keep original: Specify whether the selected face is to be retained. When cancelling the trimming of a face within a solid, the untrimmed face always generates as a new entity.

Only complementary: Create an additional face for the affected area, if the trimming of the face cannot be undone, for example, in a solid or if the face is locked. Select this area using boundaries.

Select

Faces: Select the faces to be restored . The number of selected elements is displayed.

Boundary: Only select the boundaries for which you want to specifically undo the trims . The number of selected boundaries is displayed.

Mode

Automatic: One of the following solutions is automatically suggested.

Intersection: The flanks are extended until they intersect or cut the edge of the domain .

Straight: The flank ends are connected with a straight line .

Blending: The flank ends are connected by a connecting curve, which tangentially crosses over the remaining boundary of the face.
