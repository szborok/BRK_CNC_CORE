10. Workplane

Commands to create a workplane in the Workplane menu:

Create a workplane through three points.

Workplane → Through three points

Create a workplane on a curve.

Workplane → On curve

Create a workplane on a face with the Z direction in the face normal direction.

Workplane → On face

Restore workplane that was current when a V-sketch was generated. Modify the workplane.

Workplane → On V-sketch

Create a workplane on a hole.

Workplane → On axial face / hole

Set current V-sketch reference system.

Workplane → → Set current V-sketch work plane

Save a workplane.

Workplane → Save

Reset to world workplane.

Workplane → On world

Aligns a workplane to the current view.

Workplane → On view

Rotate the workplane around the X axis.

Workplane → Rotate → Around X

Rotate the workplane around the Y axis.

Workplane → Rotate → Around Y

Rotate the workplane around the Z axis.

Workplane → Rotate → Around Z

Rotate and move a workplane.

Workplane → Move

Set a workplane to be the current workplane.

Workplane → Set current

Activate and deactivate the display of the current workplane.

Workplane → Hide / unhide

Delete workplane.

Workplane → Delete
