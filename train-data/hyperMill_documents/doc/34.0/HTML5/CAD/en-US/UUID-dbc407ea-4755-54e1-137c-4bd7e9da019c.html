Stepwise view change

Stepwise view change

Rotate the view to the right.

View → Stepwise view change → Right rotation

Rotate the view to the left.

View → Stepwise view change → Left rotation

Rotates the view upwards.

View → Stepwise view change → Up rotation

Rotates the view downwards.

View → Stepwise view change → Down rotation

Rotate the view counterclockwise (2D rotation).

View → Stepwise view change → Counterclockwise rotation

Rotate the view clockwise (2D rotation).

View → Stepwise view change → Clockwise rotation

Moves the view to the right.

View → Stepwise view change → Right pan

Moves the view to the left.

View → Stepwise view change → Left pan

Moves the view up.

View → Stepwise view change → Up pan

Moves the view down.

View → Stepwise view change → Down pan

Zoom in. Zoom into the model.

View → Stepwise view change → Zoom in

Zoom out. Zoom out of the model.

View → Stepwise view change → Zoom out

The step size that is used to move is specified using 'Pan step, Rotate step and Zoom step'.
