Select entities

Select entities for hyperMILL® job.

Available from inside hyperMILL® commands

Select entities (for example, curves, contours, boundaries, faces, solids, points, information about the position of the clearance plane or the stock model) for a calculation in a hyperMILL® job.

Select: Select the entities. The number of selected entities is displayed.
