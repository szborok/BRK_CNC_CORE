Mouse operation

The following commands are assigned to different buttons and scroll wheel of the mouse:

Left mouse button:

  • Entity selection, for example.

  • Holding down the SHIFT key at the same time allows you to add further entities to a selection.

  • Holding down the CTRL key at the same time allows you to add an additional entity to a selection.

  • Holding down the CTRL key at the same time allows you to remove an entity from a selection. This action automatically considers all identical curves, boundaries and edges for the click position, even if an entity lying below was previously selected.

  • It is possible to change the name of an entry after a simple (long) click on the entry, for example, a layer name or the name of a workplane.

  • Double clicking activates an entry, for example, updates a layer, activates an entity type to filter during selection or only makes the required entity type visible.

Right mouse button:

  • Clicking a button in the command dialog opens a menu with the ‘Reset’ command that allows you to discard a selection.

  • Right-click to open a context menu that displays a selection of commands. The commands that are available depend on the position of the mouse pointer.

  • If the mouse pointer is positioned on a tab edge and an icon, right-click to display the tab and toolbar selection.

Scroll wheel:

  • The view in the graphics area changes depending on which command and which buttons are active.

  • If the mouse is positioned on the tab headers of a tab, you can scroll through the tabs.

The view stops changing if an entity is clicked using a 3D input device during rotation. The mouse click is handled preferentially. Dynamic highlighting with a 3D input device during rotation is also possible if you move the mouse slightly.
