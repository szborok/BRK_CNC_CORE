Calculating a value

If not yet available, create a row first . To do this, select Merge or Append or Insert row or Append row on the context menu.

Enter a name for the variable in the Variable column . To do this, double-click the cell with the left mouse button . If you have just created a new row, you can change the name straight away. Enter a name for the variable, such as L1 . Confirm your entry . To do this, press the ENTER or TAB key, click Ok or click another cell.

Enter a value for the variable in the Formula / value column . To do this, double-click the cell with the left mouse button . Enter a value, such as 50 . Confirm your entry. Continue to set up variables until all the values and constants you require have been defined. Also provide variables for calculations in formulas, for example, Area. When editing, a complete numerical value is displayed and processed, otherwise its is displayed as a rounded number.

Important

A decimal point must be used as the decimal separator. There is no function to automatically convert a comma into a point.

Enter formulas for calculations in the Formula / value column . To do this, double-click the cell with the left mouse button . Enter a formula, such as pow (L1,2) . It is not necessary to include an equals sign at the beginning. Confirm your entry.

Note

A variable cannot be modified inside a formula.

The calculation is performed. The result is displayed in the Result column .

To delete a variable, click the row with the variable and select Delete . If a variable that is being used is deleted, cells with formulas that depend on it will be highlighted in red. The variable must then be defined again or the formula will have to be corrected.
