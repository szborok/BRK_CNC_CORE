Dialog: Configure import

Configure which information should be read from which cells.

File name: The path and file name of the selected Excel file are displayed. Use ... to open a file dialog in order to select another Excel file.

Sheet name: Select an Excel sheet if the Excel file contains several sheets.

Link name: Enter a short descriptive name for the link. This option is only displayed for Manage link.

Rows

Enter from which row (Starting) to which row (Ending) the information is located. If Ending is equal to 0, all rows of the file are read.

Columns

For Name of a variable, Value, Data type, Physical quantity and Unit of measure of the parameter list, enter from which column of the file this information should be read.

Default settings

Select Data type, Physical quantity and Unit of measure as defaults in case the information cannot be read from the file because the specified cell is missing or empty.

Example 7. Ignore type specification from Excel file

If, for example, a value of ‘0 / Ignore’ is specified for the type for the column address, this means that all imported variables should be set to the value specified under Default settings, Data type (for example, ‘Double’).



Show a preview for checking purposes. The preview can be accessed via the magnifying glass in the upper part of the dialog. display of warnings and errors with the help of symbols – for example, when importing into already existing data, if variables differ only in their spelling or are the same but the data type differs, for example.

Import data once

Use Import file to import data from an Excel file once. No link is generated to this file.

Select the Microsoft Excel file format *.xlsx, *.xls or *.csv in the file dialog. Then select the file and Open.

In the dialog that follows, configure which information should be read from which cells.

Transfer data permanently

Use Manage link to permanently transfer data from several Excel files and manage links to these files.

The linked Excel files are monitored if the *.hmc document is open in which the Excel files are linked. Information is displayed when an Excel file has been saved.

A link to an Excel file consists of the following:

  • Link name: Enter a freely selectable name for the link.

  • On / off: Select a state for the activation. If the specified Excel file with the associated configuration is active, it means that the data will be read in again when updating.

  • Imported file and Path: File folder and file name of the Excel file are displayed.

+ : Add a link to a file or a data folder. Proceed in the same way as for Import file.

- : Remove the link to the selected file.

Clear all: Remove all links to files.

Update selection: Update data of the selected files.

Update all: Update all existing data.

Modify: For the selected file, change the settings for transferring the data. Proceed in the same way as for Import file.

Update link

Update imported data of permanently linked Excel files.

Note

A spreadsheet with an external link cannot be updated or changed if the Excel file is open. Close the Excel file or release the Excel workbook for collaboration. The Excel file must be opened via the share link. Only then can collaboration take place during processing.

Maintain links

Variables from permanently linked data are shown in italics. If their link is lost, the background is displayed in red.

The tooltip of the variable shows the folder with the link target. When opening the document, a warning is issued if the Excel file of the link is missing. Linked variables are read only. If the name of such a variable is changed, the link to the Excel file is removed and the variable is changed to a “local” variable.
