Polygon

Create a polygon.

Drafting → Polygon

Create a polygon.

Select

Edges: Enter the number of edges of the polygon .

Center: Select a middle point.

Modes

Circumscribed: The polygon lies around the circle specified by the radius or diameter .

Inscribed: The polygon lies inside the circle specified by the radius or diameter .

Input mode

Enter a Value and specify whether the value is a diameter Diam. or a Radius .

Rotation

Angle: The polygon is aligned by entering an angle .

Direction: Align the polygon to an entity to be selected .

Reference plane

Selecting On current workplane creates the entity on the current workplane . Alternatively, select an Entity as the reference plane for the position of the polygon .
