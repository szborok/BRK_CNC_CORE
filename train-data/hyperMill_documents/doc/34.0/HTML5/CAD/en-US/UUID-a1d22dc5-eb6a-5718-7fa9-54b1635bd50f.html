OMSL_COMPANY_SETTINGS

Import company-wide default settings for options/properties, interfaces, snap options, tooltip content, dimensions, title blocks, frames, feature settings and color libraries.

Set up the Windows environment variable. Enter the OMSL_COMPANY_SETTINGS environment variable under Variable name and enter the path, e.g. W:\Companyfolder\ under Variable value.

The folder structure on the server must be set up in accordance with the following rules.

Use a subfolder with the software version number as its name to hold the company settings assigned to a specific software version.
