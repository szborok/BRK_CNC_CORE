Layer

Layers provide a way to organize entities. Each entity is assigned to a layer.

The Layer section allows you to manage layers in a tree structure. You can create new layers, delete existing layers, specify the work layer and control visibility. New documents contain a blank layer.

The commands are selected using the context menu (right-click to open) or directly via an icon.

Simply click the Names field with the left mouse button to change the name of the layer. Layers are displayed sorted by name. If a layer is renamed, it is reordered in the list according to the sorting and the display of the layer list is updated by automatic scrolling so that the layer is visible in the list.

Click the Descriptions field with the left mouse button to insert a descriptive text. The Entities column displays the number of entities on the layer. A distinction is made between visible and invisible entities. The number of visible entities appears before the backward slash, and the number of invisible entities appears after the slash. The number of hidden entities does not only include the hidden entities, but also the entities that are not displayed as a result of the visibility filter itself.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

The search displays results only for layers, not for layer container names and current layers.

Set current layer

There are various ways to set the current layer:

  • In the Layer section of the Visibility tab, select a layer, right-click to obtain the context menu and then select Set current layer.

  • Click the button and select a layer in the layer list with the left mouse button.

  • Or create a new current layer.

  • Click File → Options → Options / properties and select the layer under Model → Attributes.

Assigning entities to another layer

There are various options for assigning entities to a different layer:

  • Select entities. Click the button and select a layer from the layer list.

  • Select entities. Call the Entity properties command. Set the desired layer in the Layer branch. The entities are assigned to the selected layer.

  • Assign selected entities to the current layer. To do this, on the context menu of the Model tab, select the Assign entities to current layer command.

Layer containers

Group layers in containers. To do this, create containers and move one or more layers to a container using a drag-and-drop operation. For this purpose, select the layers and drag them to the required container while holding down the mouse button. Now release the left mouse button. If the position of the cursor is suitable, the cursor changes from to . If the layers are organized using containers, they can be switched on and off in groups. To do this, click the icon in front of the container name.

If the current layer is located in a layer container, this layer container will also be highlighted in bold.

Exclude layers from selection operations

Exclude all entities of a layer or layer container from all selection operations. All entities of these layers are displayed in simplified form.

Switch off the selection of entities of the layer or layer container.

Exclude entities assigned to the selected layer from all selection operations.

Switch on the selection of entities of the layer or layer container.

Entities that are assigned to the selected layer can be selected again.

Some layers of the layer container are excluded from selection.

All entities that change layers automatically assume the status of the target layer. The selection of other entities works as if these entities were transparent. It is not possible to select or make selectable a subordinate entity of a non-selectable entity. A group and its subordinate faces should be on the same layer. If not, the group must be opened to set the layer to not selectable.

The simplified representation eliminates the hierarchy of entities, the boundaries, the texture and various other attributes. This improves rendering performance, but some commands are not available in this state, e.g., changing attributes, tessellation and visibility filtering. In particular, color filtering of faces of a solid is not available.
