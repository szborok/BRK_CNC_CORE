Check double entities

Use rules to find and handle identical entities.

Analysis → Check double entities

Use rules to find and handle identical entities (curves, faces and faces within solids). If a face is identified as a double entity within a solid, all of the other faces of the solid will also be automatically selected. The double face will not be removed individually from the solid. The command can be used to help analyze the different versions of a construction.

Points can also be checked in addition to curves, faces and faces within solids.

Select

Select entities. The number of selected entities is displayed.

Action on target entities

Delete: Delete duplicates.

Hide: Hide duplicates.

Tag: Identify duplicates with tag from Tag name.

Select: Select duplicates.

Tolerance

Tolerance controls the deviation from which (seemingly) identical entities are classified as different entities.

Tag name

Identify duplicates with Tag from Tag name.

Keep

Analytical before NURBS: Retain the ‘analytical’ entity or preferably deal with (for example, delete) NURBS duplicates.

First created: Keep the entity that was created first.

Last modified: Keep the entity that was created last.

The three rules are given an order of priority – Analytical before NURBS has the highest priority. The other two rules follow, each with a lower priority.

Target entities

Select the entities that are intended for Action on target entities. Duplicates deletes all duplicates, for example. Use Basic entities to process the entities that have not been identified as duplicates. The number of selected entities is displayed.

List

The list shows the analysis results as a list of all basic entities and their duplicates on the basis of their entity ID.
