Overview of the structure of the parameter list

The parameter list is set up as a table.

To edit a cell, click the cell with the left mouse button.

Multiple rows can be selected with the left mouse button while pressing the CTRL and/or SHIFT key at the same time.

If the name of a variable is changed, the name will also change in an assigned dimension and in the formula used.

Numerical values in the table cells are shown in blue.

Table 80. Content of the parameter list

Column

Purpose

Variable

Enter a name for the variable.

Formula / value

Enter a numerical value or a formula.

Result

The value entered or the value calculated in the formula is displayed.

Physical quantity

Select a physical quantity.

Select the quantity 'Integer' to enter a number of copies or e.g. offsets.

Unit of measure

Select a unit of measure. The selection you make here is predetermined by the physical quantity.

Data type

Select a data type for the variable.

  • Boolean value (Bool): A yes/no condition, where 'Bool' is the data type used in the software.

  • Integer value (Integer): Numerical value without decimal point, where 'Integer' is the data type used in the software.

  • Decimal value (Double): Numerical value with a decimal point, where 'Double' is the data type used in the software.

Min.

Specify the minimum lower value.

Max.

Specify the maximum upper value.

Used

Specifies how often the variable is integrated in a formula.

Assigned

Number of times the variable is assigned in other commands.

Description

Enter explanations to make it easy to understand the row later on.



Depending on the selected context, select the mathematical functions, mathematical constants and defined variables on the context menu or call the following commands:

Append row: Append an additional row to the bottom of the table.

Insert row: Insert an additional row above the selected row.

Delete row: Delete the selected row. When you delete a row, a reference error may occur if the defined variable is used again. In this case the field is highlighted in red and the formula needs to be revised.

List variables depending on it: List all the variables that are dependent on the selected variable.

Show assigned entities: Highlight entities in which a variable has been assigned.

List used variables: List all variables that are reused in the parameter list.

Evaluate variable: Force the formula to be evaluated.

Evaluate spreadsheet: Force all formulas to be evaluated. The spreadsheet is updated in real time. Not only is there an immediate recalculation; the syntax is also checked. The main purpose is to (re)check for division by zero or negative square roots and so on.

Variables: List all variables that are defined in the parameter list.

Functions: List all the available mathematical functions.

Constants: List all the available constants.
