Text line

Enter or modify a line of text along a reference entity.

Drafting → Text line

Enter or modify a line of text along a reference entity (straight line, arched curve or circle segment).

How are special characters entered in a text line?

To insert a special character in a text line, copy the character directly using the clipboard. You can list all characters by entering “charmap” at the Windows command prompt, for example.

Are engraving fonts available?

In the Text line command, fonts can be selected that may be suitable for CAM programming for engraving. The names of these fonts begin with the identifier TGS_.

Other engraving fonts (single line fonts) can be obtained over the Internet, either for a fee or free of charge.

Modes

Line generates text between two points. Click with the left mouse button to create the start point. Click again to specify the end point and the handles .

Arc generates text along an arched curve. Clicking with the left mouse button generates the start point. Clicking again creates the end point. The curvature can be varied with the cursor on the curve. Click again to display the handles .

Sector generates text along a circle segment. Clicking with the left mouse button generates the center of the circle. Clicking again creates the radius and the start point of the segment. The length of the segment can be varied with the cursor on the curve. Click again to display the handles .

Text

Enter or modify a line of text. The entry remains temporarily in the selection list for reuse until the document is exited.

If the text line is too small due to the current zoom factor and can therefore not be read, the text can be displayed using the tooltip. To do this, rest the cursor on the text line. This requires that the tooltip content is adjusted. The tag text content must be transferred for display in the tooltip content.

The text can be oriented using a horizontal or vertical snap line.

You can also modify a text line imported from the *.e3 file format.

Attributes

Specify the attributes of the Text and Text line.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

Reference plane

Selecting On current workplane creates the text on the current workplane . Alternatively, select an Entity as the reference plane of the text . This entity must be planar.

The specific position of the plane parallel to the reference plane is determined by means of the first point that is selected. If the following points are not located on this plane, they are projected onto the plane. Projection lines are displayed.
