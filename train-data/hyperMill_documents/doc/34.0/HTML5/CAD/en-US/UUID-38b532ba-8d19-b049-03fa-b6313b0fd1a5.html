3.4. Keyboard shortcuts

Change keyboard shortcuts.

File → Options → Keyboard shortcuts

You can use keyboard shortcuts to call commands. The key combinations are displayed in the menu bar after the command names and can be modified.

Some options can also be set using a keyboard shortcut. These include:

  • Toggle ‘Dynamic highlight’ on / off.

  • Toggle ‘Self shadows’ on / off.

  • Activate or deactivate ‘Anti-aliasing’.

  • Save an image of the current view with a white background in JPEG format in the document folder.

  • Save document and image of the current view in JPEG format. This is not possible for read-only documents.

Select a command in the structure tree and double-click in the row with the left mouse button. Press a key combination on the keyboard. This combination is entered in the dialog. If the key combination is already in use, select another combination. Use the button to the right of the line to delete a key combination that is no longer required. In the lower part of the dialog, select the area of the application in which the keyboard combination is to work, for example, just in the graphics area, only on the tabs or throughout the entire application. Avoid using single letters and the key combination ALT+[letter] – otherwise the input in certain text fields will override these key combinations and no longer be possible. Combinations such as ALT+SHIFT+[letter] or ALT+CTRL+[letter] can be used instead. Operating system key combinations such as CTRL+C (for copying to the clipboard) are fixed and cannot be changed.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

Use the commands at the bottom edge of the dialog to save and reset all settings:

Factory: Restores the factory system settings.

Discard: Discard selected setting again immediately.

The settings are stored in the customized.shortcut.xml file.
