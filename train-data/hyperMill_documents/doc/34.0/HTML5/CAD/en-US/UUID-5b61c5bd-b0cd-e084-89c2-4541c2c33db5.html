Blending

Create a new spline curve between two curves.

Curves → Blending

Create a new spline curve between two curves.

Same Continuity sets the same continuities on curve A and curve B. Automatic trim shortens curve A and curve B if the start points of the result do not lie on the start or end point of curve A and Curve B.

Select

Select the two initial curves Curve A and Curve B .

Continuity on A / Continuity on B

Use Position , Tangency or Curvature to specify the continuity for the transition of the existing entity to the resulting entity. The Parameter field defines the start point of the resulting entity on the existing entity. Weight determines the influence of the tangency and curvature fields of the entity on the shape of the result.
