Bisector

Create an angle-bisecting curve between two straight curves.

Drafting → Bisector

Create an angle-bisecting curve between two straight curves.

If two parallel lines are selected, a center line will be created.

Select

Select two straight Lines . If the curves are not coplanar, they cannot be selected together.

Length

You control the Length of the curve by entering a value for the position of the start and end point. The length can be modified interactively in the graphics area using handles .
