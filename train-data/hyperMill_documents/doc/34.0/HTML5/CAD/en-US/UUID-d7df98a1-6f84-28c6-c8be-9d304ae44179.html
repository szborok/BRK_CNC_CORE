Shape undercut

Look for undercut areas in the geometry.

Analysis → Shape undercut

Search for undercut areas in the geometry (faces, meshes). Only consider geometrical aspects, not aspects of machinability from a manufacturing point of view. For the analysis, select a direction from which machining could be performed. The undercut areas resulting from this direction will be colored. Curves can be created for the edges of these areas.

Select

Shapes: Select entities (1). The number of selected entities is displayed.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Options

Align view: Change the view of the graphics area in order to look at the geometry from the direction of the analysis.

Invert light direction: Align the analysis direction so that it is opposite to the currently selected direction (of the workplane) .

Compute limits: Perform an exact calculation for the transition between the undercut and non-undercut geometry.

Extract curves: Generate curves (polyline) that delineate the undercut areas .

Create markup: Assign a Markup to the extracted curves. A markup can be used to restore the view for the direction of the undercut analysis. Also see Markups.

Tessellation

Determine the precision of the calculation by changing the discretization of the entities.

Tessellation: Control the precision of the tessellation.

Edge length: Determine the edge length for the resulting triangles.

Vertical areas, which are to be regarded as undercut by other undercut entities from the point of view of machinability for manufacturing, are also recognized as undercuts.

Shadows

As a result of the analysis, the existing colors of the geometry in the undercut area are darkened .

Darkening factor: Enter a factor between 0 (no darkening) and 1 (strong darkening) to specify the intensity of the shading.

Fixed colors

As a result of the analysis, the existing colors of the geometry are temporarily replaced .

Undercut color: Select a color for the undercut areas of the geometry.

Color for visible areas: Select the color for everything else that is neither an undercut area nor a vertical entity of the geometry.
