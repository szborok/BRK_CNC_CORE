7.1. View

Commands to modify the view in the View menu:

Set the view to the current workplane (XY plane).

View → View on workplane

Set the view to the selected face.

View → View on face

Move, zoom and rotate the model view using the left mouse button.

View → Pan-zoom-rotate

Move the view a certain distance defined by entering two positions.

View → Pan

Zoom to magnify entities in the graphics area after drawing a window around the required area.

View → Zoom window

Fit entities.

View → Fit

Toggle projection of the representation.

View → Perspective view on / off

Set 3D stereo display.

View → 3D stereo display

Adjust the display of entities to the size of the graphics area.

View → Zoom entities

Set a rotation target.

View → Rotation target → Set

Remove a rotation target.

View → Rotation target → Reset

Go to previous view.

View → Previous

View → Next

Go to next view.

Reset all view changes to the default settings.

View → Reset view

Zoom, rotate and pan the view in steps.

View → Stepwise view change → ...

Set axial view.

View → World views → ...

View → Workplane views → ...

Set isometric view.

View → World views → Isometric (world) → ...

View → Workplane views → Isometric → ...

Set a view for turn machining.

View → Workplane views → Turn machining views → ...
