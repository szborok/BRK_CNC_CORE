Shape - shape distance

Analyze distances between two faces, meshes or face groups in a context.

Analysis → Shape - shape distance

Analyze distances between two faces, meshes, or face groups in a context. Determines the distance to the stock, the simulated machining state, and the model. The distances are measured based on a tessellated model of Shape A and Shape B.

Select

Shape A: Select entities, such as entities to be analyzed . The number of selected entities is displayed.

Shape B: Select entities, such as reference entities . The number of selected entities is displayed.

Transparency: Set a transparency for the entities selected in Shape A or Shape B . The selection is determined by the direction defined by Normal to A and/or Normal to B.

Normal to A: Measurement is done in the direction of the face normal vectors of the entities selected in Shape A .

Normal to B: Measurement is done in the direction of the face normal vectors of the entities selected in Shape B .

Use entities tessellation

Use Tolerance to specify the required level of precision or use Use entities tessellation to apply the required level of precision from the selected geometry.

Color map

Min. and Max. display the determined minimum and maximum values.

The X axis shows a color map and assigns a color to each value in the analysis area. On the analyzed faces, a color symbolizes the value assigned to it. The Y axis provides qualitative information about the scope of the respective value in relation to all of the determined values. A thin, vertical line shows the analyzed position.

Use Limits to limit the analysis area and find a color map that is suitable for the analysis. Select a minimum Min. value and a maximum Max. value. Values below Min. are shown in dark grey . Values above Max. are displayed in light grey . Use Reset to set a restricted analysis area to its original size. Auto calculates an optimal analysis area.

The value at the cursor position can be written to the clipboard by right-clicking.

The value ‘inf’ is displayed when no result was determined, or if the result goes to infinity.

The analyzes in Extract curve and Color map are separate. The entries do not influence each other.

Extract curve

Generate curves (polylines) that follow selected levels .

Min. and Max. display the determined minimum and maximum values.

Use Limits to limit the analysis area and find a color map that is suitable for the analysis. Select a minimum value Min. value and a maximum value Max. value .

Number of levels: Enter the number of levels . The levels are between those in the option Limits defined minimum and maximum values.

The table shows a color for each area between the levels. The boundaries of the areas are indicated.

Exact computation: Generate curve entities from the levels displayed. The curves are on the tessellated model. Polylines are created as a result of the tessellated model. Use the polylines to delineate milling areas.

The analyzes in Extract curve and Color map are separate. The entries do not influence each other.
