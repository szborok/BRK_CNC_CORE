15.1. Direct modeling

Only select the faces or features of a solid that are to be modified and change the values for position and shape specifically without knowing all of the other values. Changes can be made interactively by dragging the handles and the manipulator. The shape of the selected entities and / or all other faces and features that are linked to the selected entities is adjusted automatically.

If faces become or remain a part of a solid, the faces will be highlighted (in red) in the preview.

Note

Direct modeling is not possible on a parametric CAD model!

The procedure follows these principles:

Use direct modeling for features and faces in solids that were modelled directly in the software. Direct modeling also works for a feature within a feature, such as a fillet in a slot .

Direct modeling can also be used for imported or static (with no construction history) solids . In this case, all faces, such as fillets, must be removed beforehand in such a way that the next accurate trim edge can be found automatically during the calculation. The calculation cannot be performed without a trim edge.

When direct modeling is used with features that are modelled directly in the software, existing fillet features and chamfer features within a feature are not modified as well. They are generated automatically at the new position .

A feature can be modified via direct modeling or using the “Change parameters” command. After direct modeling, the feature is broken up.

Direct modeling can only be used on solids that are open or closed.

Various options are available for direct modeling. These include:

Delete a CAD feature or delete faces (no CAD feature) in the shape of a chamfer or fillet from a solid after selecting the entities with the DEL key. The solid is closed automatically.

Modify features, for example, using ‘Edit > Move / copy’ .

Modify faces within a solid, for example, using ‘Edit > Move / copy’.

Mirror features using ‘Edit > Mirror’ .

To modify the size of a chamfer or fillet , for example, double-click the feature. Alternatively, create an entity first and then edit it afterwards. To do this, select the entity, call up the Change parameters command on the context menu and modify the entity using the handles. The More parameters command on the context menu opens a dialog for adjusting all attributes and options.

Split faces within solids using curves. Modify a separated partial face, for example, using Edit → Move / copy .

Faces can be moved inside a solid into target faces that are not connected to the initial face .
