Graphic properties

Show graphic properties.

Edit → Graphic properties

List the graphic properties of one or several entities based on the entity type. Common attributes can be modified in a single step. Select the entities.

Table 86. Settings that can be made in the graphic properties.

Property

Description

Layer

Select a layer from the list.

Color

Select color from list.

Isoparametric curves along U

Enter the number of isoparametric curves in the U or V direction.

Isoparametric curves along V

Points along U

Enter the number of points for controlling the graphic display along U or V.

Points along V

Face tessellation tolerance

Adapt the precision of the entity displays to size and precision requirements.

Curve tessellation tolerance

Tessellate connected

Switch the connected tessellation on or off for each entity.

Line type

Select a line type such as a dash-dot line for a center line.

Line thickness

Select a thickness for displaying curves and boundaries.
