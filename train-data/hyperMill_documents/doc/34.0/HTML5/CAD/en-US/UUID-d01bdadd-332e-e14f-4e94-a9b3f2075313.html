Coordinates

Enter points or positions as coordinate values.

Select → Snap → Coordinates

Determine points or positions within other commands by entering coordinate values (“Second Level” command). Values can be inserted via the clipboard from an NC record, for example, whereby an attempt is made to evaluate the output format. Before inserting, select the required areas (for example, X, XY or XYZ).

Coordinates

Cartesian: Enter point or position via X, Y and Z value .

Polar: Enter point or position via Rho, Theta and Phi value (polar coordinates) .

Cylindrical: Enter point or position via Rho, Phi and Z value (cylindrical coordinates) .

Origin

The values entered under Coordinates relate to the position selected under Origin.

Use Selection to select an X, Y and Z value in the graphics area or double-click the coordinate value in the table with the right mouse button to enter a numerical value.

Use Progressive after selecting Apply to add the values entered under Coordinates to the values in Origin.
