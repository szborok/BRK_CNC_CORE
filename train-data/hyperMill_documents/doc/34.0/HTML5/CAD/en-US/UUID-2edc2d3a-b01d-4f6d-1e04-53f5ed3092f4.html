Modify control points

Modify a curve using its control points.

Modify → Modify control points

Change the curve by dragging the handles with the left mouse button. Use manipulator displays a manipulator that you can use to drag the selected control point in the X, Y and, in particular, the Z direction. Select Keep original to create a copy of the initial curve.

To delete a control point, select the control point and press SHIFT+DEL or just DEL.
