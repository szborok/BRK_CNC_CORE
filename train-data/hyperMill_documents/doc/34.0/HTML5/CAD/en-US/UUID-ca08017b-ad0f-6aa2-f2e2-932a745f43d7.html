Rotational slot

Remove a round slot that has been created by rotating a contour from a solid.

Features → Rotational slot

Solids

The faces are generated within the base entity with the start and end angle of the rotation ( Angle) or by dragging the handles.

If the Close sketch option has been selected, the areas of the start and end point of the rotation curve that are still open are closed by faces perpendicular to the rotation axis.

The Multishell solid option allows spatially separated subareas for a solid. Otherwise, individual solids are created in correlation to the number of separate subareas.

Note: If curves within V-sketches or parametrically created faces are preselected, the option is automatically activated. It cannot be deactivated and is greyed out. The option can be switched when there is no preselection or if the preselection includes static entities.

With Invert, the slot remains as a solid, and the rest of the initial solid is discarded.

Symmetric: Apply an angle from selected contours evenly in both directions.

Select

Curves: Select the curves and boundaries. The number of selected entities is displayed .

Faces: Select entities . The number of selected entities is displayed.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin of the rotation axis is initialized at the first selected point.

For the origin of the rotation axis, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.
