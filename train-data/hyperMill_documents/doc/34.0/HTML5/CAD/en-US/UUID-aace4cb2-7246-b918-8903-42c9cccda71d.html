2D undercut

Insert predefined 2D undercuts.

Drafting → Turning → 2D undercut

2024

Insert undercuts for local rewinds into a 2D contour for turning in a form standardized according to DIN 509. The inserted undercut curves are marked with a T_UNDERCUT2D_TYPE tag.

Select

Curves: Select the two curves at right angles to each other of a corner where an undercut is to be inserted . The number of selected entities is displayed.

Select undercut

Shape: Select the shape of the undercut based on the standard offered (E, F, G, H) .

Configuration: The software automatically suggests a size range if the workplane is set for turning the 2D contour. The X axis of the workplane must lie in the rotational axis of the contour. Optionally change the size range.

More options

Automatic trim: Trim the contour automatically after inserting undercuts .

Associative group: The curves of the undercut are grouped. The group is given the shape and the selected configuration of the standard as its name. When these entities are changed, the linked entities are also updated.
