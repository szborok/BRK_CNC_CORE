16.14. Print

Generate an overview of electrodes for erosion

Electrodes → Print

Electrode

Create an overview for an individual electrode or a summary overview for all electrodes for documentation purposes.

Dimension the electrodes beforehand.

Change and save the layout of the title block in the templates using the fixed names single_electrode_tb.hmc and assembly_electrode_tb.hmc. You can use tags to include electrode properties for the title block in the templates.

Single electrode sheet

Click with the left mouse button to select an electrode from the list in the dialog or in the graphics area.

Mixed single sheet: Combine two side views of a single electrode with two views of this electrode in the workpiece . Otherwise, print out three side views and a three-dimensional view .

Note

Enter the details in the title block for documentation of a single electrode beforehand using the Electrode options command and make adjustments when creating an electrode with the Create or Create user defined commands. To make subsequent changes, select a whole electrode and modify the required attribute under Tags → Edit info. Please note that changes are not permitted for all attributes!

The views always refer to the current workplane. Therefore, the derivations cannot be influenced.

Assembly electrode sheet

To display an overview of all the electrodes, select the Print assembly option . The electrodes should not be hidden. After printing, the option switches itself off again automatically.

Add the document properties for the title block of the global overview in the table in the lower area of the dialog.

These properties will be automatically added to a report.

Print settings

Printer: Select an installed printer and specify the printer settings . Select the settings on the basis of the printer description. The printer name is displayed.

Page: Set up the page for printing . Select the paper size, orientation and width of the margins.

The unit of measure for the margins and the ISO / ANSI page size are synchronized with the operating system settings. Use the ‘Region and Language’ and ‘Devices and Printers’ settings of the operating system to make changes.
