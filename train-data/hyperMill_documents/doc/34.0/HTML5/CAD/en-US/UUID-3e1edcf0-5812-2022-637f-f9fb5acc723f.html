Difference

Using a solid, remove the areas of other solids that this solid intersects in the other solids.

Booleans → Difference

Solids

Select

A: Select entities .

Solids B: Select entities . The number of selected entities is displayed.

Region: Select entities to be retained. The number of selected entities is displayed.

Modes

A - B: Entities from the ‘B’ selection are removed from the ‘Solids A’ selection .

B - A: Entities from the ‘Solids A’ selection are removed from the ‘B’ selection .

Multishell solids

Number of solids: The number of parts of a resulting multishell solid is displayed.

Separate solids: Allow a number of individual solids that corresponds to the number of subareas that are spatially separated from one another. Otherwise, subareas that are spatially separated from one another are generated in one solid.

Keep original solids

Solid A: Specify whether the selected entity should be retained.

Solids B: Specify whether the selected entities should be retained.

Faces as an open solid

The volume is located on the rear side of a face. The volume is determined by the direction of the solid normals. If no closed solid is generated, an infinite volume size is defined.
