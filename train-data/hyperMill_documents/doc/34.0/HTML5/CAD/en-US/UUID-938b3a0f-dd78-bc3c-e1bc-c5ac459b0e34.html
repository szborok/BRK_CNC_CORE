2.2. Graphics area

The graphics area allows you to interact with the entities of a document as geometric representations.

Edges and boundaries

Face boundaries represent the borders of a face and are indicated by a dashed line when the mouse is positioned within the face a short distance away from the border. Borders within solids are edges and are indicated by a thin solid line when the mouse is positioned directly on the edge or in close proximity to it.

Manipulator

Within various commands, you can use the manipulator to edit input interactively. To do this, click the manipulator with the left mouse button and move it to the desired location while holding down the mouse button. The position at which you click influences the range of motion. If you select an axis, the manipulator moves along the direction of this axis. If you select an arc, the manipulator is rotated around the axis . If you choose an origin, the manipulator can be freely positioned .

To ‘latch’ precisely to a geometrically significant position of an entity, activate the Snap for manipulator / handles option. Select the option in the context menu under Selection or preset in theSnap selection filter using the Use for manipulator / handles option.

Distances or positions can optionally be selected only in a specified step size. Enable the Grid for manipulator / handles option under Options / properties and specify the step sizes.

Handles

Within commands, handles are provided with which changes can be made interactively. Click a handle with the left mouse button and move the handle while holding down the mouse button until the required size is reached.

To ‘latch’ precisely to a geometrically significant position of an entity, activate the Snap for manipulator / handles option. Select the option in the context menu under Selection or preset in theSnap selection filter using the Use for manipulator / handles option.

Distances or positions can optionally be selected only in a specified step size. Enable the Grid for manipulator / handles option under Options / properties and specify the step sizes.

Mini dialog boxes

A mini dialog box is displayed near to the entity. Enter the value here rather than in the command dialog. The context menu (right-click to open) provides a selection of options for the respective command that allow you to switch between different input options, such as between the angle of an arc and its radius.

Switch to the mini dialog box using the TAB key If the input field is not active for an entry, the value is displayed rounded. During an input, a value is displayed fully accurately.

Angles can be input as deg:min:sec or deg min sec (separated by spaces) and will then be converted to the selected format.

Tooltip

The entity ID, type and geometric attributes, for example, can be displayed for dynamically highlighted entities. A workplane can be identified via its name. Switching on is possible under Options / properties, configuring is possible under File → Options → Tooltip content.

An error message can also be shown in a tooltip when directly selecting entities or positions during dimensioning, for example.
