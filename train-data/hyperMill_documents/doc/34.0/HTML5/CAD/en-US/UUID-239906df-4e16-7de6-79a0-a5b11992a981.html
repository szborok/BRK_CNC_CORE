Tabs

Toggle tabs on/off. All activated tabs are visible.

Tabs can be toggled on/off in the menu bar or title bar of a tab. By right-clicking, a context menu with all available tabs can be opened. Clicking on an entry with the left mouse button switches the display of a tab on or off.

Reset: Restore the original condition. A modified tab is marked by (*).

Reset graphic area icons: Restore the upper and lower toolbars of the graphics area to the original state.

You can freely position, dock, and change the size of all tabs. They can be placed outside the program window. This is recommended if you work with multiple monitors, for example. The position of the tabs is saved when you exit the software.

The tabs are arranged using drag-and-drop operations. A preview of the possible position is provided . To do this, move the mouse to the title bar of the tab and drag the tab while holding down the left mouse button. Tabs can also be dragged into each other . To do this, drag the tab directly to its target. The tabs are then given tab headers and are displayed within the same frame. Double-click the title bar of a tab with the left mouse button to remove it from the layout and reinsert it into the previous position in the tab layout.

Vary the size of the icons continuously between 16 x 16 pixels and 96 x 96 pixels. Position the cursor on one of the icons and, while holding down the CTRL key, set the required size with the scroll wheel.

Set the number of columns of the Swiping grid tab using the Number of columns in swiping grid tab option under Options / properties → User interface.
