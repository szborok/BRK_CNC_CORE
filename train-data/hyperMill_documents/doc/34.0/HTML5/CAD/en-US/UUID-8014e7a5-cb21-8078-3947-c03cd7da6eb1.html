Filter selection

Set a filter for the selection of entities using the following commands:

Set selection filter for points.

Select → Filter entities → Set points filter

Set selection filter for line, arc, ellipse and NURBS.

Select → Filter entities → Set curves filter

Set selection filter for face boundaries.

Select → Filter entities → Set boundaries filter

Set selection filter for V-sketches.

Select → Filter entities → Set V-sketches filter

Set selection filter for faces.

Select → Filter entities → Set faces filter

Set selection filter for features.

Select → Filter entities → Set features filter

Set selection filter for the selection of solids.

Select → Filter entities → Set solids filter

Set selection filter for meshes.

Select → Filter entities → Set meshes filter

Set selection filter for groups.

Select → Filter entities → Set groups filter
