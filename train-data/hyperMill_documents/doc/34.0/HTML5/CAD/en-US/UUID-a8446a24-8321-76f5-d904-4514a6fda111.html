Through sections

Create a face from curves that are adopted as isoparametric curves.

Shapes → Through sections

Create a face by means of a set of curves that are taken over into the face as isoparametric curves.

Each support curve can consist of multiple adjoining curves or boundaries (chain). The orientation of the curves within a chain is adjusted automatically. The orientation of an entire chain is adapted to fit the orientation of another chain.

The selected sections (chain) must all be open or closed. If a single open or closed section is found among the selected sections, a warning will be issued.

Select

Entities: Select one or several orthogonal curves . The number of selected entities is displayed.

The first and last entity can also be defined via a point.

Modes

If Fit is selected, the section curves that are used internally for calculation are interpolated. If Smooth is selected, the section curves are approximated to ensure that the face that is created swings less.

Tolerance

To create the face, the transitions between the support curve segments are adjusted with a constant curvature (C2) within the specified Tolerance. It is not possible to create a high-quality smooth face for a positional transition (C0). The selected entities are not modified.

The size can also be specified by entering a Value or using the slide control. The value range moves between the 'Approximation tolerance' and the hundredfold 'Approximation tolerance'.

Options

Section counter shows the number of usable chains .

Closed face: Close the face with continuity G0 . The 'end' of the face is positioned precisely at the 'start' of the face. The precise course of the nearby face is not defined in any more detail. Select at least three chains or only one point and at least two chains.
