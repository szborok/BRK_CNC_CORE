Ruled

Create faces from two curves that face each other.

Shapes → Ruled

Create one or several faces from two opposite curve sets and face boundaries.

Select

Curve A and Curve B : The curve sets A and B must ‘face each other’. You can click with the left mouse button to alternately select the boundaries that are facing each other ( Curve A / Curve B).

You can also use the chain selection command . Use the chain to first select all of the boundaries of curve set A followed by those of curve set B.

The number of selected entities is displayed. The number must be equal in both curve sets in order for a ruled face to be created.

Invert: Reverse the alignment of curve set B if the result you desire is not displayed in the preview.

The transitions of the faces depend on the transitions of the curves that were used to create the face. A curve set can be rotated in the preview by clicking with the left mouse button.

Modes

Control the alignment of the isoparametric curves of the face in the U direction.

By length: Divides the respective length of curve sets A and B into an equal number of sections and connects the sections with the same ‘ordinal number’.

Parameter: Takes the existing division based on the curve parameters from curve set A and B and connects the sections with the same parameters.
