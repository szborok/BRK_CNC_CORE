Boundaries

Create curves at face boundaries and mesh edges.

Curves → Boundaries

Create curves that correspond to a face boundary of one or more faces or with the edge of a mesh.

The number of curves is displayed under Total curves. The Sewing tolerance option controls the permissible size of face gaps within all selected faces for which face boundaries are still regarded as continuous. This means that no curve arises in the gap.

Convert lines / arcs to analytical: Convert straight and arc-shaped NURBS entities to analytical geometry.

If you select Boundaries recognition, the curves will only be created on the outer face boundaries of all the selected faces .

Longest chain: Determines and generates the longest boundary from all continuous face boundaries .

The curves are not linked with the original boundary of the face. The curves can be trimmed or shortened in order to create filling faces, for example, or the machining direction specified for CAM strategies.

Select

Boundary: Create curves on the selected boundaries of a face. Select the face boundary . A preview is displayed Chain can be used to select the outer boundaries of face groups, for example.

Shapes: Create curves on all boundaries of a face or mesh . The number of selected elements is displayed.

Outer chain from view direction

Creates the Outer chain from all continuous face boundaries . If multiple solutions are possible, the outermost chain from the direction of the current view is created.

View: Update the calculation after the view has been changed in such a way that the required area of the geometry is viewed.
