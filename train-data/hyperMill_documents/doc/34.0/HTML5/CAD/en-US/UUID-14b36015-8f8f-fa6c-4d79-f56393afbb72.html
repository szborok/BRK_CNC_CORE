4.1. File

Commands to open and close documents in the File menu and other document management commands:

Create a new document.

File → New

Generate new document from template.

File → New from template → ...

Open files with different file formats.

File → Open

Open document.

File → [Document]

Open files with different file formats and merge the data into the current document.

File → Merge

PLM-supported document management with the hyperMILL® PLM Connector.

PLM Connector → Browser

Compare processing states and optionally transfer modified geometric entities.

File → Compare and merge

Save the current document.

File → Save

Save the current document under a new name.

File → Save as

Save entities from the current document in a new document, which can then be saved in a different file format.

File → Save selection

Protect a document by entering a user-defined password.

File → Protect

Close current document and its program window.

File → Close

Exit the current application.

File → Exit
