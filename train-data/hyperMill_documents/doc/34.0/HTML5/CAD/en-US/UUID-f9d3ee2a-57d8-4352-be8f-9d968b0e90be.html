Chamfer

Create or modify a chamfer as a CAD feature along a continuous edge in a solid.

Features → Chamfer

Solids

Create or modify a chamfer as a CAD feature along a continuous edge in a solid.

Select

Edge: Select edges of a solid . The number of selected entities is displayed.

Invert: Chamfers with different distances or an angle not equal to 45° can be inverted. Click the Invert button and then select the flank faces in the section of the chamfer to be inverted.

Modes

Symmetric: Create symmetric chamfers with Distance , for example, for faces at right angles to each other at an angle of 45° .

Distance: Create chamfers with Distance 1 and Distance 2 .

D. + angle: Create chamfers in Distance 1 and Angle .

More options

Tangential faces chain automatically continues the chamfer, in case a tangential extension can be found automatically.

Triangle face on vertex: Flatten corners .
