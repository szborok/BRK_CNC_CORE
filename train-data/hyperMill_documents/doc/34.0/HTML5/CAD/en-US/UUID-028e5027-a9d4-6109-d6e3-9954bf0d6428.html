Helical protrusion

Create a protrusion via the helical sweep of a contour from curves and boundaries on a face within a solid.

Feature → Helical protrusion

Solid

2025

Create a protrusion via the helical sweep of a contour from curves and boundaries on a face within a solid. The curves must form a single closed and plane chain. Parametric modeling is supported.

Select

Curves: Define the starting point and profile of the helix. Select curves . The distance between the curve and the origin defines the radius of the helix at its base. The number of selected entities is displayed.

Faces: Select entities . The number of selected entities is displayed.

Select the direction and the origin of the rotation axis .

More options

The Height describes the total height of the helix . The Pitch and Taper set the angle, further defining the shape of the helix. The direction of rotation is reversed using Clockwise.

Variable pitch: Optionally, change the pitch continuously from the beginning to the end of the helix. Enter a value for the Final pitch .

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.
