Projection points

Project points onto entities or onto a workplane.

Drafting → Projection points

Project points onto entities or onto a workplane.

Select

Points: Select the points to be projected . The number of selected entities is displayed.

Entities: Select the target entities for the point projection . The number of selected entities is displayed.

Projection direction

Normal: Projects the points in the direction of the respective face normal .

Or, you can choose other projection directions :

Forward only: The projection is only made in the selected direction .

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Plane definition

Use On plane to temporarily create a plane for the projection in Modes:

Entity: Select a planar entity (planar curve, plane face, workplane) .

3 points: Select three points that are not collinear .

Direction + origin: Select a direction. A preview of the plane will be displayed perpendicular to the direction at the selected origin :

Now select an Origin:

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.

More options

Keep original: Specify whether the selected entities should be retained.

Keep attributes: Color attributes are retained.
