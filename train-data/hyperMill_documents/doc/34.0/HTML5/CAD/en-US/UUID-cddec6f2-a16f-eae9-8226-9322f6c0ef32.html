Close holes

Close openings within faces.

Shapes → Close holes

Close openings within trimmed faces with new faces on the basis of the original, untrimmed face .

The openings for which a calculation has failed are displayed by clicking the message on the Info tab.

Select

Faces: Select faces . The number of selected entities is displayed.

Resulting faces

The number of new entities is displayed. Planar shows the number of planar faces and Non planar shows all the non-planar faces.

Layer

A layer can be assigned to new entities. To do this, enter a layer name. If the layer does not exist, it will be created. If you leave the field empty, the current layer will be used.

Advanced

Max. perimeter length: Maximum length of the circumference of an opening that is still to be closed.

Detect and close multi-faces holes: Detect and close cross-face openings between faces. The selection should be targeted.
