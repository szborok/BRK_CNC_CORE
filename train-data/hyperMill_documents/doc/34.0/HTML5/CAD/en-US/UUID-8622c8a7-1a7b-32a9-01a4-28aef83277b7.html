16.3. Create reference system

Create a positional reference between the workpiece and electrodes.

Electrodes → Create reference system

Electrode

Create a workplane for the positional reference between the workpiece and electrodes. As an aid, a bounding box with snap positions that is exactly large enough to fit all selected workpiece entities inside the box will be displayed. Selecting a snap position in the center of each side of the snap area will suggest the adjusted Z axis orientation for this side.

The selected Entities are applied as a collision model for the collision check and collision simulation in SIMULATION Center and are necessary for the contour-aided selection of the electrode shape when creating the electrode.

The workplane is automatically assigned with a name. Specify the composition of the name in the Electrode options.

Important

The position and direction that serve as a point of reference for a reference system may not be subsequently changed if it has already been used to generate electrodes. The positions of an electrode in X, Y and Z are not associatively linked to the reference system. More faces can be added to the bounding box when the identical reference point is selected to update the existing reference system by overwriting it. Changing this reference point leads to incorrect erosion measures. Likewise, the electrode workplane linked to the electrode must not be changed manually.

The selected geometry is automatically locked by this command. Unintentional shifting or scaling of the workpiece is prevented. The lock can be released manually at any time at one’s own risk using the command Edit → Unlock entities. If manually unlocked, the lock will no longer be set automatically afterwards. Only as long as the lock is present can the software inform that the workpiece is being used for eroding and should not be changed.

Note

It is advisable to lock the workpiece again after making a change. To lock the part again, reassign the eroding reference to all faces and select the existing name.

Example 20. Add faces to expand the collision model

Faces can be added to a collision model by adding TAG information.

  1. Determine the value of a face’s TAG that already belongs to the desired collision model. To do this, select this face (not a whole solid!) and open the command TAGs → Edit info. There, search for the TAG T_ELECTRODE_PART_FOR_EDM_* in the TAG name column. Where * is a variable part of the TAG name that was assigned when the collision model was created. Determine the value in the Value column and note it.

  2. In the tab sheet TAGs in the tree structure, find the TAG TAGs → T_ELECTRODE_CATEGORY → T_ELECTRODE_PART_FOR_EDM_* again. Right-click to open the context menu and select the Assign command. Within this command, select the faces to be added to the collision model. And enter the noted value in the line T_ELECTRODE_PART_FOR_EDM_*! Apply the entries.



Select

Entities: Select faces or solids . The number of selected entities is displayed. The selected entities are applied as a collision model for the collision check and collision simulation in SIMULATION Center and are necessary for the contour-aided selection of the electrode shape when creating the electrode.

Reference point

Point: Select or snap a position as the reference position of the electrodes in relation to the workpiece .

Note

In some special cases, the suggested snap positions for the electrode reference may not be precise enough. For this reason, the selection of these positions is disabled to be forced to select a point of an entity. It is important to avoid creating the bounding box and selecting the corners without checking the values. The reason is that the bounding box is based on tessellation and a coarse tessellation might not be precise enough for the process. Tip: Avoid selecting faces that are trimmed on the measured side.

Approach point

Point: Select an additional reference point for the eroding machine if the reference point of the part cannot be reached by the machine. This point will be output in the electrode summary.

Motion

From the selected origin, move and rotate the origin further with the manipulator. When the handles of the manipulator are moved in the graphics area, the values in the dialog are updated. If an arc or an axis is clicked, the cursor jumps to the associated input.

Start: Select start point .

End: Select end point .

Reference: Relate the specifications for the linear translation in the X, Y and Z direction to another point. Select a point.

Delta: Enter or check the linear translation in the X, Y and Z direction .

Angle: Enter or check the rotation around the X, Y and Z axis .

Align start X axis

The X axis of the manipulator can be aligned under Direction.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Reference plane

The edges of the bounding box will be aligned in relation to the axes of the current workplane. Alternatively, specify a suitable plane face as an Entity. The position of the face in the space and the isoparametric curves of the face in the center of the domain in the U direction are used to align the bounding box.

More options

Clearance plane offset: Add a distance to the highest determined Z-value of the workpiece geometry to guarantee safe movement of the eroding machine from the safety position of one electrode to a safety position of another electrode. Enter a value .

Save as

Accept or modify the suggested name. To modify the name, click in the field and adjust it.
