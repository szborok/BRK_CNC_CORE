Create solid

Construct a solid from faces and solids.

Shapes → Create solid

Assemble a solid from faces and solids. The selected entities do not have to describe a closed volume - open solids with edges are created.

Keep face layer: Keep face layers inside the solid as well.

Select

Entities: Select entities . Selected faces are displayed transparently. The number of selected entities is displayed.

Entity accuracy

Determine the Entity accuracy of the base entities in their maximum (Max.) and minimum (Min.) form and display in the graphics area using the Display option. The minimum accuracy is shown in green.

Open edges are highlighted in red. The minimum entity accuracy is green, the maximum entity accuracy is highlighted in blue.

Use the Show inconsistencies option to mark all places where the volume is described incorrectly. If base entities are outside the tolerance, use the option Separate multishell solids to select whether a multishell solid or several individual solids are created.

Largest gap

Largest gap: The maximum distance between all merged and all potentially mergeable edges is displayed. This distance can be used to close a solid that would not be closed if only the accuracy of the faces and edges is considered.

Two conditions apply to edges that can potentially be merged.

  • There must be pairs of open edges that have two vertices in common.

  • The edge contours must be compatible in terms of the course of their shape. They must almost lie ‘parallel next to each other’.

In this way, an attempt is made to only look for edge pairs that make sense.

Example 14. Hole

Potentially mergeable edge pairs such as two half circles, which form a hole, are excluded. Although they have two vertices in common, they do not almost lie parallel next to each other. When two common corners are present, there is only a necessary (but not a sufficient) condition for regarding the two edges as potentially mergeable.



In order to close a solid, the largest gap can be regarded as the still permissible error. As obviously, solids with real open edges (that is, are not mergeable) cannot be closed even if the largest gap is assigned as the distance.

Set value: Apply the value determined in Largest gap as the value for Assign user defined tolerance in order to still attain a closed solid.

Assign user defined tolerance: Re(calculate) the solid with the value determined for the user-defined tolerance. The solid is generated without any gaps.

Open edges

The number of Open edges is displayed. Create curves generates curves here.

Assign tag: A Keyword tag can be assigned to new entities. Specify text for this.

Multishell solids

Number of solids: The number of parts of a resulting multishell solid is displayed.

Separate solids: Allow a number of individual solids that corresponds to the number of subareas that are spatially separated from one another. Otherwise, subareas that are spatially separated from one another are generated in one solid.
