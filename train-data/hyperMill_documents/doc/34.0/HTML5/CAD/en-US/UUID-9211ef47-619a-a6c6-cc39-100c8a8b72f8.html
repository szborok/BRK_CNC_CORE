Arc / circle

Create a circle or a circular arc.

Drafting → Arc / circle

Create a circle or a circular arc.

Input mode

Enter a Value and specify whether the value is a diameter Diam. or a Radius .

Point mode

If you select Tangent , the new entity will contact the selected entity tangentially. If you select Snap , the new entity passes directly through the selected point.

Modes

C. + R.: Enter a value and select a middle point .

C. + P.: Select a middle point and a point on the circumference .

3 P.: Select three points on the circumference .

R. + 2 P.: Enter a value and select two points on the circumference . Two entities are generated if the position of the circle or circular arc cannot be uniquely calculated. Delete the unnecessary entity.

Shape option

Specify whether a Circle or an Arc should be created. For a circular arc, enter the start angle and end angle in Start and End or, for example, in the case of 3P., specify by selecting the points.

Reference plane

Selecting On current workplane creates the entity on the current workplane . Alternatively, select an Entity as the reference plane for the position of the circle or circular arc .

Context menu

Next solution: If multiple solutions are possible, select the solution you require. A solutions preview is displayed.
