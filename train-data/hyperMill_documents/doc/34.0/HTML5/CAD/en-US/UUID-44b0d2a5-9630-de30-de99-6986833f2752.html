Section

Create curves and points through entities with the help of section planes.

Curves → Section

Create curves and points through shapes, faces and curves with the help of section planes. Use the Section curves as polylines option to specify whether a spline or polyline should be created for a section through a mesh.

Select

Select the Entities (shapes, faces, curves) through which sections are to be generated . The number of selected entities is displayed.

Section mode

Parallel: Creates sections parallel to the selected plane . Specify the section plane by selecting the respective axis of the workplane, for example. You can also select planar curves or planar faces as section planes .

Radial: Creates radial sections (B). Select a straight line as the rotation axis or select a circular curve .

On curve: Arranges the section(s) perpendicular to a curve on the basis of the curve parameters . Select a curve or boundary .

First section point

Origin: You must specify a point through which the section plane will be laid, irrespective of the section mode that has been selected .

Number of sections

The Number of sections option allows you to create multiple entities simultaneously (5). If the number of sections is greater than 1, the preview is updated and, depending on the section mode, handles are provided that allow you to modify the following options interactively:

Distance: Enter a distance between two section planes .

Angle: Define an angle between two section planes .

Parameter: Specify a parameter interval between two section planes .

Plane definition

Use On plane to temporarily create a plane for the projection in Modes:

Entity: Select a planar entity (planar curve, plane face, workplane) .

3 points: Select three points that are not collinear .

Direction + origin: Select a direction. Now select an Origin. A preview of the plane will be displayed perpendicular to the direction and origin :

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.
