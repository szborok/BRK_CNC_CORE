Simplify

Simplify faces, boundaries and curves.

Modify → Simplify

Simplify faces and faces within a solid or an entire such solid by combining. The following are supported:

  • Cylinder faces

  • Planar faces

  • Cone faces

  • Rotational faces

  • NURBS faces (both faces must have the same control points as the underlying face).

  • Linearly extruded faces (Only if the base curves can be simplified. Mostly only for lines and arcs. Then the linear sweep is a plane or a cylinder.)

Simplify curves, boundaries and edges.

Features are broken up if necessary.

Note

Overlapping entities cannot be deleted.

Keep original: Specify whether the selected face is to be retained and a copy is to be created. The selected faces are always replaced with the new face within solids.

Keep attributes: Color attributes are retained.

Tolerance for curves simplification: Enter the permissible maximum size of gaps within all selected entities for which the entities are to be considered as belonging to each other when simplifying.

Select

Entities: Select faces inside a solid or select an entire solid. The number of selected entities is displayed.
