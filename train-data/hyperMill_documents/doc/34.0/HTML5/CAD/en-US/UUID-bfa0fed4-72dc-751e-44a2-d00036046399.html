A tool electrode is divided into sections. It consists of erosion faces with the contact faces as the faces at the highest point of the erosion face area. There are three extensions of the electrode shaft, the reduction with the reduction top face , the block as the remainder of the raw material and the electrode holder .

The electrode can optionally be fitted with a joint to create a side electrode, which makes it possible to reach otherwise inaccessible areas of the geometry.

An electrode can be moved and rotated along an eroding path. The eroding path can be spatial.

The head of an eroding machine can be used for the collision check. The movement of the electrode can be simulated in an eroding machine model and checked for collisions.

Important

The generation of an electrode is based on a 1:1 scaling of the model geometry. It is not possible to change the scaling. All values, such as the start and end points as well as the dimensions in the electrode documentation, are therefore generated.

An electrode is non-parametric and non-associative to the information about the electrode that is carried in TAGs. If changes are made to the geometry of an electrode after it has been generated, this can lead to incorrect tag values and must be checked and corrected accordingly.

The following changes should no longer be made after the first electrode generation, as these changes no longer have any influence on existing values and therefore imply incorrect values.

  • Transformation of the EDM workplane

  • Transformation of the electrode workplane

  • Transformation of all solids or faces within the electrode group

  • Transformation of the workpiece

Prepare the model geometry to derive electrodes. The undersizes (spark gap) are created by milling. The erosion faces thereby correspond to the model geometry. Features such as holes and breakthroughs inside a Domain (with a single continuous face boundary) are optionally closed during derivation, meaning they are not included when generating the electrode geometry. Exclude other breakthroughs (that is, those with multiple face boundaries) manually using auxiliary faces .

Some preparatory work is necessary. Information about the purpose of the faces is managed using an electrode color palette. Read in these colors using the Import colors command, or modify the *.hmct document template:

In the Electrode options, preset the usually used properties for the construction of the electrode, the technology and the reference system. A suitable electrode holder is searched for in the electrode holder library when designing an electrode. Enter all combinations of electrode holders and raw materials possible for manufacturing into the electrode holder library with the Electrode holder and raw material editor.

After the preparatory work, carry out the following steps for designing electrodes.

Electrodes can be constructed from volume and face models. The solid, open solid and face entities plus models mixed from these entities are permitted as the reference geometry. Determine an area for an electrode by selecting faces of the model geometry , by limiting with a 2D chain of curves or manually prepare the electrode geometry directly .

Imported CAD geometry data often does not have the quality required to create an electrode. Despite undercut and missing faces, gaps between faces, and overlapping faces, the software allows electrodes to be created on the basis of such CAD geometry data. A warning is only issued if the electrode geometry does not specify a closed shape. It is up to the user to assess whether such a situation will have a negative impact on the generation, NC programming, and production of the electrode, and whether it needs to be corrected in advance.

Determine areas to be eroded with the help of the Analysis → Shape spherical command.

The Create or Create user defined commands can be used to merge the electrode geometry and the calculations for the dimensions of the raw material and the electrode holder to form a complete electrode .

Use the Create partial command to convert the faces prepared for an electrode into a electrode geometry . In doing so, the electrode block and electrode holder are not yet allocated.

The length of the raw material in Z is calculated based on the size of the erosion faces, the shaft length with any correction necessitated by collision avoidance and the safety distance , the reduction, the block and the clamping depth of the raw material in the electrode holder.

The dimensions of the raw material in X and Y are influenced by the size of the erosion faces, optionally optimized by rotating the electrode – using Optimized C angle for the best fit. The width of the spark gap, a negative dimension, is not taken into consideration.

Based on these dimensions, a raw material that is suitable for X and Y or the diameter is searched for in the *.holders.xml electrode holder library and then the appropriate Z length of this raw material is selected. This results in the assigned electrode holder. If there is no sufficient Z length, the system moves on to the next larger raw material. The necessary shaft length for the safety distance is calculated again for each search step. The overhang of the raw material can be controlled using the Raw material → Z length option .

Rotate an electrode to combine roughing and finishing electrodes for an electrode holder. For this purpose, use the Rotational option of the Create command to place multiple identical electrode geometries on an electrode block.

For an identical or mirrored workpiece geometry, break the electrode geometry out from a complete electrode, reposition it using Edit → Mirror and Edit → Move / copy and attach it to an electrode holder again using Create user defined.

Manage identical electrodes at various positions as references (virtual electrodes) to a master electrode . For this purpose, use the Edit → Move / copy command to copy the electrodes to the positions and then use the Virtual electrode command to convert them to virtual electrodes.

The target position for the erosion process is calculated.

An eroding path can be made visible and changed to influence the movement of the electrode through the workpiece and the erosion movement during the erosion process. The initial situation is the situation after the generation of an electrode in which an electrode is moved back and forth from three positions on the same eroding path. This eroding path can be changed so that an electrode can also be moved back and forth on the same eroding path from more than three positions or moved from a start position to a target position. The electrode can be rotated selectively or continuously.

Dimensioning and TAGs can be used to document the values in the electrode overview. Use the Derive and milling command to view the electrodes individually in *.hmc documents for the milling NC programming.

Electrode measurement can be integrated into the programming process with hyperMILL® Electrode, and is supported for the CERTA® QMeasure software. This is only available with the corresponding licence.

Use the Export to EDM command to output the data for eroding and convert it into a specific file format for specific eroding machines. This is only available with the corresponding licence.
