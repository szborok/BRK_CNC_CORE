Company-wide default settings

The software settings can be managed centrally for all users. All local instances read the required settings from various *.xml files and the *.ini files that are located in folders specifically set up for this purpose.

Procedure

First specify and save the company settings locally in the software. All files are stored in C:\Users\[user]\AppData\Roaming\OPEN MIND\[Software].... Make any further changes to the settings here. Delete any files and entries that are not required. When deleting an entry, make sure that the XML structure remains valid. Set up a folder on a computer that acts as the server computer and can be accessed by all local computers. Copy the customized files and their folder structure to this folder. It is advisable to map the server as a drive, for example, W:\Companyfolder\. After that, refer to the folder on this drive.

Set up the Windows environment variablesHC_INI_FILE, OMSL_COMPANY_SETTINGS and HC_ADDITIONAL_HCCONFIG. Then start the software.

Tip

To display the current environment variable settings, select the Analysis → Document info command. The result appears on the Output tab.
