Tags

To filter the display of entities, you can capture and combine graphic and geometric attributes plus system properties using conditions.

Define condition

You formulate conditions in the condition list. A condition comprises a property, an operator and a value or value range.

Use + to add a further condition and - to remove a condition. When you use the OR operator, multiple conditions act independently of each other. If you want multiple conditions to apply jointly, you must link them using the AND operator.

In addition, an (Complementary) condition is inserted automatically. This applies to all entities outside of the defined conditions.

Applying an OR operator

All entities to which an activated condition applies are visible. If all the conditions are activated, all the entities will be visible. If all the conditions are deactivated, all the elements will be invisible.

This behavior is similar to the functionality in the Entities, Layer and Color sections, where conditions are implicitly linked with an OR operator.

Applying an AND operator

In AND operations, the conditions are connected with each other and act together. If a condition is activated or deactivated, all the other conditions are also changed accordingly.

Table 14. Commands in the Tags visibility filter

Command

Explanation

All active

Activates all the conditions.

All inactive

Deactivates all the conditions.

Activate only this

Activates the selected condition.

Activate all except this

Activates all conditions apart from the selected condition.

Toggle

Activates all inactive conditions and deactivates all active conditions.

Suspend

You can use the Suspend command to temporarily deactivate a condition in a group of AND conditions. This means that you do not need to delete a condition when it is not needed for a while.

Activate

Reactivates an inoperative condition in a group of AND conditions.

Delete

Deletes the selected condition.



The commands are selected using the context menu (right-click to open) or directly via an icon.
