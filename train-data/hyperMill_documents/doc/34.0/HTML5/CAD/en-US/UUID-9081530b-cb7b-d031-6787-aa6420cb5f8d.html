Parallel line

From a straight line or a straight curve segment, create a parallel translated line or a line at an angle.

Drafting → Parallel line

From a straight line or a straight curve segment, create a parallel translated line or a line at an angle.

Invert reverses the alignment of the parallel translation.

Select

Base line: Select the line from which a parallel line is to be created .

Angle

Fixed: When performing a simple analysis on possible tool positions with regard to machinability, for example, set and place the ‘parallel translated’ straight line at a fixed angle .

Length

If Equal is selected, the new line will be identical in length to the Base line and is translated parallel. If Editable is selected , the start and end point of the new line can be extended or shortened by dragging the handles.

Definition mode

Through point: The parallel translated line is placed with its middle point on an entity to be selected .

Tangent to: The parallel translated line is placed on an entity to be selected in such a way that it is tangent to the entity at one point .

Distance: The parallel line is inserted by entering the distance .

Copies

Number of Copies – If you enter a value that is greater than one in this field, an additional line is created with the value from the field for the distance. If you select Symmetric , parallel lines are created on both sides of the Base line.

Plane

Parallel to XY: The parallel translation of the line takes place parallel to the XY plane of the workplane.

Entity: If the Parallel to XY option has not been enabled, an entity (point, line, plane face) can be selected for aligning the line.
