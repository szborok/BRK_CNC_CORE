Deformation

Deform entities.

Iterations specifies the number of calculation loops for the optimization. The distribution of points placed on the boundaries can be improved by specifying a larger number of loops.

Keep attributes: Color attributes are retained.

Select

Entities: Select the faces to be deformed . The number of selected entities is displayed.

Use the Automatic computation option to generate a preview.

Keep original: Specify whether the selected entities should be retained.

Constraints

Specify where the shape of the entities to be deformed should remain unchanged and where they should be adapted to the shape of the target entities.

Joined curves: Collect several selected curves to one curve.

Fixed

Use Points and Curves to select boundaries, curves or points at the place where the entities to be deformed are to remain unchanged in terms of their original position and their shape . The number of selected entities is displayed.

For face boundaries, select whether only the Position or the G1 continuity constraint, in other words, a Tangency must be adhered to.

Matching points

Select points where positions of the entities to be deformed should be made to match the target entities. Here, the number of target entities must be the same as the number of base entities.

Initial

Points: Select the base entities . The number of selected entities is displayed.

Target

Points: Select the target entities of the deformation . The number of selected entities is displayed.

Matching curves / boundaries

Use curves and boundaries to choose where the entities to be deformed are to be made to match the target entities.

Initial

Curves: Select the base entities . The number of selected entities is displayed.

Target

Curves: Select the target entities of the deformation . The number of selected entities is displayed.

For face boundaries, select whether only the Position or the G1 continuity constraint, in other words, a Tangency must be adhered to.

Invert orientation

Curves: Invert the orientation of an initial or target curve. Select the curve. The number of selected entities is displayed.

Parameter

Vary the settings if the result does not meet requirements. Information about the precision that has been achieved is output on the Info tab.

Position: Precision G0 specifies a maximum permissible distance between the target entities of the deformation and the result.

Tangency: Precision G1 specifies a maximum permissible deviation from the tangency for the transition to the target entities of the deformation and the result.

Points: Specify the number of points placed on the boundaries that are to be used in the calculation. Choosing a greater number of points helps to increase precision but also increases the calculating time.

Rigidity specifies the resistance of the entities that are being deformed to the deformation for each coordinate axial direction.

NURBS output

Tolerance specifies a positional conversion tolerance for the NURBS entities. Use Max. degree to specify the maximum degree for the NURBS entities, Min. cont. to define the minimum continuity and Max. patches to specify the maximum number of face patches.

Example 10. Precise deformation

To achieve a very precise deformation, use the following values as the starting point for the NURBS output:

  • Tolerance: 0.001 mm

  • Max. degree: 7

  • Min. continuity: 2

  • Max. patches: 3600



Example 11. Fast deformation

To achieve a fast calculation in order to gain an initial overview, use the following values as the starting point for the NURBS output:

  • Tolerance: 0.004 mm

  • Max. degree: 7

  • Min. continuity: 0

  • Max. patches: 800
