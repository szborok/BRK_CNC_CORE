Dimension

Dimension entities.

Drafting → Dimension

Dimension entities.

Modes

Distance: Select two entities. Snap points or positions . On the context menu, select whether the measure should be applied Horizontally, Vertically or directly From point to point. Use the Minimum distance and Maximum distance commands on the context menu to specify whether the minimum or maximum distance between

  • two circles or circular arcs

  • a line and a circular arc

  • a plane face and a cylinder

  • two cylinders

is to be dimensioned.

Angle: Select two straight lines that intersect. On the context menu, specify which angle – Major explementary or Minor explementary should be dimensioned.

Radius or Diameter: Select a circle, circular arc or circular boundary parallel to the reference plane. In the context menu, specify whether a Diameter or Radius should be dimensioned.

Length: Select a straight line or circular arc or a straight boundary or circular boundary .

Click with the left mouse button to place the measure line or measure text in a suitable position.

More options

Symmetry dimensioning: Display the dimension as half for symmetrical workpieces in the Distance mode. First select a symmetry line , then snap a point where the reference line is to start.

Radius / diameter: Specify whether a Diameter or a Radius is to be dimensioned.

Create projected angle: When straight curves outside of a reference plane to be determined are selected, the curves are projected internally onto this reference plane in order to generate a dimension in the Angle mode .

Specify the attributes of Arrow head, Reference line, Measure position, Measure text, Units of Measure and Tolerances as well as General.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

Reference plane

Select the Automatically defined reference plane option if you want the software to automatically calculate the position of the dimension in the space either from the selected information (snapped positions, selected points, curves, boundaries) or, if necessary, with the help of the current workplane.

Select On current workplane if you want the dimension to apply to the current workplane . Alternatively, select an Entity as the reference plane of the text . This entity must be planar.

How can a dimension be changed?

To change the position of several dimension entities, select the Edit → Move text / dimension command.

For a single modification, select the dimension entity and open the context menu using the right mouse button.

To change the position of the dimension entity, select the Move dimensions command or double-click the dimension with the left mouse button. To move the arrow head, double-click the tip.

Use the Edit dimensions command on the context menu to modify further attributes of the dimension entity. All entities created during a specific work process are handled as a “family” of dimensions. Selecting one entity in this family will activate all entities for the purpose of modifying attributes. The attributes of an individual entity in a “family” can be modified individually using the Entity properties command.

How are special characters entered in the notes and dimensions?

To enter a special character in the Notes and Dimensions, find the HTML code for the desired character at e.g. HTML Symbols.

Select a Unicode font name, e.g., 'Arial Unicode MS'.

Type in the code in the &[Name]; format.

Ensure that there is an '&' at the beginning and a semicolon at the end.
