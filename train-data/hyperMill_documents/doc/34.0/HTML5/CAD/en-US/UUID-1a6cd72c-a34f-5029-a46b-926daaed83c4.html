Lasso

Select entities within an area delineated by a polygonal line.

Select → Lasso

Select entities within an area delineated by a polygonal line. After selecting a start point and multiple intermediate points , which are automatically linked with straight lines, a closed selection polygon is created. The end point of the polygonal line is also the start point.

The selection is completed as soon as the last position in which you click is identical to the first click position. A small asterisk appears and the last line is highlighted in bold . All entities that lie completely within the polygon or are intersected by the polygon are selected. The entities that intersect with the polygon are not selected if you press the ALT key when you close the selection polygon.
