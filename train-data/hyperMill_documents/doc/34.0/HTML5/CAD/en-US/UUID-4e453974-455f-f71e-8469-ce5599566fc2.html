Check document warnings

Analyze the warnings detected when opening or inserting.

Analysis → Check document warnings

The warnings detected when opening or inserting are logged and can be analyzed later using this command. All affected entities are listed with their ID number and the detailed warning.

The positions of the very small entities are indicated by temporary markers in dot form.

The command is active when warnings are logged in the open *.hmc document.

Highlight: Show or hide the markers of the listed entities.

Show only this: Only show the entities to which the warnings refer. Hide all other entities.

Delete: Delete the listed entities.

The Check visibility, Entity properties, Check quality / healing and Zoom entities commands can be called via the context menu.
