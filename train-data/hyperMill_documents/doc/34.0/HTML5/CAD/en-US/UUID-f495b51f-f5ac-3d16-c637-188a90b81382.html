Edit color library

Change color attributes.

Edit → Color library → Edit color library

Change color attributes.

Color library

Name shows the name of the active color library. Description displays an optionally saved explanation. In the color selection section, select the color to be changed.

Use Set default to apply changes to the color library so that they are not only available in the current document but also in other new documents with this color library. The settings are stored in the default.hcmaterials.xml file. Export writes the settings to a freely chosen *.hcmaterials.xml color library file.

Edit color

Select a color to change in the color overview.

Diffusive: Diffuse reflection adds a spatial, three-dimensional impression to the display with the help of the color that has been selected. An imaginary, infinitely distant light source with parallel rays of light is modelled. Click the left mouse button to open a dialog in which you can select a color.

Ambient: Select a color for the reflection of the ambient lighting that can be used to give the color an even and flat, two-dimensional appearance. This lighting provides basic background brightness. Click the left mouse button to open a dialog in which you can select a color.

Specular: In specular reflection, bright spots are created by means of a headlamp light source in order to depict smooth surfaces. Click the left mouse button to open a dialog in which you can select a color.

Transparency: You can set the transparency of the color and thus the respective entity to between 0% (non-transparent) and 100% (not visible).

Shininess: Vary the intensity of the shine effect.

Name: Enter a name for the color.

Description: Enter an optional, freely selectable note for the color.

Copy duplicates a color entry.

Delete removes a color entry.
