Graphic > System > Rendering

Anti-aliasing: Set anti-aliasing – reduces the aliasing (stepping effect) of curves and edges.

Background: Set colors for graphics area background as a gradient. Select a Top color and a Bottom color. This attribute is not managed on a document basis. Select Alternative colors for the graphics area, e.g., for the Unhide option.

Single boundary color: Highlight face boundaries in ‘Shaded view’ and ‘Shaded and edges view’ in a different color. Use ‘Boundary color’ to specify the color.

Boundary color: Select the color for the face boundaries in ‘Shaded view’ and ‘Shaded + edges view’.

Color for system graphic: Specify the color for the curve of a selection area drawn with the mouse, for example.

Job referenced entities color: Set the color for referenced entities of a hyperMILL® job.

Mesh with edges: Show mesh entities (not other face entities) with edges of the tessellation triangles if Shaded and edges view or Shaded with hidden lines is selected for the entity view. The system default for this option is ‘Off’.

Face boundaries line type: Select a line type for displaying the face boundary.

Global color library transparency: Use the Enabled option to display all colors from the color library of the document transparently. Use the Factor option to specify the intensity of this cross-color transparency as a percentage.

Hidden line: Choose display of hidden lines or edges in the graphics area.

Render cache: One-off rendering of entities when a document is loaded. The displayed entities are subsequently read from cache memory unless new entities need to be displayed.

Silhouettes: Activate the display of the silhouette of the model as a graphic in the current view. With ‘Edge view’ and ‘Edge view with hidden lines’, model edges are added that are created only by the current view of the model.

Solid open edges: Highlight the open edges of solids in color. No entities were created.

Solid open edges color: Select the color for the open edges of solids.

Entity density of non-selectable layers (%): Select the display density of entities on non-selectable layers, for example, for the display of virtual electrodes.

Tessellation color: Select the color of triangulation triangle boundaries for the ‘Tessellation view’.

Transparency type: Select between different forms of transparency calculation, whereby a performance-based choice is recommended. If Automatic is selected, the software automatically chooses the optimal transparency calculation for the given performance.

TrueType text quality: Modify the tessellation precision of TrueType font text elements.

Zebra lines: Select the number of zebra lines for zebra analysis.

The settings are stored in the viewViewProperties.xml file.
