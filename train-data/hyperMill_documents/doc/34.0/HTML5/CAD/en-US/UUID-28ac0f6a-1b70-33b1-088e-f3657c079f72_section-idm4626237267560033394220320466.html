Filtering

A separate filter can be applied to metadata names for each CAD interface format. The filter is formulated in metadatafilter.xml. An example file is provided in the .../files/converters/cad interfaces/conf/all/ folder.

This XML file is structured as follows:

<?xml version="1.0"?>
<MetadataFilter version="1.1">
	<Format type="cv4">
		<ConditionSet logical="or">
			<Condition operator="ISLIKE">
				<Reference>*inner*</Reference>
			</Condition>
		</ConditionSet>
	</Format>
</MetadataFilter>

Use <Format> to specify the CAD interface format (to be imported) for which the filter should be used.

  • type="cv4" for CATIA V4

  • type="cv5" for CATIA V5

  • type="cv6" for CATIA V6

  • type="par" for Parasolid

  • type="pro" for PTC Creo

  • type="slw" for SOLIDWORKS

  • type="ugx" for Siemens NX

  • type="inv" for Autodesk® Inventor®

  • type="igs" for IGES

  • type="stp" for STEP

  • type="jto" for JT-Open

  • type="3dm" for Rhinoceros®

  • type="sle" for Solid Edge

  • type="prc" for PRC (Product Representation Compact)

Conditions can be combined in <ConditionSet> with the logical operators AND or OR.

Each condition <Condition> can use a suitable operator. Various operators are available:

  • IS: Search for an exact match.

  • ISLIKE: Search for matches using wildcard characters. Here, * matches 0 or more instances of any character or string and ? matches any character.

  • STARTSWITH: Search for an existing substring at the beginning.

  • ENDSWITH: Search for an existing substring at the end.

  • CONTAINS: Search for an existing substring.

  • DOESNTCONTAIN: Search for a non-existing substring.

Formulate the search string in <Reference> .

Example 1. Filter out metadata for electrode

Using the Ele_* reference and the ISLIKE operator, transfer all metadata that begins with the Ele_... string. To do so, formulate the following XML lines:

 <Format type="pro">
        <ConditionSet logical="or">
            <Condition operator="ISLIKE">
                <Reference>Ele_*</Reference>
            </Condition>
        </ConditionSet>
</Format>
