Updating help files

Help information is output to a browser in HTML5 format. For this purpose, the software associated with the *.html file extension in the operating system is called. If the file extension is to be associated with another browser software, it is advisable to use the omAppSettingsX64.exe software under ...\Info Center\.... This means that the association only applies to OPEN MIND software. The association for other software on the computer remains unaffected.

Following procedure is valid up to and including Version 2018.2:

If an older Help version is opened, even though a current Help version is available, delete the temporary Help files in C:\Users\[USER]\AppData\Local\OPEN MIND\[Software]\help\....

To view the data, select the Folder options – Advanced Settings → Show hidden files, folders and drives option in Windows. Check the version of the Help displayed on the homepage.
