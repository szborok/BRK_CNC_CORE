Text

Insert text.

Drafting → Text

Enter text and insert it into the graphics area.

Text box

2 points: Specify the size of the reference frame in length and width using two diagonally opposite positions . Click the graphics area with the left mouse button or snap the required position, drag the mouse and click the left mouse button again. Change the size by dragging the handles.

The text will wrap automatically at punctuation marks and space characters.

Rotation

Align the text box by entering a value under Angle or by dragging the handles. Use the Direction option to orient the alignment of the text box to an entity to be selected. If the Parallel to view option is selected, the text will always be displayed horizontally and parallel to the view, even if the view changes.

Frame

Select a Type . Specify a Color . Use the Diameter option to specify the space between the text and frame for circular frames. The size of the frame remains constant until the text reaches the frame size. Only then is the size automatically adapted to the content. The Distance option also creates a gap between the text and frame. However, the frame is always adapted to the size of the content. The frame only follows the shape of the reference frame if you select the Text box option. Use Line thickness to change the line thickness of the frame.

Reference plane

Selecting On current workplane creates the text on the current workplane . Alternatively, select an Entity as the reference plane of the text . This entity must be planar.

How are special characters entered in the notes and dimensions?

To enter a special character in the Notes and Dimensions, find the HTML code for the desired character at e.g. HTML Symbols.

Select a Unicode font name, e.g., 'Arial Unicode MS'.

Type in the code in the &[Name]; format.

Ensure that there is an '&' at the beginning and a semicolon at the end.
