Graphic > System > Print

Improves thin lines: Thin lines are automatically widened for the purpose of printing. This is often necessary when you work with data that has been imported. It is no longer necessary to adapt the line thickness manually. After printing, the original line thicknesses are restored.

Improve text with TrueType fonts: For title blocks and drawing frames, improve the quality for lines and text in print commands such as Print, Quick print and Electrode → Print.

Silhouettes: Also activate the calculation of the model silhouette for print commands. Calculating the silhouettes during print preparation can take some time.

The settings are stored in the viewViewProperties.xml file.
