5.2. Solid editing options

Influence the behavior during direct modeling and when modifying solids.

File → Options → Solid editing options

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 20. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The settings are stored in the applicationDMProperties.xml file.

The settings are stored in the applicationParametricProperties.xml file for parametric modeling.

Fillet

Linear tolerance: Specify the permissible deviation within a fillet face.

Use geometric recognition: Also restore fillets as features in imported and static data or in NURBS faces.

Max. radius: Maximum radius for restoring a feature.

Chamfer

Linear tolerance: Specify the permissible deviation within a chamfer face.

Use geometric recognition: Restore chamfers as features in imported and static data or in NURBS faces.

Recognize only 45° chamfers: Only process 45° chamfers. The geometry must meet vertically.

Max. diagonal length: Restore chamfers as features up to the specified size.

Direct modeling

Repair solid before modify: Try to close a solid before modifying it using automatic methods to join all faces in a topologically correct way.

More options

Assign solid layer to solid faces: When this option is enabled, the solid layer will be used when adding a feature with its faces. If a face inside a feature or the feature itself is changed, the solid layer will be used again.

A group, solids and faces can be managed on separate layers. If this option is disabled, a face or a feature with all its faces will be assigned with the current layer when it is added to a solid.

Solids

Face simplification: Collect resulting coplanar faces into an entity. As a result of a feature command, faces, but not features, are inserted in a solid.

Solids

Allow non manifold results: Allow open solids as results.

Solids

Extension factor: Limit the extension of faces to find a new trim edge. Increase the factor if faces need to be extended more in order to obtain a manifold solid. Reduce the factor if excessive extensions in a complex solid result in undesired trim results between parts of the solid.

Solids

Iterate: If the standard method for calculating direct modeling is not satisfactory, other and different methods will be used to generate a result. These methods depend on the quality of the data included in the calculation. If such multiple calculation is not desired, these repeated attempts can be disabled with this option.

Solids

Enable direct editing: Switch the direct modeling of solids on or off. In the commands Move / copy or, for example, Offset, the shape of all other faces and features, which are linked to the selected entities, is adjusted automatically.

Icon in the upper toolbar (state for off and on)
