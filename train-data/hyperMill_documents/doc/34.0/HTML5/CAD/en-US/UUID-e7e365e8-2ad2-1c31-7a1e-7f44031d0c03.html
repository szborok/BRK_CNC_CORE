7.5. Tags

User-defined tags allow you to manage user-defined attributes and information that can be linked to entities in the graphics area.

You can use drag-and-drop operations to assign tags to other categories. Tags that are in the root of the category tree must be assigned to a category to be able to use them in the visibility filter.

When a new tag is created, an existing parent category can be selected or created if it does not yet exist.

Commands for tags in the Tags menu:

Create tag category

Tags → Create → Tag category

Create keyword tag.

Tags → Create → Keyword tag

Create quantified tag.

Tags → Create → Quantified tag

Modify a tag.

Tags → Redefine

Delete tag.

Tags → Delete

Assign tag.

Tags → Assign

Remove assigned tags.

Tags → Unassign

Modify tag information.

Tags → Edit info
