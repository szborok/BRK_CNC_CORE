3.6. Layout manager

Save and restore the user interface.

File → Options → Layout manager → ...

Save current settings as configuration

Save current layout of the user interface.

Save the currently set user interface as an individual configuration. The settings are stored in the ‘hypercad.ini’ file.

Restore configuration settings

Restore saved layout.

Restore the user-defined configuration of the user interface.

Restore default settings

Reset user interface.

Reset the layout of the user interface to the basic settings.

Restore settings from previous version

Restore a layout from a previous version

Restore the user interface layout, the options / settings and keyboard shortcuts from the saved configuration of the last version of the software found on the computer. To do this, first save the configuration in its previous version.

Note

Please note that you can only restore the settings with this command and not by copying and pasting files from the Roaming folder.

Transfer settings

The hyperMILL®CONFIGURATION Center can be used to transfer data and settings from one computer to another computer. This may be necessary, for example,

  • when a computer is replaced or

  • hyperMILL® is to have the same settings on several computers.
