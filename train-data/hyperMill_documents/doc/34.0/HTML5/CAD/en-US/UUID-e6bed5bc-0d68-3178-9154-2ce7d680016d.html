Shape draft angle

Analyze draft angles and the splitting process.

Analysis → Shape draft angle

Analyze the draft angles and the mold parting line of a geometry to be drafted using angles. The entities are shown shaded in different colors.

The tooltip shows the value at the cursor position.

Select

Shapes: Select one or more faces, solids or meshes for analysis.

Direction

Select the reference direction with the aid of an entity , specify using 2 points or by selecting an axial direction of the current ( X WP, Y WP, Z WP) . The Invert option reverses the reference direction. If View is selected, the reference direction will be the same as the current view in the graphics area .

Use entities tessellation

Use Tolerance to specify the required level of precision or use Use entities tessellation to apply the required level of precision from the selected geometry.

Fixed steps

The Draft angle and the Transition angle determine the splitting process.

All of the areas that are split by the angles are assigned colors for the purpose of identification. Click with the left mouse button to open a dialog in which the colors can be selected and defined.

One or two curves (polyline) are generated automatically at the transition of the areas, for example, as a silhouette line with a draft angle of 0 and a transition angle of 0.

The analyzes in Fixed steps and Color map are separate; the entries do not influence each other.

Color map

Min. and Max. display the determined minimum and maximum values.

The X axis shows a color map and assigns a color to each value in the analysis area. A color symbolizes the value assigned to an analyzed entity. The Y axis provides qualitative information about the scope of the respective value in relation to all determined values. A thin, vertical line shows the value of the analyzed cursor position.

Use Limits to limit the analysis area and find a color map that is suitable for the analysis. Select a minimum Min. value and a maximum Max. value. Use Reset to restore a restricted analysis area to its original size.

The analyzes in Fixed steps and Color map are separate. The entries do not influence each other.

Specify a desired analysis area. All entities outside the analysis area are colored grey.
