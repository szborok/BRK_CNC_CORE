Convert to NURBS

Convert one or several curves and faces to NURBS.

Modify → Convert to NURBS

Multiple entity types can be selected together.

Distance: Displays the maximum distance between the converted entity and the original entity.

Select

Entities: Select entities (circles, circular arcs, faces). The number of selected entities is displayed.

Options

Reparametrise: Repair entities by reparametrizing if irregular parameterization is detected in the Analysis → Check → quality / healing command.

Skip NURBS: Entities that are already NURBS entities are skipped.

Keep domain: The untrimmed initial face is retained.

Rational: Creates rational entities (NURBS: Non Uniform Rational Bezier Spline) from analytical cylinder and rotational faces.

Settings

Use Tolerance to specify the maximum distance for the conversion of the new entities to the original entities. Use Max. degree to specify the maximum degree for the conversion, Min. cont. to define the minimum continuity and Max. patches to specify the maximum number of face patches.
