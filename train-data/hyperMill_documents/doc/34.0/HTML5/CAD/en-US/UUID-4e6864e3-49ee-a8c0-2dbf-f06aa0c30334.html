Smooth meshes

Remove incorrect outliers with minimum distortion of the original surface.

Modify → Smooth meshes

Solids

Remove incorrect outliers with minimum distortion of the original surface. The smoothing algorithm is newly applied for each iteration and new positions for all nodes are calculated. The number of triangles remains the same. For smoothing, a distance vector between two nodes is calculated in each case and projected on an underlying plane. If the length of the projection is greater than the maximum distance allowed, the distance between the nodes is reduced to the length of the projection.

Max. distance allowed: The newly calculated nodes are compared with the original nodes. If the distance is greater than the maximum distance allowed, it is reduced to ensure the maximum distance is adhered to.

Iterations: Enter the number of repetitions of the calculation for the approximation to the desired result.

Select

Mesh: Select an entity . If Keep attributes has not been selected, the attributes that are active for new entities will be used. If Keep original is selected, the original entities are retained.

Keep boundary continuity

Maintain continuity at the edge of the mesh accurately with the Position option or tangentially with the Tangency option. The outer notes remain unchanged with the Position option. Other nodes are not moved to the 'second row' with the Tangency option.

Keep character lines

If the Reference angle between triangles is greater than the specified value, the triangles are not smoothed against one another . Select the option and enter a value.
