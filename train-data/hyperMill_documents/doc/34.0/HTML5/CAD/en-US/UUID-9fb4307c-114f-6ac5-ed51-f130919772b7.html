Output PDF

Create a PDF document.

Output 3D geometric information as tessellated faces, curves, workplanes, text, dimensions and toolpaths in a PDF document. Note that only workplanes with a name are output.

Select the File → Save as command and then select the PDF file file type. Specify the Settings for saving a PDF file. Enter a name for the file. Select a folder.

Table 105. Settings for saving a PDF document

Property

Explanation

Save only visible entities

Hidden entities can be optionally saved or ignored.

Tessellation tolerance

Adjust the accuracy for face tessellation.

PDF export mode

Also generate and output curves on the model edges ( Edges as cures option).

View size

Create the output in the size of the current graphics area (Yes). Otherwise in width and height (No).

Width (pixel) and Height (pixel)

Create the output in width and height in the specified size.



To save and restore settings, right-click to open a context menu containing the following commands:

Table 106. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Reset all settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

Copy the text or value of the selected area to the clipboard.



All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The settings are stored in the pdfsavefilesettingsvalues.xml file.
