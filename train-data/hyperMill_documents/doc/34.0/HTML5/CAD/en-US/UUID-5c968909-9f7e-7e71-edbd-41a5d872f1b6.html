16.16. Export reports

Export electrode and raw material summaries.

Electrodes → Export reports

Electrode

Output electrode and raw material reports for the electrodes of the document .

The reports are saved according to the settings specified in the File → Options → Electrode options command in the folders branch. When using the EDM converter software to convert electrode reports into the formats used by other eroding machine manufacturers, it is advisable to use the Export reports command to select a folder for permanent use.

The reports generated using the templates electrode_summary.xml or full_electrode_summary.xml contain information that indicates whether the referenced erosion path shape is 3 points, Blind or Through. The waypoint coordinates and rotation angles of eroding paths are not given in these reports.

Note

If it is not possible to change and print a report in Microsoft Excel, check with the software administrator in the Microsoft Excel Trust Center whether the settings can be optimized for this purpose.

Format

Determine the purpose or format of the report by selecting a template.

In this template, set the flag attribute to 1 to write the required information to the reports .

In order to be offered an *.xml template for selection, it must be stored in the Summaries subfolder of the folder specified in the Electrode options, in electrode holders and XML files. The template for the *.xslt file must also be saved at this point.

An *.xslt file is also output to this folder. This file is evaluated in order to format the data when a report is opened in a browser or in table management. A custom logo and a stylesheet *.css can be added to this template.

Special reports

Note

Export for EDM converter: Data from the EDM converter software is output into the EDM_Converter_summary_2.0.xml template. Do not alter this English XML file.

The output eroding path shapes are 3 points, Blind and Through.

Only selectable with an EDMcon license.

Note

The CERTA_JMS-V11_0.xml file is used to transfer data to the JMS Version 11 software from CERTA Systems GmbH. The file must not be changed.

File names

Enter a name *.xml for both theElectrode summary and theRaw material summary. The output electrode reports with the EDM_Converter_summary_2.0.xml template for the EDM converter software must contain the file extension*.ElectrodeSummary.xml.

Electrode list

Only selected: Optionally generate a report with selected electrodes only. To do so, select the option. Select the relevant electrodes with the left mouse button. Otherwise, all listed electrodes will be included in the report. The list is sorted by name.
