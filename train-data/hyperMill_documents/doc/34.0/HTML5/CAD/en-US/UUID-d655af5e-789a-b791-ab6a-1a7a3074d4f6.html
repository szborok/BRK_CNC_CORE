2.3. Dialogs

In numerous commands, data is entered in a dialog. These dialogs are similar in structure and can also be stored permanently outside of the program window. They are divided into a title bar, an upper area and an input area.

Title bar

Right-click the title bar to display the context menu:

Save as default: First specify your value as a default setting for an option, then save as a default.

Reset default: Reset your value specified for an option to the original setting.

Default values are automatically converted between metric and inch when the system of units is changed. The settings are stored separately for metric and inch for each command in the ...\commands\defaults subfolder of the user settings.

Upper area

Confirm

Exits the command. The dialog closes.

Apply

Entries are applied. The dialog remains open. The ENTER key can also be used to accept entries.

Show message

Warnings, error messages and information are output on the Info tab. Click the icon with the left mouse button. The Info tab opens displaying the message.

Default settings

Options are reset to the default settings. This also takes place after the software has been restarted.

Redo

Restores the settings from the last command selection.

Preview

Switches the results preview on / off.

Input area

The input area contains embedded sections such as the direction selection, whose layout and operation are identical in different commands. Decimal places can be entered with a comma or decimal point. A comma is converted automatically to a point. Unnecessary decimal places are automatically removed. Mathematical calculations are possible during value input.

Angles can be input as deg:min:sec or deg min sec (separated by spaces) and will then be converted to the selected format.

Depending on the context, you can double-click with the left mouse button in the graphics area to automatically jump within commands to the next input required for the command sequence. The Continue command (highlighted in bold) in the context menu indicates this option. If the Apply command is highlighted in bold, the entries will be applied and the calculation will be performed.

Buttons are marked with different colors in order to indicate the input status:

No selection has been made yet.

Selection is sufficient to execute the command.

No further selection can be processed. The command can be executed.

A further selection must be made.

Too much information in the selection. The software waits for the input to be corrected.

The different colors used for the button labels mean:

  • Neutral: Input is possible.

  • Blue: Input is possible or necessary. Option selected.

  • Grey: No input is possible.

You can use the TAB key to confirm an entry and move between the input fields.

Right-click the respective button and call the command.

Context menu

Different context menus with a selection of commands, such as entering text and values, are offered in the different areas. Right-click to open the context menu.

Reset: Undo a selection.

Save selection: A selection of curves and a selection of faces can be saved separately. A combination of curves and faces is not possible.

Add saved selection: Add a saved selection for curves or faces to a current selection.

The following are available for parameterizable inputs:

Variables: Select a variable from the list that is located in the Edit → Spreadsheet command. A value assigned by a variable is marked with the icon .

Functions: Select one of the available mathematical functions from the list.

Constants: Select one of the available mathematical functions from the list.

Mathematical calculations

Mathematical calculations are possible during value input. A calculation may be preceded by an equals sign, but does not have to be.
