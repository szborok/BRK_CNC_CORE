Split meshes

Split meshes with a planar entity or along a curve.

Modify → Split meshes

Solids

Split meshes with a planar entity or along a curve.

Select

Meshes: Select entities .

Keep: Select the clusters to be retained. The number of selected entities is displayed .

Keep original: Select whether the original mesh should be retained.

All results: The mesh is split and all clusters are retained.

Split by curve

Split or trim meshes along a curve or boundaries. Cut out a closed contour (for example, a hole) as a circle. Use Direction to project the selected curve onto the mesh.

Select

Limits: Select external boundaries (curves, face boundaries) . The boundaries must be closed. The number of selected elements is displayed. The boundaries are projected onto the mesh by using Direction. Select a direction :

Direction

Select the reference direction with the aid of an entity , specify using 2 points or by selecting an axial direction of the current ( X WP, Y WP, Z WP) . The Invert option reverses the reference direction. If View is selected, the reference direction will be the same as the current view in the graphics area .

Split by plane

Split meshes with a planar entity. In the case of a face, the face is automatically extended for the calculation. A check is carried out for all selected mesh entities to see whether a section is generated. All triangles that are cut are rebuilt. A smooth section is created.

Modes

Entity: Select an entity (such as a planar curve or face) that will define the section plane .

3 points: Select three points that do not lie on a straight line. The section plane is created based on these points.

Direction + origin: Define the section plane via the Origin and Direction. The preview of the section plane is displayed perpendicular to the selected direction. The point defines the passing through point of the plane to the direction.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

For the origin of the Mirror plane , Select a point in the graphics area , enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.
