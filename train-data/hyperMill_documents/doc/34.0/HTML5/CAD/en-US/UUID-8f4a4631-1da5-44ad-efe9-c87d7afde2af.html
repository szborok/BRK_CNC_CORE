Assign material

Assign material.

Edit → Assign material

Assign a material to shape entities.

A list with material attributes is maintained in the default.densities.xml file. When editing this, make sure that unique, unambiguous material names are used. A value, which is greater than zero, must always be present for the mass density.

Select

Shape: Select entities (solid, mesh, stock model) . The number of selected entities is displayed.

Physical attributes

Mass density: Displays the density of the selected material.

Mass: Displays the weight force based on gravity, which is simplified as the mass, for the selected entities.

Description: Displays a description that has been stored for the selected material.

Material

Select material from the list .
