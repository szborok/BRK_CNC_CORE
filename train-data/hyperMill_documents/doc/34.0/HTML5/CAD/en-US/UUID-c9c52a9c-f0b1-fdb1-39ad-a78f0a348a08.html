Keyboard shortcuts

You can also use keyboard shortcuts to call commands. The key combinations are displayed in the menu bar after the command names and they can be modified using the Customize shortcuts command.

The following options are also available:

  • F11 Display borderless program window.

  • ALT+I ‘Invert selection’.

  • ESC Reset selection filter to default setting.

  • CTRL+F Open the Search and start command tab and set the input focus on the tab.

Different tabs can be opened directly using keyboard shortcuts. The TAB key can be used to switch between input fields in a dialog.

Some keyboard shortcuts always work, even if the current input focus is on a window, dialog or a tab (for example, in a chain selection).

Press the ENTER key to recall the previous command.
