Separate mesh clusters

Separate a mesh entity with mesh clusters into individual mesh entities and remove very small clusters.

Modify → Separate mesh clusters

Solids

Separate a mesh entity with mesh clusters into individual mesh entities and remove very small clusters. Determine the cluster with the largest number of triangles using the Keep biggest cluster option . All other clusters are deleted.

Select

Mesh: Select an entity . If Keep attributes has not been selected, the attributes that are active for new entities will be used. If Keep original is selected, the original entities are retained.

Entities

Specify the desired cluster using the Based on entities option.

Min. number of triangles: Clusters with a smaller number of triangles are discarded .

Identified clusters: The number of identified clusters is displayed.

Select: Click with the left mouse button to select the Regions that are to be retained . The number of selected entities is displayed. If no selection is made, all of the clusters with more than the minimum number of triangles are retained. All other clusters are deleted.
