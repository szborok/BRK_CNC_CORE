Points on face

Create one or several points on a face.

Drafting → Points on face

Create one or several points on a face.

If On domain is selected, the result for a trimmed face can also be created outside, in the area of the original untrimmed face. Create point cloud combines several points into a point cloud entity.

Select

Face: Select the target faces . The number of selected entities is displayed.

Parameter

The points of the face are aligned based on the isoparametric curves in the U and V direction . The parameters for the Start and End of the U and V direction can be modified. You can set the number of points ( No. of points) .
