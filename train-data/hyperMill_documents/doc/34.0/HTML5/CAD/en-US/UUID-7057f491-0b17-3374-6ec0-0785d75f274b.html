Mating

Mate faces, groups and solids and define the relationship between them.

Edit → Mating

Mate faces, groups and solids. Define a mating pair relationship between individual faces, groups and solids. This should be a stepwise process. Once the two faces have been selected and the identified relationship is correct, confirm the result by clicking Apply and create the next mating pair. The mating pairs appear on the Model tab.

The software will automatically attempt to suggest a meaningful relationship for a mating pair. This depends on the selected entities. As an example, two cylinders will be arranged to be Concentric. For two planes, for example, Coincident will be assumed automatically. This suggestion can then be changed to Parallel, for example, and a Distance can be inserted.

To mate multiple faces together, collect the faces into a group. To mate meshes, collect the meshes and faces together into a group. Place the faces on the mesh in advance so that they can be used as mating aids for the mesh.

The relationships between multiple solids and groups can be established in any order.

Persistent: Permanently save the relationship defined between a mating pair. Otherwise, the entities will only be moved and/or rotated.

When moving entities, the relationships saved as mating pairs are observed. If a group or a solid is moved, all child entities will always move with it. In turn, the movement of these groups or solids is only possible within the bounds of the constraints defined for the parent entity. This can mean that a movement executed in the XY direction results in movement only actually occurring in the X direction, for example. A warning is displayed.

In direct modeling, affected mating pairs are adjusted automatically. The mating continues to remain in place after direct modeling.

When you use Move / Copy and Paste, the matings are also transferred via the clipboard.

Using File → Save selection, it is possible not to save selected mating pairs or, using File → Merge, it is possible not to reimport selected mating pairs. Disable the display of matings on the Model tab in Options → Options / properties in Model structure → Entities to show.

Select

First entity: Select a face of the entity to be moved.

Second entity: Select a second fixed entity. The entity type depends on the mating type. It is usually a face, but may also be a point or a position, as in the case of the point-plane constraint.

Always work in a ‘from - to’ sequence. The software will automatically suggest an optimal relationship for the mating pair. Keep or modify the condition. Always Apply the correct result before proceeding to the next mating step.

Relationships

The software will attempt to automatically suggest a meaningful relationship for a mating pair. This depends on the selected entities. The active relationship is highlighted in color. Inapplicable relationships are greyed out. Click to change one of the applicable relationships.

Fixed entity

Fix an entity.

Coincident

Arrange the two selected planar faces of the mating pair so that they are coincident to one another. Create a parallel mating pair by combining this with a Distance.

Parallel

Arrange the two selected planar faces of the mating pair so that they are parallel to one another.

Perpendicular

Arrange the two selected planar faces of the mating pair so that they are perpendicular to one another.

Concentric

Arrange the two selected rotationally symmetric faces of the mating pair so that they are concentric to one another.

Tangential

Arrange the selected plane and the selected cylindrical face of the mating pair so that they are tangent to one another.

Angular

Arrange the normal vectors of the two selected planar faces of the mating pair in an angle to one another.

Selecting a direction axis projects the normal vectors of the two faces in the plane defined by the direction axis. The axis determines the normal vector of this plane. The angle is specified there. The orientation of the axis influences the result. If the direction is reversed, then the sign of the angle will change. A positive angle denotes anticlockwise.

Point-Plane

The selected plane is arranged in such a way that the selected point, mesh vertex point or the selected position is located on the plane.

Direction

Specify a direction axis for a projection plane using Select for an entity or using 2 points.

Distance and angle

Distance: Maintain a distance between the First entity and the Second entity . Enter a value.

Angle: Maintain an angle between the normal vectors or projected normal vectors of the First entity and the Second entity . Enter a value.

Orientation

Aligned: Maintain the automatically determined alignment .

Anti aligned: Invert the automatically determined alignment .

How can a mating pair be changed?

Permanent mating pairs can be modified. To do so, select the entry under Mating in the model tree on the Model tab and right-click to open the context menu. Then select the Change parameters command.

You can delete mating pairs. To do so, select the entry under Mating in the model tree on the Model tab and right-click to open the context menu. Then select the Delete command and delete using the DEL key.

What happens when a mated entity is modified?

Icon

If mated solids or groups are deleted, the mating pairs remain in the model tree. This is known as ‘dangling’ mating. After inserting another solid or another group, use Change parameters to assign the missing faces to the dangling mating pairs. Dangling mating is indicated by an icon in the model tree.

Icon

The mating can no longer be used as changing an entity will result in a contradiction. The unused mating is shown by an icon in the model tree.
