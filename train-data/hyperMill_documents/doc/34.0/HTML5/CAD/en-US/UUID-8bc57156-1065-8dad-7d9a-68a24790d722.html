Shape continuities

Analyze continuities at edge transitions and internal continuities.

Analysis → Shape continuities

2023.2 | Update 3

Examine the continuities of the entire model or selected model spaces at edge transitions for gaps, G0, G1, and G2 continuities of the faces and the internal continuities for NURBS faces. Find convex or concave edges to differentiate the limits for machining, e.g., for milling or eroding.

Select

Shapes: Select entities .

Mapping

Select the continuities to be examined:

  • Gap 

  • Sharp edges

  • Tangent continuous

  • Curvature continuous

The number of curves found is displayed. The results are highlighted with the colors shown in the dialog. The colors cannot be changed.

Sharp edges

Search for everything, only convex edges or only concave edges.

Invert orientation Invert the orientation of the shape’s faces.

Reference values

Tolerance: Enter a value for the permissible deviation in the position for the size of the gap.

Angular tolerance: Enter a value for the maximum permissible deviation of the transition’s tangency.

Curvature difference (%) Enter a value for the maximum permissible deviation of the transition’s curvature.

More options

Create curves: Generate the analysis result as curves . Result curves are generated in four additional colors for the document. The colors are “SC CurvatureCurves,” “SC GapCurves,” “SC SharpCurves,” and “SC TangentCurves.”

Internal continuities: Visualize continuities within NURBS surfaces . Both internal C0 and C1 continuities can thus be identified directly. Milling operations can be better prepared and validated.
