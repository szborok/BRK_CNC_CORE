Wrapping side shell

Wrap or unwrap a label for a side shell.

CAM → Utilities → Wrapping side shell

Unwrapping or wrapping of curves (for example, for tires) in order to transform a label on a side shell. The planar lettering must be bent in a circular arc and lie on the XZ plane of the world workplane. The radius is determined by the required target . The curves are unwrapped in the Y direction.

Parameter

Center line: Select a line along the Y axis of the world workplane .

Section curve: Select an individual curve as the profile of the side shell . This curve must lie on the YZ plane of the world workplane.

Radial reference point: Specify a reference point for the position of the entities along the section curve . The position of the label is influenced by means of another position for the radial reference point on the section curve .

Curves to be projected: Select entities . For the purpose of wrapping, the entities must lie on the XZ plane.

Wrapping: Specify whether planar entities are to be wrapped or curved entities are to be unwrapped.

Output layer: Enter a name for a resulting layer.

Tolerance: Specify a maximum permissible deviation for the approximation.
