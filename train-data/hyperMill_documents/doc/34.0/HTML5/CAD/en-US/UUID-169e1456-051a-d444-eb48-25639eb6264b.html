Mating clamping device

Mating clamping device.

Edit → Mating clamping device

Mate solids or groups as a clamping device. First align the individual solids or groups using Mating.

Note

The command is a specialization of the Mating command. See the description of this command for more information on the mode of operation.

Persistent: Permanently maintain the defined relationship.

Linear

Move two entities (group, solid) directly towards one another. These entities may represent two clamping devices, such as clamp jaws for example.

Distance: Enter a value for the distance that should be maintained between the entities .

Select

Entities: Select entities . The number of entities is displayed.

Modes

Select an entity or a direction for a plane that determines the movement of the entities towards one another .

Entity: Select an entity to define the plane (e.g., a planar curve of face).

3 points: Select three points that do not lie on a straight line. The plane is created based on these points.

Direction + origin: Use Origin and Direction to define the plane:

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.

Radial

Move entities that are initially radially arranged as a clamping device (group, solid) towards one another at a shared center point.

If the selected faces belong to a closed solid, this results in the inside and outside from the topology. This leads to the direction for the Distance. The face normal vector is evaluated for individual faces. If this vector points in the opposite direction, use the Modify → Invert orientation command to adjust the direction beforehand.

Diameter: Enter a value for the distance that should be maintained between the entities .

Select

Entities: Select entities . The number of entities is displayed.

Direction

Specify a center for the movement towards one another . To do so, use an entity to select a direction using Select or 2 points .
