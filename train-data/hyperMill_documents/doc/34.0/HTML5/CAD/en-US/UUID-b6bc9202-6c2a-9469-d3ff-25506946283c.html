Pan-zoom-rotate

Move, zoom and rotate the model view using the left mouse button.

View → Pan-zoom-rotate

Press the following keys to modify the type of view change carried out with the help of the mouse.

Table 74. Pan, zoom and rotate with the mouse

Keys

Cursor

Command

-

Rotate the view.

CTRL

Move the view.

SHIFT

Zoom: Change the magnification factor of the view.

ALT

Tilt the view.



Press the ESC key to exit the command.
