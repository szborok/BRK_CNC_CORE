Trim curve

Shorten or extend a curve via a boundary.

Modify → Trim curve

If the Curve and Boundary lie on the same plane without an intersection point, an attempt is made to generate an intersection point for trimming by extending the boundary .

If the Curve and Boundary are spatially separated from one another so that an intersection point can also not be generated by extending the boundary, a message is output.

The position at which you click influences the result. The first click position on the first curve determines which intersection point is used and which section is retained. The second click position on the second entity determines which section of the second entity is retained.

The Compute on view option trims the curve at the place where its projection on the current view plane intersects the boundary .

If several possible intersection points are determined, the clicked curve portion is retained .

If several curves are selected with a window selection, the curve portions are retained where the imaginary middle point of the selection area is located .

Modes

Boundary: Select the boundary at which the curve should be shortened. Boundaries must intersect the curve. You can select points, curves, faces, face boundaries and meshes .

Each other: The curve and boundary are both trimmed at the intersection point. Select both entities .

Select

Curve: Select the curve to be shortened. The curve is shortened according to the position at which you click the curve. The side that you select is retained.
