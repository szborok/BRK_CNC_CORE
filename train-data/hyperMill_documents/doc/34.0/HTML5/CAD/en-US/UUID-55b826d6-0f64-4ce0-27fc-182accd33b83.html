1.1. Installation

Please follow the installation information for installing the software. If you have any questions concerning the installation, please contact Support first.

Hardware requirements

The software requires a 64-bit operating system and uses OpenGL. For up-to-date information about hardware requirements and supported operating systems, please visit www.openmind-tech.com and go to ‘Support > Software requirements’ in the ‘Service’ area of our website.

Licensing

The software is copy-protected via hardware. A dongle and an activation code are provided for operating the system. Each license is unique. Protect your license against loss.

For more information, please refer to the installation and licensing information on the website and in the CAM software documentation.

Select system of units

Select a system of units in the hyperMILL®CONFIGURATION Center.The setting is also valid for the software. Set the Metric option for the unit of measure in millimeters. Set the Inch option for the unit of measure in inches.
