Open

Open files with different file formats.

File → Open

Set the file format and select a document. Specific options can be defined depending on the file format.

All (*.*) and All supported formats are used to display all file formats or all supported file formats. In both cases, no interface format options can be selected.

As is usual in WINDOWS, a visual preview of the file contents is provided in the Windows File Explorer. Set ‘Large Icons’ for this, for example.

In Windows File Explorer, system information can be determined for hyperMILL® *.hmc documents and *.hmct templates. As such, prior to opening a document, ensure that the correct software version can be run.

  • The Windows tooltip displays hyperMILL® version information.

  • The file properties in Windows show an additional ‘OPEN MIND Info’ tab, which includes all the available information. The tab is only shown if a *.hmc document or a *.hmct template is selected. If several files are selected, the tab is not shown in order to avoid any confusion.

If the document is password protected, it may not be possible to retrieve some information, for example the version.

The prerequisite is that the dynamic program library thumbnailcom.dll is registered in Windows. The registration is performed automatically during installation using the standard hyperMILL® installation kit.

If a document is opened more than once, the second and all further instances of the document will be read-only. Even if the first instance is closed, it cannot be overwritten by the second or further instances of the document.

Warning for file opening!

A report is opened for defective entities automatically if there is a need to remove these entities. To this end, the ID and type of the deleted entities and the errors found are output.
