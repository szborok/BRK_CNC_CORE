Point file formats

*.pt, *.asc, *.xyz, *.txt: Read in a text file with points and point clouds. Save points and point clouds to a text file in the form XYZ, with or without separators and quotes.

Table 51. Options for opening or inserting a point file

Property

Description

Load mode

Specify how the data is processed when opened:

Point cloud generates a “Shape – Point cloud” entity .

Simple points generates single point entities .

Points and lines: Read in points and create connecting lines between the points, in the order of the points of the file .

Interpolating: Read in points and interpolate curves from them.

Approximating: Read in points and approximate curves from them.

If there are only a few points, interpolation makes sense because the curves run exactly through the points .

If there are many points, approximation is useful if the data is subject to measurement errors and these should not be reflected. The approximation tolerance controls the maximum deviation of the generated curves from the points. The generated curve is smoothed .

Separator

Enter the character, for example, , that separates the individual point coordinates.

Line comment

Enter the character, for example, # with which lines are commented out.

Place within quotes

Select the Yes option if the point coordinate values are enclosed in quotes ".

Points are numbered

Select the Yes option if the points or their rows are numbered.

Entity precision

Enter a value.

Significant digits

Enter a value.

Maximum number of points per spline

Enter a value. Only available when the Interpolating or Approximating load mode is selected.

Separate splines

When loading, an additional separate spline is created in accordance with Maximum number of points per spline.

Approximation tolerance

The approximation tolerance controls the maximum deviation of the generated curves from the points. Enter a value.

Unit of measure

Select the Millimeter or Inch option. If necessary, the data is converted for insertion into an open document.

Template

Merge: Insert CAD data to be imported into ahyperMILL® document template.

hyperMILL document template name (*.hmct): Select document template.



The settings are stored in the pointcloudloadptfilesettingsvalues.xml file.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 53. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Restore the last saved settings

Reset all settings

Restore the installation settings.

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

Copy the text or value of the selected area to the clipboard.



All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.
