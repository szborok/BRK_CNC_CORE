Environment > Lights

Ambient: Use Enabled to activate the light and Intensity to specify an intensity between 0 (off) and 1 (100%).

Light 1 – Light 4: Use Enabled to activate the light and Intensity to specify an intensity between 0 (off) and 1 (100%). If the Fixed option is selected, the direction of the light from the perspective of the viewer will remain constant despite view changes. Use Cast shadow to create a shadow on another entity. To do this, also select the ‘Self shadows’ option under ‘Graphic’ > ‘View’. If Cast shadow has been activated for several lights, a cast shadow will only be created for the light with the lowest number. Select a Color for the light. Specify the Direction of the light.

The settings are stored in the viewgraphicsenvironmentproperties.xml file.
