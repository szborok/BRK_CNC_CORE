Linear slot

Create a slot via the linear sweep of a contour from curves and boundaries, based on a face within a solid.

Features → Linear slot

Solids

Enter a Height or specify using the handle. Both sides can be optionally used to split the height in both directions of the contour. Mirror mode will then work . The same mirror-image slot will be generated in both directions. This is visible when an Angle is entered or specified with the handle for the lateral inclination.

The Multishell solid option allows spatially separated subareas for a solid. Otherwise, individual solids are created in correlation to the number of separate subareas.

Note: If curves within V-sketches or parametrically created faces are preselected, the option is automatically activated. It cannot be deactivated and is greyed out. The option can be switched when there is no preselection or if the preselection includes static entities.

The Through all option creates a slot that runs through the entire solid . With the Invert cut side option, the slot remains as a solid, and the rest of the initial solid is discarded . An open cut contour is required.

Select

Curves: Select the curves and boundaries . The number of selected entities is displayed.

Faces: Select entities . The number of selected entities is displayed.

Modes

If Normal is selected, the contour will be projected in the direction of the face normals. Alternatively, the Direction can be specified:

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Transitions

If the sides of the slot are inclined positively, the transition between the faces can be controlled.

Rounded: Creates rounded transitions .

Sharp: Creates transitions with sharp edges .
