Tags

Tags allow you to manage attributes and information that can be linked to entities. User-defined tags can be created. Categories, Keyword tags and Quantified tags are available for the purpose of input and organization.

Procedure:

  1. Create one or more tag categories (such as ‘Tool’).

  2. Create keyword tags or quantified tags in the tag categories (for example, ‘Nozzle side’).

  3. Select the entities and assign tags to the entities.

  4. Apply the tags. For example, tags can be used on the Visibility tab to control the visibility of complex workpieces.

Tag category

Create a new tag category as a new branch in the tree structure. A tag category may contain keyword tags, quantified tags and further tag categories.

To create a new entry, right-click and select the root of the tree structure or an existing category on the Tags tab. A context menu opens with the following commands: Create category, Create keyword tag and Create quantified tag. Select the required command.

A new tag category or a new keyword tag.

The icon indicates a temporary state. An identifier must be entered. This ensures that the entry is saved permanently.

The input of an identifier can be enabled by clicking the entry with the left mouse button. Alternatively, right-click to open the context menu. Then choose the Rename command.

Tag category with its own identifier.

Keyword tag

Use the Create keyword tag Keyword tag command to create an entry for a freely selectable keyword, for example, ‘Nozzle side’, in the selected category.

To create a new entry, right-click and select the root of the tree structure or an existing category on the Tags tab. A context menu opens with the following commands: Create category, Create keyword tag and Create quantified tag. Select the required command.

Quantified tag

Use the Create quantified tag Quantified tag command to create an entry with further attributes.

To create a new entry, right-click and select the root of the tree structure or an existing category on the Tags tab. A context menu opens with the following commands: Create category, Create keyword tag and Create quantified tag. Select the required command.

Sort, assign and delete

If a tag has been selected, you can right-click to open a context menu containing the following commands:

Table 15. Commands on the Tags tab

Command

Explanation

Delete

Deletes the tag. To select multiple tag entries, press the CTRL and SHIFT keys while clicking with the left mouse button.

Assign

All entities that have been chosen using Select are assigned the selected tag entries.



You can use drag-and-drop operations to assign tags to other categories. Tags that are in the root of the category tree must be assigned to a category to be able to use them in the visibility filter.

When a new tag is created, an existing parent category can be selected or created if it does not yet exist.
