13.1. Create / modify

Create or modify V-sketch.

Drafting → V-sketch → Create / modify

Create a V-sketch by incorporating curve entities. Define the horizontal direction. Modify a V-sketch.

Automatic constraints: The Drafting → V-sketch → Automatic constraints command is executed automatically .

Simplify: The Modify → Simplify command is executed automatically .

Tolerance: Enter the permissible maximum size of gaps within all selected entities for which the entities are to be considered as belonging to each other when simplifying.

Gaps and overdefined areas are marked by red points. Make corrections. Click the left mouse button and press the CTRL key at the same time to separate entities from the V-sketch.

Note

When making changes, the X direction cannot be adapted. It is only possible to select a new Origin.

Select

Entities: Select entities . The number of selected entities is displayed. All entities must be coplanar. The entities move to the V-sketch layer when in the Sketcher options the Assign curves to V-sketch layer option has been selected.

X direction

Define a horizontal. To do so, select a direction . The direction is indicated by an arrow. The direction also determines the constraints Vertical, Coincidence X and Coincidence Y. If no selection is made, the horizontal is determined from the X axis direction of a workplane . A direction arrow is shown in the workplane origin in the X axis direction.

Select the direction based on an entity . Via 2 points or origin and selection of a point determine the Point or enter the direction via a Vector . The Invert option reverses the direction.

Select

Origin: Select a point for the origin of the X direction .

Reference

Curve: Use additional curves, which do not belong to the model contour, as a reference for the dimensioning within the V-sketch. Select the entities . The number of selected entities is displayed. The line type for these entities is changed to dash-dot.

If entities, which are not drawn in the ‘continuous line’ line type, exist under Select → Entities, these entities are automatically suggested as a reference.

Change the position of a hole, based on a reference curve not belonging to the model contour. For this purpose, generate a dimensional constraint and change the value.
