Sphere

Create a sphere.

Shapes → Sphere

Create a surface in the shape of a sphere.

Select

Center: Select a point or a position for the center of the sphere .

Input mode

Value: Enter a value for the size of the sphere .

Diameter / Radius: Interpret the value for the size of the sphere as diameter or radius .

More options

Check collisions: Check if other faces intersect . A warning is returned on the Info tab. Clicking on the warning highlights the affected faces in color and displays the smallest distance.
