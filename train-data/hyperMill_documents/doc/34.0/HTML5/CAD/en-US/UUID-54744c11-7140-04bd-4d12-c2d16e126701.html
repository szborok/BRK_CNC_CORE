5.1. Options / properties

Load and locally modify the defaults for the model, the model structure and the graphical attributes of the document and the software.

File → Options → Options / properties

Load and locally modify the defaults for the model, the model structure and the graphical attributes of the document and the software.

When starting the software, all the factory system settings are loaded first. After that, all the local settings are applied. These settings are then updated with the company settings.

If an option has not been specified in the company settings, the previously loaded setting is used. If company settings are used, all settings can still be changed locally, including the original system settings that were loaded and the company settings. The settings are effective from when a change is made and do not affect how existing entities are displayed.

In the Search line at the top of the tree structure, search for text within this structure. Enter the text to be searched for. Suitable results are displayed as the user is typing. Confirm the entry by pressing the ENTER key. The search history will be saved temporarily. Separate text fragments with spaces to display several different contents at the same time. The search is case insensitive.

Clear: Remove the search string again. To do this, right-click the line to display the context menu: Select the command there.

The commands at the bottom edge of the dialog can be used to save and load all options and settings:

Reset: Restores the factory system settings.

Save settings: Save all settings locally.

Load settings: The local user settings are loaded and updated with the company settings.

If several values are available for selection for an option, click the selection with the left mouse button and simply scroll through the values using the scroll wheel.

To save and restore settings, right-click to open a context menu containing the following commands:

Table 19. Commands on the context menu:

Command

Explanation

Collapse all

Fold structure tree.

Expand all

Unfold structure tree.

Save all settings

Save the modified settings.

Load all settings

Copy to clipboard

Copy the settings of all sections to the clipboard as text in the form ‘name = value’.

Copy cell to clipboard

All settings are stored in files in XML format. For further information about this and about standard settings for multiple users, please refer to the ‘Company-wide default settings’ section.
