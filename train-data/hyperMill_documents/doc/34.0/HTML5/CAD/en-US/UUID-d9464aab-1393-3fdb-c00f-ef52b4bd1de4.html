Milling areas

Faces that are used in milling areas are protected from being modified by the user. The Unlock entities command can be used to disable this protection so that it is possible to edit these faces.

A V-sketch can be selected as a machining contour. It is used as a closed polygonal line for job calculations. If the V-sketch is changed (for example, fillets or chamfers are inserted) and the contour line remains closed, the machining contour of the job is updated. However, this only applies if the V-sketch was generated from Version 2021.1.

Coordinate system

Set WP to frame: Create a coordinate system for multi-side machining (hyperMILL® Frame) once or several times for a workplane in the same position and orientation.

Link frame to Wp: For an associative workplane in the same position and orientation, create a coordinate system for multi-side machining (hyperMILL®) once and link it to this selected workplane. Only available if parametric modeling is switched on and the workplane is associative.

Example 27. How to Link a coordinate system for multi-side machining (hyperMILL® Frame) with a workplane

  1. Check whether parametric modeling is switched on.

  2. Create a workplane, e.g. with the command Workplane → On face. Select the option Associative and enter a name for the workplane.

  3. There are two ways to link the workplane to a frame:

    • If the frame already exists:

      Select a frame and call the Edit command, enable the Associative workplane option in Frame definition → Definition, and select the relevant associative workplane from the drop-down menu.

    • If a new frame is required:

      On the WP tab, select the Link frame to WP command on the context menu of the associative workplane. A new frame is created in the hyperMILL® frame browser. The Associative workplane option is enabled and the associative workplane is selected.

  4. To be able to make a change to the geometry, check whether the Activate option is switched off in hyperMILL → Setup → Settings in Document → Locking.



Toolpath entities

Toolpath entities are managed without layer information, and do not appear in the model tree on the Model tab. Display settings are not managed as for all CAD entities.

Specify settings for toolpath entities:

  • CAM → Setup → Settings on the Document tab in Feedbacks → Colors

  • CAM → Setup → Settings on the CAD document tab in Toolpath display

  • In the hyperMILL® job on the General tab

Geometric attributes within a toolpath entity can be determined using CAD commands from the Analysis menu. To do this, activate the Toolpath polyline-point snap filter and the Inside toolpath snap option in the Snap selection filter.

Toolpath entities are not reflected in the visibility filter on the Visibility tab. Toolpath entities cannot be filtered by color and layer in the Selection filter on the Selection tab.

The Edit → Graphic properties command does not display any information on the graphic properties of a toolpath entity and its segments.

The command cannot be used for a toolpath because a toolpath entity is only managed as a CAM data object.

Click a toolpath entity with the left mouse button in the graphics area to select the associated job in the hyperMILL® browser. Double-click the hyperMILL® job with the left mouse button to open it for editing.

Commands can be run for the selected toolpath entity on the context menu:

  • Select a single toolpath entity: The Modify command is shown.

  • Select one or several toolpath entities: Commands that apply to toolpaths are shown.

  • If no toolpath entity has been selected, no toolpath-related commands are shown.
