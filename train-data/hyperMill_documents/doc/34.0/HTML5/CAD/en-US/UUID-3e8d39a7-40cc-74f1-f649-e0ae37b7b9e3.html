Align faces orientation

Invert the orientation of an individual face or orient all faces in the same direction.

Modify → Align faces orientation

Invert the face normal vector of an individual face or orientate all faces so that all face normal vectors are oriented “inwards” or “outwards.” Applicable for faces and faces in open solids, primarily used to correct imported data. Needed to calculate the tool position based on face normal vectors.

Uniform orientation

Align: Click the button! Select all necessary faces . The face closest to the user is automatically aligned so that the face normal vector points out of the monitor plane . All other faces are oriented accordingly based on topological relations. The number of selected entities is displayed.

Swap orientation: Invert the orientation of all faces .

Single invert

Invert: Select a face . Only the orientation of the selected entity will be reversed.
