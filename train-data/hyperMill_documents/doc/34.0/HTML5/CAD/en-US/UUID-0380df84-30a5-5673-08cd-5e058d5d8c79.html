Configure tool report

Select and configure template for tool report.

CAM → Reports → Configure tool report

Call up the command and carry out the following steps to make presettings for exporting a tool report:

  1. Choose a template. A simple and a detailed tool report are available.

  2. Report mode (model / joblist): Select whether the tool report should be generated for each job list or for the entire model.

  3. Tool order: Select the order in which the tools are to be listed in the report.

  4. For two images (Define model view (Graphics 1), Define model view (Graphics 2)) select in which form the images are generated. The following options are available:

    • No screenshot – so no image

    • 2D view

    • 3D view

    • Setting up the view manually

    • Manually setting up and changing the view

    Use the entries in the Start Excel cell, Width of bitmap and Height of bitmap options to influence the position and size of the images.

  5. If necessary, adjust Folder and File name to use your own logo. The OPEN MIND logo is used by default.

  6. Set the following options:

    • Joblists selection: Select the required job lists

    • Each job list in a separate Excel file?

    • Automatic clamp display?

    • Stock dimensioning?

    • Sub joblists separated?

  7. Result display: A PDF document or Excel can be optionally opened with the generated job report.
