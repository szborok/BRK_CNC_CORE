6.5. Importing, filtering and renaming metadata names

Transfer user-defined properties (metadata) of a document, from solids and faces from external CAD software as tags. To do so, select the Model conversion mode → Create model structure option. Metadata is assigned as tags to the document or entities.

Note

Filtering takes place first before renaming. This means that you need to work with the original names to filter out metadata.
