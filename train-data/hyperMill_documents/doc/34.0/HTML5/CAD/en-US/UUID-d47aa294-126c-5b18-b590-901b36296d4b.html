Automatically generated variable

If, for example, for the area of a square with an edge length of 50 pow(50,2) is entered in an option in a command dialog, the result 2500 will be calculated and used.

If, however, an equals sign is placed in front of the formula, for example, =pow(50,2) the calculated result will be used and, in addition, the formula will be transferred to the parameter list and a variable will be generated, such as _1.

Select the Show automatically generated variables option to display all formulas entered in commands . A row is inserted into the table for each automatically generated variable. The variable has an automatically generated name.

To edit the name, double-click the respective cell with the left mouse button , edit the name of the variable and apply . By renaming the variable, it will be saved permanently with all its data. If you do not rename it, the automatic variable will also be automatically removed from the parameter list after you have removed your entries in the respective command dialog.
