Copy attributes from entity

You can use this command to transfer properties from a source entity to target entities. First select the target entities and then the source entity.

Note: Automatic preselection (5.1) should be activated.

Procedure:

  1. Use entities selected for the last command (automatic preselection) – target entities.

  2. Start the command

  3. Then select the entity whose properties you wish to apply – source entity.

Transferred properties:

  • Layer

  • Color

  • Isoparametric along U / Isoparametric along V

  • Points along U / Points along V

  • Line type

  • Line thickness

  • Properties that define the style of dimensions, such as arrow heads, dimension lines, text properties, and tolerances.

Note:

  • If the selection is invalid (e.g., due to multiple or unsupported source entities), it is automatically discarded and must be made again.

  • The command cannot be used for toolpath entities.
