Ellipse

Generate an ellipse.

Drafting → Ellipse

Generate an ellipse using two axes for the minimum and maximum diameter of the ellipse.

To change an existing entity, double-click the entity with the left mouse button or select the entity and choose the Change parameters command on the context menu.

Modes

Point: Select or snap a point or a position for the intersection point of the two axes .

First axis: Select length for major radius of the main axis .

Second axis: Select length for minor radius of the secondary axis .

Rotation

Rotate the ellipse. At 0° the main axis lies parallel to the X axis of the current workplane.

Angle: Select an angle .

Direction: Align the main axis parallel to a straight entity .

Reference plane

Selecting On current workplane creates the curve on the current workplane . Alternatively, select an Entity as the reference plane for the position . This entity must be planar.
