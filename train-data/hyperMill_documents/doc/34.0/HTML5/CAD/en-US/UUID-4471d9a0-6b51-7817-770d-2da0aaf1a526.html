2D chamfer

Create chamfers between several lines.

Drafting → 2D chamfer

Create chamfers between several lines. The lines must be planar but do not have to lie on the workplane. Distance and depending on the mode, Distance or Angle define the position of the chamfer. The clicking order is taken into account. If Automatic trim is selected, the curves will be automatically trimmed at the transitions to the chamfer , otherwise only the chamfer is inserted .

Select

Curves: Select the entities . The number of selected entities is displayed.

Modes

Symmetric: Create symmetric chamfers, usually at a 45° angle with Distance.

Distance: Create chamfers with Distance and Distance .

D. + angle: Create chamfers in Distance and Angle .

When a chamfer is inserted between arched curves, the position of the chamfer is calculated in different ways. In Symmetric mode, the calculation takes place in the same way as the insertion of a fillet. The only difference is that a straight line is inserted instead of an arc . The Distance value defines the radius of the circular arc. In Distance mode, the values for both distances are measured along the path of the curve . In D. + angle mode, the chamfer is the result of the Distance along the first curve. At the resulting point, a curve normal is calculated to which the value entered for Angle refers .
