Adjusting title block and frame

Title block and frames are stored in templates in *.hmc format.

The company settings (highest priority), user settings and finally, installation settings (lowest priority) evaluated for title block and frames. Templates for the title block must be located in the Printing title blocks folder. Templates for frames must be located in the Printing frames folder. All templates are collected. In the event of duplicates, the template from the folder with the higher priority is provided.

Drawing frames

Enter a frame in the desired shape. The outer contour describes the edge of the page. The workplane must be on the bottom left corner. Draw another contour for the frame within this contour. Use a point to specify which corner the title block should be aligned to. Hide this point.

Save the document as a *.hmc file. The file name must comply with the rules, as the template is assigned using the file name of one of paper sizes selected in the print settings:

It is possible to generate templates for ANSI Y14.1 paper sizes from A to E and the DIN EN ISO 216 paper sizes of the A to C series from class 4...0, 2...0, 0 to 5. Identify vertical alignment with the letter P. Identify horizontal alignment with the letter L. The frame from the template is fitted while maintaining the aspect ratio in the paper size.

Title block

Enter the title block in the desired form.

User defined information can be directly applied in the title block using TAGs. To do so, perform the following steps in the template:

Create a tag with Tags → Create → Quantified tag.

Determine the position within the title block and insert a link to the tag in Drafting → Text.

Assign the tag to the text in Tags → Assign.

Save the document in the Printing title blocks folder as an *.hmc file. The file name is freely selectable.

The tag is then displayed in the value table for the Title block when printing.
