16.1. Import colors

Import a color palette for electrodes.

Electrodes → Import colors

Electrode

Import predefined colors for electrodes into the color library of a document or transfer to an *.hmct document template.

The color palette can be read in multiple times. The colors will be updated by matching the color names.

Import to an open document

  1. Open document.

  2. In File → Options → Electrode options, go to the Folders → XML file for electrode color import option and enter an *.hcmaterials.xml electrode color palette .

  3. Then import the color palette into the opened document using the Electrode → Import colors command. The successful import is confirmed on the Info tab.

  4. Under File → Options → Electrode options → Colors, assign the colors of the color palette to the options .

Transfer to a document template

  1. Create a new document or open an existing document template.

  2. In File → Options → Electrode options, go to the Folders → XML file for electrode color import option and enter an *.hcmaterials.xml electrode color palette .

  3. Then import the color palette into the opened document using the Electrode → Import colors command. The successful import is confirmed on the Info tab.

  4. Under File → Options → Electrode options → Colors, assign the colors of the color palette to the options .

  5. Save the opened document as an *.hmct document template under File → Save as.

  6. Then enter this template as a document template. There are two ways of doing this:

    • File → Options → Options / properties → Application → Template

    • File → New from template → Add templates to the list

  7. Use the template for a new document.
