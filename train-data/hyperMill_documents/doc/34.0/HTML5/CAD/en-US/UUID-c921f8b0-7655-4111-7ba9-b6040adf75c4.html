17.3. Adaptive construction

Commands for preparing geometries for CAM-oriented construction tasks.

Close openings within faces.

Shapes → Close holes

Create faces to guide the tool during swarf milling.

Shapes → Swarf cutting

Create extension faces - for milling as well.

Shapes → Extension faces

Join multiple free-form or analytical faces into a face.

Shapes → Global fitting

Wrap or unwrap a label for a side shell.

CAM → Utilities → Wrapping side shell

Wrap or unwrap a profile pattern.

CAM → Utilities → Wrapping tread
