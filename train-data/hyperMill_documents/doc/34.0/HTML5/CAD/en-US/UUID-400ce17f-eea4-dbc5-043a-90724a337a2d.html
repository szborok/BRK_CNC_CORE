13.3. Geometric constraints

Add geometric constraints.

Drafting → V-sketch → Geometric constraint

Assign constraints manually; in addition to any constraints already assigned automatically. The V-sketch is adjusted accordingly. The constraints should be able to be applied in a meaningful way.

Select

First entity: Select an entity within the V-sketch to which the selected constraint is to apply.

Second entity: A second entity must be selected if a constraint is created between two entities.

Select a point and curve for the Point-Line constraint type.

Constraints

Select the desired constraint. An active constraint is shown in green.

Fixed position

The position selected is not moved in the case of modifications.

Horizontal

The entity is arranged horizontally and maintains its horizontal arrangement in case of modifications.

Vertical

The entity is arranged vertically and maintains its vertical arrangement in case of modifications.

Parallel

The entities are arranged parallel to each other and maintain their parallel arrangement in case of modifications.

Tangential

The entities are arranged tangentially to each other and maintain their tangential arrangement in case of modifications.

Perpendicular

The entities are arranged perpendicular to each other and maintain their perpendicular arrangement in case of modifications.

Point-Line

A point or position is arranged in such a way that it is at the same ‘height’ with respect to the selected curve and its imaginary extension. This height is maintained in case of modifications.

Coincident

Two points or positions are modified in such a way that they assume and maintain a joint position. The first position determines the location.

Coincidence X

Two points or positions are arranged in such a way that they are at the same ‘height’ with respect to the X direction (horizontal). This height is maintained in case of modifications. The first position determines the ‘height’.

Coincidence Y

Two points or positions are arranged in such a way that they are at the same ‘height’ with respect to the vertical. This height is maintained in case of modifications. The first position determines the ‘height’.

Coincidence radius

Two arcs have the same radius.

Coincidence length

Two lines have the same length.

Fixed angle

The angle of the entity remains constant if changes are made.

Options

External references: You can create a relation to an entity not contained in the edited V-sketch.

Example 16. Parallel constraint between two V-sketches

Create a constraint for two V-sketches, so that two curves remain parallel to each other. First select the curve in the first V-sketch, then select the curve in the second V-sketch. Then specify the type of constraint. In the first V-sketch, the curve will change so that it runs parallel to the curve selected from the second V-sketch.



Select series: Multiple application of constraints (Parallel, Coincidence XY, Coincidence X, Coincidence Y, Coincidence radius, Coincidence length) in succession without interruption. All constraints assigned in this way are treated as a single constraint and can be changed simultaneously, e.g., with a single dimension.
