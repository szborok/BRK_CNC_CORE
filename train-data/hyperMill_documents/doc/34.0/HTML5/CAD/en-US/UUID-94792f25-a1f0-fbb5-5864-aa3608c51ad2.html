2. User interface

You can start each document as a separate process. The software is adapted to the use of wide screen formats and for working with multiple screens. You can also place variable areas outside of the program window, that is, you can move them to another screen. When you exit a document the last window positions are saved.

The user interface is composed of the following elements:

Title bar

The title bar displays the name of the currently opened document. To hide the title bar, press the F11 key.

Menu bar

The software's commands are accessible from the menu bar . The menu bar is static and cannot be changed.

Brief information

Brief information on commands at positions where the cursor rests or on entries required within a command – on the right in the menu bar .

Progress bar

Progress is displayed for processes that require considerable processing power and information is provided about the current action within the process (if possible).

Toolbars

Toolbars are used to call software commands. Click the icon with the left mouse button.

The user can create own toolbars, and the content of toolbars can be modified.

Dialogs

In numerous commands, data is entered in a dialog.

Graphics area

The graphics area allows you to interact with the entities of a document as geometric representations.

Toolbars in the graphics area

The software's commands are accessible from the menu bar. Toolbars are fixed in their position. The contents of a toolbar can be changed.

Context menu

Context menus provide a selection of commands that are relevant for the work that is currently being carried out, also in areas for entering text and values. To open a context menu, briefly press the right mouse button.

Depending on the context, you can double-click with the left mouse button in the graphics area within a command or selection to automatically jump to the next input that is appropriate for the sequence. The Continue command or another command highlighted in bold on the context menu indicates this option. If the Apply command is highlighted in bold, the entries will be applied and the calculation will be performed.

Click OK or Exit to exit the command. Recent commands allows you to call up a command from a list of most recently used commands. The length of this list can be controlled using the Max. recent commands option under File → Options → Options / properties → User interface.

Press the ENTER key to recall the previous command.

Swiping tabs

On swiping tabs , icons can be used to call commands. If the content of a tab is partly hidden, you can scroll through all of the available commands with the mouse wheel while holding down the ALT key at the same time.

Tabs

Tabs are used for structural information, for filtering, for displaying messages and attributes, and for the hyperMILL® browser.
