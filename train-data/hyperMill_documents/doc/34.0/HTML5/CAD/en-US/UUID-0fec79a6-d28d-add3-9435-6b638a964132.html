13. V-sketch

Provide 2D contours, such as milling boundaries or turning contours, with geometric constraints. If changes are made to curves within a V-sketch, the V-sketch is updated automatically by means of the dependencies defined via the constraints. Edit the values of the V-sketch via dimensional constraints and the parameter list.

Overview V-sketch

Create or modify V-sketch.

Drafting → V-sketch → Create / modify

Add automatically defined constraints.

Drafting → V-sketch → Automatic constraints

Add geometric constraints.

Drafting → V-sketch → Geometric constraint

V-sketch dimensioning.

Drafting → V-sketch → Dimensional constraints

Change a measure of a V-sketch.

Drafting → V-sketch → Change dimensional constraint

Hide the highlights for V-sketch constraints.

View → Hide / unhide → Hide V-sketch constraints

Unhide the highlights for V-sketch constraints.

View → Hide / unhide → Show V-sketch constraints

Options for the Sketch and V-sketch commands

File → Options → Sketcher options

There are three types of constraint:

Implicit constraint

An implicit constraint is always recognized automatically by the software. It is not marked with a symbol. An implicit constraint arises, for example, when a curve end point and a curve start point match, in other words, both points are coincident.

Geometric constraint

A geometric constraint must be assigned manually. The V-sketch is adjusted accordingly. The constraints should be able to be applied in a meaningful way. The software can search automatically for geometric constraints. Geometric constraints are marked with symbols. These symbols indicate the type of constraint.

Dimensional constraint

The dimensions in a V-sketch. Modifying a measure in a dimension assigned to entities changes the geometry of the entity accordingly.

A V-sketch can be created in the following sequence:

  1. Create the curve entities, for example, using the Drafting → Sketch command.

  2. Convert these entities to a V-sketch entity using the Drafting → V-sketch → Create / modify command. In the model tree on the Model tab, the selected entities are added to his V-sketch entity.

  3. Using the Drafting → V-sketch → Automatic constraints command, geometric constraints are determined from the existing position of the entities to each other. This is also a possible option in the Create / modify command.

  4. Using the Drafting → V-sketch → Geometric constraints command, further geometric constraints between entities are defined. The selected constraint may change the position of the entities relative to each other.

Dimension a V-sketch to be able to vary it:

  • Make sure that the V-sketch entities and their constraints are present in the desired form.

  • Call the Drafting → V-sketch → Dimensional constraints command and insert a dimension for all geometric properties of the entities that are to be modified later on. This is also a possible option for radii in the Automatic constraints command.

To modify the V-sketch, double-click with the left mouse button on a measure or call the Drafting → V-sketch → Change dimensional constraint command and select the measure.

Generate a variable optionally under Edit → Spreadsheet. Use this variable as a value for a dimension. Vary the V-sketch by changing the value of the variable in the parameter list.

The geometry of a V-sketch can be modified subsequently:

  • Variant A

    • Use a command to modify the geometry, for example, Fillet, to insert a fillet. Permit the selection on the Selection tab to select an entity within the V-sketch.

    • Then call the Create / modify command and select the V-sketch. Gaps and over-defined areas are marked by red points. Make corrections. Click the left mouse button and press the CTRL key at the same time to separate entities from the V-sketch.

  • Variant B

    Change the V-sketch interactively. Make sure that the selection within the V-sketch entity is permitted. Double-click a curve within the V-sketch with the left mouse button. Generate the required shape of the V-sketch using the preview. Click once with the left mouse button or press ENTER to confirm the entry.

When the cursor rests on a dimension or a curve, the related entities are highlighted in color. Make sure that the selection within the V-sketch entity is permitted.

Change the position of a dimension using the Edit → Move text / dimension command.

Remove a constraint within a V-sketch using the Edit → Delete command. Call the command and select the symbol of the constraint using the left mouse button. Make sure that the selection within the V-sketch entity is permitted.

Delete a V-sketch using the Edit → Break command. The entities are retained.

Redundancy – if there are two identical symbols on an entity, it can be assumed that a redundant relationship has been created. To check, allow the selection within the V-sketch and rest the cursor on one of the symbols. The related entities are highlighted. If they are the same entities, one of the redundant relationships can be removed.

Number of Degrees of Freedom - The number of degrees of freedom of a V-Sketch can be determined in analysis → Element properties . It is also possible to include the number of degrees of freedom in the tooltip.

Free dependencies within a V-sketch are displayed as points or positions if the display of Number of degrees of freedom has been configured for the tooltip.
