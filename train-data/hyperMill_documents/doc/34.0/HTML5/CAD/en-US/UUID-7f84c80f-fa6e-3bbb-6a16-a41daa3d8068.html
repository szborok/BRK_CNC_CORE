Circular sector visible entities

Select all entities arranged in an arc that are visible from the viewing direction.

Select → Circular sector visible entities

Select entities that are visible from the viewing direction with an arc-shaped selection area. Press the ALT key to only select entities that are completely within the selection area.

Select

Center: Click the center of the arc with the left mouse button, snap a position or select a point.

Vertices: Click two diagonally opposite corners with the left mouse button, snap the positions or select two points to draw the selection area.

The number of selected entities is displayed.
