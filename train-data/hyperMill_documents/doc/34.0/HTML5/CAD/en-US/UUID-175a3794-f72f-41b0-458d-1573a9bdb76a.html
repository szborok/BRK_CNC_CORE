Spiral

Create a spiral curve.

Curves → Spiral

Create a spiral curve.

Height specifies the overall height of the spiral. The spiral will be cut at this height and may be incomplete. Use Pitch and Taper (as the tapering angle) to specify the shape of the spiral further. If you select Clockwise, the helix will be oriented in a clockwise direction. Flat creates a projection on the current workplane. The projection is placed at the start point of the helix and only occurs if the helix has a taper. Define the alignment of the helix . The distances (a) are the same.

Split: Split the spiral after each 360°.

Variable pitch: It is possible to modify the slope continuously from the beginning to the end of the spiral. Enter a value for the Final pitch.

Select

Point specifies the start of the spiral. The distance between the point and origin defines the radius of the spiral at its base.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.
