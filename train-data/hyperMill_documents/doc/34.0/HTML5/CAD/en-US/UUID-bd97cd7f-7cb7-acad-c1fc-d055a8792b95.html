Split curve

Split one or several curves via boundaries or at their NURBS arc joints.

Modify → Split curve

If no intersection points can be calculated for the entities, a message appears.

The Compute on view command splits the curve at an intersection point that appears to exist from the view direction when the curve is projected onto the current view.

Select

Curves: Select the curves that are to be split. The number of selected entities is displayed .

Modes

Limits: Select the entities on which the curves are to be split .

Spans: Resets the selection of boundaries and splits the curves at the NURBS arc joints. The arc joints are displayed in the preview .

Select the preview curves on which the curve should be split. The number of joints is displayed.
