Helix

Insert helical face.

Shapes → Helix

2022.1

Insert helical face.

The Height describes the total height of the helix . The shape of the helix is further specified using the Pitch and the Taper as the tapering angle. Clockwise aligns the helix clockwise.

Variable pitch: It is possible to modify the slope continuously from the beginning to the end of the helix. Enter a value for the Final pitch .

Select

Curves: Define the starting point and profile of the helix. Select curves . The distance between the curve and the origin defines the radius of the helix at its base.

Direction

Select the direction by using an entity, specify it by using 2 points or selecting an axis direction of the current workplane (X Wp, Y Wp, Z Wp) or enter the direction with a Vector . The Invert option inverts the direction.

Origin

The origin is initialized at the first selected point.

For the origin of the plane, Select a point in the graphics area, enter a coordinate Coords. or select WP 0 0 0 for the origin of the current workplane.

More options

Split: Split the helical faces after each 360°.

With bases: It is possible to close the helix with flat faces at both ends.

Solids The Solid option is used to combine the faces into a solid .
