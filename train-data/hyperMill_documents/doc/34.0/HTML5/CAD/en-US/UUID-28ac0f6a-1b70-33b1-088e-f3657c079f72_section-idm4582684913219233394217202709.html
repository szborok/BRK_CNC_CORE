Renaming

Metadata names can be renamed to tag names for each format. The instruction is formulated in metadatarename.xml. An example file is provided in the .../files/converters/cad interfaces/conf/all/ folder.

This XML file is structured as follows:

<?xml version="1.0"?>
<MetadataRename version="1.1">
	<Format type="pro">
		<NamesMap>
			<Pair original="Original_Name">New_Name</Pair>
		</NamesMap>
	</Format>
</MetadataRename>

Use <Format> to specify the CAD interface format (to be imported) for which the filter should be used. The abbreviations that can be used for this are described in the Filter section

Within <NamesMap> with <Pair> enter a list of all metadata to be renamed. In the attribute original the original name and as a property of <Pair> insert the new name.

Example 2. Rename metadata for electrode

Insert the original name in the original attribute and enter the new name as a property of <Pair>. To do so, formulate the following XML lines:

<NamesMap>
	<Pair original="Elec_gap01">T_ELECTRODE_ROUGHNESS_V</Pair>
	<Pair original="Elec_gap02">T_ELECTRODE_ROUGHNESS_VV</Pair>
        <Pair original="Elec_gap03">T_ELECTRODE_ROUGHNESS_VVV</Pair>
        <Pair original="Elec_gap04">T_ELECTRODE_ROUGHNESS_VVVV</Pair>
        <Pair original="Elec_num01">T_NUMBER_V</Pair>
        <Pair original="Elec_num02">T_NUMBER_VV</Pair>
        <Pair original="Elec_num03">T_NUMBER_VVV</Pair>
        <Pair original="Elec_num04">T_NUMBER_VVV</Pair>
        <Pair original="Elec_mat">T_ELECTRODE_MATERIAL</Pair>
</NamesMap>
