Continuity face

Modify the transitions between two individual faces.

Modify → Continuity face

It is possible to select trimmed and untrimmed faces.

Tension weights the tension of the new face. Projection mode retains the size of the face. If Projection mode is not set, the size is adapted to the reference face . Distance shows the maximum distance between the new face and the reference face and thus indicates the quality of the calculation. You can use Optimize to trigger a new calculation in order to achieve a more favorable result. Use Keep original to retain the original face.

Select

Face: Select the face that you want to adapt. The area to be modified is specified by selecting the relevant boundary .

Reference: Select the face that is to be referenced. The referencing area on the face is specified by selecting the boundary .

Free boundaries

Select: Select the positions in order to specify the area in which the face can be varied .

Continuity on target

Use Position to create a positional transition between the two faces or Tangency to create a tangential transition .
