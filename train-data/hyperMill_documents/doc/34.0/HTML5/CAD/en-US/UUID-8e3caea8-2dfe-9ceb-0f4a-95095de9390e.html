Intersection points

Create one or several points at the intersections of curves, curves with a face or a mesh.

Drafting → Intersection points

Create one or several points at the intersections of curves, curves with a face or a mesh.

Extensions enabled: Calculate intersection points, including on the virtual extensions of the first curve with further entities in a plane selected as the second curve .

All intersections: Calculate all intersections of multiple selected curves (2D and 3D curves) . Only make the selection with Second.

Total points shows the number of intersection points that have been found.

Select

First: Select one or more entities that are needed to create the intersection points .

Second: Select the curves on which you want to create the points. The number of selected entities is displayed .
