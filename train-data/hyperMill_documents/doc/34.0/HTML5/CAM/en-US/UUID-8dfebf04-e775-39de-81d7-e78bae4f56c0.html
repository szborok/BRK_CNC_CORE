Document dialogue page

Units

Metric/Inch: The selected system of units applies to all input and output values.

Note

Do not change the measurement system during CAM programming, as definition values that have already been created will not be converted. Copying jobs between hyperMILL documents with different measurement systems is not allowed.

Directories

Project: Create project-specific directories to store your generated files.

Use Path to select a project folder. All generated files are stored in subfolders.

If you select Model path, all generated files will be stored in subfolders relative to the model file.

Joblist specific subdirectories: Subdirectories with the names of the job lists are used to store the generated files.

Global working space: All files generated are stored under the default path set under Application in the Setup hyperMILL dialog.

Feedbacks

Feedbacks visualize toolpaths, the start and end of machining in the vertical direction (top, bottom), clearance plane, tool or coordinate systems.

Size: Enter the required factor (visualization size).

colors: Open the color setting dialog, then click the respective color to open the color selection dialog and define the required color.

Color table

Set the colors for various hyperMILL elements.

Toolpath

Define the Toolpath colors for G0, G1, points, clipped toolpaths, NC path G0, NC path G1, tool axis, collision, and contact axis.

Tool

For Tool feedback, define the colors for the following elements: dimensions, spindle, holder, cutting area extension, tool, tool holder, center line, insert, and insert cutting area

General

Define the colors for the following elements under hyperMILL feedback: clearance plane, top, bottom, contours, boundaries, profiles, NCS, frame, highlight, synchronization lines, cutting direction, and infeed.

Stock model

Define the colors for the Material feedback for the following elements: stock model, stock coordinate system, removed material, and material to be removed.

Feature

Define the Colors for list display. Activate Used by Feature Mapping if the colors should also be used by the feature mapping.

Function

Max. number of origins: Limits the maximum permissible number of origins.

Use global clearance plane: Select this option to use the global clearance plane of the frame instead of the clearance plane from the job definition. The formula value for the global clearance plane will then be firmly linked with the Clearance plane parameter for the job definition (greyed out).

To machine with a job-related clearance plane, adapt the Clearance plane job parameter by right-clicking and then clicking Unlink on the shortcut menu.

Prefer featurelist frame: Activate to copy the frame assigned to a feature to the job.

Requirement: During feature mapping (hole), the Use feature list function is activated and the Frame creation mode has been set to 2D.

If the Prefer featurelist frame function is not activated, the NCS is assigned to the job if its orientation is identical to that for the frame orientation of the feature.

Use parameter: This function must be activated when local cycle parameters or user variables are to be used as part of the job definition (also see Input field values).

Calculation

These settings apply to toolpath generation.

Display temporary graphics : Temporary toolpaths created during the calculation are shown on screen in the interim and are then replaced by the final toolpaths.

Erase existing toolpaths: The next time a job is calculated, the existing toolpath is deleted. The newly generated toolpath is output graphically on the screen.

Automatic geometry check: After a geometry that is to be machined has been modified, the outdated geometry data is deleted from the cache before each calculation and the modified geometry is converted again.

Note

This function should always be enabled (it is active by default) in order to avoid faulty calculations.

Notify changed geometry: A message is generated for the geometry to be machined before the job is recalculated to notify the user that modified geometry data has been found and that the stored data will be deleted.

Delete temporary files: Deletes the temporary files ( *.tmp) created during job list calculation. Do not select this option if you are having problems with a component that requires the assistance of your OPEN MIND dealer.

Preserve reports: Save your report file in the folder defined on the Application dialog page of the Setup hyperMILL dialog (default paths).

Check toolpath status

Check geometry: When geometry changes are made, the system checks whether the toolpath is still up-to-date.

A modified geometry is displayed in the hyperMILL browser following a Check toolpath status. As this function uses up a lot of processing time, it is disabled by default.
