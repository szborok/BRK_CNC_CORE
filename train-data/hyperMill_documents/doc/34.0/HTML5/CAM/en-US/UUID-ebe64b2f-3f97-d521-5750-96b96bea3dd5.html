Multiple selections

Select multiple elements in the browser at the same time. To do this, click to select the required elements and hold down the CTRL key at the same time. Depending on the type of elements selected, different functions are available in the shortcut menu (right-click) (including Copy/ Export).

You can also use the multiple selections function to transfer several tools to hyperMILL. For further information, see section Import tool from database.
