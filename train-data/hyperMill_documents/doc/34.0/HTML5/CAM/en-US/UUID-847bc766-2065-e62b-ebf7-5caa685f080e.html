Import tool from database

In the hyperMILL browser, switch to the Tools / Jobs tab or go to the Tool dialog page in the job definition.

Select NC tool from external tools database.

Select the tool you require in the browser of the external OPEN MIND tool database. If you hold down the left mouse button and the CTRL key simultaneously, you can select more than one tool at once.

In the NC tool view of the external tool database, all input fields not directly belonging to the NC tool are write-protected.

If necessary, use a filter to restrict the display of the tools. For this purpose, define the appropriate filter conditions in the areas General, Tool, Holder and Technology. Right-click the mouse, select Transfer to hyper MILL from the shortcut menu and close the dialog box.

The NC tool is displayed in the list of document tools and is linked to the external OPEN MIND tool database. When importing a tool via the hyperMILL browser, new tools can be created in the import dialog. This is not possible when importing via the job definition. When importing a tool via the job definition and when opening the import dialog successively, the tool previously defined is already pre-selected. If, based on a unique ID, hyperMILL ID) recognizes that this tool has already been imported, you will be prompted whether you want to update the tool in question.
