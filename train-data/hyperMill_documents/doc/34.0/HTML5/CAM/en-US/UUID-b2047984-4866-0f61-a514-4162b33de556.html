Mirrors and stock models

Stock models which are to be reflected, must be in *. format. To mirror a job list that uses a stock model:

  1. On the Mirror dialog page in the joblist, select the mirror plane and enable the Copy option to create a (mirrored) copy of the joblist that you have already defined.

  2. On the Models dialog page in the mirrored job list, define the stock model that you want to use for the mirror. The resulting stock model of the original job list can also be used for this purpose.

  3. Assign this stock model to the first job contained in the job list and enable the Generate resulting stock option.

  4. Check the job status of the jobs contained in the job list and calculate the mirrored job list.

Resulting stock

Resulting stock is also mirrored during mirroring ( Job list → Mirror) if the Manufacturing parameters → Copy function is activated.

Mirror stock chain

A stock chain is also mirrored during mirroring ( Job list → Mirror) if the Manufacturing parameters → Copy function is activated.
