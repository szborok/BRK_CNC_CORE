Define and apply transformations

Define transformations and apply them in a job.

Define

To define a transformation, switch to the Frames tab in the hyperMILL browser and select the desired transformation from the shortcut menu in the Transformation area or create a new transformation.

Apply it to a job

In order to apply a transformation in a job, change to the Transformation dialog page and select the Active checkbox.

Now select the type of transformation under Selection and then click the icon to select a transformation or click the icon to create a new transformation.

Note

For toolpaths generated by a transformation, no precise safety check will be carried out. Collisions with a sufficiently small overlap may not be recognized under certain circumstances. Please check the geometric conditions and parameters of the transformation carefully before machining.

Output origin

Available if the Allow multiple origins option has been enabled in the job list (Job list settings dialog page).

Select the required origin from the list or click the icon (1) to define a new origin. Click the icon (2) to modify an origin.
