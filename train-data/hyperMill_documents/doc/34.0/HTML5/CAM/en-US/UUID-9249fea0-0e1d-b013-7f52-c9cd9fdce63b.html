Define milling area

  1. In the hyperMILL browser, double-click to open the job list.

  2. On the Part data dialog page, select the Defined option and select a milling area.

  		3.

    Define a new milling area or modify a milling area.

Alternatively, open the dialog via the job definition on the Setup dialog page or using the New milling area shortcut menu in the hyperMILL browser on the Models dialog page.

The Milling area dialog

The dialog consists of the Definition and Milling area dialog pages.

On the Definition dialog page, define the elements that belong to the milling area. On the Milling area dialog page, define the name and storage location for the milling area.

File: Click the Path button to select a 3DF or STL file.

Definition dialog page

Mode

Define the mode for the milling area definition.

Surface selection: hyperMILL automatically calculates the resolution for the selected milling areas depending on the machining tolerance. You can select a total of five surface groups with different resolutions and stock allowances.

Note

You cannot use an external polyhedron file to define the milling area for the following cycles: 3D Profile Finishing, 5X Profile Finishing, 3D Z Level Finishing, 5X Z Level Finishing, 3D Automatic Rest Machining and 5X Rest Machining.

Current selection

Group name: Specify the Surface Group’s name. A surface can only be allocated to one Surface Group. You assign the groups in the job definition.

Surfaces: Click one of the icons to select the surfaces to be converted.

Allowance: Height of the allowance that will be used in the cycles with multiple allowances. The value is added to the allowance that is defined within the cycle.

Define filter

Use a filter to define geometric elements.

  		1.

    Click the icon to open the Filter definition dialog, select the Active checkbox and enable the required filter option: Use color filter, Use tag filter or Use layer filter.

  2. Then click the icons to Add / select color filter or Add / select tag filter and define the color or tag class. If required, click the icon to select the surfaces that are to be used as the basis for the filter definition.

  3. If a layer filter or a layer container filter is to be used, select the Layer filter or Layer container filter checkbox. Then click the icons to Add / select layer filter orAdd/Select layer container filter and define the layer/layer container.

  4. To do this, in the Layer / Layer container line, click the icon to select a layer or a layer container that will be used for the filter definition. Enable the Use wildcard (\\*) option to search for character strings that appear before or after the placeholder in a name. You can also use two placeholders to search for strings in between.

  5. To delete one or all filters, click the color, Layer or Tag class line and use the Delete / Delete all function on the shortcut menu.

  		6.

    After completing the filter definition, click the icon to include the elements found in the CAD model on the basis of the filter definition in the selection.

No filter function is available for hyperMILL for Autodesk® Inventor®. Only the color filter is available for hyperMILL for SOLIDWORKS.

Milling area dialog page

Milling area information

Name: Define the name of the milling area. This is the same as the name specified in the hyperMILL browser.

Comment: If necessary, enter a comment.

File settings

Path: Define the storage location of the milling area. Corresponds to the value defined in the Setup hyperMILL dialog, on the Application tab (Default paths).

File prefix: Part of the file name of the calculated 3DF file. File name structure: File prefix-$-resolution.3df.

3DF file calculation

A 3DF file is calculated automatically when a new job is calculated, provided all the entries in the Milling area dialog have been made correctly beforehand.
