5X Deburring

Deburring of sharp edges. This cycle generates toolpaths for deburring sharp edges using 3-axis or 5-axis mode.

CAM Plan

To use this cycle, the edge geometry must be created in the CAM Plan before the toolpaths are calculated. Information on this can be found in section Basics of CAM editing → Structure CAM project → CAM Plan.
