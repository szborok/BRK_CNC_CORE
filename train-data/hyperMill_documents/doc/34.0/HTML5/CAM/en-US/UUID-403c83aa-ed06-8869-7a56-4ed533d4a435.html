Functions on the shortcut menu: Tools

New: Create a new tool.

Edit: Edit the selected tool.

Copy / Paste: Copy the selected tool and paste to another document. If the NC number or ID already exists, it is automatically incremented by 1.

Delete: Delete the selected tool from the current document. If there is a link to an external database, the linked tool will not be deleted from the external database.

Update state / Update all states: Update the status of the selected tool or all tools of the document.

Tool / Update all tools: Update the selected tool or all tools with the data from the tool database.

Update technology: Update the selected tool with technology data from the tool database. The following are updated: technology records, cutting profiles, cutter material, spindle direction and cutting edges.

Unlink / Unlink all tools: Unlink the selected tool or all tools of the document from the external database.

Relink/ Relink all tools: Link the selected tool or all tools of the document with the database.

Partial unlink: Partially disconnect the document tool from the external tool database. The parameter Tool reach, as well as all parameters relevant to the tool call (NC number, ID, name) can be changed in the document. The other tool parameters remain linked to the external tool database and their status remains current.

Difference: Show the differences between local and (external) database tool. The differences are highlighted in color. Use the two buttons previous and next to navigate between the differences.

Export to tool database: export selected tools to external database.

Load from file: import tools from a text file in *.tst or *.xml format in the current hyperMILL document.

Export ... / Export all...: Export the selected tools or all tools of a document to an *.XML file.

Messages ...: Display of warning(s) stored for the tool. This is also displayed as a tooltip in the Status column of the tool list.

Usage...: Display of Job ID and Job name. Open the job definition with a double-click.

Note

hyperMILL automatically calculates the matching tool length once the test tool option and the options to reduce/extend the tool are enabled (see Setup dialog page). The calculated tool length is displayed in the dialog box usage.

hyper MILL Setup..: Open the hyperMILL settings dialog.

Feedbacks/No feedbacks: Show/hide the tool feedback.

Feedback position: Set the base point for the feedback in the CAD model.
