2.2. The hyperMILL browser

The hyperMILL browser is used to define job lists, jobs, tools, machining systems (frames) and features.

Layout of the browser

The hyperMILL browser consists of the following elements:

(1) Tabs (CAM Plan, Jobs, Tools, Frames, Models, Features, Macros).

(2) Upper browser window in a tree structure.

(3) Toolbar with functions for editing elements, calculating and deleting toolpaths, or opening the tool database, among others.

(4) Lower browser window.

The browser view and layout change depending on the selected tab.

Open the selected entry (job/job list) for editing.

Delete the selected entry (job/job list).

Display messages for the selected entry (job/joblist).

Save the current view in the graphics area using the hyperMILL bookmark. CAD visibility settings and toolpaths are taken into account. Associative to hyperMILL jobs. Can be assigned to any hyperMILL job.

Note

A visual bookmark is not copied with an associative job copy and must therefore be recreated.

Delete the toolpath for the selected entry (job/joblist).

Open the tool database to select an NC tool.

Display the model in a transparent manner in the graphics area.

Show the removed material automatically when a job has been selected.

Check the status of the toolpaths for the selected entry (job/joblist).

Calculate the toolpaths for the selected entry (job/joblist).

Configure browser

The tabs can be arranged anywhere in the browser or undocked from the browser window. To do this, click the tab with the left mouse button and drag the tab to the required place while holding down the mouse button. Possible docking positions are indicated by a graphical feedback as blue, transparent areas. Dock or undock the tab from the browser by double-clicking (left mouse button) the title bar in the browser.

Show or hide a tab. Right-click the tab and enable or disable the respective command on the shortcut menu.

View manager

Use the View manager to control the visibility of tabs in the hyperMILL browser.

CAM → Setup View manager

By default, all tabs are shown in the browser. To hide a tab, disable the respective option. Use Reset layout to restore the default settings.

Jobs tab

The top section of the browser displays joblists and jobs and the bottom section shows the milling areas and stockmodels.

Note

Meaning of the text color for displaying stocks in the lower area of the browser:

Red: The start stock of the selected job. Prerequisite for a resulting stock (highlighted in blue).

Blue: The resulting stock of the selected job.

Black: The stock generated last in a machining sequence.

The start stock or last generated stock of a machining sequence is used in hyperVIEW or in VIRTUAL Machining Center.

The entry's marking in the browser indicates whether or not a job definition is complete and correct.

Definition free of errors/warning. Calculate job with these parameters.

Definition free of errors/warning. Definition free of errors/warning calculation status currently unknown (can be checked explicitly using command Toolpath status check).

Definition free of errors/warning. Job must be calculated.

Definition contains warnings. Definition free of errors/warning calculation status currently unknown (can be checked explicitly using command Toolpath status check).

Definition contains warnings job must be calculated.

Element (toolpath/stockmodel) does not exist.

Element (toolpath/stockmodel) is calculated and hidden. Click the icon to show the element.

Element (toolpath/stockmodel) is calculated and shown. Click the icon to hide the element. Double-click the icon to automatically display the toolpath when the job name is clicked.

Display the toolpath automatically. Click the job name to automatically show the toolpath for this job. Double-click the icon to disable the automatic display of the toolpath.

Turning with turret

The following information is available for turning jobs with a turret:

A machine with turret is used in the job list and the definition of the turret setup is free of errors and warnings.

A machine with turret is used in the job list and the definition of the turret setup contains errors.

A machine with turret is used in the job list and the definition of the turret setup is free of errors and warnings. No NC file has been created yet.

A machine with turret is used in the job list and the definition of the turret setup contains errors. No NC file has been generated yet.

A machine with turret is used in the job list and the tool used in the job is set up on the turret.

A machine with turret is used in the job list and the tool used in the job is not set up on the turret.

No machine for turning jobs with a turret has yet been set up in the job list
