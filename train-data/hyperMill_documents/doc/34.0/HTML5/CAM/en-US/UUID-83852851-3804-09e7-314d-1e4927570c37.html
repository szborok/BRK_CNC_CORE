4.2. User interface

The user interface of the feature module comprises the following:

  • The feature browser (Feature tab in the hyperMILL browser) with representation of the feature lists and features.

  • The Feature editor (Feature dialog) for defining feature parameters and

  • The Feature job connector (Feature dialog page of the job definition) for selecting and using features in a job.
