Point Milling

Strategy

Milling reference

For Milling reference, set the area of the blade to be machined. Machining starts from the hub (1) or shroud (2). Machining of the entire blade takes place with the Complete machining option (3). When using ball mills, complete machining is the preferred method for seamless transitions from hub to foot.

Contact mode

The last path is milled in contact with the contour. The radius of the selected tool thus corresponds to the radius between blade and side surface.

If the option is enabled, the last circumferential path is calculated so that the ball (tool tip) is rolled over the surface boundary of the platform if it cannot be touched tangentially.

The Contact mode option can only be combined with the Reverse rotation function. The combinations Standard (1), Contact mode active (2), Rotation reversed (3) and Contact mode active with Rotation reversed (4) are possible.

Edge curves

Use the Manually defined boundary option to specify the edge curves in the model.

The option Surface boundary is only available (and should then preferably be used) if exactly four surfaces have been defined under Geometry → Blade surfaces or Geometry → Drive surfaces, one each for the pressure and suction side and one each for the leading and trailing edge area.

Automatic: The edge area is delimited from the pressure and suction side via the Length / thickness ratio, so that manual sketching of the curves can be dispensed with even if the data quality is poor.

It is preferable to use automatic curve recognition in the edge area (shown in grey in the graphic) if the blade is not made up of four surfaces, but is made up of any number of surface patches.

The Length (a) and Thickness (b) of the edge area are specified by the Skeleton curve (2) of the blade and a line that runs perpendicular to the skeleton curve (3), see figure: Blade cross section, Edge area. Feasible values for the length/thickness factor lie between 0.5 (radii) and 1 (ellipses).

Figure: Blade cross-section (edge area)

Infeed mode

Reverse stepover: Machining is performed from the platform in the direction of the blade. This is a frequently-used certified machining strategy that needs to be reproduced in this way for the manufacture of new parts.

This type of machining prevents the tool from cutting over center. The more material that needs to be removed, the greater the effect.

Reverse rotation: The Reverse stepover function can be combined with the Reverse rotation function. The combinations Standard (1), Rotation reversed (2), Stepover direction reversed (3), Stepover direction reversed, Rotation reversed (4) are possible.
