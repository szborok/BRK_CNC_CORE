Parameters

Allowance

Allowance: Spatial offset from the surface (3DF file required).

Axial offset: The tool tip or the tool reference point remains this amount away from the selected contour.

Tip

A negative value means that the tool tip is below the contour by the selected amount (for example, for engraving).

Example: (1) Selected contour, (2-4) Generated toolpath, (2) Axial offset = 2, (3) Axial offset = 0, (4) Axial offset = -2

Infeed

Infeed and machining depth are relative to the tool axis.

Machining depth: Machining depth of the contour.

Stepdown: Specifies the number of machining passes. Z infeed to the next machining pass. The value of the last infeed is automatically adjusted to the machining depth and the stock allowance in Z.

Ramp axial infeed: Ramp-shaped stepdown from one machining plane to the next machining plane.
