5 axes

Setup A/B axis

Tilt angle (1): Tool axis angle to the Z axis of the frame in use. The value should be greater than the lead angle of the helix, so that the tool mills using the top face. The resulting reduced axial load helps to protect the tool and machine.

Lead angle (2): Angle by which the tool is tilted in the infeed direction. Relative angle to the tangent on the helical toolpath.

Hole collision avoidance A/B

Hole collision avoidance: If this option is enabled, collision avoidance occurs around the A/B axis. First, the tilt angle is reduced. If this is not sufficient, the lead angle is reduced as well.

Tilt angle max. range/Lead angle: For collision avoidance, the defined tilt angle/lead angle may be changed only by this value at the most in both directions.

Example: Tilt angle 10, Tilt angle max. range 4. As a result, the minimum permitted tilt angle is 6 and the maximum permitted tilt angle is 14.

Hole check offset: Offset for checking the tool for collisions using the model geometry of the hole.
