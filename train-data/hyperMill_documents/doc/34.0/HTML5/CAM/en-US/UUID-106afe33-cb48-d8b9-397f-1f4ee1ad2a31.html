Strategy

To generate the toolpaths, cuts are made along a central curve. Compared to the tube cycles, the tool axis does not run along the central curve, but more or less perpendicular to it.

Geometry type

The cycle supports the following geometry types: (1) 2 open ends, (2) 1 open end, (3) No open end, and Infinite (4).

Strategy

Machining can be performed along the halfpipe (option: Along) or across it (option: Across).

Reverse direction: Reverse the cutting direction.

Below is an overview of the machining options for the three geometry types:

Central curves / Boundaries

Top curve: (1) The upper curve corresponds by default to the 0.5 iso curve of a regular surface above the halfpipe. It is the basic reference for the maximum distance of the ball center (see Parameters dialog page).

Bottom curve: (3) The lower curve divides the halfpipe at the line (regular surface) between the upper and lower central curve into a left and a right side.

(2) = Boundary.

Note

The height of the central curve must be selected so that the tool contact covers the entire required machining area on the model.

The greater the distance of the upper curve (1) to the edge of the halfpipe or to the defined boundary (2), the more the tool ‘rolls’ around the edge of the halfpipe.

Boundaries

The selection of Boundaries (1) depends on the geometry type of the forming groove. The following are permitted for the geometry type

  • No open end A closed boundary around the central curve (A)

  • 1 open end An open U-shaped boundary around the central curve (B)

  • 2 open ends Two open boundaries on both sides of the central curve (C)

  • Infinite Two closed boundaries inside and outside the central curve (D)

(2) = Central curve.

Start point: (3) Available for the two geometry types No open end and Infinite. Activate to define the starting point of machining for closed geometries by selecting a point on the boundary. If the selected point is not on the profile, it is moved to the profile at the shortest possible distance. When using a feature, the starting point can also be moved/defined by editing the profile in the feature.

In the case of open geometries (B, C), machining starts by default at the starting point on the first or only boundary.

Note

The profile selection has been technologically improved for the 5X Halfpipe Machining cycle. The selection

  • is saved in full and generates an error message if the number of profiles or profile type is incorrect.

  • can be subsequently edited to close missing areas or remove elements selected by mistake.

Reverse direction: By default, the direction of the machining or the infeed is always the direction of the boundary. Select the Reverse direction option to reverse the machining direction.

Start opposite boundary: Only available for geometry type 2 open ends. Select to move the start of machining to the other boundary.

Contact mode: Enable the Contact mode option, to stop tool path already with the contact to the boundary profile - means before the tool center reaches the line to the top center curve.

Tip

Preferred option if tool must not roll into neighboring region that can be machined more efficient with different strategy/tool. Simplifies location of top central curve and boundary profile.

(A) function is not activated, (B) Function is activated. (1) Center curve → Top curve, (2) Boundaries → Profile

Synchronize toolpaths

Synchronization lines Synchronization lines run between the two boundaries or between the boundary and the upper curve. They are to be used to divide the halfpipe into areas and so influence the course of slicing as well as the course of the paths running across. The interpolation of the slices/paths then takes place – as with other machining strategies – over the areas limited by the synchronization lines. Slicing improvements improve the quality of the toolpaths – including those along the halfpipe.

Grouping toolpaths

Similar to other machining strategies - e.g. 3D ISO-Machining - toolpaths along a halfpipe can be symmetric to the bottom center curve (A) or with uniform number of path on left/right side (B).

Note

Preferred to be used if the split into left and right side should fairly follow the bottom center curve - e. g. along the deepest position along a halfpipe to achieve the same cutting condition per each side.
