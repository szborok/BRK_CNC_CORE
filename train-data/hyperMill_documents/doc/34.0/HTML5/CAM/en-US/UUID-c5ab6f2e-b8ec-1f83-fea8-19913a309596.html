The Feature editor

The dialog for defining feature parameters is called the Feature editor. The Feature editor comprises the following:

  • An input area with feature parameters (1).

  • A 3D preview with zoom and rotate functions (2).

  • A toolbar for saving inputs and calling the online help (3).

Define feature parameters

Feature parameters are associatively linked to the feature and can be defined by selecting geometric elements in the CAD model. If a geometric element is changed in the CAD model, the change is copied to the feature.
