hyperMILL for SOLIDWORKS

The following CAD utilities are available:

Incorrect converted faces

hyperMILL runs a plausibility check on the selected surfaces before each milling area recognition. If one or more surfaces with errors are recognized, you can use this function to save the found surfaces with errors to a layer.

You can repair the surfaces (for example, reconstruct) or remove them from the milling area if they are not relevant to CAM editing.

Zoom all: Zooms all incorrectly converted faces in the current CAD window.

Zoom selected: Zooms the incorrectly converted faces that have been selected in a list in the current CAD window.

All off: Hides all incorrectly converted faces in the current CAD window.

Show all: Shows all incorrectly converted faces in the current CAD window.

Show selected: Shows the incorrectly converted faces that have been selected in a list in the current CAD window.

Delete all: Deletes all incorrectly converted faces in the current CAD window and deletes them from the list of incorrect faces.

Move all to excluded face list: If the incorrectly converted faces are faces that you do not require for machining, you can exclude them from the selection. You must then convert the surfaces again, so that the polyhedron is generated correctly.

Update

Changes in the model geometry are forwarded to hyperMILL.
