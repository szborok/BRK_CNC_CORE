Plane

Geometric data from the Plane feature can be used for 2D, 3D and 5X cycles in the job definition.

The following parameters can be defined:

Plane: Define plane(s) in the CAD system.

Manual Boundary: Activate to define the boundary manually and use Boundary to define a closed contour in the CAD system. Otherwise, the boundary of the selected surface is used,

Top / Bottom: Enter values or derive from the CAD system. When the feature is used in the job definition, determine the vertical machining range.

Frame associative: Allows you to move the model to a different position when using the feature in the job definition. The frame used in the job associated to the model is also moved if the Application - Model check function in the Job shortcut menu is used.

For information on selecting contours, see section Selecting elements in the CAD system.

For information on defining attributes for geometric elements, see section The Feature editor.
