10.2. 3D collision check

Collision checking is ensured by the following factors:

  1. Milling area/turning area: 3D collision models are the part of the CAD model defined as a milling area or turning area (also see section Prepare manufacturing geometry).

  2. Frame: The orientation of the frame system determines which surfaces of a machining area can be milled. The system machines only those surfaces that are visible from the tool axis.

  3. Clearance plane and clearance distance: These parameters limit the areas for rapid tool movements (see section Clearance).

In addition to the automatic collision check performed for the tool tip, the tool shank, tool holder and extension can also be checked for collisions (see section Check tool).

Warning

No collision check is made for tools with free geometry.
