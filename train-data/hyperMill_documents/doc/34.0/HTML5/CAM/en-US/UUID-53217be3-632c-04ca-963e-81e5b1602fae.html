Parameters

Allowance

Hub allowance / Blade allowance: Remaining minimum allowance on the hub surface / blade surface.

Axial infeed

Manual stepdown: If the axial infeed is not limited manually, the system will only calculate one infeed in the axial direction. When specifying a maximum axial stepdown, the number of axial steps will be calculated from the height of the blade.

Lateral infeed / infeed strategy

Side width: In the lateral direction, the thickness of the material defines the machining area. The number of paths is derived from the lateral infeed.

Technology: edge behavior

Fan distance: Define the length for the smooth transition between the suction/pressure and edge areas.

Feedrate: Define the feedrate for edge rolling.

If the Strategy → Edge machining → None is activated, the following functions are available:

  • Edge clearance (1): The tool is moved around the edge (1) with the defined distance.

  • Edge lift (2): Avoid damaging the cutting lines along hub surface by entering an additional clearance distance relative to the hub in the edge area  (2).

If the option Strategy → Edge machining → None is enabled, no sheet edges are processed and no edge face may be selected at the start position (normally it is a blunt edge). The area of the enclosed edge is bypassed with the specified minimum distance in front of the edge. Edge clearance and Edge lift should therefore be > 0. Otherwise, a warning is issued and machining is hardcoded with the Enclosed edge only option.

If the option Strategy → Edge machining → Enclosed edge only is enabled, no special parameters are available in the Technology: Edge behavior area. When Enclosed edge only is enabled, the enclosed sheet edge is processed. As for the None option, no edge surface may be selected at the start position (normally it is a blunt edge).

If the option Strategy → Edge machining → Both is enabled, both (round) sheet edges are processed and a smooth overlap can be defined at the entry position:

  • Overlap length: (1) Define the length of the overlap of the toolpaths in the blade edge area (leading edge).

  • Overlap distance: (2) Define the maximum height of the lifting movement during the overlap.

Technology: Final paths

If the pre-machining job used a larger tool, this may mean that more material is left in the transition radius between the blade and hub surface.

Using this feature, you can define technology values for a Path number which meet the requirements for this increase in material removal.

Technology: Compensation spring path

Compensation path: Milling forces may cause the tools and blades to bend, resulting in defects in the form. One way to compensate spring paths is to repeatedly mill the last path. It is duplicated automatically if the feature is enabled. You can define the feedrate and RPM for this toolpath separately.
