Parameters

Vertical machining area

The vertical machining area is defined by default using the values for the Top (1) and Bottom (2) of the component.

Manual top (3) / Manual bottom (4): Specify the values for top and bottom directly on the model by clicking the icon. Manually defined values are not associative. If changes are made to the model geometry, these values will not automatically change as well.

Use the Material height parameter to set the height of the material to be removed.

Select the Trim to stock option to trim the toolpaths on the stock.

Pocket mode

Two methods are available for calculating the pocket toolpaths. Use the  High feed mode (1)  to realize large side infeeds and machine pockets with islands. Preferably use the Standard mode (2) with smaller side infeeds and consequently fewer repositioning movements.

Vertical infeed mode

Vertical stepdown: The vertical stepdown determines the number of machining levels.

Finish allowance: Material that is removed during a final finish pass.

Horizontal infeed mode

Infeed in the machining plane (XY plane) as a length dimension ( Horizontal stepover) or as a factor of the tool diameter ( Horizontal stepover factor).

Stock allowance

Allowance: Remaining material on the workpiece in the direction of the surface normals.

Additional allowance XY: Additional horizontal stock allowance. Allows machining to take place with different stock allowances for the bottom and side walls of the machining area.

Toolpath fillets

Fillet radius: Abrupt changes in direction within the cutting toolpaths are rounded. The value defined in the user interface is used to round the toolpaths that are in (lateral) contact with the model (1). The toolpaths that are not in (lateral) contact with the model are automatically rounded on the basis of the tool diameter (2).

Retract mode

The retract mode defines the Z level where the system executes horizontal infeed movements. The mode set is valid for all machining directions (incl. approach, retract and return movement macros) and for the profiles machined with them. Exception: Machining in zigzag mode without return macro.

Clearance distance (1): all retract and infeed movements are executed via the clearance distance. Starting and end positions of an infeed movement in rapid are displaced in Z direction in order to guarantee a collision-free linear infeed movement. The clearance distance is added to these positions.

Clearance plane (2): all retract and infeed movements are executed via the clearance plane.

Clearance

Clearance plane and clearance distance apply in the direction of the Z axis of the current frame.

Clearance plane: Plane for rapid tool movements. Specification in absolute dimensions. The clearance plane can be defined by right-clicking on the icon. Select the required contour and confirm the selection.

Warning

Traversing movements on the clearance plane are not collision-checked. Therefore, this plane must be placed at a sufficient distance above the surface of the workpiece.

Clearance distance: Distance to be defined to the toolpath to be milled. Above the clearance distance, infeed takes place as rapid in the Z direction; below the clearance distance infeed takes place at the Z feedrate.
