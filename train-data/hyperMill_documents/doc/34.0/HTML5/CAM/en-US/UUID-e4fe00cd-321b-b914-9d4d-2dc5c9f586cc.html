Define attributes

Click the icon to change geometric elements with attributes.

Point: Orientation (X, Y, Z), offset (in the direction of the defined orientation),

Vector: Rotate, Length:Scale (entered value is multiplied by the length value)

Boundary: Offset.

Contour: top, bottom, start point, end point, plunge point, Offset, reverse, overlapping.

Length: Scale factor

Surface, plane: exchange UV parameters, Use solid normal, Reverse normal of face, ignore inner boundary.

The offset for 2D Contour, 3D Contour and boundary is considered to be perpendicular to the defined orientation.
