Surface Group

Geometric data from the Surface Group feature can be used for 2D, 3D and 5X and drilling cycles in the job definition.

The following parameters can be defined:

Surfaces: Specify surfaces in the CAD system.

Strategy type: Define the machining surfaces and additional surfaces if required. You can define as many reference points as needed.

To change attributes for separate surfaces, you can combine several Surface features into one Compound Feature.

For information on selecting contours, see section Selecting elements in the CAD system.
