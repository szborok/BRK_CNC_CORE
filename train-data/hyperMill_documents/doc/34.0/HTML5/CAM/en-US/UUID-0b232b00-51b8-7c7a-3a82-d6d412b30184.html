Functions on the shortcut menu: Features

Automatic Zoom-In On: Automatically zoom in on a selected feature in the graphics window.

Zoom In: Zoom in on a selected feature in the graphics window.

_______________________________________________________________

Edit...: Edit selected feature(s).

Copy / Paste / Delete / Cut: Entries are managed in the standard Windows manner.

_______________________________________________________________

Create Compound Feature: Create a new Compound Feature with the selected feature.

_______________________________________________________________

Contour OMX export: Function on the contour feature shortcut menu for exporting contours in *.omx format. For file, specify the storage location path and name for the export. If necessary, determine the reference frame to which the coordinates refer. By default, coordinates of the world system are used. If Start point and/or End point are defined, the contour is trimmed at these points and the direction is adjusted as defined.

Hide referenced features: Hide the referenced features of a Compound Feature.

Select referenced feature: Select the feature referenced in a Compound Feature.

Combine pockets: Combine generic pockets manually into a pocket.

Export hole / Import hole: Export or import one or multiple holes in *.omx format.

Add hole to library: Transfer a generic hole to the standard hole library of the CAD system (Features → Holes → Advanced → Library → [User Library]).

Reverse hole: Reverse the direction of a Simple Hole or Sink Hole.

Replace by Sink Hole / Replace by thread: Convert a Simple Hole feature into a Sink Hole feature or a Sink Hole feature into a thread feature.

Fit hole to stock model: Adjust a hole to an existing or modified stockmodel geometry.

Sort Holes: Sort the list of intersecting holes of a compound feature in the feature browser.

Apply macros: Apply the macro(s) linked to the feature.

New macro from feature: Create a new macro based on the selected feature.

View macro database: Open the macro database.

_______________________________________________________________

New frame on feature: Create a new frame based on the selected feature.

New Job with feature: Create a new job based on the selected feature.

_______________________________________________________________

Set feature IDs: Define new feature IDs for the selected elements, beginning with ID1.

_______________________________________________________________

Usage: Display the jobs in which the selected element is used in a separate window.

Check model: Check the consistency of the contours assigned to the job.

hyperMILL setup: Open the dialog for inputting the basic settings of hyperMILL.
