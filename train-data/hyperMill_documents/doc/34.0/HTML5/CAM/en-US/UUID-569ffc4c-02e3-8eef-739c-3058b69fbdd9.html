Import WinTool tools

The tool IDs are checked in the process. When importing tools from WinTool, hyperMILL assigns NC numbers automatically. The next NC number is the highest existing number + 1.

Start via the CAM → Utilities → Import WinTool tools menu.

Export WinTool tools

Start via the CAM → Utilities → Export WinTool tools menu.
