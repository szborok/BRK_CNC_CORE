Edit

New NC-Job: In the hyperVIEW browser, create an empty NC job (data format: *.hv) that is assigned the following: machine, stock, model, result, rest material, clamp and toolpaths. The machine that is used is displayed as a default machine in the Machine administration dialog. The name of the default machine is also adopted for the name of the NC job.

Delete all NC-Jobs: Delete all NC jobs in the hyperVIEW browser.

Machine administration: Open the dialog for creating and editing machines.

The default machine is also displayed in the Machines toolbar.

Note

By default, the Machines toolbar is hidden when you first start the program.
