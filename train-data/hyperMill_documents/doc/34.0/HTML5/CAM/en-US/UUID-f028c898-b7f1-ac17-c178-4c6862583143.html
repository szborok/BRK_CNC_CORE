General dialog page

Frame information

Enter the Name , ID (ID number) and, if necessary, a Comment. All entries can also be changed later in the frame list's tree structure.

Global clearance plane: clearance plane of the frame. To change the default value, enter a new value, right-click the left side of the dialog header and select the Save as default option from the shortcut menu.

Note

If a frame is subsequently changed, check all jobs that use this frame again. A frame is not moved on a job-specific basis. Instead, the translation operations for the frame are adopted in the frame definition and therefore in all the jobs that use the frame.
