Parameters

Define infeed, approach and clearance parameters.

Infeed

Vertical stepdown (1): choose the stepdown to the next machining plane, from the Z direction of the active frame. This gives the number of machining planes. If a top is situated between two defined machining planes, the system will automatically reduce the vertical stepdown and insert an intermediate step.

Horizontal stepover factor (2): stepover in the XY plane is the distance between the center points of two adjacent milling tool paths and is generally given as a factor of the milling tool's diameter

Allowance

Allowance XY: Enter horizontal stock allowance.

Cutting mode

Select the Climb milling or Conventional milling option for the tool cutting mode. The cutting mode relates to tools rotating clockwise.

For climb milling, the following cutting modes apply to tools that rotate clockwise:

  • Counterclockwise for pocket contours

  • Clockwise for island contours

To perform climb milling with an anticlockwise rotating tool, select conventional milling. Climb milling and conventional milling are also available for High Performance Mode.

Clearance

Select the clearance plane (1) as plane for rapid travel movements along all three axes (absolute dimensions). Below this plane, only the infeed in Z direction is in fast travel mode up to the clearance distance. For a 3D model, the clearance plane value can be taken from the drawing.

Warning

The clearance plane is always situated above the workpiece boundaries in the Z direction.

Define a clearance distance (2) as a distance from the current surface (3), for fast travel Z-stepdown infeed (relative dimensions). Below the clearance distance, Z-stepdown takes place at feedrate.
