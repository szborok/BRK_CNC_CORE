Contours

Define the drilling strategy, contour selection for the holes and the vertical machining area (top and depth) of the holes.

Drilling mode

2D Drilling: The direction of the drill holes is always aligned with the Z axis of the defined frame system. When using features, the frame system is part of the feature definition. All drill holes can be collision checked using the Milling area.

2D multi index drilling: the direction of any number of drillings can be aligned with the different frames/planes. For infeed movements in XY direction, a relative clearance (automatically calculated for each frame) can be defined. All drill holes can be collision checked using the Milling area.

5X Drilling: The direction of the drill holes is aligned with the normals of the selected surfaces or with the selected lines. When using features, the frame system is part of the feature definition. All drill holes can be collision checked using the milling area.

The following applies to multi-axis indexing:

  1. In Drilling mode: The frame specifies the machining direction. The linking movements between the frames are controlled by the postprocessor.

  2. In 5X Drilling mode: Several sides can be machined in one job. The linking movements between the machining sides are checked for collisions and are executed as 5X simultaneous movements. A machine is required that supports this type of movement.

Turning: With this option, turning jobs can be carried out with a rotating spindle and fixed tool. The machining direction is defined by the Z axis of the frame. The drill position is always located in the center point of the workpiece being turned.

Frame dependent: Available for the 5X Helical Drilling cycle. The direction of the drill holes is aligned with the Z axis of the defined frame system. When using features, the frame system is part of the feature definition. All drill holes can be collision checked using the Milling area.

Note

Drilling in 5X Drilling mode in combination with tool reference points is no longer supported. If you want to machine with reference points, OPEN MIND recommends using the2D multi index drilling mode.

Contour selection – general information

Drill positions are selected either directly on the Contours input page or by allocating a previously defined feature on the Feature input page.

For information on linking features and the machining cycle, see section From job to feature.

The following CAD geometries are recognized as drill holes:

points,

lines, but no line points (for example, end points), no grid points.

For information on the general procedure for selecting contours, see section Selecting elements in the CAD system.

You can select all the drill holes for a drilling job that are produced with the same tools and have the same collision parameters (clearance distance, clearance plane).

Contour selection – 2D Drilling

Lines: The selected contour defines the top and depth of the drill hole. The machining area (top/depth) can be changed on the dialog page by entering values. Positive values will enlarge the drill hole and negative values will reduce the size of the drill hole. The direction of the drill hole is determined by the Z-axis of the defined frame.

Points: The following methods are available for defining the top and depth: Absolute (job frame), Top of contour, Bottom of contour, Relative to the top (depth only).

Absolute (jobframe) (A): The definitions for Top (1) and Depth (2) are absolute values relative to the zero point of the defined frame (3). They are independent of the actual position of the selected geometries.

Top / Depth (B): The dimension entered relates to the top edge and/or bottom edge of the selected contour (4).

Relative to top (C): The dimension entered for the depth relates to the defined top of the drill hole (5).

Note

The direction of the drill hole is determined by the Z axis of the defined frame (-Z) in all procedures.

Diameter: Define the required diameter of the drill hole.

Additional distance: Clearance distance for an individual drill hole (as opposed to the clearance distance parameter on the Parameters dialog page) when collision check is disabled.

Points and surfaces: Set the drill position with the selected points and surfaces and set the direction of the drill hole using the frame. Enter depth and diameter in the respective field.

Contour selection – 2D multi index drilling

Lines: The selected contour (line) automatically defines the Top and Depth of the drill hole.

Points and surfaces: Set the drill position with the selected points and surfaces and set the direction using the frame (Tool dialog page). Enter depth and diameter in the respective field.

Contour selection – Turning

Outer line: The start and end of machining in the Z direction and the machining diameter are specified (depending on the applied cycle) by defining a contour (outer line).

Base point: The start of machining in the Z direction is specified with the definition of a base point. If the base point does not lie on the turning axis, it is projected onto the turning axis in the X direction automatically.

Contour attributes

To define contour attributes (top, bottom), select the machining contour in the contour list. By making multiple selections in this contour list, you can quickly define the same top and bottom values for several contours.

Use the following buttons to change the order of selected contours or delete a contour. (1) To the top, (2) Up, (3) Delete, (4) Down, (5) To the bottom.
