Parameters

Infeed

Infeed length (1): Distance between probing point and ball of the touch probe. Traversing movement in probing feedrate.

Warning

Define the infeed length so that is large enough when there are large position deviations of the setup. There is a high risk of collision!

Retract mode

Define the Z level for retract movements.

Clearance plane: All retract movements take place via the clearance plane. In the case of interfering contours (for example, as a result of clamping elements), an additional approach or retract movement can be performed.

Clearance

Clearance plane (2): Plane for rapid tool movements. Specification in absolute dimensions Define the clearance plane by clicking the icon. Select a point and confirm the selection.

NC

Create log file: Creates a measuring log.

Select Default if the machine is to create a measuring log with the structure specified by OPEN MIND. Use the Extra function to generate a measuring log that can be customized. The Best Fit function (only 3D Point Probing) generates a measuring log that can be read in with the VIRTUAL Machining Center in order to determine the real part position with the help of the Best Fit function. For further information on the Best Fit function, see the VIRTUAL Machining Center product documentation.
