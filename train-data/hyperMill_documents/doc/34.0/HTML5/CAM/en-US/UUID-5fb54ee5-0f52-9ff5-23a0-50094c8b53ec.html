External simulation

hyperMILL currently supports the following external simulation programs:

  • VERICUT

  • COLLISION SAFEGUARD

  • NCSIMUL SOLUTIONS

Note

The following section contains information on activating and configuring external simulation programs in hyperMILL. For information on how the simulation programs work, please consult the product documentation supplied by the respective manufacturer.

If you use an external simulation program, you must activate it first. Afterwards, you can specify various settings for its use.

Please note that a license is necessary to use the program. Please contact your OPEN MIND partner for details.

Activate VERICUT

  • First open the hyperMILL CONFIGURATION Center program and select Connect applications.

  • Select the VERICUT entry under Key, then go to Program, browse to the correct folder in your VERICUT installation and select vericut.bat.

  • Confirm by selecting OK.

  • On the Postprocessor tab in the job list, select the Machine option.

  		•

    Click the icon to open the Machine administration dialog.

  • Define a machine whose NC output you want to check using VERICUT.

  • Activate the required simulation in the General → External simulation section.

  		•

    Click the icon to open the configuration dialog.

Configure VERICUT / COLLISION SAFEGUARD

Machine

Select a machine model (file format: *.mch/ *.xmhc).

Controller

Select a controller (file format: *.ctl/ *.xctl).

Project

Select a project file. This is a template file in which the VERICUT start configuration is defined (including window view, tolerances, collision checking, etc.). To change the configuration, open the project file in VERICUT and then save your changes.

Options

Default correction list: This value will be used if it has not already been defined in hyperMILL (job definition, Tool tab). Numerical values (integers), variables (tool numbers) and calculation options (such as T:No+10) may be used.

Tool reference point ID: Definition of a unique tool name (with formula support). Prerequisite for a correct tool change.

Attach to: Specify whether the components are to be placed automatically ( Automatic option). The VERICUT interface will then automatically search for the intended clamping position.

Position: Specify the position manually. The name of the VERICUT element at which the workpieces are placed must be defined.

Model: Close STL specifies whether a closed solid should be created for the STL files of the model.

Option Close fixture STL creates a closed solid for the STL files of the clamping part.

Closing the files can prevent display problems in VERICUT.

Program zero: Specify whether the VERICUT interface Automatically moves the program zero point or whether the setting is imported from the Project template.

This concludes the settings required to use the VERICUT interface.

Activate NCSIMUL

  • Open the hyperMILL CONFIGURATION Center program and select Connect applications.

  • Select the entry under Key, then go to Program, browse to the correct folder in your installation and select ConnectorFromNXF.exe.

  • Confirm by selecting OK.

  • On the Post processor tab in the job list, select the machine option.

  		•

    Click the icon to open the Machine administration dialog.

  • Define a machine whose NC output you want to check using .

  • Activate as a simulation in the General → External simulation section.

  		•

    Click the icon to open the configuration dialog.

Configure

Station: Select a machine station (only available if the Use stations option is activated in the NCSimul Configuration dialog).

Clamping position automatically: Activate if the clamping position is to be defined automatically by the software.

Clamping position manually: Activate to select a frame from the list to define the clamping position.

Output project: Path to create the *.nxf file and the *.stl file.

Generate project data: The files are created without starting NCSIMUL.

Run NCSIMUL: The files are created and NCSimul is started.

Start external simulation

Use the Utilities function on the shortcut menu of the job list or job and select the required external simulation.
