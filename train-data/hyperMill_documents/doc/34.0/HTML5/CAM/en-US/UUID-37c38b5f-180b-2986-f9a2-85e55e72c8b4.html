Tool

Supported tool types: Ball mill, end mill, bullnose end mill, lollipop, indexable cutter with round inserts, indexable cutter with high feed inserts, T-Slot Cutter, Woodruff Cutter.

The Use adaptive pocket option is not available for ball mills.

Consider corner radius: Function not enabled (1). The infeed distance is based on the total tool diameter. When working with bullnose endmills, material ridges may remain. Function enabled (2): The infeed distance is based on the tool's inner shank (shank diameter â€“ 2x corner radius). Material ridges are avoided.

Note

The Consider corner radius function is supported for both tool types Bullnose End Mill, Radiusmill and Indexable cutter with round inserts.

Free geometry

Use free tip geometry for calculation: Use a free tip geometry to calculate the toolpath. The exact positioning makes it possible to optimally incline the area to be machined.
