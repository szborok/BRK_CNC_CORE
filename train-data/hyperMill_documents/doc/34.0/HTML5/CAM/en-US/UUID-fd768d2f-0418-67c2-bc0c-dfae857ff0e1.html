Update calculated toolpaths

Changes to the feedrate, spindle RPM or comment do not require a recalculation of the job for the following cycles.

- all 3D cycles except Form Pocket

- all 5X cavity cycles except Z-level roughing and cutting edge

- all 5X surface cycles

- 5X Blade Top Milling

- 5X MB Point Milling

To change the spindle RPM or feedrate of the tool, choose one of the following options:

  1. Change the value for this job only directly on the Tool tab of the job definition. To do so, select the Unlink command from the Spindle RPM / Feedrate shortcut menu in the Cutting profile section. Enter the value directly into the dialog.

  2. Change the value for the tool in the Technology tab of the tool definition. Click the icon to open the Edit tool dialog and enter the required changes on the Technology tab.

Then update the toolpath file in the hyperMILL browser using the Update shortcut menu.

Information on calculated toolpaths

Information is available for the following via a quickinfo (tooltip): frame, job ID, job name, comment, job type, linked job, tool ID and tool name.

Configure the information displayed on menu File → Options → Tooltip content konfigurieren. To do this, move the elements from the Available TAGs area to the Tooltip content area.
