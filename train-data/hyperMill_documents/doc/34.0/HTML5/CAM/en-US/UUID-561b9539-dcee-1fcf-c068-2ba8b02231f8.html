Define material/cutter material

  1. Select the database entry in the database browser and then select the menu item Materials... / Cutter materials...

  2. To define a new Cutter material, right-click and select New or select a New standard cutter material from the list of existing cutting materials.

  3. To define a new Material, right-click and select New or select New from Library, to select an existing material.

  4. Enter the Name of the required material/cutter material and, if necessary, enter a suitable Comment.

Note

The name of the material/cutter material must be unique. Otherwise, you will get an error message when you close the window.

Because the cutting conditions for the tool are largely affected by the cutter material, you can assign a defined cutter material to each material in the cutter material dialog.

Assign cutter material to a material

In the lower part of the Materials dialog, you can assign each material one or more cutter materials.

In such an assignment, a distinction is made between the two measurement systems metric and inch as well as between milling technologies and turning technologies.

Note

If the cutting material Default is assigned to tools when converting to the current version of the tool database, this material is converted to Undefined.
