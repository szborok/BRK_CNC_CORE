Tool

The T-slot cutter tool type is supported. The cutting edges of the disc may be either square, rounded or chamfered.

It is possible to select different tool reference points.

Note

An adjusted postprocessor is required to use different tool reference points (Tip and Disc top). To adjust your postprocessor, please contact your OPEN MIND partner.

A distinction is made between the following options when selecting the Tool reference point:

The tool reference point is:

  • Disabled: The toolpath is calculated for the tool tip.

  • Enabled and Tip (1) is selected: The toolpath is calculated for the tool tip ( Tip is the default name used for the tool reference point that can be adjusted!).

  • Enabled and Disc top (2) is selected: The toolpath is output for the top of the disc ( Disc top is the default name used for the tool reference point that can be adjusted!).

(A) Upper cutting edge (2) / Lower cutting edge (1) = chamfer

(B) Upper cutting edge (2) / Lower cutting edge (1) = radius

(C) Upper cutting edge (2) / Lower cutting edge (1)= sharp

Free geometry

Use free tip geometry for collision check: A free tip geometry is taken into account during the collision check and simulation.
