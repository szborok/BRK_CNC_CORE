5 axes

Peripheral Path / Infill

Information about the tilt strategies Fixed, Keep, Manual curve and Radial Z in the hyperMILL product documentation.

The Normal to surface option (1) causes a toolpath to be created in which the tool axis is aligned with the surface normal at the contact point.

Use the Previous path function (2) to point the additive tool tip toward the previously generated path (B). For the first path (A), the tool axis will stay oriented to the job frame or o the surface normals if a drive surface is selected.

Machine limitation

Max. angle to Z: Measured at the Z-axis of the frame. It can be used to prevent the output of tilt angles that exceed the position range of the machine when path smoothing or collision avoidance is activated.

Definition:

  1. Z-axes of the selected frame and NCS are parallel: The maximum tilt angle corresponds to the maximum permissible angle of the machine axis.

  2. Z-axes of the selected frame and NCS are not parallel: For the maximum tilt angle, the angle between both Z-axes has to be subtracted from the maximum permissible angle of the machine axis.

Vector smoothing

Use vector smoothing: Activate to smooth tool orientations within a given angle range. Abrupt changes of orientation are thus avoided.

Delta A/B axis: The angle difference for the tool setting of the A / B axis must not exceed the defined value. Standard = 15 °.

Delta C axis: The angle difference for the tool setting of the C axis must not exceed the defined value. Standard = 15 °.
