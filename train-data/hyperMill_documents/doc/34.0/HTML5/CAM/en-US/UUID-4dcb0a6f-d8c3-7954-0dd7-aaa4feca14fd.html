Boundary Recognition

Extract edges and curves from the surfaces of a solid and generate a boundary from these.

Selection

Define the geometric elements (curve/surface) with which you want to generate a boundary.

Start edge: Define the first part of the boundary.

Surfaces: Define the surfaces from which the boundary is to be generated.

Single: The contour element that was highlighted with visual feedback is added to the selection and the next contour element is highlighted.

Auto: The contour element that was highlighted with visual feedback is automatically added to the selection and the next contour element is highlighted. The next contour element to be highlighted is always the one with the largest change in direction (angle) to the currently selected element.

Change: Click this button to highlight the other available contour elements (to be added to the contour selection in the next step).

Undo: The last selection step is undone and the last selected element is highlighted with visual feedback.

Result

Specify which feature is saved in the feature browser as the selection result.

Create CAD curves: A CAD curve is generated from the selected boundary. These are retained when the surfaces of the solid are ungrouped.

Feature: Select the options to generate a hyperMILL feature. You can generate either a Strategy feature (boundary/profile) or a Contour feature (2D/3D) from the selected boundary.

Apply: Generate a hyperMILL feature that is created as an entry in the feature browser. Contours that are not closed are also generated as boundaries. The start and end points of the contour are simply connected with a straight line.
