Parameters

Infeed

Constant: The maximum stepover is assigned by manual input.

Scallop height: The maximum side path distance for the current tool is derived from the maximum scallop height.

For an optimized infeed strategy, it is also necessary to specify the:

  • Min. stepover and/or

  • Min. scallop height

Allowance

Hub allowance / Blade allowance: Defines the remaining minimum allowance on the hub or the distance of the last path to the blade.

Partial machining

Number of paths : In the case of partial machining, only the last paths to the blades are machined according to the value specified.

Overlapping paths, Start distance: Additional paths are calculated according to the number specified. Beginning with a start distance relative to the hub, these paths get closer to the hub with each path. In the case of consecutive machining jobs, this smooth overlapping prevents different deflections and tool tolerances from appearing on the hub surface.

Technology: Edge behavior

Edge tolerance: The precise edge rolling of a splitter produces sharp changes in direction, which in turn lead to reduced feedrates. Entering an edge tolerance that is not equal to zero makes it possible to round the path to produce a smoother movement.

Example: (1) Large edge tolerance, (2) Small edge tolerance.

Technology: Opening cut

This function, which is only available for complete cutting, makes it possible to define other technology values for the first step.

In contrast to roughing, multiple infeeds are not provided here.

Technology: Final paths

If the pre-machining job used a larger tool, this may mean that more material is left in the transition radius between the blade and hub surface.

Using this feature, you can define technology values for a Path number which meet the requirements for this increase in material removal.
