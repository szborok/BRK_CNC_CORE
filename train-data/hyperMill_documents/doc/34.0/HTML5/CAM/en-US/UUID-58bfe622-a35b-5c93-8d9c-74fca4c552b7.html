The Job list dialog

The Job list dialog consists of the following dialog pages: Job list settings, Comments, Models, Turning, Mirror and Postprocessor.

Job list settings

Joblist

Name: Enter the name required for the joblist.

Compound job IDs/Job IDs: Use Start to define the start ID and Increment to define the incremental increase of the start ID for compound jobs/jobs.

Use Min. digit count to define the minimum number of digits for numbering jobs/compound jobs in the job list.

Toolpath

Check default path and name; change if necessary.

NCS

You must specify the program zero point before creating the toolpath. Subsequent moving of the NCS invalidates the toolpaths previously created.

Calculation

Compensated center path: Applies to the 2D Contour Milling, 2D Contour Milling on 3D Model and Inclined Contouring cycles.

Origins

Allow multiple origins: Select this option if you require multiple origins for machining, in order, for example, to machine a contour on four different sides of a cube with the help of transformations.

Note

The Best Fit function is only available with an appropriate license.

Probing

Best Fit: Activate if the Parameter → NC → Best Fit function in the 3D Point Probing cycle is to generate a measuring log for further processing with the VIRTUAL Machining Center. The measuring log records the position values on the component that is not yet optimally aligned.

It is used to calculate a corrected output system that will be used to create the NC file. The calculation is made on the Corrected output system dialog page of the frame definition.

Note

The Best Fit function can also be used with the help of the SHOP Viewer, for example, when CAM programming and machine operation are carried out by different people or on different workstations.

Create logs for CAD import: Visualize the measured values generated by the 3D Point Probing cycle in the CAD system (via the Probing tab). Various filter functions are available. For further information, see the product documentation for the CAD system.

Important

The Create logs for CAD import function is executed via the VIRTUAL Machining Center. The prerequisite for this is a corresponding Virtual Machine that is activated in the Job list → Postprocessor → Machine → Machine administration dialog and is available for selected machine controls. Please contact your OPEN MIND dealer to arrange this.

Stock chain

Note

The Stock chain → Automatic function is available for milling, turning, drilling, and additive machining operations.

Calculate stock chain

Automatic: Activate if a stock chain with a 3D start stockmodel is to be created automatically for all jobs in the job list. The Resolution defines the default for the calculation for all stock models in the job list if no other value is defined in the job definition on the General (Stock chain → Resolution) dialog page. The stock models are calculated in the order of the jobs in the job list.

If in a job in the job list the

  • option Generate NC data (dialog page General) is deactivated or the

  • option Exclude from stock chain (dialog page General) is activated,

there is no stock model calculation.

If a job is selected in the browser, the corresponding stock model with the same name is automatically displayed in the lower browser area.

Click the icon (1) to perform a stock calculation for the selected job. The calculated stock model is then displayed in the graphics area.

The calculated stock model is displayed in the graphics area.

The calculated stock model is hidden in the graphics area.

Update stock chain

If individual jobs are re-sorted in the job list, use the function Update automatic stock chain on the shortcut menu of the job list to recalculate the stock models for all required jobs. The stock models are only recalculated for those jobs that have been changed. Jobs that do not use a stock model are not recalculated.

Simulation / Generate NC file

If a simulation is carried out with the functions Toolpath preview or SIMULATION Center, or if the VIRTUAL Machining Center is started with the function Generate NC file, a correctly calculated stock model is automatically accepted if the function Update automatic stock chain has been carried out beforehand.

Comments

General information on a hyperMILL project that is transferred to the NC program as an NC comment. You can also add further information on the CAD file or CNC machine to the NC file by defining a prologue in the hyperVIEW Machine definition dialog.
