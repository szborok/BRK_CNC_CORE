hyperMILL for Autodesk® Inventor®

The following CAD utilities are available:

Show surfaces with errors

hyperMILL runs a plausibility check on the selected surfaces before each milling area recognition. If one or more surfaces with errors are recognized, you can use this function to save the found surfaces with errors to a layer. You can repair the surfaces (for example, reconstruct) or remove them from the milling area if they are not relevant to CAM editing.

Hide surfaces with errors

You can use this function to hide all surfaces that could not be converted correctly when generating the polyhedron.

Clear information about surfaces with errors

You can use this function to delete information about surfaces that could not be converted correctly when generating the polyhedron.
