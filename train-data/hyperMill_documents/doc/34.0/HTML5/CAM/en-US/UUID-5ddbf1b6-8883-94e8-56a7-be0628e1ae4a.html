Linear pattern

You can use a linear pattern in a job, compound job or linking job.

Define

  1. Define the Transformation name and a Comment in the Transformations dialog.

  2. Use X direction / Y direction to define the direction of the pattern.

    Structure: If the Fixed distance option is selected, the Number of elements defined and the Distance specified between each element are taken into account when creating the transformation. If the Fit elements option is selected, the Number of elements defined is fitted to the Length of the entire pattern.

Apply it to a job

To apply a linear pattern to the job:

  1. Navigate to the Transformation dialog page, enable the Active checkbox and select the required linear pattern under Selection.

  2. Under Start reference, specify the start coordinate system to be used as a basis for the relative movement of the transformation. Utilize the Use job frame option to use the active job frame. Alternatively, select a frame or origin.

    Target reference: The target coordinate system to be used as a basis for the relative movement of the transformation. Use the Use start reference option to utilize the start reference system.

Examples of use

2D Pocket Milling, identical start and target reference. Pattern: X direction, specifications: Fixed distance, Number of elements: 3, Distance: 50.

Note

The Number of elements option always accounts for the base element. With other words, two transformations and one base element = three elements.

2D Pocket Milling, deviating (1) Start reference and (2) Target reference. Pattern: X direction, specifications: Fixed distance, Number of elements: 3, Distance: 50.
