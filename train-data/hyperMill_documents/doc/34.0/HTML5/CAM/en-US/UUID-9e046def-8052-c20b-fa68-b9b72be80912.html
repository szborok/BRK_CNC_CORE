Input dialog: NC

NC

Enter user-defined text blocks such as control instructions or other similar parameters. This function requires an appropriately configured postprocessor.

As hyperMILL does not perform any syntax or collision checks on the information, the function should only be used if the machine used can interpret the data correctly.

Position: Position of the NC text in the NC program. Enable the required position and enter your text directly in the window on the right. Use Load file to import information from a text file to the selected location. Select Save file to save the NC text from all of the specified positions to a text file. Select Edit external to open an editor in which to enter your NC text.

Note

Free NC text is now also supported by VIRTUAL Machining. Only input in VNC format is permitted. Otherwise, an error message is generated in the VIRTUAL Machining Center. It is then no longer possible to release the NC program.
