Define job

To create a new job, right-click in the CAM browser and select New → Job ..., or use the New function to open the required job dialog directly.

A job (machining step) contains different dialog pages, depending on the strategy in use. The following table provides an overview of the dialog pages.

Overview (only associative job copy)

Summary of parameters where the parameters of the associative job copy differ from those of the original job.

Tool

Define a new tool for machining or select a tool from the tool database.

Contours

Define the contours, top, bottom, start and end points for machining (2D Milling) and specify the drill positions (Drilling).

optimization (only drilling)

Optimize the machining paths. Depends on the drilling strategy selected.

Strategy

Assign the machining strategy.

Parameters

Define the infeed values, top, bottom, clearance distance, clearance plane and infeed mode.

Boundary

Specify the horizontal limit of the machining area (if this is not defined by a contour selection).

Macros

Specify approach, retract, and (where applicable) plunge strategies

5 xes

Define strategies and parameters for 5-axis machining.

Setup

Define a model for the collision check. Define a stockmodel for roughing strategies. Specify parameters for tool checks and other NC parameters.

Feature

Assign or define a new feature.

Transformation

Define transformation of toolpaths for symmetrical components.

General

Enter job name and comments. Job-specific settings for displaying the tool during the computer run or the simulation.

NC

Define NC text (control instructions to the machine and so on) plus free machine parameters.

Create a new job:

  1. Use the New function on the browser's shortcut menu.

  2. Select the job you require and specify the required settings on all the dialog pages.

  		3.

    Save your settings and close the job definition.

Users who are not yet familiar with the functions of the various cycles can proceed as follows to create a new job:

  1. Use the New... → Job... function on the browser’s shortcut menu.

  2. Select the cycle you require in the selection window.

Note

The lower part of the window provides a short description with information on how the respective cycle functions. On the right side of the dialog, an image provides information on the function.

Text in dialogs that cannot be displayed in full due to its length is displayed with three points at the end. A tool tip appears that contains the complete text.

Transfer job parameters

You can transfer the parameters of a defined job to another job. To do this, use the Copy and Paste functions on the shortcut menu. Parameters that cannot logically be transferred are simply ignored.
