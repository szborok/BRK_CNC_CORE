Setup

Make basic settings that are valid for all subsequent actions and definitions in hyperMILL .

Model

Specify the model (milling area) required for the current job. For further information, see section Collision check preparations.

Additional surfaces

Additional surfaces: Temporary safety surfaces to avoid unnecessary rapid travel movements.

Stock model

Specify the stock model required for the current job.

Check tool

If the Compensated center path option is enabled, the offset contour is used for the collision check.

Tool check setup

Options

You can enable the Check spindle option separately.

Clearance

Clearance value added for each tool component (retained minimum distance to model). The following tool components are checked: 1. Spindle, 2. Holder, 3. Extension, 4. Thick shank (see image below). The tool tip is checked without clearance.

When using machining cycles that work with stock models, the defined clearance distance may deviate and will be increased if necessary.

Stop/Clip tolerance

This tolerance value specifies the retract (and approach) point for collision-free movement. (5) Clearance, (6) stop/clip tolerance.

Length calculation increment

Accuracy when extending or reducing the tool (see the Calculate length option).

NC parameters

Check tolerance: Permissible violation of the model geometry. Factor of the tool radius.

Machining tolerance: Enter the required machining tolerance. The value defines the accuracy with which the calculation for the generation of the toolpaths is carried out.

Stop before execution: A stop marker in the toolpath causes the tool to stop.

In addition to activating this function, go to the Machine properties → Program halt dialog and enter the command to stop the tool (by means of M function or command chain). Open the Machine properties dialog via Job list → Postprocessor → Machine → Machine administration → Edit.

For the new PP technology (VIRTUAL Machining), open the Virtual Machine configuration dialog under Machine properties → General → Virtual Machine directory by clicking the icon and configure the stop of the tool in the NC → General → Output area with the available options.
