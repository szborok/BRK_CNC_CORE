Parameters

Infeed / machining area

Lateral infeed / Axial infeed: Infeed in lateral (1) or axial direction (3) for each milling path. If there are several infeeds, the paths are trimmed (optimized) against each other.

Additional axial distance: Additional toolpath distance in the axial direction (5).

Total lateral infeed / Total axial infeed: Total infeed in lateral (2) or axial direction (4) over all milling paths. Determines the lateral or axial machining area.

If the total lateral infeed / total axial infeed is not an integer multiple of the infeed, then the final infeed will be smaller. This allows a reduced milling tool load for the final infeed and an improved surface quality (Roughing with finishing pass).

Trim axial infeed: By trimming the toolpaths in the axial direction, ‘empty paths’ are avoided and machining is faster due to axial retraction.

Example: (A) Function is not enabled, (B) Function is enabled.

Strategy → Mode = Curve on surfaces, (1) Geometry → Side surfaces, (2) Profiles, (3) Additional surfaces → Bottom surfaces

Allowance

Allowance (1) / Additional side allowance (2): Remaining material to be removed in subsequent fine machining. Distance that the cutter must keep from the model geometry. The path of the section curve (3) generated in base surfaces mode will be automatically adjusted in interior corners.

Clearance mode

Specify the type of retract movement.

Planar (1), Radial (2).

Retract mode

The retract mode defines the Z level where the system executes horizontal infeed movements. The mode set is valid for all machining directions (incl. approach, retract and return movement macros) and for the profiles machined with them. Exception: Machining in zigzag mode without return macro.

Clearance distance (1): all retract and infeed movements are executed via the clearance distance. Starting and end positions of an infeed movement in rapid are displaced in Z direction in order to guarantee a collision-free linear infeed movement. The clearance distance is added to these positions.

Clearance plane (2): all retract and infeed movements are executed via the clearance plane.

Safety

Clearance plane and clearance distance apply in the direction of the Z axis of the current frame.

Clearance plane (1): Plane for rapid tool movements. Specification in absolute dimensions Define the clearance plane by right-clicking on the icon. Select a point and confirm the selection.

Warning

Traversing movements on the clearance plane are not checked with regard to collisions. Therefore, this plane must be placed at a sufficient distance above the surface of the workpiece.

Clearance distance (2): Distance to the current toolpath to be milled. Above the clearance distance, infeed takes place as rapid in the Z direction (A); below the clearance distance infeed takes place at the Z feedrate (B).
