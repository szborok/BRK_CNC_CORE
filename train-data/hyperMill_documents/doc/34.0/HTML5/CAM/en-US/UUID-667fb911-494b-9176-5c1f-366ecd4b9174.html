Contours

Select contour

Point: Use Select first hole / Select second hole to select the Point position of the first and second hole that are used for probing. The point positions of the first (1) and second (2) point as well as the direction of the contour defined in this way are displayed as graphical feedback (3). Select Reverse to reverse the direction of the contour and the machining direction (4).

Auto direction: The touch probe always moves in the ‘+’ direction, regardless of the first and second selected point position. (1) = first selected point position, (2) = second selected point position.

First point/second point

The coordinates of the first and second selected point are displayed under Point position.

Measuring depth

Measuring depth: The depth of the contact point of the touch probe and workpiece. It corresponds to the coordinate of the ball center in the axis of the touch probe (A) and is defined Relative to first point (1), Relative to second point (2) or as an absolute value relative to the job frame with Absolute (jobframe) (3).
