Contours

Define machining contours, the vertical machining area and cycle-specific contour parameters.

Contour selection

The number of selected contours is displayed.

Contour attributes

Top: (1) Start of machining in Z direction, in relation to the current frame.

Bottom: (2) Final dimension of machining in Z direction, in relation to the current frame.

To define the top and bottom, select the machining contour in the contour list. Different vertical machining areas are possible for each machining contour. Make multiple selections to define the same top and bottom definitions for different contours.

Define the top and bottom either as absolute values or as thicknesses. The bottom can also be defined relative to the top.

Absolute (jobframe) (A): The values that are set are relative to the current frame. Use the absolute dimensions when a drawing has no defined Z coordinates (2D drawing). In these cases, define the top (1) as absolute and the bottom as relative or as absolute depending on the dimensions in the drawing (3). Selected contour (2)

Relative to top (B): Relative values are relative to the position of the selected CAD geometry. The values for the top (1) can be defined as absolute (job frame) or as the top or bottom of the contour. The bottom (3) should be defined as a value relative to the top. Use for drawings that have Z information (3D contours, solids, defined thicknesses). (2) = Top of the geometry (Z = 0).

Top of contour / Bottom of contour (C): Thicknesses are relative to the position of the selected CAD geometry. (1) Top, (2) = Bottom. Use for drawings that have Z information (3D contours, solids, defined thicknesses).

Specify the Preferred start point, End point and Plunge point parameters. Enable the checkbox for the desired option and select a point.

Preferred start point (1): Freely select for each contour.

End point (2): Set an end point if the contour is to be machined only partially.

Plunge point: The plunge point (1) applies for the infeed to the first workplane. The cutter moves directly from the plunge point to the starting point or to the approach macro position. The plunge point is not collision checked.

Additional allowance: You can define a horizontal (X, Y) and vertical (Z) allowance for each individual contour.

Note

The values for the Additional allowance for contours are taken into account for the path generation but not for collision checks for rapid link movements. The Allowance values defined on the Parameters dialog page are used for this.

Reverse: In the default setting the machining direction follows the direction of the selected contour. This can be seen in the graphic preview. If the generated toolpaths do not result in the intended machining direction, select the relevant contours and select the Reverse option.

Overlap: An overlap is only permissible for closed contours. After milling the contour, the tool moves past the user-defined start point (1) to the set end point (2).

Optimization

Soft contours: If several contours are selected, these are sorted in such a way that the linking fast travel movements (rapids) are as short as possible (A). If this option is not enabled, the contours are linked with each other in the order in which they are selected (without sorting) (A).
