Tools

Supported tool types: Ball mill, end mill, bullnose end mill, lollipop, indexable cutter with round inserts, indexable cutter with high feed inserts, T-Slot Cutter, Woodruff Cutter.

Exceptions: Profile Finishing in Scallop height mode only supports ball mills and slope-dependent machining only supports bullnose end mills and ball mills. With tapered tools, collision avoidance is possible by taking into account the conical shape of the tool.

Use free tip geometry for calculation: Enable to use the freely defined tool tip for toolpath calculation and simulation. Enables the use of tools with a freely defined geometry, such as high feed milling cutters for pre-finishing.

Note

The Use free tip geometry for calculation function cannot be used in combination with the Strategy → Machining mode → Slope mode, Strategy → XY optimization → Slope mode, Boundary → Smooth overlap at boundary and Settings → 3D path compensation options.
