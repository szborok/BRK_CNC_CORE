Contours

Define the drilling strategy, the contour selection for the drill holes and the vertical machining area (top and depth) of the holes.

Drilling mode

2D Drilling: The direction of the drill holes is always aligned with the Z axis of the defined frame system. When using features, the frame system is part of the feature definition. All drill holes can be collision checked using the Milling area.

2D multi index drilling: the direction of any number of drillings can be aligned with the different frames/planes. For infeed movements in XY direction, a relative clearance (automatically calculated for each frame) can be defined. All drill holes can be collision checked using the Milling area.

The following applies to multi-axis indexing: The frame specifies the machining direction. The linking movements between the frames are controlled by the postprocessor.

Contour selection

Drill positions are selected either directly on the Contours input page or by allocating a previously defined feature on the Feature input page.

For information on linking features and the machining cycle, see section From job to feature.

The following CAD geometries are recognized as drill holes:

points,

lines, but no line points (for example, end points), no grid points.

For information on the general procedure for selecting contours, see section Selecting elements in the CAD system.

You can select all the drill holes for a drilling job the are produced with the same tools and have the same collision parameters (clearance distance, clearance plane).

2D Drilling

Contour selection

Lines: In the case of contour selection using lines: The selected contour (line) automatically defines the Top and Depth of the drill hole. You can also define absolute values for these two parameters (Top/Depth) (Parameter dialog page, Absolute top/depth). This option is available for all 2D drilling cycles except for centering.

The direction of the drill hole is determined by the Z-axis of the defined frame (-Z).

Points: In the case of contour selection using points: The following methods are available for defining the Top and Depth: Absolute, Top of contour, Bottom of contour, Relative to the top (depth only).

Absolute (jobframe) (A): The definitions for Top (1) and Depth (2) are absolute values relative to the zero point of the defined frame (3). They are independent of the actual position of the selected geometries.

Top / Depth (B): The dimension entered relates to the top edge and/or bottom edge of the selected contour (4).

Relative to top (C): The dimension entered for the depth relates to the defined top of the drill hole (5).

Note

The direction of the drill hole is determined by the Z axis of the defined frame (-Z) in all procedures.

Major diameter / Minor diameter: Define the required diameter of the drill hole for internal threads (major diameter) or external threads (minor diameter).

Additional distance: Clearance distance for an individual drill hole (as opposed to the clearance distance parameter on the Parameters dialog page) when collision check is disabled.

2D multi index drilling

Lines: The selected contour (line) automatically defines the Top and Depth of the drill hole.

Points and surfaces: Set the drill position with the selected points and surfaces and set the direction using the frame (Tool dialog page). Enter depth and diameter in the respective field.
