MB Edge Milling

Strategy

Milling reference

Main blade / Splitter blade: Specify the blade whose edge you want to mill. The selection of the splitter will not be available, if it has not been defined on the geometry model.

Edge

Leading edge / Trailing edge: Specify the edge you want to machine.

Infeed strategy

Zigzag: Machining with alternating orientation.

Oneway: Edge milling starts in the center above the edge surface. Material is always removed using climb milling.

To ensure climb milling can be performed, the material is removed as follows:

  • In the area left of center, material is removed (from the hub to the shroud) with a drawing cut.

  • In the area right of center, material is removed (from the shroud to the hub) with a plunging cut.

The lift off paths are performed with feedrates that can each be defined separately using a given clearance distance parallel to the cutting toolpath.
