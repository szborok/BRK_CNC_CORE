Plane Recognition

User: Plane Recognition is performed based on the defined surfaces. Select the surfaces to be used for the recognition.

Automatic: Plane Recognition is performed automatically based on the entire CAD model.

Filter

Several filter functions are available.

Use frame: When this option is enabled, only planes that have the same orientation as the selected frame are recognized.

Top / Bottom: Specifies the range (Z direction) that is relevant to the Plane Recognition.

Ignore circular planes: Enable this option when circular planes (of holes) are to be excluded from recognition.

Start: Start the recognition process.

Result

Recognized planes are shown in a list display. Use the Reverse function on the shortcut menu for the recognized planes to reverse the direction of the surface normals (machining direction) of the recognized planes. Use Reverse all to reverse all the surface normals. Select Delete to delete the feature for a recognized plane.

Top / Bottom: Define the height of the top and bottom for recognized planes. The values are transferred as parameters to the Feature dialog.

Apply: Create a feature from recognized planes. All recognized planes are transferred as features to a feature list.
