OMX data

OMX is the internal hyperMILL geometry data format. You can transfer geometry data such as boundaries or drilling positions from one job list to another using the Export OMX and Import OMX additional functions.

Import OMX

Load omx-format geometry data into a model.

Start via the CAM → Utilities → Import OMX menu.

File: Specify the source file.

Color/Layer: Define color and layer.

Export OMX

Save selected geometry data in a separate *.omx file.

Start via the CAM → Utilities → Export OMX menu.

If you want to use this function to export a freely defined holder/spindle as a contour to the OPEN MIND tool database, enable the Data to clipboard option. For further information on importing a freely defined holder or freely defined spindle, see section Create geometry with the geometry editor.

Contours: Select contours that you want to save in the *.omx file.

Approx. tolerance: Approximated tolerance. Default value for converting 2D contours with elliptic arcs or splines into a set of connected lines (for profile curves or boundaries).

File: Enter the path and name of the *.omxfile you want to create.

Write data to clipboard: This function allows you to transfer geometry data directly to hyperVIEW.
