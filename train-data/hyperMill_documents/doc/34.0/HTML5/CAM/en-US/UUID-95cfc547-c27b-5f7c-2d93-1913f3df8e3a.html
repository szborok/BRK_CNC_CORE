Create geometry with the geometry editor

Use the geometry editor to create freely definable geometries for tool components.

To do this, click the Geometry line in the data view of the component and then click the icon.

Specify a name for the geometry and define whether you want the geometry to be active and whether a coupling exists.

To edit a geometry, click the icon and open the geometry editor.

Click the icon to delete a geometry.

Note

Only one geometry can be active for each component. You cannot delete an active geometry.

The geometry editor comprises an input area for defining line segments and a graphical preview (2D). To add a geometry, right-click in the input area and specify the Height for the Origin and the Diameter and Height for the line segment. Add a line segment using the Paste shortcut menu.

Note

As the component geometries are always rotationally symmetrical models, you can uniquely define a line segment by simply defining a Diameter and Height. The radius is displayed in the graphical preview.

Warning

Component geometries must not contain any undercuts in the X direction.

Select the checkbox in the Report column to copy the line segment to a report. The first and last line segment are always selected by default, so that they are output in a report.

If the check box in the Report column is enabled, its dimensions are also displayed in the geometry editor. The same also applies to the components Spindle, Holder and Extension.

To edit or delete a line segment, right-click in the input area and select Edit or Delete.

Load: Load a contour from a *dxf file.

The following requirements must be met in order to be able to correctly load the dxf file:

  • The contour must be on layer 1,

  • the contour must be closed,

  • both halves must be present for rotating bodies.

  • The axis must be horizontally aligned.

Insert: Insert a contour in the *.omx format and import freely defined tool geometries directly from hyperMILL, for example.

Copy: Copy a contour from the database to the clipboard.

Mirror: Mirror a contour.

Delete: Delete a contour.

Complete your entries by clicking OK. A contour is now displayed in the graphical preview as a 3D bounding geometry.
