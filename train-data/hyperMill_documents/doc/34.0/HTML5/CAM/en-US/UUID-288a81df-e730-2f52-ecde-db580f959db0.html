3.5. Tool management

An external database and a document database are available for managing tools. Tools stored in the external database are called external tools. Tools stored in the document database are called document tools.

Supported tool types

hyperMILL generally only supports the tool types listed in the documentation. Other tool types, in particular, angle tools or angle heads are not supported.

External tool

Standard tools for manufacturing tasks are stored in the external OPEN MIND tool database with all data specific to them, such as tool number, machining type and so on. While defining tools in the CAM programming, you can access the external OPEN MIND tool database at any time to import the appropriate tools into the current document.

The external OPEN MIND tool database is displayed in the hyperMILL settings dialog, Database dialog page.

Transfer NC tool from database to hyperMILL

Select an NC tool from the OPEN MIND tool database:

  		1.

    Go to the Jobs or Tools dialog page of the hyperMILL browser and click the icon to open the Select NC-Tools dialog.

  2. Switch to the NC Tools dialog page and select the required tool. If changes are made to tool parameters before transfer to hyperMILL, click the icon to confirm the changes.

  3. To transfer the tool to hyperMILL, use the Transfer to hyperMILL function on the shortcut menu.

Document tool

If the external database does not contain a tool you need, you can set it up as a new document tool in the current document (model file). Document tools are stored in list form on the Tools dialog page of the hyperMILL browser in the Milling tools, Drilling tools and Turning tools sections.

Right-click the Tools dialog page to show all available tabs of the hyperMILL browser. Enable the desired view options.
