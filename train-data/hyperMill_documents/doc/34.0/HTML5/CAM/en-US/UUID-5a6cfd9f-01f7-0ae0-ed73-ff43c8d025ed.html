5 axes

Contact point setup

The contact point of the tool with the machining surface can either be calculated automatically by hyperMILL or manually defined.

Bounding path parameter / Filling path parameter: Specify the contact point of the tool with the surface to be machined using the Bounding path parameter or the Filling path parameter. Values between 0 and 1 are allowed. The value 1 guides the tool on the upper contact point (1) of the barrel radius (2) on the surface. The value 0 guides the tool on the lower contact point (3) of the barrel radius.

Automatic: The value 1 is used as the contact point for the bounding path (4). The contact point for the filling path (5) is calculated based on the lower contact point of the barrel radius as follows: 0.5 * vertical stepdown (6).

Collision avoidance filling

The tool is guided in the area specified under Min. filling angle (1) and Max. filling angle (2) for collision avoidance.

Collision avoidance bounding

The tool is guided in the area specified under Min. bounding angle (3) and Max. bounding angle (4) for collision avoidance.

Avoidance mode

Automatic indexing: For collision avoidance, the area to be machined is split into sections that can be machined with a fixed tool inclination.

Simultaneous: The tool is moved in 5 axes simultaneously for collision avoidance.

Blending

For indexed machining with a minimum of two orientations, specify how the toolpaths in the limit area of the two orientations overlap.

Auto: The toolpaths are automatically calculated so that complete machining is ensured (1). Overlap: Activate when the overlap degree is to be defined. (2) Overlap = 0, (3) Overlap = 3.
