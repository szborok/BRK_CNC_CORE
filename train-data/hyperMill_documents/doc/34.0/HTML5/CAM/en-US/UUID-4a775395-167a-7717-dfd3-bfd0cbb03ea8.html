Internal simulation

Warning

The internal simulation is based on the calculated toolpaths. It is not an NC code-based simulation.

To start the internal simulation, select the required job list or required job and select the Internal simulation option from the shortcut menu.

Job status: This indicates the job that is currently being simulated.

Frame: Shows the frame of the simulated job.

Tool/Origin: Shows the tool and origin of the simulated job.

Speed: Set the simulation speed using 50 steps.

Tool: Configure the tool display.

Display the tool holder (enabled by default).

Display a freely defined tool geometry (enabled by default).

Generate an image of the tool at the current position. To delete the image, use the shortcut menu.

Start the toolpath viewer. The toolpath viewer displays the complete toolpath in G0/G1 steps. Each entry corresponds to one toolpath step. Z levels and macro movements are displayed separately. This means that you can locate and analyze the individual tool movements very accurately. To address a position of the tool directly, double-click the respective row.

Open the dialog to define the stop conditions.

Stop at job change: If you simulate several jobs of a joblist, the simulation stops at each job change.

Technology: Displays G0 movements and feedrates as well as spindle speed and coolant information.

Axis coordinates: Displays the coordinates of the linear and rotation axes.

Frame: Displays the axis coordinates in relation to the current job frame. If the option is not selected, the display of the axis coordinates will refer to the NCS.

Display toolpaths / Hide toolpaths: This function (right-click the top of the dialog) is used to show or hide the toolpaths during the simulation.
