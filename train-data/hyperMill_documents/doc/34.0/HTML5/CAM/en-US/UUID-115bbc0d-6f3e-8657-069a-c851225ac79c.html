The cycle Edit toolpath

Use the Edit toolpath cycle to delete certain sections in calculated toolpaths using a boundary and link the remaining parts of the toolpath together or split a toolpath on the basis of the tool life in order to exchange the tool or the inserts.

A collision check is available for the following tool types: ball mill, end mill, bullnose end mill. A warning message appears if machining is carried out with a different tool. The job to be edited and the cycle Toolpath Edit represent a jointly managed unit. Changes in the original job are reproducible with the same settings in the Toolpath Edit cycle.
