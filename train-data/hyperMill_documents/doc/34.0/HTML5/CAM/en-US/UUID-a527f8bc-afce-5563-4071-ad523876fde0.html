Macros

Approach and retract macros are used to define the tool approach at the start point and the tool retraction from the end point of the contour. This behavior can be defined manually or calculated automatically by hyperMILL.

Ramp height: The ramp height can be specified for the approach macro. Prevents the tool from plunging vertically into any rest material that may remain on the bottom of the workpiece (see section Macros, Using the ramp as a plunge macro).
