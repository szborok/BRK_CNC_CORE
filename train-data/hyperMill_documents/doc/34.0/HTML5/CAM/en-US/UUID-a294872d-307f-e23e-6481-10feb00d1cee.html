Define fixture

  1. In the hyperMILL browser, go to the Jobs tab and open the required Joblist by double-clicking. On the Models tab, activate the Defined option and select a fixture.

  		2.

    Click the icon to define a new fixture area or edit an existing fixture area.

Alternatively, open the dialog via the New fixture area shortcut menu in the hyperMILL browser on the Models tab.

The Fixture area dialog

The dialog consists of the Definition and Fixture area dialog pages.

Definition dialog page

Mode

Surface selection: Select the surfaces that are to be used as fixtures.

File: Click the Path button to select a 3DF or STL file. If necessary, adjust the allowance.

Current selection

Surfaces: Click one of the icons to select the surfaces to be used as fixtures.

Define filter

Use a filter to define geometric elements.

  		1.

    Click the icon to open the Filter definition dialog, select the Active checkbox and enable the required filter option ( Colors, Layer, Tag).

  2. Click the icons to define a new filter and

  3. Click the icon to select the surfaces that are to be used as the basis for the filter definition or

  4. click the icon in the Layer line to select a layer that will be used for the filter definition. Enable the Use wildcard (\\*) option to search for character strings that appear before or after the placeholder in a layer name. Two placeholders can also be used to search for character strings in between.

  5. Click the icon to update the filter definition.

To delete a filter, click the color, Layer or Tag class line and use the Delete function on the shortcut menu.

No filter function is available for hyperMILL for Autodesk® Inventor®. Only the color filter is available for hyperMILL for SOLIDWORKS.

Tool safe distance: Minimum distance between the tool and the fixture.

Warning

The Tool safe distance defined for the Fixture area should not fall below 0.1 mm (0.004 inch). Values between 0.1 mm (0.004 inch) and 0.5 mm (0.02 inch) may lead to long calculation times if the collision check is on and the stop/clip tolerance is small.

A collision check against the defined fixture area is not supported for jobs with the following tools: Chamfered Cutter, T-Slot Cutter, Woodruff Cutter, Barrel Cutter, Turning Tool, Recessing Tool, Axial Recessing Tool and Threading Tool.

Turning fixture

Turning fixture:

Select the Frame for turning axis from the list of existing frames.

Define a new frame.

Edit the selected frame.

Preselect: Preselect the marked origin or frame. This can then be selected directly from the list in the job definition (see ToolFrame) by clicking the icon. A preselected frame or origin is displayed in bold in the browser.

Calculate: Create a 2D silhouette from the selected 3D body or the selected surfaces for the turning jobs and the simulation.
