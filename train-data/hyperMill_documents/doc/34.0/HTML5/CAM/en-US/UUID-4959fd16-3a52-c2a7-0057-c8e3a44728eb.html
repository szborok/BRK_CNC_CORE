Contours

Contour selection

Contours: Define contours for turning operations. These may correspond to the contour of the turning model.

Contour attributes

Start point / endpoint: define the start and end points for machining, either by selecting a point or by entering coordinates.

Reverse: Invert the machining direction of the selected contour.

Auto-creating a contour

Use Feature → 2D Contour to automatically derive the appropriate contour for a machining operation (roughing, grooving, etc.) from the contour of the turning model.

Procedure:

  1. Create the contour for the turning model as Feature → 2D contour.

  2. Create a turning job for the desired machining operation (such as grooving) and navigate to the Feature dialog page.

  3. Select the Feature 2D Contour created in the first step and double-click to move it to the window on the right.

  4. The part of the contour of the turning model used for machining is automatically set to Active and displayed to provide graphical feedback.

Example

(1) Turning stock contour, (2) Turning model contour

(A) Roughing, (3) Turn contour outside, (4) Turn contour plane

(B) Groove Plunging, (5) Groove contour outside

(C) Face Groove Turning, (6) Groove contour front plane

(D) Parting, (7) Parting back face

Note

The function for the automatic creation of a machining contour using Feature → 2D Contour is available for all turning cycles except 3X simultaneous Roughing and 3X simultaneous Finishing.
