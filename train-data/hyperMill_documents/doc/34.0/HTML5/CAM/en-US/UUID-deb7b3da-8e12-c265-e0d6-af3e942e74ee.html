Contour

The Contour feature contains the following feature types: 2D Contour (1), 3D Contour (2), Slot Contour (3) and Point (4).

Geometric information for the 2D Contour, 3D Contour, T-Slot and Point contour types can be used within the scope of the job definition for 2D, 3D and 5X Machining as well as during Turning and Probing.

Parameters of the 2D Contour, 3D Contour, T-Slot and Point feature types:

Orientation: Enter coordinate values or determine the orientation vector from CAD system.

Type: Select the feature type.

Contours: Specify contours in the CAD system.

For information on selecting contours, see section Selecting elements in the CAD system.

For information on defining attributes for geometric elements, see section The Feature editor.
