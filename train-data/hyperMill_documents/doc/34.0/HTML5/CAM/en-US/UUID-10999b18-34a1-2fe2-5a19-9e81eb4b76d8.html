hyperMILL for SOLIDWORKS

Select point.

The following can be selected as geometric elements: Points, Snap points and Curves (drilling cycles only). Curves can be specified in greater detail using Geometry filters. The following elements can be filtered: circle, arc, circles and arcs, miscellaneous.

Under Diameter, determine the max. value / min. value for the defined filter elements.

In addition, Automatic selection is also available. Use the Select all points or Select all curves options to copy all points/curves of the selected faces and sketches to the selection, according to the filter settings.

When defining the top, bottom, start, end or plunge point, Snap points can be selected as geometric elements. These snap points can be specified in greater detail.

End point: The end point of the marked curve is selected.

center point: The center point of the marked curve is selected.

Circle center point: The circle center point of the marked circle is selected.

Next: The next point of the marked curve/face is selected.

Select contour.

Edges, Sketch segments (including automatically connected segments) as well as Sketch contours can be selected as geometric elements.

Select face.

Faces, Surface body, Solid and Feature can be selected as geometric elements. Automatic selection is also available, which can be used to select all visible surfaces.
