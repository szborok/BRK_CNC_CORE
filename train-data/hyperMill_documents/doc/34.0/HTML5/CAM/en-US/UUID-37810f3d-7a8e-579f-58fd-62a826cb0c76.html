Functions on the shortcut menu: Macros

Update macro from database: Update the selected macro with the parameters from the macro database.

Save changes to database: Save the parameters of the selected macro to the macro database.

View macro database: Display the macro database.

Calculate: Calculate the toolpath(s) for the selected job(s).

Internal simulation...: Simulate the toolpath using the internal simulation.

Set toolpath status: Check whether the toolpath is still current for the geometry.

Toolpaths >

Show / Hide / Autoshow / Delete: Show, hide, automatically display or delete the toolpaths of the selected job.

Check model: Check the consistency of the contours assigned to the job.

Delete: Delete the selected macro.

View >

Expand / Collapse: Hide or show the jobs linked to the selected macro.

Expand all / Collapse all: Show or hide all the jobs in the macro browser.
