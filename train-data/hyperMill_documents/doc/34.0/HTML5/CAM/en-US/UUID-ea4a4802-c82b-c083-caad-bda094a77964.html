The New macro dialog

Macro

Name: The name of the source feature. This is part of the macro name. The name is written to the macro database when the macro is saved. The name is also used for automatic macro searches. The names of feature-based macros contain the prefix F; the names of job-based macros contain the prefix J.

Comment: You can also enter a comment for easier identification of the macro.

Macro group: Each macro has to be assigned to a macro group. Macro groups allow you to combine macros that are typical for a particular machining situation, for example.

Define a new macro group.

Modify a macro group.

Machine group / Material group: Enable if the macro is only to be used for one specific machine or material group.

Define a new machine group/material group.

Modify a machine group/material group.

Macro jobs

The bottom section of the dialog box displays the jobs linked to the selected feature and their main tool parameters. Click OK to save the macro in the database.
