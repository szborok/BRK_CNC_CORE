2.1. hyperMILL and CAD program

hyperMILL is a CAM module that is fully integrated into the CAD program and is used for generating 2D, 3D, 5-axis, milling and turning programs. Individual machining steps (jobs) are created on the basis of machining cycles with the open CAD model in the form of one or more work schedules (joblists).

All functions of the hyperMILL toolkit, as well as all input pages, menus and programs that can be opened with icons can also be accessed using keyboard commands and repeated using <return>.

When you open a document, the hyperMILL browser opens automatically. All CAM functionalities can be accessed via the CAM entry in the menu bar.
