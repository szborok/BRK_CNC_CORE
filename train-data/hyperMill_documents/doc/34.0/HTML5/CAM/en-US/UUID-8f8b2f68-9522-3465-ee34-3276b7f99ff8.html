hyperMILL for Autodesk® Inventor®

Select point.

Points: Select any single points of a model.

Arcs/circles from sketches: Select any arcs/circles from sketches.

Arcs/circles from edges: Select any arcs/circles from edges (of a model).

All arcs/circles from sketches: All arcs/circles of the selected sketch are automatically selected.

All arcs/circles from faces: All arcs/circles of the selected face are automatically selected.

All point entities from sketches: All points of a sketch are automatically selected.

Filters: All geometric elements to be selected can be specified in greater detail using filters.

Geometry: Mark the geometric elements you wish to select.

Diameter: Define a diameter for the size of the elements to be selected.

None: No diameter filter is applied.

Value: Use operators (=, <=, >=) to define a fixed value for the diameter.

Area: All elements are selected within the value range.

Confirm selection: Confirm the selection by adding and clicking on OK.

If you want to reject the selection made, click on Cancel or remove the tick in the checkbox. To delete a previous selection, click on Remove.

Select contour.

Auto chained sketch curve: Automatic selection of linked part curves of a sketch.

Manual chained sketch curve: Selection of one or more part curves of a sketch (which must represent a linked curve).

Auto chained edges: Selection of one or more edges specified in the model.

Manual chained edges: Selection of one or more edges specified in the model (which must represent a linked curve).

Confirm selection: Confirm the selection by adding and clicking on OK.

If you want to reject the selection made, click on Cancel or remove the tick in the checkbox. To delete a previous selection, click on Remove.

Select face.

Individual faces: Select any faces of a model.

Select faces belonging to a feature: Automatic selection of all features for the selected face within the scope of the respective features belonging to the faces.

You can select features directly in the browser of the CAD program. In doing so, multiple selections are possible.

Confirm selection: Confirm the selection by adding and clicking on OK.

If you want to reject the selection made, click on Cancel or remove the tick in the checkbox. To delete a previous selection, click on Remove.
