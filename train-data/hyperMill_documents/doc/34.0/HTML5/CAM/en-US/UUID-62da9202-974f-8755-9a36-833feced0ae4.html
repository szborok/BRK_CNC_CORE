Job list

A job list is automatically created when you call up a machining cycle for a job definition.

In order to create another job list, in the hyperMILL browser, switch to the Jobs tab, right-click New → Select job list.

Job list settings can be defined separately for each job list. You must make the settings before you calculate the toolpaths.

To edit, double-click the joblist or select the Edit function from the shortcut menu.

Export/import job list

Save a joblist with all its parameters and settings in *.jst format and transfer to other CAD models.

Advantages: Job archiving is independent of the drawings, fast programming of frequently occurring procedures, transfer of jobs to workpieces with the same drawing parts when machining sequence is the same.

To export a job list:

  1. Select job list and then select the Export function from the shortcut menu.

  2. Specify the required export options as well as the name and path (storage location) of the file.

Include tools / Include geometry / Include frames: Tools, geometries (boundaries, profiles) and frames are also exported along with the machining parameters.

Include milling area references / Include stock references / Include feature references: Milling areas, stock models and features are also exported.

To import a job list:

  1. Use the New → Load from file function on the browser's shortcut menu.

  2. Select the *.jst file and enable the required import options.

The Export and Import functions are also available for the individual jobs in a job list.

Direct job transfer

Direct transfer of job lists or selected jobs to other CAD models using the Copy and Paste functions on the shortcut menu. When pasting into a job list, select the ( Include tools / Include geometry) import options: All tools assigned to the job or all defined geometries (boundaries, profiles) of the job are imported with the machining parameters.
