Strategy

Probing mode

Probing: Measure the angle of an edge in relation to the X axis of the job frame (1).

Set origin: Specify the origin that is to be set or adjusted according to the machine settings (2). For this purpose, select the required ID.

Note

The differing workpiece position in the XY plane (angle) is stored in the controller (origin table).

If you use the Set origin function in combination with an old postprocessor, the system will display the following error message:

Your postprocessor does not support the Set origin probing mode.

Please contact your OPEN MIND partner. In this case, you need to update your postprocessor in order to retain the ability to use the function.
