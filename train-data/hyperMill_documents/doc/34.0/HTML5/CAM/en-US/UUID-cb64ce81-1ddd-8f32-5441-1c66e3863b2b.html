5 axes

Inclination strategy

Define the mode for the tool inclination.

Use the Iso option (1) to set the tool orientation to the Iso lines of the surface (A).

Select Normal to curve mode (2) if the tool should be aligned along the normals of the toolpath (B).

Radial Z (3) is the preferred selection for machining rotationally symmetrical models. The tool is aligned to the Z axis of the frame (C).

Tilt parameters

Control the inclination depending on the tool type.

For barrel tools and lollipops, use Tilt parameters â†’ Contact parameters to specify the contact point of the tool with the surface to be machined. The permissible value range is from 0 to 1. A setting of 1 will guide the tool along the surface based on the upper contact point (1) of the barrel radius (2). A setting of 0 will guide the tool based on the lower contact point (3) of the barrel radius.

Activate the Dynamic contact point option in order to dynamically adjust the tool inclination to the surface to be machined.

The contact point of the tool with the machining surface can vary between the Contact parameter (1) and the Contact end parameter (3). The permissible value range is between 0 and 1 for both parameters. A setting of 1 will guide the tool along the surface based on the upper contact point (1), a setting of 0 along the lower contact point (3).

The current contact point is calculated automatically between the two values and enables optimum utilization of the tool insert shape. Hub fillet machining is also possible.

Use the Lead angle to specify the tool inclination (4) in the infeed direction (5).

For ball mills, end mills and bullnose end mills, use the Tilt angle (1) to specify the tool inclination with reference to the surface to be machined. Use the Lead angle (2) to specify the tool inclination in the infeed direction (3).

Use the Contact offset (4) to move the (contact point) of the tool in the direction of the tool axis. The advantage of this is that surfaces can be efficiently finished tangentially, even when using a standard tool (end mill).

Collision avoidance

Enable the Collision avoidance option to automatically adjust the inclination of the tool in the case of a possible collision. If machining without a collision is not possible, the toolpath is only calculated and executed up to the collision event.

Use the Adjust lead angle to change the lead angle of the tool to avoid collisions, and use Avoid angle (1) to specify by how many degrees the tool is permitted to adjust the lead angle. (2) Collision avoidance not enabled.

Use Adjust tilt angle (contact point) to adjust the tilt angle dependent on the surface slope using the contact point.
