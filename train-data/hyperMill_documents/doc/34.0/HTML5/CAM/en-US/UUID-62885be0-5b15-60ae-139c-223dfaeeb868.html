Strategy

Strategy

The machining strategy is determined by the parting tool definition. If the parting insert position on the tool is left, Front face is automatically enabled as a strategy (1); if the position is right, Back face is activated as a strategy (2). If the parting insert position on the tool is neutral, the strategy can be chosen freely (3). In this case, the parameter front angle of the insert is not available.

Options

With chamfer:enable when a chamfer is to be created at the trailing edge of the model and define the desired chamfer dimension (1). The chamfer angle is fixed at 45Â°.

Type

Defines how the tool should proceed after parting off.

Parting off and rapid (1): A retraction movement is performed at feedrate in Z direction to clearance distance, then rapid retract to the clearance radius.

Parting off and rapid (2): a retraction movement is performed at feedrate in Z direction to clearance distance, then rapid retract at feedrate up to the clearance radius.

Machining mode

Normal (1): Parting off is performed in one pass without interruption over the entire contour length in feedrate.

Retract clearance distance (2): Parting off is performed in multiple passes, each with the defined Plunge depth (4). Between each parting, there is a retraction to the clearance distance

Retract to clearance radius (3): parting off is done in multiple passes, each with the defined plunge depth (4). Between each run, a retracting movement is performed at feedrate in Z direction to clearance distance, then rapid retract to the clearance radius.

Reduce speed/feedrate

Reduction: Enable when a reduction in the speed and feedrate is required from a certain workpiece diameter and define the diameter with From diameter.

Use part catcher: Enable when a part catcher should be swung in. If the reduction option is enabled as well, the part catcher is swung in on reaching the reduction diameter. If the reduction option is not enabled, the part catcher is swung in for parting off after the positioning.

The pivoting back of the part catcher takes place between parting off and retraction movement in the Z direction. With the dwell time parameter ( parameter tab) specify the time between the parting off and the retraction movement.
