Functions on the shortcut menu: Coordinate systems

Preselect: Preselect the marked origin or frame. This can then be selected directly from the list in the job definition (see ToolFrame) by clicking the icon. A preselected frame or origin is displayed in bold in the browser.

Remove preselection: Cancel the preselection for the selected origin or frame.

New frame: Create a new frame.

New origin: Create a new origin.

New origin from frame: Create a new origin based on the current frame.

Edit...: Edit the selected coordinate system.

Copy, Cut, Paste, Delete: Entries are managed in the standard Windows manner.

Set workplane: Set the workplane to the current frame/origin.

Set turn workplane: Set the workplane for turning jobs to the current frame/origin.

Generate graphical entities: Generate non-associative CAD elements for the display of the coordinate system (for example, for 2D drafting). If the frame is changed, the elements must be deleted manually and re-generated.

Messages: Display messages (notices, errors, warnings) for the selected entry.

Usage: Display jobs in which the selected element is used.

hyperMILL setup: Open the dialog for inputting the basic settings of hyperMILL.

Feedbacks / No feedbacks: Turn feedback on/off for the currently selected coordinate system.
