Type of cut parameter and associated calculation rules

Type of cut parameters are tool technology parameters that are required in a hyperMILL job or are necessary to calculate the technology parameters. Only the type of cut parameters that are actually required are available. The following table contains all the type of cut parameters that can be defined in hyperMILL and their calculation rule(s).
