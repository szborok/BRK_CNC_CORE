Tool

For information on selecting a tool and defining tool parameters, see section Components for NC tool.

Correction list: Inputs for correction list are only possible for the selected reference side.

Reference side: The insert side that is used as a reference to calculate the toolpath can either be left or right. The insert side is only freely selectable when the position of the insert on the tool holder is neutral.
