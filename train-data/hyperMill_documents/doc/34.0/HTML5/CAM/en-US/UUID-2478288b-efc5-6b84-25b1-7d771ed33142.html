Functions on the shortcut menu: Models (stockmodels)

New stock: Generate a new stock model.

New compound stock: Generate a new compound stock model.

Mirror stock: Generate a new stockmodel by mirroring the selected stockmodel.

New stock through transformation: Generate a new stockmodel by transforming the selected stockmodel.

Modify stock: Split or clean the selected stockmodel.

Save stock file as: Save the selected stock model in *.vis format.

Rest material display: Show the rest material of the selected stockmodel.

Show / Hide: Show or hide the selected stock model.

Autoshow: The stock model is automatically displayed when selecting.

Hide all: All stock models visible in the graphics area are hidden.

Edit: Open the Stockmodel dialog and edit the selected stockmodel.

Copy: Copy the selected stock model to the clipboard.

Paste: Paste the stock model from the clipboard and create a new entry in the stock browser.

Delete: Delete the selected stock model.

Calculate: Calculate the selected stock model.

Close stockmodel:

Remove unused stock files: All stock files not referenced in hyperMILL are deleted from the default directory. This also deletes *.stl files that were created from existing *.vis files. As *.stl files are required for display in CAD, it may take longer to display the stock files afterwards.

Remove generated stock files: All stock files created by hyperMILL during calculation are deleted. These stock files can be recreated at any time with the existing toolpath files using the Calculate stock function.

Tip

Both functions for deleting stock files are used to clean up when a project is finished. The files are not absolutely necessary for archiving, as they are either not used or can be created again at any time. Both functions significantly reduce the size of projects and can be used to send project information (for example, to OPEN MIND support).

Messages: Display messages (notices, errors, warnings) for the selected entry.

Usage ...: Show the usage of the selected stock model (job list/job lists).

hyperMILL settings: Open the hyperMILL Settings dialog.

Feedbacks / No feedbacks: Turn the feedback display on/off for the selected stockmodel.

Functions on the shortcut menu of the icons

In the shortcut menu (right mouse button) of the icon, select the Show/Hide function to show or hide the selected stock model in the graphics window.

In the shortcut menu (right mouse button) of the icon, select the Autoshow function to automatically display the stock model when the corresponding job is selected in the hyperMILL browser.

When the Autoshow function is enabled, the entry in the shortcut menu is marked with an icon.

In the shortcut menu (right mouse button) of the icon, select the All off function to hide all stock models in the graphics window.

Show with hyperVIEW: Display the selected stockmodel in hyperVIEW.

Show with hyperVIEW only: Display the selected stockmodel only in hyperVIEW.
