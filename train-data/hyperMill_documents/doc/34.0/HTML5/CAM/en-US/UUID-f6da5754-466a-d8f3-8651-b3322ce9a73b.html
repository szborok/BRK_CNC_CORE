Parameters

Infeed

Infeed length (1): Length of the toolpath at the feedrate to the surface contact of the ball of the probing tool.

Additional infeed length : Added to the infeed length in the component direction to ensure that the touch probe contacts the component. This should be used in particular in cases with a negative offset (electrode machining).

Danger of collision

When entering the value for the Additional infeed length for your component, please watch out for possible interfering geometries, as the parameter cannot currently be simulated and there is therefore a risk of collision.

Tolerances

Symmetric: Lower allowance and upper allowance are the same. Use Tolerance -/+ to define the minimum/maximum permissible deviation of the actual dimension to the target dimension.

Two sides: Define Lower allowance and Upper allowance. The values may be different.

NC

Create log file: Creates a measuring log.

Select Default if the machine is to create a measuring log with the structure specified by OPEN MIND. Use the Extra function to generate a measuring log that can be customized.

Note

The Best Fit and Best Fit verification functions are only available with an appropriate license.

The Best Fit function (only 3D Point Probing) generates a measuring log that can be read in with the VIRTUAL Machining Center. The measuring log contains the position values on the component that is not yet aligned.

In VIRTUAL Machining Center, the measuring log is read in and the actual component position is calculated.

The real position determined in VIRTUAL Machining Center is finally verified using the Best Fit verification function so that safe machining can take place on the machine.

For further information on the Best Fit function, also see the VIRTUAL Machining Center product documentation.

Stop program if out of tolerance: Stop measuring if a measured value is outside of the specified tolerance range.

Note

Some controllers do not support automatic log generation/Message mode. Please contact your OPEN MIND partner if you have any questions.

Retract mode

Defines the Z level where retract movements are executed.

Safety distance/ clearance plane: All retract movements are executed via the clearance distance/clearance plane. .

Clearance

Security plane: Specify the level of the security plane.

Axial clearance/lateral clearance: Minimum distance in an axial or lateral direction from the surface of the model.

Reposition feedrate: Feedrate of the tool during the repositioning between two measurements.
