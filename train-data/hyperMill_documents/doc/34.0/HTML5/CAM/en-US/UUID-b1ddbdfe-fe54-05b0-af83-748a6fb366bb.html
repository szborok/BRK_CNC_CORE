Link macro

Only for Parallel to hub strategy. The link macro connects the toolpaths of both flanks â€“ which are trimmed against the shroud. The Height (1) and Radius (2) define how the link movement is performed over the shroud.

Split paths: During 4X Machining, continuous machining around the blade can be obtained by automatically splitting the paths at both edges and connecting these paths again with an additional macro. The advantage is that you can use tools with cone angles on 4-axis machines that would have damaged the edge(s) using uninterrupted circumferential machining. This allows you to increase the process parameters and thus reduce the machining time.
