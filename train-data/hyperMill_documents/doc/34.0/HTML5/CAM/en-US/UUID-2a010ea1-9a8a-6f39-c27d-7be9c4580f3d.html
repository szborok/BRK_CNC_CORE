Inclination strategy

Fixed: Fixed tool inclination in the entire machining area.

Automatic: All milling movements in G1 are, if possible, carried out automatically with a fixed orientation in order to achieve maximum speed and stability. When using rotary axes, a constant simultaneous movement of the rotary axes is possible thanks to the pre-analysis and evaluation of the entire toolpath segment.

Preferred tilt angle: Define a starting inclination which – if possible – is maintained for the entire toolpath. If the preferred tilt angle is not used, a vertical alignment is preferred.

Using Holder / Tip, specify the area of the tool that is to be aligned to the Z axis of the machining frame.

Rigid guiding: The tool is rigidly aligned to the Z axis of the frame. If the option is disabled, the orientation of this axis may be changed due to automatic smoothing and/or collision avoidance.

Tilt angle control: The A/B axis requires tilt angle definition.

Additional lead angle: In addition to tilt angle control, a lead angle can be specified for the tool.

Offset on XY plane: In the case of end mills and bullnose end mills, you can offset the tool axis to the Z axis as an alternative to using the lead angle.

Collision avoidance: If required, hyperMILL may change the tilt angle control at a later point if collision avoidance for the A/B axis has been enabled.

Manual inclination curve

The movement of both rotary axes can largely be guided independently of the surface formation of the machining area, using the user-defined inclination curve.

Compared to the offset strategy, a more harmonious movement sequence around the Z axis of the selected frame can be achieved. The inclination curve should be only roughly oriented to the profile of the surfaces, have a smooth profile without abrupt changes in direction, enable continuous movements of the rotary axes.

Orientation of 4th axis (C) is calculated as shortest distance towards curve from Z-view of current frame.

3D-information of the curve does not influence tool orientation around 4th axis (C)!

(1) C axis, (2) Toolpath (length), (A) Offset curve, (B) Manual curve

Setup: C axis

Holder / Tip: Defines whether the curve guides the tool tip or tool holder.

Rigid guiding: If rigid guiding for the curve is not activated, the orientation can be modified by means of smoothing and/or collision avoidance around the C axis by hyperMILL®. If strict orientation to manual curve or frame Z-axis is required, this switch must not be disabled!

Setup A/B axis

A distinction is made between three modes: For all 3 options, the tool orientation can be modified by hyperMILL® subsequently for the A/B axis if collision avoidance is activated.

Consider curve: If Consider curve is not activated, the 3D information of the curve is not used. The height of the curve has no influence on the angle of the A/B axis, because the tilt angle of the milling tool will stay unchanged. If Consider curve is enabled, two different options are available.

Through: Creates a tool inclination at which the tool axis intersects the curve at every point.

Toward: If the tool axis arrives at the curve, the tool is positioned to it, possibly up to a perpendicular position. Depending on the machining conditions, this can allow the tilt angle and the movement guidance of the second rotary axis to be influenced using the Z height of the manual inclination curve.

Toward (in connection with Tool holder): The manual inclination curve (1) is used for orienting the tool shank/tool holder. It normally lies in front of the surface to be machined from the tool axis perspective.

Constant tilt angle (1): To attain a constant tilt angle, the manual curve must be defined so that overlapping with the tool axis is avoided during machining. The manual inclination curve should normally be as deep as possible. If necessary, create several jobs with different radial curves.

Deactivate the Consider curve option.

Modifiable tilt angle (2): To avoid collisions between tool shank and model surfaces of the opposite wall, set the manual inclination curve high enough in order to position the tool onto it while the machining depth is increasing.

Smooth tilting of the second rotary axis

For back-and-forth machining of basic surfaces, it is also recommended to set the manual inclination curve higher. Since the positioning of the tool on the manual inclination curve then begins earlier and ends later, it results in a smoother movement guidance of the second rotary axis.

Toward (in connection with Tool tip): The manual inclination curve (1) is used for orienting the tool tip. It normally lies behind the surface to be machined from the tool axis perspective.

The manual inclination curve allows for, in particular, a specific definition of the position direction for convex surfaces.

Constant tilt angle

To attain a constant tilt angle, the manual inclination curve must be defined such that overlapping with the tool axis is avoided during machining. If necessary, create several jobs with different manual curves. Deactivate the Consider curve option.

Modifiable tilt angle

If the tilt angle changes as machining progresses, then the machining direction and the shape and positioning of the radial curve should be selected such that the tool axis has to cut the radial curve.

For example, it is recommended for back-and-forth machining that the radial curve is set lower. Since the positioning of the tool on the radial curve begins earlier and ends later, it results in a smoother movement guidance of the second rotary axis.

Offset curve: The offset curve (1) is used for orienting the tool shank.

With this strategy, the system forms a curve that corresponds to the defined offset value in each machining level for the calculated center path of the tool. Comparing to former version tool paths smoothing of 4th axis (C) as well as automatic collision avoidance is always switched on.

Warning

The surfaces to be machined must not be too flat, as there is then the possibility that no useful offset curves can be formed.

Offset: Using the offset value, you can control how precisely the inclination of the milling tool follows the surface profiles.

Large offset values reduce the rotary motion around the Z axis in small narrow areas. However, they increase the danger of the offset curve breaking down into numerous, separate curves.

If the offset strategy results in a large number rotary motions around the Z axis that are not necessary for milling, the Manual curve option should be used. If it is not possible to avoid a breakdown into individual curves, the machining should be divided up into several jobs with different machining areas and different offsets.

Setup: A/B axis

Tilt angle control: The current frame is the reference coordinate system for the tilt angle control. The frame can be tilted with respect to the NCS for machining undercuts and achieving an optimum flow of the NC data.
