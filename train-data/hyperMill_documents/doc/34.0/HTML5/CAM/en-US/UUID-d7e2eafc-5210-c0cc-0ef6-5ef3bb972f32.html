Free Path Milling

Free Path Milling performs collision-checked machining along a freely specifiable center point path with a multiple axial infeed. It is possible to machine undercuts.

Tool

If the Tip option is selected in the tool panel, then the toolpath is output offset in the axial direction in relation to the contour. The tool is always kept on the guiding contour with a center reference, irrespective of the reference point setting for the path output in the tool panel.
