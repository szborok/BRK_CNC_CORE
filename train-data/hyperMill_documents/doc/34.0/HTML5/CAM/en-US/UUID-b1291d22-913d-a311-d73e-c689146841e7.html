Define stock size and allowances

Stock: Define the size of the stock.

Milling: Model length, model width and model height are displayed. Define the stock length, stock width and stock height. The size of the stock is calculated automatically from the values defined for Offset X+ / Offset X-, Offset Y+ / Offset Y- and Offset Z+/ Offset Z-.

The offset is displayed. (1) Offset in X direction, (2) Offset in Y direction, (3) Offset in Z direction.

MillTurn: Model length and model diameter are displayed. Define the stock length and stock diameter. The size of the stock is calculated automatically from the values defined for Offset Z+ / Offset Z- and Radial offset.

The offset is displayed. (1) Offset in Z direction, (2) Radial offset.

Click the icon to lock a value so that it can no longer be changed.

Click the icon to unlock a value so that it can be changed again.
