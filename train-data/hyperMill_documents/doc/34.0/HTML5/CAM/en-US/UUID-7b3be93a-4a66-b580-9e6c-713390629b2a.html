Functions on the shortcut menu: Jobs (stockmodels)

Save stock file as ...: Save the current stock model in *.vis format.

Autoshow: Automatically display the stock when the associated job is selected.

Messages: Display messages (notices, errors, warnings) for the selected entry.

Usage ...: Show the usage of the selected stock model (job list/job lists).

hyperMILL settings: Open the hyperMILL Settings dialog.

Feedbacks / No feedbacks: Turn the feedback on/off for the selected milling area/stockmodel.

Functions on the shortcut menu of the icons

In the shortcut menu (right mouse button) of the icon, select the Show/Hide function to show or hide the selected stock model in the graphics window.

In the shortcut menu (right mouse button) of the icon, select the Autoshow function to automatically display the stock model when the corresponding job is selected in the hyperMILL browser.

When the Autoshow function is enabled, the entry in the shortcut menu is marked with an icon.

In the shortcut menu (right mouse button) of the icon, select the All off function to hide all stock models in the graphics window.

Show with hyperVIEW: Display the selected stockmodel in hyperVIEW.

Show with hyperVIEW only: Display the selected stockmodel only in hyperVIEW.
