Parameters

Machining mode

Centering (1): Longer G1 movement, longer machining time.

Chamfer existing hole (2): Shorter G1 movement, reduced machining time.

Machining depth

The drilling depth can be defined in several different ways:

Related to depth: Drilling stops when the specified depth is reached.

Related to diameter: Drilling stops when the specified diameter (< tool diameter!) is reached.

Related to hole diameter: Drilling stops when the specified drill hole diameter is reached.

The drill hole diameter parameter is made up of the Diameter parameter (Contours dialog page) and the chamfer width.

Top offset mode

Relative offset: Use the Top offset parameter to set the value for the relative offset for centering.

Absolute top: Use the Absolute top parameter or select points on the model to specify the value to start centering.

With Offset top only, you can limit the offset to the top (start of G1 movement) (1). The Offset centering complete option defines the offset for the top and depth of the centering (G1 movement) (2).
