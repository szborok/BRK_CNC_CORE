Define technology usage

Assign different technology parameters to a cutter material, depending on the machining (Roughing, Finishing). This assignment is made in the data view of the tool definition in the technology parameters area (Material > Cutter material).

To define a technology usage:

  1. Select the database entry in the database browser and then select Technology Usages....

  2. Right-click in the Technology Usages window and select New.

  3. Specify the Usage and, if necessary, enter a comment.
