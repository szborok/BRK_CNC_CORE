Associative job copy

Copy of a job in which all parameters correspond to the parameters of the original job. A job copy is associatively linked with the original job. If a parameter is changed in the original job, then this parameter will also change in the associative job copy. Parameters cannot be edited. Exception: transformations and contours.

To create an associative job copy:

  1. In the hyperMILL browser, select the job in the job list, right-click and

  2. choose the Associative job copy function on the shortcut menu.

To unlink the copied job and original job:

  1. In the hyperMILL browser, select the copied job, right-click and

  2. choose the Unlink associative copy function on the shortcut menu.

All changes to the original job are summarized on the Overview dialog page.

The associative job copy corresponds to the Copy option when using a mirror within a job definition (Transformation dialog page). If you define a mirror on job list level and have enabled the Copy option, a mirrored associative copy of the original job is created from all jobs in the job list with the parameters and settings of the original job.

To deactivate the associative link between the original job and the associative job copy for individual parameters, right-click the relevant parameter and select Unlink.

Changes to this parameter in the original job no longer affect the associative job copy. Conversely, the parameter can now be edited directly in the job copy without affecting the original job. The associativity between both jobs is restored using the Link function.

If you recalculate a job for which an associative job copy exists, you can also recalculate the job copy.
