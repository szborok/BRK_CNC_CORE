Contours

Define the drilling strategy, contour selection for the holes and the vertical machining area (top and depth) of the holes.

Drilling mode

2D Drilling: The direction of the drill holes is always aligned with the Z axis of the defined frame system. When using features, the frame system is part of the feature definition. All drill holes can be collision checked using the Milling area.

2D multi index drilling: the direction of any number of drillings can be aligned with the different frames/planes. For infeed movements in XY direction, a relative clearance (automatically calculated for each frame) can be defined. All drill holes can be collision checked using the Milling area.

The following applies to multi-axis indexing: The frame specifies the machining direction. The linking movements between the frames are controlled by the postprocessor.

You can select all the drill holes for a drilling job that are produced with the same tools and have the same collision parameters (clearance distance, clearance plane).

Contour selection

In Lines mode (A), select a contour (1) to define the top (2) and depth (3) of the sink as well as the position of the drill hole (4). Use the Additional distance parameter to define the depth of the drill hole (5) and specify the diameter of the sink (6) under Diameter.

In Points mode (B), select a point (1) to define the position of the drill hole (1). Under Top and Depth, select a point in each case to define the top (2) and depth (3) of the sink. Use the Additional distance parameter to define the depth of the drill hole (4) and specify the diameter of the sink (5) under Diameter.

Note

The direction of the drill hole is determined by the Z axis of the frame in all procedures.
