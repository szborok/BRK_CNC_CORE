Importing/exporting data

  1. Select the required database entry in the database browser.

  2. Select Import / Export on the shortcut menu.

  3. Then choose Neutral Tool Exchange Format *.xml for file type.

Please note: The Import shortcut menu function is only available for database nodes.

Tip

Importing data via drag-and-drop: You can easily import tool and technology data using drag-and-drop. This is done by dragging the .xml file from the file explorer to the desired database node in the file browser. This function is available for all tool data that was previously exported from the tool database in hyperMILL .xmlformat, as well as for tool manufacturer data that can be saved in hyperMILL .xml format.
