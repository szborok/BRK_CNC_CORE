General

Tolerance cycle setting

Depending on the machining situation, use different tolerance values to generate the NC file. The values influence the machining accuracy on the machine via the NC Generator and so enable an optimal combination of precision and dynamics adapted to the machining situation.

Roughing, Semi finishing or Finishing are already preset depending on the job and can be can be adapted as required. Define the required tolerance values in the NC Generator → NC → General → Accuracy dialog.

Additional options

Relevant for two or more jobs with identical tool in direct machining sequence.

Force tool call: If the function is enabled for all jobs, the tool is called from all jobs at the beginning in the postprocessor run. Safety positioning is carried out before and after each tool call. All relevant technology commands are regenerated after each tool call.

If the function is not activated, the tool is called only at the beginning of the first job. Safety positioning is carried out before and after the tool call. All relevant technology commands are regenerated after the tool call.

Note

The Force tool call function is available with an appropriate license for the VIRTUAL Machining Center. Please contact your OPEN MIND partner.
