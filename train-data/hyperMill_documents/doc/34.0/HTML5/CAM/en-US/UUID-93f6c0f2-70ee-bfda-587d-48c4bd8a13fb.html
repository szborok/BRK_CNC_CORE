Parameters

Define machining parameters.

For information on the machining parameters, see section Parameters:

Infeed

Vertical stepdown (1): choose the stepdown to the next machining plane, from the Z direction of the active frame. This gives the number of machining planes. If a top is situated between two defined machining planes, the system will automatically reduce the vertical stepdown and insert an intermediate step.

Horizontal stepover (2): Stepover in the XY plane as the distance between the center points of two adjacent milling tool paths.

Default = [tool diameter * 0.5].

Horizontal stepover factor (2): stepover in the XY plane is the distance between the center points of two adjacent milling tool paths and is generally given as a factor of the milling tool's diameter

Allowance

Choose Allowance XY (1) and Allowance Z (2). Remaining material that is to be removed in subsequent fine machining. Corresponds to the distance from the final contour that the cutter has to keep

Warning

Negative Z allowance is added to vertical stepdown, might cause collision!

Retract mode

Clearance plane: The tool retracts to the clearance plane after each plane has been machined.

Lead-in keep position: The tool plunges to the next deepest plane at the current position after each machined plane. This reduces machining times.

Lead-in reposition: the tool plunges to the next deepest plane at the current position after each machined plane.

Path compensation

Important

The 2D/3D/5X path compensation is only intended for a fine correction of the real tool. If large corrections are made, especially if the radius is increased, there is a risk of collision.

Compensation finish path: When calculating the finishing path, path compensation (compensated path / compensated center path) is taken into account.

Vertical stepdown finish path / Semi path allowance: The values for the vertical stepdown and the allowance can be defined separately for the finishing path.

Example: Open pocket, (1) Roughing path, (2) Corrected finishing path = center path / compensated center path, (3) Compensated path

Toolpath fillets

The toolpaths in the plane can be smoothed.

Fillet radius: Abrupt changes in direction within the cutting toolpaths are rounded with the specified radius (2).

Radius for horizontal infeed: The horizontal infeed between the toolpaths is rounded with the horizontal infeed radius (1).

Fillet all toolpaths: model contour corners are rounded with the radius defined above, with the result that the machining contour deviates from the model contour at the corners (3).
