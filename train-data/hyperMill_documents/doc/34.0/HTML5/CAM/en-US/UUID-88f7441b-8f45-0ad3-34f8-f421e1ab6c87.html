Functions on the shortcut menu: Job lists/Jobs

New >

Project Assistant: Start the Project Assistant.

Create a new entry (joblist, job, compound job, linking job, NC-event).

Electrode milling: Open the Electrode milling dialog to generate the data for NC programming (job list, milling area, stock, clamp, NC system and electrode feature).

Job list: Create a new job list.

Job: Create a new job

Tip

Use the function Job → Select new operation to open an overview which, in addition to a graphical representation, also contains a brief description of the function and possible uses of the respective cycle. This is helpful for users who are not yet familiar with the software.

Compound job: Create a new compound job.

Linking job: Create a new linking job.

Linking job turning: Create a new linking job turning.

NC event: Create a new NC event.

Probing, Turning, Drilling,

2D Milling, 3D Milling, 3D Advanced Milling,

5X Cavity Milling, 5X Surface Milling, 5X Tube Milling,

5X Blade Milling, 5X Multiblade Milling, 5X Dental Milling:

Open a job definition dialog.

Load from file: Load data of a job or job list that was previously exported with the Utilities → Export function in *.jst format.

Macros >

Apply macros: Open the Apply macros dialog.

View macro database: Open the current macro database.

New macro from job: Open the New macro dialog and create a new macro based on the selected job.

Optimization: Optimization of jobs in a job list created by macros according to different criteria with the aim of reducing the number of tool changes and shortening machine running times.

Optimization (only compound jobs): Optimization of manually created job steps of a job list. The jobs are grouped into compound jobs either by tool or by machining step.

Replace by: Replace the selected job with another job.

Associative job copy: Create an associative job copy from the currently selected job.

Add to compound job: Create a new compound job and add the selected job to the compound job.

Add to linking job: Create a new linking job and add the selected job to the linking job.

Unlink associative copy: Unlink the job and associative job copy.

Edit: Edit the current element. If several jobs or job lists are selected, parameters can be edited together (see Global editing).

Copy: Copy the selected entry to the clipboard.

Paste: Paste the clipboard contents

Delete: Delete the selected entry.

Stock >

Create stock chain: Create a stock chain for the selected jobs. You can select any jobs in a job list or compound jobs.

A stock object is created for each job in the hyperMILL browser. To calculate the stock models, use the Stock → Update function on the Jobs shortcut menu or the Calculate function on the shortcut menu for the stock model entry. The stock models are calculated based on the preceding stock model and the toolpath of the current job.

Remove stock chain: Delete all unused stock models in a stock chain. The original stock remains intact.

Update: Update the stock models for the selected jobs.

Create: Start the Stock model calculation dialog to create a new stock model.

Calculate: Calculate the toolpath file.

Update: Update the toolpath file, transferring modified job parameters to it.

Update clamping position: If subsequent changes are made to the clamping position (Job list → NCS → Clamping position), update all jobs in the job list. It is not necessary to recalculate the toolpaths.

Statistics...: Displays the length and duration of G0, G1 and G2/G3 movements, as well as the dwell times and number of tool changes for the selected jobs. This also applies to outdated jobs. It is not necessary to recalculate the toolpaths.

Note

The following simulation functions are only available with the standard postprocessor.

Internal simulation: Start the internal simulation.

Simulation Center: Simulate the machining in the SIMULATION Center for the machine defined in the postprocessor. No linking movements are taken into account.

Internal machine simulation: Start the internal machine simulation.

Note

The functions described below for the toolpath preview, NC simulation and NC file are available only with a VIRTUAL Machining Center licence and the appropriate postprocessor technology (Job list → Postprocessor → Machine administration → NC Generator).

Toolpath preview: Display the machining in the SIMULATION Center in a preview.

The following restrictions apply:

  • No linking movements are simulated.

  • No optimisation is available with the VIRTUAL Machining Optimizer technology.

Machining planner: Start the VIRTUAL Machining Center in Machining planner mode and equip a turret for turning jobs with graphical support. No toolpaths are loaded. The function is available if, in the job list (Postprocessor dialog page) a corresponding Virtual Machine has been loaded.

NC simulation: Use to load an already generated NC program into the VIRTUAL Machining Center. If the loaded NC file is not up-to-date, the following message appears:

  • Existing NC files are not up to date, continue?

    Detailed information on all changes that affect the NC file are listed in the dialog.

  • The function Generate new NC files is activated by default. By confirming the query, new NC files are generated and loaded into the VIRTUAL Machining Center. If the function is deactivated, the VIRTUAL Machining Center is started with the outdated NC files.

Postprocessor messages: Display the postprocessor messages in a text editor and save in a text format if required.

Use the Check NC status function to check the current status of the NC file. The status is displayed as follows:

NC file does not exist.

NC file exists and is up to date.

NC file exists and is approved.

NC file exists and is approved but is no longer up to date.

NC file exists but is no longer up to date.

The NC status does not change when the software version is changed.

Tip

Double-clicking on one of the icons shows the exact status of the NC file with all changes that affect the NC program, such as changes to the settings of the Virtual Machine.

Show detailed NC Status: Only available if an NC program has already been created. Deviations from the previous status are displayed. The NC status is updated automatically.

Generate NC file: Generate an NC file for the machining selected in the hyperMILL browser. The postprocessor messages are displayed in the Postprocessor dialog. If required, start the VIRTUAL Machining Center by double-clicking the entry. If the function is used again in the same job list and with the VIRTUAL Machining Center open, double-clicking the entry again will start a new instance of the VIRTUAL Machining Center.

Generate NC file (Batch mode): Generate an NC file for the machining selected in the hyperMILL browser. The postprocessor messages are displayed in the Postprocessor dialog and the VIRTUAL Machining Center is started automatically afterwards. If the function is used again in the same job list and with the VIRTUAL Machining Center open, the machining will open in the same instance of the VIRTUAL Machining Center. Any existing main program will be overwritten.

Generate NC file (Keep mode): Generate an NC file for the machining selected in the hyperMILL browser. The postprocessor messages are displayed in the Postprocessor dialog and the VIRTUAL Machining Center is started automatically afterwards. If the function is used again in the same job list and with the VIRTUAL Machining Center open, the machining will open in the same instance of the VIRTUAL Machining Center and will be added to an existing main program. Requirement: The Job list → Postprocessor → Single mode function is enabled.

Generate NC file (Manual stock selection): Generate an NC file for the machining selected in the hyperMILL browser. First select a stock from the job list manually. The postprocessor messages are displayed in the Postprocessor dialog. If required, start the VIRTUAL Machining Center by double-clicking the entry.

Job view >

Messages: Display messages (notices, errors, warnings) for the selected entry.

Display calculation info: Display information on the current calculation.

Select jobs to calculate / Select all jobs / Select jobs to check: Select the jobs to calculate, all jobs or the jobs to check in a joblist.

Search job by ID: Select this option to search for a job in an extensive machining sequence with the help of the Job-ID. The job that is found is opened.

Open / Expand all: Open the detailed view for the selected job or all the jobs in a joblist in the hyperMILL browser.

Collapse / Collapse all: Close the detailed view for the selected job or all the jobs in a joblist in the hyperMILL browser.

Expand linking jobs / Expand compound jobs: Display the sub-jobs of linking jobs or compound jobs.

Sort by frame and tool / Sort by frame / Sort by tool / Sort by job IDs: Sort the jobs in a joblist, in order, for example, to perform a tool change optimization.

Utilities >

hyperVIEW...: Start the hyperVIEW simulation program.

hyperVIEW report: Create a report with information on the tool and job parameters. hyperVIEW will start, after which you can select which report should be created (joblist, tool list with graphics or tool list). To do so, select the desired template file in the *.xsl format.

hyperMILL Report: Create a report directly from hyperMILL with information on the tool and job parameters. Select which report should be created (joblist, tool list with graphics or tool list). To do so, select the desired template file in the *.xsl format. Job graphics are currently not supported in the joblist report.

Analysis: Start the analysis dialog with functions for surface analysis.

Rest material display: Display the remaining rest material. For further information, see the product documentation for the CAD system .

TOOLPATH.EDIT... : Start the TOOLPATH.EDIT program for machining toolpaths.

VERICUT ... / NCSIMUL ...:/ COLLISION SAFEGUARD ... Start an external simulation program (VERICUT /NCSIMUL / COLLISION SAFEGUARD).

Export: Save a job or job list to an external file (file format: *.jst for jobs, *.xml for tools). You can import the data to another job list or another model using the New → Load from file shortcut menu function.

User variables: Open the User variables dialog in order to define a user variable. Use user variables to specify parameter values in the job definition.

Electrode user variables: Open the Electrode user variables dialog to check or display production-relevant parameters (spark gap) (CAD parameters, milling parameters).

Check model: Check the consistency of the contours assigned to the job.

Explode: Expand macro job lists for drilling jobs into individual jobs.

Toolpaths >

Check toolpath status: Check whether the entry (job/joblist) was calculated using the current definition or whether it has to be recalculated.

Delete toolpath files: Delete the toolpaths for the selected jobs or job lists.

Show tool: Only available for the Toolpath Edit cycle. Displays the tool that is currently used in the cycle.

Show / Hide / Autoshow / Read / Delete: Show, hide, automatically display, read or delete the toolpaths of the current job.

Show all / All off / Delete all: Show, hide or delete all toolpaths of the current model.

Read: Read in the calculated toolpaths.

Delete: Delete the toolpath of the selected machining operation.

Delete All: Delete all toolpaths of the *.hmc file.

Show G0 only / Show G1 only: Only display the G1 or G0 movements of the toolpaths.

hyperMILL settings: Open the hyperMILL settings dialog with the hyperMILL basic settings.

License status ...: Open the License status dialog with the overview of licensed products.

Open directory

Global working space: Open the Global working space directory. The directory is set in the hyperMILL Settings → Applications → Default paths → Global working space dialog.

Model: Open the directory in which the current model file (*.hmc) is saved.

Project: Open the project directory. The directory is set in the hyperMILL Settings → Applications → Default paths → Projects dialog.

Feedbacks / No feedbacks: Turn the feedback on/off (when the mouse pointer is in the browser window).

Show referenced geometries: The geometric elements used in the job are highlighted visually. The other geometry is displayed transparently.

A solid edge is represented by a dotted line, a surface boundary by a "fanned" dotted line. In a job list, info / elements of all jobs involved are displayed. When switching to a single job, job-related info / elements are automatically adjusted.
