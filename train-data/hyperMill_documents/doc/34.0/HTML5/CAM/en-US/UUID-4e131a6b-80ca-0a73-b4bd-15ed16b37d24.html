5 axes

Inclination strategy

Define the mode for the tool inclination.

Use the Iso option to set the tool orientation to the Iso lines of the surface.

Select Normal to curve mode if the tool should be aligned along the toolpath normal.

Radial Z is the preferred selection for machining rotationally symmetrical models. The tool is aligned to the Z axis of the frame.

Tilt parameters

Control the inclination depending on the tool type.

Use the Tilt angle (1) to specify the tool inclination with reference to the surface to be machined. Use the Lead angle (2) to specify the tool inclination in the infeed direction (3).

Use the Contact height to specify the contact point of the tool with the surface to be machined. Only available if a free tip geometry is defined for the tool and the Use free tip geometry for calculation option is enabled.

Use the Contact offset (4) to move the contact point of the tool in the direction of the tool axis. The advantage of this is that surfaces can be efficiently finished tangentially, even when using a standard tool (end mill).

Toolpath smoothing / Fanning

Tilt smoothing: Smooths tool movement within the fan distance. This results in a smoother machining behavior of the milling tool.

Fan distance: Area in which the tool movement is smoothed. (1) Synchronisation line, (2) Fan distance.
