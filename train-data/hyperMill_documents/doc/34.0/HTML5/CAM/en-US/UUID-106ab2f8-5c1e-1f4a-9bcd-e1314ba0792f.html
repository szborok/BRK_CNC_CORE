Free Defined Hole

Geometric data from the Free defined Hole feature can be used for 2D, 3D and drilling cycles in the job definition.

A distinction is made between the basic types Through (1), Flat (2), Tip (3) and Through hidden (4).

Depending on the basic type of hole chosen, you can define the following parameters for the Free defined Hole feature:

Comment: If necessary, enter a comment.

Orientation, Group and Position:

ISO fit: Yes if ISO fit required. No, if ISO fit not required. Fit value: Define tolerance value according to norm (DIN ISO 286-2).

ISO fit offset: Enter the value or derive it from the CAD system. Fit length: Enter the value or derive it from the CAD system.

Thread: Designation, diameter, length, pitch:

Thread offset: Enter value or derive from the CAD system.

Bottom type: Through, flat, tip, through hidden (see figure: Basic types of free defined hole). Offset bottom: Enter value or derive it from the CAD system.

Number of steps: Up to 32 steps can be defined.

Step: Type: cylinder, cone, fillet, cutout. Diameter (1): Enter value or derive from the CAD system. Depth (2): Enter value or derive from the CAD system. Angle (3): Cone type only. Enter value or derive it from the CAD system.
