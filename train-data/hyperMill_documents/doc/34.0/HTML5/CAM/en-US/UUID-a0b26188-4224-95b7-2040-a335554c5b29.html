From feature to job

To use a feature during job definition:

  1. Start the hyperMILL browser and switch to the Feature tab.

  2. Select the required feature and right-click.

  3. Choose the New job with feature function on the shortcut menu. A list appears with the cycles in which you can use the selected feature.

  4. Select the cycle you require and go to the Feature dialog page.

  5. The feature currently connected with the job and the status of the geometric data are displayed in the Feature Job Connector.

To edit a feature:

  1. Select the feature, right-click and select Edit.

  2. Make the required changes in the Feature editor and close the editor.

To select a feature:

  		1.

    Click the icon to open the Select feature dialog.

  2. Drag the feature into the right window and confirm with OK.

Note

In 2D cycles, only those features with the same frame as the one already being used in the job can be added.

To change the geometry status of a feature, select the geometry to be changed in the right window, click the right mouse button and select Active or Inactive.

If there are several geometries in the Feature Job Connector, but only one of these is to be active, select the geometry to be changed in the right window, click the right mouse button and select Active only.
