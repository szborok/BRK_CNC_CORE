Define structure+#

  1. In the hyperMILL browser, switch to the Feature tab.

  2. Right-click and select New → Customized Process Feature → Machining object. The Customized Process Feature dialog opens.

  3. Name the CPF according to the machining planned.

  4. Specify the CPF elements to be used in the Select area.

  5. Select 1 for Orientation number.

  		6.

    Click the icon to display further options. Leave the default settings Frame for Usage in job and W (‘write’) for Mode.

  7. Select 2 for Surface group number, click the icon and use the predefined settings.

Rename parameters

You can rename the parameters (for example: Orientation 1 > Frame pocket). To do this, enter the name of your choice in the right-hand part of the line.

In addition, you can assign a graphic to each parameter that clearly shows which geometric elements make up the parameter. This simplifies the manual selection of geometric elements in the CAD model.

Define filter

By defining a filter, you specify how elements are recognized in the CAD model. Manual assignment is then no longer necessary. This increases the degree of automation when using the CPF.

To define a filter

  		1.

    Click the icon to open the Filter definition dialog, select the Active checkbox and enable the required filter option: Use color filter, Use tag filter or Use layer filter.

  2. Then click the icons to Add / select color filter or Add / select tag filter and define the color or tag class. If required, click the icon to select the surfaces that are to be used as the basis for the filter definition.

  3. If a layer filter or a layer container filter is to be used, select the Layer filter or Layer container filter checkbox. Then click the icons to Add / select layer filter orAdd/Select layer container filter and define the layer/layer container.

  4. To do this, in the Layer / Layer container line, click the icon to select a layer or a layer container that will be used for the filter definition. Enable the Use wildcard (\\*) option to search for character strings that appear before or after the placeholder in a name. You can also use two placeholders to search for strings in between.

  5. To delete one or all filters, click the color, Layer or Tag class line and use the Delete / Delete all function on the shortcut menu.

  		6.

    After completing the filter definition, click the icon to include the elements found in the CAD model on the basis of the filter definition in the selection.

Prepare CAD model

For the automatic reading of the geometry elements to work correctly using filters, the CAD models must be created according to the filter definition.

  1. Select 1 for Profile group number.

  		2.

    Click the icon and select Boundary for Usage in Job and W for Mode.

  3. Enable the Milling area and Stock options and name.

  4. Now confirm the CPF definition.

The CPF structure is now completely defined. The newly created CPF appears as an entry in the list under Customized Process Feature.
