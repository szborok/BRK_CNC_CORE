Strategy

Reference path sorting

Usually, the toolpath for milling strategies is created from top to bottom. In additive manufacturing, the normal sequence of adding material is from bottom to top. Therefore, the sequence of the reference toolpath must be re-ordered. The following methods are available for this purpose.

From bottom to top (1): Use this method to re-sort the sequence of the reference job toolpaths from bottom to top.

Important

Valid only if all movements of the reference job between two G0 movements are lying on a frame parallel to the additive manufacturing job frame.

Invert sequence (2): Use this method to invert the sequence of the reference job toolpaths. The first path of the reference toolpath becomes the last path of the additive toolpath.

None (3): The machining sequence (of the reference job) remains unchanged.

Reference path options

Layer: Group of toolpaths at the same Z depth or at the same distance from the drive surface. The number is independent of whether an infill takes place or not (see options under Additive â†’ Infill).

Use Number layers to skip to define the number of layers that do not generate any additive toolpaths (1) and Max. number of layers to define the maximum number of layers that are created (2).

Group: If  Infill is activated a pocket (including islands - if present) and an open contour are regarded as a toolpath/group. If Infill is deactivated, each contour is considered a toolpath/group.

Use Number paths to skip to define the number of paths that do not generate any additive paths (1) and Max. number of paths to define the maximum number of toolpaths that are created (2)
