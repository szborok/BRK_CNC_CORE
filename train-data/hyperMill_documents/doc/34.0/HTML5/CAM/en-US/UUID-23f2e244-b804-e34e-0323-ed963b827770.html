Input dialog: General

Define job-specific details for identification and display purposes.

Job information

Define the job name.

Name, Comment: Enter a name and, if required, a comment. Name patterns can be defined for both elements.

Note

If Name and/or Comment are changed, no recalculation of the toolpath file is required. The changed designations are automatically transferred to the toolpath file.

Use name patterns for jobs and comments

Use any job parameters to create name patterns for naming jobs and comments. Proceed as follows:

  1. Click the icon to open the Define name pattern dialog and enable the Use pattern function.

  2. Click the icon under Name pattern to select an entry from the Job, Frame, Tool, Machining or Parameter areas.

Any combinations of parameters and free text are possible. The selected entries are displayed in a preview in the Preview line.

Parameter values that are used change dynamically.

Any other job parameters are available in addition to the standard available parameters.

Tip

Using a job list comment in the job

Simply go to Job information → Comment, open the Define name pattern dialog and select the required comment in the Name pattern → Job list option. This will directly apply the comment from the job list to the job.

Using any job parameters in the name pattern

Proceed as follows to use any job parameters in the name pattern:

  1. Select the parameter in the job definition.

  2. CTRL key + right-click.

  3. Use the Copy property name to clipboard function on the shortcut menu.

  4. In the Use name pattern dialog, use the Parameters → Paste property name from clipboard function to use the parameter in the name pattern.

Click the icon to save the name pattern for this job. You can also save a name pattern as a default, see Dialog control functions.

ID: The identification number is set by the system and does not change even when the job sequence is changed. Click Set job IDs on the joblist shortcut menu to set new job ID numbers.

Generate NC file: Enabled by default. An NC file is generated for the cycle if the Generate NC file function is used on the shortcut menu of the job in hyperMILL or in hyperVIEW.

Important

Generate NC file: 3D and 5X Rework Machining – linking job – compound job

Initial situation: Reference job: Generate NC file is deactivated. Rework Machining: Generate NC file is activated.

  1. Reference job and related Rework job are used within a Linking job. Generate NC file is automatically disabled in both jobs.

  2. Reference job and related Rework job are moved from the Linking job into a Compound job. By doing so, the Generate NC file option is automatically enabled in both jobs. It has to be manually disabled within the Reference job again.

Use for additive: Activate if the cycle is to be used as a reference cycle for additive manufacturing. For further information on additive manufacturing and the possible reference cycles, see section 5X Machining → Additive Manufacturing.

Note

The function Use for additive is only available if the cycle can be used as a reference cycle for additive manufacturing.

Tool feedback

If the feedback display is activated, the tool, tool holder and tool head are shown by default:

  • at the zero point of the WCS if the tool is defined in the browser.

  • at the zero point of the selected frame if the tool is defined directly in the Job dialog.

For checking purposes, the tool position can be relocated on a job-specific basis. To do this, switch to the General dialog page in the job definition input dialog.

Base point: Right-click the icon to define the position of the tool tip.

Tool holder: If defined, the tool holder and tool head are displayed along with the tool. The tool feedback can also be placed at any location.

Right-click the icon, choose Point and direction. The selected Base point determines the position. The selected point on axis defines the orientation of the tool.

Toolpath visualization

Use global colors: Use the global color definitions from the Setup hyperMILL dialog ( Document → Feedbacks → Colors). Disable this option if other colors should be used for the job.

G0 / G1: Set the colors for rapid tool movements (G0) and infeed movements (G1).

Point: Define the size and color for displaying the toolpath points.

Show NC-Path: The calculated NC path is also displayed in addition to the toolpath. The colors for rapid tool movements (G0) and infeed movements (G1) can be defined.

Note

The Show NC-Path function is available for the following cycles: Thread Milling, Helical Drilling, Pocket Milling, Contour Milling, Contour Milling on 3D Model, T-Slot Milling on 3D Model, Chamfer Milling on 3D Model, Contour Grinding on 3D Model, Turning: Finishing, Groove Finishing, Face Groove Finishing.

Show clipped paths: Show the parts of the toolpaths that are in collision (clipped). Specify the color for clipped areas.

Note

The Show clipped paths function is not available for all cycles.

Tool axis / Collision: Define the display size and color for the tool axis and collisions.

Stock chain

Use global resolution: Activate to adopt the accuracy for the stock model calculation from the Job list → Settings → Stock chain. If a different accuracy is required for the job, deactivate the function and define a value for Resolution.

Exclude from stock chain: Activate if no stock model calculation is to be carried out for the job.

If the Job information → Generate NC file function is activated, the Allow stock without NC function can be used to perform a stock model calculation for the job.
