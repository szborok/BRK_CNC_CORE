5 axes

Machine kinematics

5X Machine: This mode generates 3+2 and optionally 5-axis toolpaths (if the Allow simultaneous option is enabled). This mode requires a 5 Axis post-processor and should be use only for 5X machines supporting simultaneous motions as the connection movements between the different frames are 5X simultaneous movements.

3X Machine: The 3X machine mode generates deburring toolpaths in the selected job frame. Toolpaths generated are compatible with 3 Axis post-processors if the Frame selected in the job is the NCS or is parallel to the NCS. This mode should be use for 5X machines not supporting simultaneous motions.

5X

Allow simultaneous: Activate to enable simultaneous 5-axis movements.

Machine limitation

Max. angle to Z: Can only be enabled if the machine used has 5-axis kinematics. Permissible values for the Max. angle to Z are between greater than/equal to 0° and less than/equal to 180°. The default value is 90°.
