Update external tool

In the hyperMILL browser, switch to the Tools tab and highlight tool(s).

Update state / Update all states: Function on the shortcut menu that updates the status of the linked tool. This process also checks whether the linked tool is still in the external database and if the tool definition is up to date. No changes are made to the tool definition.

Tool / Update all tools: Function on the shortcut menu that updates the linked tool with data from the external tool database. As part of this process, the data of the linked tool is changed if necessary to make it match the tool in the external database. The status of the linked tool is then updated.

Warning

The selection of a different tool type can cause problems with reference jobs. For this reason, the basic type of an NC tool cannot be subsequently modified.
