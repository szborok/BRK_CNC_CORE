Transformation Pattern Recognition

Use the Transformation Pattern Recognition function to recognize equal subareas of the CAD model and transform toolpaths, which have already been generated for a subarea, to the recognized equal subareas of the CAD model. To use the function, proceed as follows:

  1. Select Transformation Pattern Recognition from the shortcut menu in the feature browser (feature lists) and open the dialog.

  2. In the dialog, select the frame under Frame and the surfaces of the previously edited geometry under Selection → Surfaces. The Sorting → Shortest distance function searches for the same geometries at the shortest distance from the selected geometry, taking the current frame view into account. If the Grouping function is also activated, the same geometries found are grouped within the selected frame and the groups are sorted according to the shortest distance.

  3. As a result of the calculation, a new transformation is created in the Frames browser under Transformations. It contains a correct frame for each geometry found. The frames are automatically used for machining when the transformation is used.

  4. For further processing, activate the Transformation function in the job definition and select the newly created transformation under Selection → General pattern.
