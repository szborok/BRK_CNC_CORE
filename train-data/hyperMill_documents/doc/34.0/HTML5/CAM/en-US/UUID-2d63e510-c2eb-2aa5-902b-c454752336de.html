Global editing

Edit job definition parameters for multiple jobs at the same time.

In the hyperMILL browser, select the Jobs tab, select all jobs, for which common parameters should be determined.

To edit, click the icon or right-click and select Edit.

The Global editing input window opens. Here, you can specify definitions in the following areas for several jobs.

Edit a parameter by double-clicking the corresponding line. If you move the mouse over a parameter in the column on the left, the jobs that use this parameter are shown as a tooltip.

If different values are defined for a parameter in the jobs to be edited together, this parameter is highlighted in yellow and can be adjusted accordingly.

If you want to adjust different values for a parameter, click in the line highlighted in yellow.
