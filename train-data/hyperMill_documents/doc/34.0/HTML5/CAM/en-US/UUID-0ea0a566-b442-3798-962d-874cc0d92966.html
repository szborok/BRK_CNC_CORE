Stock

A stock model in *.stl (Stereo Lithography) or *.vis (Visicut) format can be defined for various machining operations. Stock models in the format *.vis are recommended if you require greater accuracy and want to use the stock models that are generated in further stock model calculations (= stock model update).

Note

If a stock model in *.stl format is used for a stock model update, it has to be a closed solid.

By default, stock models with a file size of up to 200 MB are displayed in hyperMILL. If the stock file is larger, this is indicated in the browser by the following icon. The size limit can be adjusted to suit individual requirements in the dialog hyperMILL settings → CAD  document → STL file display.

Collision check

The updated stock model is checked for collisions. This guarantees that both tracked and calculated stock models are always collision-checked.
