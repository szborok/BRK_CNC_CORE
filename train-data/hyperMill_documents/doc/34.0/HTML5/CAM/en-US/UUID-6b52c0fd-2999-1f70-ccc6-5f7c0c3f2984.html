Save NC tool in a depot

To save an NC tool in a depot:

  1. Select the relevant depot and right-click.

  2. Select New â†’ Depot item...

  3. In the Select NC-Tool dialog, select the tool you require and confirm your selection with OK.

Alternatively, proceed as follows:

  1. Select the required tools on the NC tool tab.

  2. Select the Create Depot Items... function from the shortcut menu.

  3. In the Select Depot dialog box, select the depot you require and confirm by selecting OK.

The NC tools are now available in the depot and the parameters are displayed in the data view. If some of the NC tools already exist in this depot, they are not added and a corresponding message is displayed.

Note

The Create Depot Items.. function is also available for entire folders. This allows you to transfer the content of an entire directory to a depot.

If NC tools are contained in a depot, they are identified by the following general parameters: Alternative NC-Number, Alternative ID and Alternative NC-Name. The parameters initially correspond to the NC-Number, ID and Name parameters.

You can change the alternative parameters manually. If you create a depot as a magazine, for example, the Alternative NC-Number can be used as the place number of the tool. This number therefore always needs to be unique.
