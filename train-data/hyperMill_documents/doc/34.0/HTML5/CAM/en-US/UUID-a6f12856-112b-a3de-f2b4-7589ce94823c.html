From job to feature

To use a feature during job definition:

  1. Start the hyperMILL browser, go to the Jobs tab and select the required cycle.

  2. Specify the required settings and go to the Feature dialog page.

Reverse orientation of the feature

If you use a feature in the job definition, the correct feature orientation is important. You can reverse the feature orientation directly in the feature browser for through holes and through pockets. To do so, select the feature and then select the Reverse hole or Reverse pocket function from the shortcut menu.

If the feature orientation is not identical to the frame orientation of the feature list, the feature is marked with an exclamation mark and a message is output.

Disconnect: Deactivate the link between the job and the feature. The contour which has been derived from the feature is retained as a CAD element, and can be used for job definition.

Create new frame with feature

To create a new frame with the selected feature, select the feature in the left window of the Feature job connector, right-click and select New frame on feature. The new frame appears in the hyperMILL browser (Model tab).

Replace Simple Hole with Sink Hole

Replace by Sink Hole: Only available for the Simple Hole feature. Replace a Simple Hole with a Sink Hole. Define the sink diameter and sink depth. The original feature name remains.

Replace Sink Hole with thread

Replace by thread: Only available for the Sink Hole feature, adds a thread to a Sink Hole. Here, the sink diameter and depth are transferred for the thread.
