12.11. Tool technology parameters

Technology data and cutting profiles allow you to create tools for special areas of use. If a tool is to be used for finishing and roughing, define a corresponding Usage in the cutting profile and assign this to the tool according to its use.

Technology data can be assigned either to Tools or Inserts. In doing so, the system differentiates between milling and turning jobs. This means that individual technology data can take on different values for milling jobs and turning jobs.
