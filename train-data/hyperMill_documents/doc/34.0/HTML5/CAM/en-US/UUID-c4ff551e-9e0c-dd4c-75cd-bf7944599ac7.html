Electrode user variables

Production-related data is displayed as user variables in the Utilities â†’ Electrode user variables dialog (shortcut menu of the job browser) and can be used for programming in the job definition. The dialog is divided into the areas: Spark gaps, CAD parameters, and Milling parameters.

Spark gaps

Check the size of the spark gaps for the machining tasks Roughing, Re-roughing, Semi finishing and Finishing and adjust if necessary. The size of the spark gap corresponds to the undersize in NC production (negative value).

User variables in the CAD parameters and Milling parameters areas are only displayed and cannot be changed.

CAD parameters

Minimum distance stock (1): smallest distance between the top point of the electrode surface and the stock.

C angle (1): Rotation of stock around the C-axis for the best possible fit of the electrodes.

Milling parameters

All milling parameters are based on the zero point (A) of the machining (here: top of electrode holder).
