dialog control functions

The general dialog control functions can be accessed via icons (only in Modal) mode or via the shortcut menu.

Shortcut menu: Position the mouse pointer in the left half of the input dialog header, then right-click. The content depends on the input dialog and the selected mode for dialog control (modal/modeless).

Icon

Function

Description

Ok

Save the entries and close the dialog.

Cancel

Discard the entries and close the dialog.

Apply

Save entries; dialog stays open.

Reset

Discard the entries; the dialog stays open.

Calculate

Calculate toolpaths (job) or create 3DF file ( hyperMILL Converter).

Save As Default

Save the entries as user-defined default settings in the Global Working Space under metric.cfg or inch.cfg.

Help

Open the online help.

Messages

Display job and calculation information.

Set workplane to clearance plane

Set the current workplane to the clearance plane.

Internal simulation

Start the internal simulation

Reset workplane

Reset the original workplane.

Temporary/permanent feedbacks

Control the graphical feedback during the job definition.
