Define NCS position

NCS Position: Define the position of the NCS.

Mode: Select whether the NCS should be positioned at one of the predefined Basic positions or at a Userdefined position.

Use Basic positions to select a position Relative to the model (1) or Relative to the stock.

Under Userdefined, select the Origin directly in the CAD system.

Milling: NCS position relative to the model (1), NCS position relative to the stock (2).

MillTurn: NCS position relative to the model (1), NCS position relative to the stock (2).

Available basic positions are located on the rotary axis (3), the center of the turning surface. The rotary axis corresponds to the Z axis of the NCS.

(A) NCS position = Z-, (B) NCS position = Z+.
