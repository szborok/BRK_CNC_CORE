Simple Hole

Geometric data from the Simple Hole feature can be used for 2D, 3D and drilling cycles in the job definition.

A distinction is made between the following three basic types (bottom type) of Simple Hole: Through (1), Flat (2) and Tip (3), as well as Through hidden (4).

Depending on the basic type you choose, you can define the following parameters for the Simple Hole:

Orientation (1): Enter coordinate values or determine the orientation vector from the CAD system.

Group: No if no group is to be used. Yes if group is to be used.

Position (2): Define point(s) in the CAD system.

Chamfer: Select this option (if chamfer is required).

Chamfer depth (3): Enter the value or derive it from the CAD system.

Diameter (4), Depth (5): Enter values or derive from the CAD system.

Bottom chamfer: Select this option (if bottom chamfering is required) and specify the depth of the bottom chamfer.

Bottom type: through hole, blind hole, tip hole, see figure: basic types of Simple Holes.

Offset bottom (only bottom type: through hole). Enter value or derive it from the CAD system.

Tip angle (6): Enter the angle for the tip hole bottom or derive it from the CAD system.

ISO fit: Yes if ISO fit required. No if ISO fit not required.

Fit value (norm): Define tolerance value according to norm (DIN ISO 286-2).

Fit length: Enter the value or derive it from the CAD system.

Thread: Yes if thread required. No if thread not required.

Thread designation: Enter the name of the thread. This will be part of the feature name as follows: Feature Name<Thread Name>.

Thread diameter (7), Thread length (8): Enter values or derive from the CAD system.

Thread pitch (9): Enter value or derive from the CAD system.

You can substitute the Simple Hole feature with the Sink Hole feature.

  1. Select the required feature and right-click.

  2. Select Replace by sink hole and enter sink diameter and depth.

You need to manually rename the feature after replacing it.
