Create stock chain

Use the Create stock chain function to create a resulting stock model for any job in a sequence of machining steps, based on the stock model of the preceding job and the toolpath of the current job. This function is available for joblists, any jobs in a joblist, an individual job or a compound job.

Requirements: A start stockmodel must be present. All toolpaths of the jobs for which a stockmodel is to be created must have been calculated.

Procedure:

  1. In the hyperMILL browser, select the jobs for which you want to create a stock chain.

  2. Select the Stock → Create stock chain function from the shortcut menu.

hyperMILL creates a stock object in the stock browser for each of the selected jobs.

hyperMILL calculates the stock models for each of the generated stock objects. The stock models are indicated by a corresponding icon in the stock browser.

To calculate the stock models, use the Stock → Calculate (update) function on the shortcut menu.

Note

The Stock → Create stock chain function is only available if the Job list → Settings → Stock chain → Automatic function is deactivated.
