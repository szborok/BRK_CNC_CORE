Origin

Influence your NC output by defining origins.

Make accuracy or tolerance corrections for a specific area of the part to be machined as well as precise position adjustments.

Each Origin is identified by a unique ID. The ID is translated into NC instructions during postprocessing using an origin table.

For more information on this, please contact your OPEN MIND partner.

Define origin

  1. In the job list, Job list settings dialog page, enable the Allow multiple origins option.

  2. In the hyperMILL browser, switch to the Frames dialog page and select the New origin option from the shortcut menu. Specify the Reference system and Origin.

  3. On the General dialog page, set the Name and ID. Save your entries.

The new origin appears as an entry in the frame browser. If this origin is assigned to a job as an active origin, it will be displayed in the Job dialog (Transformation dialog page).

Use the Max. number of origins function (hyperMILL settings → Document) to define the maximum permitted number of origins.

Assign an origin to a job

  1. In the Job definition, switch to the Transformation dialog page.

  2. Select the required origin from the list under Output origin or click the icon (1) to define a new origin. Click the icon (2) to modify an origin.

The active Output origin is displayed as graphical feedback in the CAD window.

Example of use

Contour Milling on 3D Model- Transformations

To machine a contour on four different sides of a cube in a job, for example, you just have to program this on one side. The other three sides of the cube are machined automatically by transformation. To do this in the job definition, proceed as follows:

  1. Create an origin for each side to be machined. Then create a frame for each origin using the New frame from origin shortcut menu.

  2. Make the required settings in each of the job dialogs. Make sure that the correct frame (defined for the contour) is set in the Tool → Frame area. Then switch to the Transformation dialog page.

  3. Define a General pattern where you specify each of the previously defined (origin-based) frames as the target reference system. Switch to the Transformation dialog page of the job definition, set the transformation to Active, disable the Keep original job option and select the Use job frame option under Start reference.

In the calculation that follows, hyperMILL transforms the contour defined on the one side of the cube to three other sides of the cube and therefore allows complete machining of all sides of the cube in a single job.

Please note, that you need a special license to use this function. Please contact your OPEN MIND partner.

Associative Workplane

An associative workplane is linked to the elements selected for creation. When these elements are changed, the linked workplane is carried over. To use an associative workplane in hyperMILL, proceed as follows:

  1. Activate Parametric Modelling,

  2. select the required Frame or Origin,

  3. activate the Associative workplane function on the dialog page Definition and select the desired workplane.

An associative workplane is marked with a lock symbol in the hyperMILL browser.
