Turning

Turning stock / Turning model

Define or select the Turning stock and Turning model. See the Stock model dialog for further information on defining a turning stock/turning model.

Main and counter spindle

Allow main and counter spindle: Activate to generate a job list structure for turning with main and counter spindle. This consists of the two sub-job list elements Mainspindle and Counterspindle as well as the Transfer job for transferring the turning model between the main and counter spindle.

Max. spindle speed

Main spindle max. / Counter spindle max.: Define the maximum speed for the main spindle / counter spindle.

Since the max. spindle speed usually depends on the part, a maximum value can be defined separately for each of the two spindles.  The value can deviate from the max. speed defined in the Technology area of the turning tool. The lower of the two values is always used.
