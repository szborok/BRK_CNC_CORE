Set up machine simulation

Proceed as follows to set up the machine simulation:

  1. Define machine.

  2. Create an NC job and insert toolpath.

  3. Define the machine simulation conditions. To do this, switch to the Simulation tab in the browser and specify the required settings.

Note

Machine origin: If you select the Automatic option, the parts used and the toolpaths will be positioned automatically. Use Machine Setup to define other origins.

Animation control

C: All tool movements of the respective job are simulated.

G1: Only tool movements on the G1 plane are simulated.

G0: Only tool movements on the G0 plane are simulated.

If you set the (Program) halt option to On, the simulation will be stopped at the point specified in the NC file.
