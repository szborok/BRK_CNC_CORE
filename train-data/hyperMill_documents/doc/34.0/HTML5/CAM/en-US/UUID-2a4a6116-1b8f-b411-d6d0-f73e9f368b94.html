Strategy

Infeed strategy

Roughing / Finishing along the selected contour. The infeed (stepover) distance is calculated for roughing (1) based on the tool reference point and the toolpath. When finishing, the infeed (stepover) distance is calculated directly on the contour (2). You can optionally machine in stepover mode Scallop height when finishing (3).

Tool position

Left / right: the tool is advanced to the left or right of the contour.

On contour: the tool moves directly on the contour. The behavior in sharp corners cannot be defined.

Automatic: The tool position is adjusted automatically to the contour direction (enabled by default).

The Model check function must be enabled.

Sharp edge behavior

Define the behavior of the tool at the outer edges of a model.

Roll (standard) (1): Edge rolling (only for Roughing infeed strategy). Trim (standard) (2): Trim toolpath on the corners (only for finishing infeed strategy).

Sharp (3): Tangential extension on the edges.

Retract strategy

Rapid retract: Retract movement is executed in rapid mode.
