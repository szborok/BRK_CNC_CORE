Macros

Approach macro / Retract macro

The following approach and retract behaviors are available: Axial (1), Circular (2) and Tangential (3). These additional tool movements only take place when this can be done without causing a collision.

Return macro

If Return macro is defined, horizontal stepover is executed between the milling paths via the clearance plane or clearance distance. The activated approach or retract macro is executed for each retract or infeed movement.
