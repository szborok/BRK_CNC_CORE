12.13. Define manufacturer

To assign a tool manufacturer to a database, proceed as follows:

  1. Right-click on the database entry in the DB browser.

  2. Select the Manufacturers... entry in the context menu.

  3. Right-click and select the desired manufacturer from the list of available manufacturers at New → [Manufacturer name].

  4. If necessary, enter a comment and confirm the entry with OK.

If required, the manufacturer can now be assigned to a tool/tool series or an Extension, a Holder, an Insert or a Tool holder. To assign a manufacturer to a tool/tool series, for example, proceed as follows:

For a single tool in the Tool → Geometry → Common → Manufacturer area or for a tool series in the Tool → Series definition → Manufacture area, select the manufacturer from the list.

In the Ordering code field, an ordering code can be assigned to a Tool, Extension, Holder, Insert or a Tool Holder.
