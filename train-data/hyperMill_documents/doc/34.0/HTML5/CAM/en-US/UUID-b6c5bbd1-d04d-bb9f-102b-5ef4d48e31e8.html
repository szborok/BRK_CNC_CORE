2.3. Job definition dialogs

Dialog control configuration

Set the dialog control in the hyperMILL settings â†’ Setup dialog, on the Application dialog page.

Dialogs

Modal: The input dialogs contain the general function icons. It is not possible to quickly switch the display between different jobs.

Modeless: The input dialogs do not contain any general function icons. It is possible to quickly switch the display between different jobs.

The dialog control functions can be accessed via the shortcut menu of the input dialog header. Input dialogs can be fixed to any chosen position by clicking on a pin in the header. Jobs can be calculated when a dialog is open

In this mode all settings are applied immediately and are always saved when the dialog is closed. It is not possible to exit without saving changes.

Text: dialog tabs contain text that displays the contents. In the case of cycles with a high number of definition parameters, the different tabs cannot be viewed side by side.

Icons: dialog tabs contain icons that display the contents. Even in the case of cycles with a high number of definition parameters, all the tabs can be viewed side by side.
