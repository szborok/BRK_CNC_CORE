Keyboard commands

Keyboard command

Tab

Description

n/N

Jobs, Frames, Models, (Milling areas/Turning areas)

N = Create a new joblist. n = Create a new job. N = Create a new joblist.

e/E

Jobs, Frames, Models, (Milling areas/Turning areas)

Edit: Change definitions for the selected tab entry.

m/M

Jobs, Frames, Models, (Milling areas/Turning areas)

Message: Display messages (notices, errors, warnings) for the selected entry.

c/C

Jobs, Job lists

Calculate: Recalculate toolpaths.

u/U

Jobs, Job lists

Update: Update the toolpath file and transfer modified job definitions.

v/V

Jobs, Job lists

View: Start the hyperVIEW simulation program.

r/R

Jobs, Job lists

Read: Read toolpaths.

l/L

Jobs, Job lists

Open the User variables dialog.

t/T

Jobs, Job lists

Start the internal simulation.

P

Jobs, Job lists

Check the toolpath status.

h

Jobs, Job lists

Hide selected paths: Hide the toolpaths of the selected jobs.

H

Jobs, Job lists

Hide all paths: Hide all toolpaths in the graphics area.

s

Jobs, Job lists

Show selected paths: Display the toolpaths of the selected jobs.

S

Jobs, Job lists

Show all paths: Display all toolpaths.

A

Jobs, Job lists

Autoshow: Automatically display the toolpaths of the selected jobs.

q/Q

Jobs, Job lists

Show referenced geometries

Highlight and display the geometry of the elements used for toolpath calculation.

Button Q: Switch on the visualization. The geometric elements are highlighted. The other geometry is displayed transparently. Press the key again to switch off the visualization.

A solid edge is represented by a dotted line, a surface boundary by a "fanned" dotted line. In a job list, info / elements of all jobs involved are displayed. When switching to a single job, job-related info / elements are automatically adjusted.

The tooltip on element shows information about the job ID, the milling area and geometric information of the element. Quickinfo on Element displays information about the job ID, milling area and geometric information of the element.

The visibility filter and the selection filter are taken into account. The Check visibility function can help to determine the visibility settings of elements.

Select the color for referenced elements in File → Options → Options / Properties in the option Graphic → System → Rendering → Color for referenced elements of a job.
