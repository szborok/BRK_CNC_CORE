T-Slot

Geometric information on the T-slot feature can be used in the following cycles: T-slot milling on 3D model and Pocket Milling.

Parameters of the T-slot feature:

Slot height (1), slot width (2), throat height (3), throat width (4), height upper tolerance (5), height lower tolerance (6), lateral upper tolerance (7), lateral lower tolerance (8),

Access type one sided (9), access type two sided (10), upper corner type (11), lower corner type (12), corner type sharp (13), corner type round (14), corner type mill (15), corner radius (16), chamfer angle (17), chamfer length (18),

Throat contour surface (19), throat contour depth (20), slot profile (21), wall profile (22), face position (23), face position depth (24), Position top of face position (25),
