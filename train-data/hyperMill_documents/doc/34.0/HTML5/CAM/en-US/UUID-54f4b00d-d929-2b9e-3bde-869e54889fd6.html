Creating a New Report

A range of report layouts for various types and languages are installed together with hyperMILL and are available to the user immediately and directly within hyperMILL.

It is often sufficient to simply adjust these templates to your company's requirements. However, in some cases it may also be necessary to design a report layout from scratch.

The dialog for this task can be found in the File > New menu item. The report type, e.g. a joblist report, the language and the output format, are all specified in this dialog.
