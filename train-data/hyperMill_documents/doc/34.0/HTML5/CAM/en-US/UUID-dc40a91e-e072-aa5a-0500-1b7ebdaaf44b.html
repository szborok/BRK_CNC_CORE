Setup

Note

Models and stock models for turning cycles are always shown in a 2D view for the sake of clarity.

Turning model

A turning model is defined on the basis of a closed, planar contour that is located in the X-Z plane of the turning frame.

Collision check: If this option is selected, the holder of the tool used is permanently checked for potential collisions against the current stock model (tracked stock model) and the current turning model.

Open the Turning area dialog. Specify a name and, if necessary, a comment for the turning area.

Generate a turning area by selecting a closed contour.

Open the Create turning area from model dialog to create a turning area from surfaces or solids.

Frame for turning axis

Select a frame for the turning axis from the list.

Define the Color and Layer of the turning contour. Select Use model colors to adopt the current colors from the model.

Surfaces

Resolution: Define the resolution (fineness) of the turning area.

Select surfaces or a solid. hyperMILL automatically creates a turning contour that defines the turning area. An entry is created in the hyperMILL browser and a Contour feature is created in the feature browser.

Turning stock

Turning stock is defined on the basis of a closed, planar contour in the X-Z plane of the turning frame and the associated turning axis in the X-Z plane.

Open the Stock model dialog. Specify a name and, if necessary, a comment for the turning stock. Enable a mode that is to be used to create the turning stock.

Generate turn resulting stock: Create a stock model for subsequent turning jobs based on the current turning job. The stock model is named after the job that was used to create it.

Generate mill resulting stock: Create a stock model for subsequent milling jobs based on the current turning job or a combination of milling and turning jobs.

From existing mill stock: Select base stock from which a mill resulting stock is to be created.

Warning

From existing mill stock: Should only be used if the following conditions are fulfilled: Prerequisite: mill/drill machining with non-rotationally symmetrical removal. The milling job begins when the turning job is complete. If these conditions are not fulfilled, computing times may increase as a result of different resolutions during stock tracking.

Generate milling stock from turning stock: Any turning stock that is represented by an entry in the hyperMILL model browser can be used to create a milling stock model.

Highlight the corresponding turning stock model in the browser and select the Create milling stock function from the shortcut menu. Specify the required resolution and click OK to confirm. The created milling stock model appears as an entry in the hyperMILL model browser.

Frame: The Z axis of the machining frame must be the same as the rotary axis of the tool. Both the turning area (turning model) and the turning stock must lie in the X-Z plane.

For further information on frames, see section NC system and frames.
