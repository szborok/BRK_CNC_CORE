12.14. Define user-defined data class

  1. Right-click the database entry and select Custom Data Classes....

  2. Select the component for which you want to define the user-defined data from the selection list.

  		3.

    Right-click, select New and define the name for the data class (for example: Tool manufacturer, order number, storage location or similar).

  4. If required, use a comment to describe the data class.

  5. If required, use the icons to change the order of the entries.

  6. Confirm your entries by clicking OK.

Note

User-defined data classes are listed in the data view of the overview window on the Geometry tab as Custom data. Each user-defined data class can be assigned a value here (maximum 128 characters).
