Structure View

The right area, known as the structure view, is used to edit a report layout. In this area, the user can select an element to edit its properties, as well as add new elements if necessary.

The context inside the report dataset, which is referenced e.g. when inserting placeholders, appears above the tree.

Logical elements, such as placeholders and loops, are highlighted blue in the tree view while visible elements are displayed in normal black.

It is possible to make direct changes to the properties of visible and logical elements in the lower Properties dialog.
