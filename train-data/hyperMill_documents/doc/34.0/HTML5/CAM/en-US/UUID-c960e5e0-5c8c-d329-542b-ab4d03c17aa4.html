Compound Feature

Geometric data from the Compound Feature can be used for 2D, 3D, 5X and drilling cycles in the job definition. The following parameters can be defined

Frame: Yes if uniform frame required. If this option is selected, only sub features with the same frame system within the Compound Feature are used. No, if uniform frame is not required.

Frame origin: Enter coordinate values or determine the orientation vector from CAD system.

Frame orientation: Enter coordinate values or determine the orientation vector from CAD system.

Sub features: Select previously defined features. Existing features can be added to a Compound Feature directly in the feature browser. To do so, select the feature and use the Compound Feature function in the shortcut menu.

Sub features of a Compound Feature can be selected directly in the feature browser and freely moved or reordered (drag-and-drop operations). A Compound Feature consists of referenced sub features. Changes to one or more sub features are automatically passed to the Compound Feature.

Functions on the shortcut menu

Point optimization: hyperMILL optimizes the paths to be travelled between the holes of the Compound Feature. To do so, select the feature and then select the Point optimization function from the shortcut menu. The machining sequence is indicated by arrows. The first step is indicated by a blue arrow.

When one of the hole features within the Compound Feature already consists of several holes, these need to be broken up first for the points to be optimized efficiently. To do so, select the feature and then select the Explode function from the shortcut menu. The optimize the points as described above.

Reverse sequence: Use this function to reverse the machining sequence for the holes in a Compound Feature.

Sort Holes: Sort the list of intersecting holes of a Compound Feature in the feature browser. Use Diameter or Depth to define the sort criterion. Use Ascending or Descending to determine the sort sequence Holes that have the same diameter or the same depth are combined into a frame (feature list). If necessary, adjust the Tolerance for the grouping.

Import hole / Export hole: Import or export all parameters defined with a hole in *.hmf format. This applies to the Simple Hole, Sink Hole or Free defined Hole features.

Reference frame: A reference frame can be selected when exporting holes. By doing so, all holes will be exported with their relative frame coordinates. If this option is not activated, the WCS coordinates are used when exporting the holes. If the model position has changed, the holes will be correctly imported, if the same Reference frame is selected.

Use feature lists: When importing the holes, you can enable the Use feature list option to preserve the allocation of the Features to the Feature lists.
