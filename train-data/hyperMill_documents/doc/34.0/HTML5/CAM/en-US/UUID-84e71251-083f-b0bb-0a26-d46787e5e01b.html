Tool

Cutting profile

The values for spindle speed and axial feedrate are taken directly from the definition of the tool in the OPEN MIND Tool database.

Technology optimization

You can also set the spindle speed and (axial) feedrate for the various drilling phases. The values in the Feedrate and Spindle speed columns correspond to the value set in the cutting profile, whereby the adjustment defined in the % Feedrate or % Spindle speed columns is taken into account.

To manually adjust a value for Feedrate / Spindle speed:

  1. Select the line, right-click the Feedrate / Spindle speed column and

  2. select the Unlink option.

For the feedrate/spindle speed in this drilling phase, only the values set in the Feedrate/Spindle speed column apply. Changes in the Cutting profile or in the columns %Feedrate and %Spindle speed are not considered.

To establish a new connection between Feedrate / Spindle speed and % values:

  1. Select the line, right-click the Feedrate or Spindle speed column and

  2. select the Link option.

Note

Threading with standing spindle, retract while turning backwards The spindle speed 0 and negative values for the spindle speed are allowed in sections 1, 7 and 8 of the hole. This is useful, for example, for the “threading” of an ELB-TLB in a pilot hole or for retracting the tool. A negative value represents the switch in the spindle orientation of the tool. When using a negative spindle speed or 0, the system checks of the postprocessor is prepared for this. If this is not the case, the generation of the NC file will terminate with the following error message: Postprocessor not ready for changing spindle orientation (Optimized deep Hole Drilling).

Movement sequence of the tool

Feedrates and spindle speeds assigned to the tool during its movement:

(1a) Guide sleeve: rapid movement up to the start of the guide sleeve; with F1 and S1 until the end of the guide sleeve.

(1b) Pilot hole: rapid movement up to the clearance distance. With F1/S1 until the end of the pilot hole.

(2) With F2/S2 until the start of the crosshole.

(3a) Crosshole: from the start of the crosshole until the end of material removal with F3/S3. For crossholes with F4/S4. From the start of material removal until full material removal with F5/S5. (3b) Crosshole: for continuous material removal in a crosshole: until the end of the crosshole/material removal with F6/S6.

(4) Outlet for a through hole/crosshole) with F3/S3 or F4/S4.

Retract movement: to the clearance distance or the start of the guide sleeve in F7/S7.

Linking movements: between drill holes with F8/G0 and S8 (reduced feedrate/spindle speed).
