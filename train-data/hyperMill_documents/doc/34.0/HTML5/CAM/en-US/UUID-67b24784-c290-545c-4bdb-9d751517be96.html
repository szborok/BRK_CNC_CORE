Parameters

Machining area

Offset top: Specify offset for the start of the Helical Drilling.

Bottom offset: material remaining on the machining base.

Absolute top / Absolute bottom: The values for top and bottom refer to the origin of the defined frame.

Break through length: Area above the lead in or the tip of the tool. Extends the toolpath in the axial direction. In the case of a through hole, approach must continue by this value to ensure that no burring occurs.

Pre-drilled hole: Activate if there is already a pre-drilled hole. The area of the pre-drilled hole is not machined. Specify the pre-drilled hole diameter via Diameter.

Finish bottom: Enable if a final horizontal circular path is to be inserted at the bottom of the hole to remove any remaining rest material.

Machining parameters

Pitch height (1): Pitch of the spiral-shaped path. Corresponds to the height of a full 360Â° travel.

Adapt pitch height: Enable if the pitch height of the spiral-shaped path is to be adapted in narrow areas and specify the lower limit for the pitch with Min. pitch height.

Stock model

Roughing allowance XY (2): Material remaining on the hole wall.

Infeed

Step height / Horizontal stepover: Machining in several infeed planes/radial intermediate steps. If the step height (3) or horizontal stepover (4) is smaller than the depth or diameter of the hole, machining will be done by way of vertical and horizontal multiple infeed.

Cutting direction

The path direction is either clockwise or counterclockwise.

Retract mode

See the information in section Retract mode.

Clearance

Clearance plane: Plane for rapid tool movements.

Clearance distance (1): Distance between the defined surface of the hole and the start of the helical drill movement.

Axial retract distance (2): Axial distance between the end of the helical drill movement and the start of the rapid movement to the security plane.

Reposition feedrate (1): Feedrate of the tool during repositioning between infeed planes (see Step height parameter) or radial intermediate steps (see Horizontal stepover parameter).
