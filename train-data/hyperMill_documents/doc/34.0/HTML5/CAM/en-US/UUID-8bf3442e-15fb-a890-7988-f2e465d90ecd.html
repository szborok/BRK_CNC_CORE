Setup

Model

All the surfaces given in the geometry section – including the blade copies calculated internally – are automatically checked for collisions, if this function is enabled. This means you do not need to select the surfaces into one 3DF file again.

To ensure the collision check is correct, however, the system must "know" all the surfaces at risk in the machining area.

These surfaces can be defined as follows:

  • As a milling area or 3DF file if you want to include these surfaces in the collision check for most operations or

  • as additional surfaces if the surfaces are only relevant to individual machining jobs

Check tool

Check on: The collision check is required for automatic collision recognition and avoidance. If this feature is not enabled, the system will not correct the tool orientation in the case of a collision!

Warning

You should only deactivate the collision check if you want to examine the path distribution in advance. It is essential, that you keep this feature activated during all final machining jobs to ensure collision-free milling.

Additional shaft allowance: You need to define an additional shaft allowance if the adjacent blades have an even larger allowance during blade milling and there is a risk that they will touch the non-cutting shaft.

The value for the additional shaft allowance should be equal at least to the difference between the current allowance and the existing allowance on the adjacent surfaces.

Note

Values that are too large will make it difficult to access the stock using thicker (more robust) tools.

NC parameters

Machining angle tolerance: If both of the machine’s rotary axes are set relatively soft, the tool orientation may deviate slightly from the collision-free orientation that has been calculated. To be on the safe side, you can use this value to internally and conically enlarge the tool and holder by this angle.

Note

Values that are too large will make it difficult to access the stock using thicker (more robust) tools.

Max. angle increment

The maximum angle increment restricts the permissible change of the tool inclination between two points.
