5-axis parameters

Holder orientation

Opposite to infeed / Toward infeed: The tool axis will tilt normal to tool contact path opposite to infeed (option: Opposite to infeed (1) or toward infeed (option: Toward infeed (2) inclines. Rotation will be adjusted to guaranty climb cutting.

Inclination

Contact parameter: The contact parameter along the ton radius determines the default tilt angle of the tool. Smoothing and collision avoidance have an impact on the contact point achieved.

  • 0 = End of barrel contour toward tool tip

  • 1 = End of barrel contour toward tool shank

Example:

(1) Contact parameter = 0.9

(2) Contact parameter = 0.5

(3) Contact parameter = 0.1

Note

The parameter should be selected so that the distance of the contact point from both ends of the barrel is more than 50% of the lateral infeed.

Lead angle: Rotation of the tool shank (around blade surface normal) toward cutting direction.
