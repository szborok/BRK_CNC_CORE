Link CPF to CAD data

To link CAD data of a model to the CPF, proceed as follows:

  1. Right-click the feature browser and open the CPF dialog.

  2. Select New → Customized Process Feature → [CPF name]. A corresponding entry is created in the lower part of the feature browser [CPF name] and all previously defined geometry elements are displayed in the CPF dialog.

  3. For Orientation 1, select the required Frame for machining.

  4. Select the hub and side surfaces of the pocket for Surfaces 1 and 2.

  5. Select the limit of the hub surface for Profiles 1.

  6. Finally, define the Milling area and Stock.

If all parameters of the CPF are clearly named and the corresponding data from the CAD model assigned, the CPF can be used in the job definition. The job and CPF are linked in the Feature Job Connector.
