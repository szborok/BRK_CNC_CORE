Use standard formulas

  1. In the Usage area, select the parameter.

  2. Use the combo box to select the required formula.

Note

The formula icon shows that the corresponding value has been defined using a formula. This also applies to the data view of the NC tool.

For the Spindle RPM parameter, there is a formula for the Metric measurement system and one for the Inch measurement system.

Define new formula

  1. Right-click the database entry and select Formulas....

  2. In the combo box in the Formula dialog, select the area for which you want to define a new formula.

  3. Right-click in the Formula dialog box and select New.

  4. Enter a name for the new formula and double-click the Formula column.

  		5.

    Then, click the icon for displaying the formula elements.

  6. Define the formula you require and select Ok to close the dialog.

Formulas can be used in all numeric input fields.

Example: Diameter: (5+2)*3 = 21.

Note

Only points, but not commas, are allowed, as decimal separators in formulas. Only certain variables are allowed for formula definition, depending on the formula type. The following restrictions apply: speed: spindle RPM (n) is not permitted. Length: ae, ap, fz, z, Vc, n and f are not permitted. Feedrate: all variants are allowed.
