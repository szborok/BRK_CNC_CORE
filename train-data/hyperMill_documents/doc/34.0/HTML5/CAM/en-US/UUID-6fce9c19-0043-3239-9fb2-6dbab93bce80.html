Generic Pocket

Generic Pockets can be used universally and are structured according to manufacturing elements and related properties.

Note

Generic Pockets offer significant advantages when using macro machining. In particular, when setting up a macro database with macro rules, OPEN MIND recommends exclusive use of generic holes.

Geometric data can be used for 2D, 3D and 5X and drilling cycles in the job definition. Geometric parameters are automatically copied for use in the job definition.
