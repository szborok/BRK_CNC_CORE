4.4. Generate feature

A feature can be created as follows:

  1. Use the Feature mapping (hole) function in the feature browser to automatically assign information from the CAD model to a hole.

  2. Use the Feature Recognition function in the feature browser to automatically or manually assign information from the CAD model to a feature.

  3. By manually selecting contours and surfaces using the feature browser. For information on the feature editor.

  4. By manually selecting contours, boundaries, milling surfaces, stop surfaces or additional surfaces in the job definition.
