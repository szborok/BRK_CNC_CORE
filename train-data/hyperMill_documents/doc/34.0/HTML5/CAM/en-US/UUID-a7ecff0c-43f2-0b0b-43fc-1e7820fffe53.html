The feature browser

The Feature tab in the hyperMILL browser is known as the feature browser. The feature browser comprises the following:

  • The top area with feature lists and shortcut menu functions (1)

  • The bottom area with feature and shortcut menu functions (2)

  • A freely definable filter (3)

  • A toolbar with filter functions and functions to control the visibility of features (4).

The display of features

Use the following options to control the display of features:

1. V- By showing or hiding the features of a feature list.

Activate (show features) or deactivate (hide features) the function in the top area of the feature browser.

2. By entering any numeric or alphanumeric characters into the filter

Enter any digits or characters that are contained in the feature name to control the display of comprehensive feature lists and thus improve the transparency of the feature browser.

3. Using feature filters.

Open the feature filter by clicking the icon on the toolbar. A list of available feature filters is displayed in the Filter name column.

Select the type of geometric elements for the filter definition and then select the desired element / desired element directly in the CAD model. Alternatively, enter a value in the corresponding line. A filter is only active if the Feature feedback function is activated.

Move to feature list → New: Create a new feature list with the features that correspond to the filter conditions.

Move to feature list → [L1], ... [L2]: Select a feature list to include the features that correspond to the filter conditions.

(1) Filter enabled, (2) Filter selected, (3) Filter disabled.

6. Via the shortcut menu of the feature browser in the bottom area

Zoom In

Zoom in on a selected feature in the graphics window.

Automatic zoom-in on

Always automatically zoom in on a selected feature in the graphics window. The zoom function remains in place until it is deactivated (see Automatic zoom-in off).

Display the model in transparent mode

Assign the transparent color to the model with highlighted representation of the feature. The CAD system's first available transparent color is used.

Set the feedback color.

C -

Color coding of the feature feedback in a feature list.

Change the color for the feature feedback of the feature list by double-clicking the icon. Feedback for all the features of this feature list is colored accordingly in the model.

B -

Create a visual bookmark.

Store a defined view of the model. A visual bookmark can be assigned to a feature list or a feature and makes it easier to work with a large number of features. To define a new visual bookmark, set the required model view, select the feature or feature list and choose the New visual bookmark function from the shortcut menu by double-clicking the Visual bookmark icon.

Assign frame to a feature list

Assign a frame to the feature list by double-clicking the icon.

Change the frame of the feature list by double-clicking the icon.

If a feature from this feature list is used for the job definition, the frame of the feature list is analyzed. Frames associated with individual features are not analyzed.

Edit a frame with the Edit function in the shortcut menu.

Delete a frame with the Delete function in the shortcut menu.

Sort features

Use the drop-down list on the left of the toolbar to sort the features in the browser according to Type, Name or Number. If you select the None option, the features will be sorted according to the generating order.

Functions in the feature browser

Create a new feature list using the Feature list function on the shortcut menu.

Delete a feature list with the Delete function in the shortcut menu.

Change a feature list name by double-clicking and directly editing.
