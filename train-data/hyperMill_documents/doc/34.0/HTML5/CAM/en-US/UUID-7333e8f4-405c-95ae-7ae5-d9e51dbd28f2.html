Strategy

Probing mode

Measure: Measure slots and/or ribs.

Measure and adjust: Measure slots and/or ribs and adjust the tool according to the machine settings.

Set origin: Specify the origin that is to be set or adjusted according to the machine settings. For this purpose, select the required ID.

If you use the Set origin function in combination with an old postprocessor, the system will display the following error message:

Your postprocessor does not support the Set origin probing mode.

Please contact your OPEN MIND partner. In this case, you need to update your postprocessor in order to retain the ability to use the function.

Reference job

Define a reference tool for the adjustment for the Measure and adjust option via the reference job. The 3D Point Probing, Rectangular Probing, Circular Probing and Axis dependent Probing cycles cannot be selected as the reference job.
