Sample pocket dialog page

Define the pocket type to be recognized. Select the surfaces of a sample pocket. Define search options and filter criteria. Recognized bottom surfaces and pockets are presented in a list.

Option

Pocket type: Define a pocket type for recognition purposes. Distinguish between the following:

Pocket with bottom: Pockets with a bottom are recognized.

T-Slot: T-Slots are recognized.

Pockets with bottom and T-slots: Pockets with bottom and T-slot are recognized.

Through pocket: Through pockets are recognized.

Tolerance: The required tolerance of the curve resolution for the selected feature.

Name: Define a name for the pockets. This is used in the hyperMILL feature browser; it also contains the smallest radius in the pocket. This name is used when performing a tool search in the macro database.

Example:

If a pocket has this name, it can be machined using tools with a radius of 3 mm or less.

If you want pockets with bottoms to be recognized, you can use the following options to more accurately specify the sample pockets:

Combine pockets: Enable if individual pockets of a model are to be automatically combined into a more complex pocket.

When used in a job, combined pockets are processed more efficiently than individual pockets

It is possible to combine the following: pockets with bottoms (1), through pockets (2) and open (2) and closed (3) pockets. The pockets must have a common edge (4) and the same frame in order to be combined.

Note

Pockets connected to one another through an open pocket (5) are not considered for automatic combination.

If the pockets have chamfers or fillets, the values for the corner radius (fillet) or chamfer angle and chamfer height parameters must be identical. It is not possible to join chamfers and fillets for combination.

The individually recognized pockets, the combined pockets and the Generic Pocket are displayed in the feature browser as a result of the Pocket Recognition.

Generate O-Rings: If this function is activated, the following variants of an O-Ring are recognized: (1) O-Ring with pocket, (2) O-Ring with non-centric hole, (3) O-Ring with centric hole, (4) O-Ring without additional geometry.

The following Feature parameters are also recognized: (5) Center, (6) Diameter, (7) Diameter inside, (8) Depth, (9) Hight inside, (10) Inclined angle outside, (11) Inclined angle inside, (12) Hole position, (13) Hole diameter, ( 14) Hole height

Suppress chamfers/ planes: Enable if you want to exclude planes and chamfers from recognition. In this case, the following will be excluded from recognition

  • Planes (pocket bottoms) that do not have any pocket walls and

  • Planes that have been recognized as chamfers (see Setup dialog page).

Open pocket reduction: Enable to reduce the number of open pockets that are recognized. If there are several adjacent planes which can potentially be used as bottom surfaces, the largest plane (by area) (1) is used as the pocket bottom.

Manually combine pockets

Generic Pockets can be manually combined either in the feature browser via the shortcut menu or using the color table. The pockets must have a common edge and the same frame in order to be manually combined.

  1. Highlight the pockets to be combined in the feature browser and select the Combine pockets function on the shortcut menu.

  2. Use the Setup hyperMILL (dialog page Database) to open the Color table dialog.

    Define the colors to be used for the combination in the Pockets section. The color of the hub surface is relevant for pockets with bottoms and the color of the wall surface is relevant for through pockets.

  3. Alternatively, click the icon to select a sample element (surface), whose color is to be used. Enable the Use color table option on the Pocket Recognition Setup dialog page and disable the Use all colors option.

For pockets that are combined using the Combine pockets shortcut menu in the feature browser, a new feature list is created that is marked with the [ Combined] notation.

Bottom

Select all: Searches the entire model for pockets.

Surfaces: Select surfaces to be recognized as pocket bottom surfaces.

Bottom: Recognized bottom surfaces are presented in a list.

Reverse: With closed solids, the correct direction is usually detected automatically. In the case of ambiguous solids or individual surfaces, reverse the direction manually. To do so:

  1. Double-click or select the list item.

  2. Right-click and select Reverse if the direction of the selected bottom surface is to be reversed or select

  3. Reverse all if the direction of all of the bottom surfaces is to be reversed.

When the direction of a surface is reversed, pockets that have already been recognized will be removed from the list and Pocket Recognition must be repeated from the beginning.

Pocket

Recognized pockets are presented in a list. In addition, the parameters Open/Closed, Height, Edge type (Chamfer/Fillet), Radius/Height, Chamfer angle of the respective pocket will be displayed.

For each pocket recognized, the functions Close corners and gaps and Close slots are also available in the shortcut menu.

Close corners and gaps (A): Boundary contours of pockets (1) whose holes interrupt the contour (2) are closed.

Close slots (B): Boundary contours of slot-like pockets that are the same length and run parallel to each other (1) are closed in a semicircle (2).

During Pocket Recognition, a check is also automatically carried out to see whether a recognized pocket meets the conditions for a slot. If this is the case and the pocket contour has a start and end point, then the slot profile may be used as a contour during contour milling (example 1) and the slot ends may be used as drill points during drilling (example 2).

Example 1: Model (1), Pocket feature (2) and Slot profile as contour (3).

Example 2: Model (1), Pocket feature (2) and Slot end as drill points (3). The Pocket features thus created contain the designation Slot in the feature name.

Example 3: Generic Pocket <Closed R10 Slot> with Slot profile (1) and Slot width (2).

The Slot, Slot profile and Slot width feature parameters are displayed in the feature editor. Proceed as follows to machine the recognized slot:

  1. Create a new job (2D Contour Milling) using the corresponding Pocket feature.

  2. Define the tool diameter according to the slot width and specify the On contour option as the tool position.

  3. Select the Active only option from the shortcut menu of the slot profile on the Feature dialog page.

  4. Define all other parameters.
