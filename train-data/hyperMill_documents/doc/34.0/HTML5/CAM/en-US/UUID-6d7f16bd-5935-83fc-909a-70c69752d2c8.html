TDM Systems integration

Manage the exchange of data with the TDM Systems tool management system using the TDM Systems integration.

Set up the TDM Systems database from the following dialog: hyperMILL → Setup → hyperMILL → settings → Database → Settings Wizard.

Import tools from the TDM Systems database

Open the TDM Systems database

via the entry CAM → Utilities → Import TDM Systems tools or

via the job definition, job browser or tool browser.

Click a tool icon to first select the required tool type and then the tools to be imported. If the TDM Systems database was opened from the job dialog, then only the tool type already selected in the job dialog is selectable.

Note

Tools which were imported using the TDM Tool Manager can be used in macro programming. A prerequisite is that the tools are saved in the database as an NC tool with a unique ID. In the Apply macro dialog, NC tools used in macro jobs which come from the tool database and have a unique ID are marked as DB.

Export tools to the TDM Systems database

Open the TDM Systems database via the menu hyperMILL → Tools → TDM Systems tools.

Define the name of the tool list to be exported using List ID.

Note

Either all local tools defined in the ( *.hmc) model file are exported or all tools in a job list. To only export the tools in a job list, select the respective job list beforehand.
