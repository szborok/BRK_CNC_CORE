Define tool

  1. Switch to the Tools tab.

  2. Choose New â†’ [Tool type] from the shortcut menu.

  3. Define all required parameters in the Edit [tool type] input dialog. Make sure that the tool has a unique ID ( General section).

  4. Close the dialog box when all entries are made.

A click in the rows Spindle, Holder, Extension or Tool enables functions for selecting and editing the components.

The tool appears in the list of document tools and can be selected during job definition.

You can edit and save document tools in the document, even if you have not defined a (valid) external database.
