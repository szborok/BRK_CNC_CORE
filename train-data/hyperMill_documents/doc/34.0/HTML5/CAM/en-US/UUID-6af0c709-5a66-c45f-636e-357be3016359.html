3.6. Prepare manufacturing geometry

Both 2D contours and CAD models can be used for 2D machining.

3D, 5-axis and turning jobs are based on a CAD model.

A stock model can be defined for various machining operations.

2D machining

You can select the following contours:

  • Lines, circles, polylines, splines, graphics loaded in *.omxformat (Contour Milling, Pocket Milling and Helical Drilling).

  • Points (drill positions)

Generate contours from toolpaths: Curves extracted from previously calculated toolpaths can be used both as 2D contours and as help contours for 3D or 5-axis machining. For information on selecting contours, see section Selecting elements in the CAD system.
