12.6. Define tool

Once the folder structure has been defined, you can create new tools.

  1. On the Tools tab, right-click the database entry and select New.

  2. Select the required tool type.

  3. Specify a unique name in the data view. A comment can make it easier to select the correct tool later on. Define all other parameters.

  4. A newly defined tool appears as an entry in the database browser.

Note

If you define a non-permitted value for a parameter, the corresponding line in the data view is marked in red.
