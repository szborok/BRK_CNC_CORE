Linking job

A linking job contains other jobs that can be linked with each other in such a way that approach and retract movements for the entire machining sequence and all linking movements between the individual jobs can be optimized and monitored for collisions.

Advantages

  • Shortened machining times.

  • Shorter programming times.

  • Improved product flexibility.

Requirements

To combine multiple jobs into a linking job,

  • for 2D/3D machining jobs, all sub-jobs must use the same tool,

  • for turning jobs, all sub-jobs must use the same tool, the tool angles and tool orientation (Invert tool) must be the same for all sub-jobs,

  • for 3D/5X machining jobs, all sub-jobs must use the same tool.

Important

For 3D operations with the 3D path compensation function activated (Settings → Tolerance parameters → True Shape point distribution → 3D path compensation): All jobs of the linking job must have the 3D path compensation function either switched on or off. Different settings for this function are not supported.
