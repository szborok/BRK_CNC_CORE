Text Areas and Properties

The main text area in the ReportDesigner is the paragraph ( p). A paragraph initially only contains the letters of the text contained inside this paragraph.

The format properties for the paragraph can be displayed and if necessary changed by selecting the p in the structure view. Note that every change affects the entire paragraph.

If the user wants to highlight a single word in bold or italic, for example, a separate structure element must first be inserted into a text range ( span) for this specific purpose and then formatted with the bold or italic property.

Cells in a table or positions in a list can also be regarded as text ranges and their format properties adjusted correspondingly.
