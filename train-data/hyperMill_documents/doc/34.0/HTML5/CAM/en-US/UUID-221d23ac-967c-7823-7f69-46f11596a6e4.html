12.7. Define turning tool

Note

A turning tool is always constructed from the Insert and Tool holder components.

To define a turning tool:

  1. On the Tools tab, right-click the corresponding database entry and select New â†’ [Tool type].

  2. Select or redefine the Insert and Tool Holder components.

A tool holder is made up of the shank geometry (part 1 and 2) and the extruded geometry. It contains all placement information to support inserts.

By defining the Insert support you also set the Insert type to be supported by the tool holder, and via the Clamping method, the manner in which an insert is to be mounted on the tool holder.

The Insert component is also characterized by geometric parameters as well as a Mounting type. In addition, it is possible to assign to an insert the technology parameters Material, Cutter material and Purpose.
