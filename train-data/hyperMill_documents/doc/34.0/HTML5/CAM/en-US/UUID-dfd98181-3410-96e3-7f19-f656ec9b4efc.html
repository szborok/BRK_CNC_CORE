Defining a linking job

Right-click and select the New → Linking job / New → Linking job turning function.

Add job to a linking job

Select the job(s) and then select the Add to linking job function on the shortcut menu or hold down the SHIFT key to move the jobs to the previously created linking job.

The Linking job dialog

Linking jobs are defined on the following dialog pages: Tool, Parameters, Setup, Feature, Transformation, General and NC.

Tool dialog page

All of the jobs combined into a linking job must contain the same tool. This also applies to 2D mode and 5X mode.

Parameters dialog page

Linking mode

The following linking methods may be used:

2D: Linking of 2D, 3D, drill jobs and turning jobs that share the same frame orientation.

5X: Linking of 3D and 5-axis jobs. It is also possible to link the 2D cycles Pocket Milling and Rest Machining in 5X mode.

Rapids

Specify the mode for the fast travel movements (rapids) of linked individual jobs.

Skip first and last: Skip the first infeed movement and last retract movement of the individual sub-jobs.

Keep all rapids: Keep all infeed and retract movements of sub-jobs within the linking job.

Clearance mode

Planar/ Radial: Specify how the toolpaths of the linked sub-jobs are linked. (1) Planar, (2) Radial.

Rapid smoothing

Available for the Clearance distance retract mode. Enable the High speed option to activate Rapid smoothing (1).

When machining in 5X mode, the linking movements are defined without abrupt changes of direction. The Smooth factor affects the shape of the smoothed movement as a ratio of the length of the connection line (2) to the required "height" of the movement (3). If the selected smooth factor potentially leads to a collision, the movement is automatically rendered collision free.

The Clearance distance retract mode must be active. The Linking movement with the Distance angle limit ( Setup dialog page) must match the rapid smoothing.

Transformation dialog page

The current transformation is executed by calculating the linking job and all copies are linked to collision-checked rapid link movements. A 5-axis compatible NC file is output.

Example:

Complete machining of a closed pocket with inclined walls in 5X linking mode without a tool change. The following jobs are combined in the linking job:

  • 3D Free Path Milling,

  • 5X Rest Machining, 3D Profile Finishing and

  • 5X SWARF Cutting 1 Curve.
