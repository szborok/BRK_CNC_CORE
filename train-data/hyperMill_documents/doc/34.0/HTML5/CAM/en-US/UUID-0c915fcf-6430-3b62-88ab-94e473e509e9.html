Strategy

Milling reference

Specify the area to be milled with Hub / Shroud. The selection of platform surfaces (hub and shroud surfaces) takes place using a feature definition (see Turbomachinery feature).

Edge curves

Use the Manually defined boundary option to specify the edge curves in the model.

The option Surface boundary is only available (and should then preferably be used) if exactly four surfaces have been defined under Geometry → Blade surfaces or Geometry → Drive surfaces, one each for the pressure and suction side and one each for the leading and trailing edge area.

Automatic: The edge area is delimited from the pressure and suction side via the Length / thickness ratio, so that manual sketching of the curves can be dispensed with even if the data quality is poor.

It is preferable to use automatic curve recognition in the edge area (shown in grey in the graphic) if the blade is not made up of four surfaces, but is made up of any number of surface patches.

The Length (a) and Thickness (b) of the edge area are specified by the Skeleton curve (2) of the blade and a line that runs perpendicular to the skeleton curve (3), see figure: Blade cross section, Edge area. Feasible values for the length/thickness factor lie between 0.5 (radii) and 1 (ellipses).

Figure: Blade cross-section (edge area)

Milling strategy

Point Milling: Using this machining strategy, it is possible to machine blade surfaces of any shape (including concave shroud surfaces) with ball mills and bullnose end mills.

Flank Milling: This strategy is suitable for rapid machining of all regular surfaces. In particular, convex bottom surfaces can be machined faster in this way.

Infeed strategy

Diagonal (1): This infeed strategy sorts all areas which may be machined separately. These are processed first. Then, the remaining areas are processed with movements moving completely around the blade.

Zigzag direct (2): Only available for Point Milling milling strategy. In this strategy, the tool is always engaged, resulting in a corresponding reduction in the cutting time.
