Roughing

In Roughing mode, rest material areas from preceding machining are machined completely or limited to the cavities.

Roughing mode

Enable: Enable function to activate the Roughing mode.

Roughing and finishing (A): Enable if the rest material areas (1) are to be finished after roughing.

Roughing only (B): Enable if no finishing is required.

Roughing allowances

In addition to the usual allowance (2), see Parameters dialog page, you can define an Allowance in Z direction (3) and an additional XY allowance.

Cavity depth (4): is calculated from the radius of the reference and machining tools. Example: R5 = radius reference tool, R1.5 = radius machining tool
