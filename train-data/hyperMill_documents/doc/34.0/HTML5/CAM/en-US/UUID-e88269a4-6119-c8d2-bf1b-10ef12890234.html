Define turning tool

To define a turning tool:

  1. On the Tools tab, in the lower area ( Turning tools), click with the right mouse button.

  2. Choose New â†’ [Tool type] from the shortcut menu.

  3. In the New turning tool dialog:

    Select an Insert from the external tool database or define a new one.

    Select a Tool holder from the external tool database or define a new one.

  4. Specify the tool parameters Tool angle, Tool reference and Cutting edge position by selecting the parameter set.

  5. Define all required parameters in the Edit turning tool dialog. Make sure that the tool has a unique ID ( General section).

Close the dialog box when all entries are made. The tool appears in the list of Document tools and can be selected during job definition.

Note

If, in the NC file, the position of the cutting edge and location are to be output, the postprocessor must be adjusted. Please contact your OPEN MIND partner.
