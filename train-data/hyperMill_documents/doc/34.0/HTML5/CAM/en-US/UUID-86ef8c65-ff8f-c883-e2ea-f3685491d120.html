Boundary

X limit 1 (shroud side) / X limit 2 (hub side)

The machining boundary can be defined separately by selecting any point for the shroud surface ( X Limit 1) and the bottom plane ( X Limit 2). The corresponding Y value is automatically determined by hyperMILL. The maximum extension of the corresponding 3DF model (plus the tool diameter of the tool used) is always transferred.

On/To: Based on the center point of the tool used (tool tip), the boundary for the To option is moved inwards by the cutter radius. If you use the On option, the toolpath is calculated as if the tool tip were positioned on the boundary.

The limit for the center point path of the milling tool is always displayed as feedback. The limit – relative to the selected point – is therefore moved into the machining area by the tool radius for the To option.

Clearance: Any given value can be chosen. A positive value reduces the boundary, while a negative value enlarges it.
