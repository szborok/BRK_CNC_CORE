Create reports

Generate report files with the parameters of the defined tools and/or tool components. You can generate reports for tools, NC tools, extensions, depots and holders.

To create a report:

  1. Select the element you require in the browser (you can also make multiple selections).

  2. Right-click and

  3. select Report... from the shortcut menu .

The data is initially temporarily saved in the *.html format and displayed in your default browser. You can now back this data up anywhere on your system.

The creation of reports can be controlled from the Options â†’ General menu.

Show report after generation

The generated report file is opened automatically in the default program.

Overwrite report output directory

When creating a new report, an existing report directory is deleted and a new directory is created. By default, reports are saved in the [tooldbReport] directory of the Global working space. If you use project paths, they are saved to the [tooldbReport] directory of the project path defined.

For information on creating reports, see section ReportDesigner.
