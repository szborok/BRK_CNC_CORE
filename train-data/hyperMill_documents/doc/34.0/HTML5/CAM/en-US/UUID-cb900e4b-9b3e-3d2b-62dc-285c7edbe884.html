Parameters

Infeed / Max. distance from central curve

Lateral infeed / Axial infeed: When finishing with a spiral machining strategy, an axial infeed (A) is performed that is either determined on the central curve (1) (option: At curve) or on the model surface (2) (option: At surface). When finishing with a parallel machining strategy, a lateral infeed (B) is used, where the value entered determines the greatest distance between two paths (3).

For roughing, lateral infeed is carried out in spirals around the central curve or around optionally selected tilt curves. The number of paths is determined by the selected lateral infeed and the path's maximum distance from the tilt curve.

Max. distance: (4) Limits the maximum distance of the path to the central curve. This allows for the machining of partially open tubes without constructing any auxiliary surfaces.

Allowance

Allowance: Remaining material on the tube's outside surface to be machined.

Add. shank clearance: Increases the shaft radius for collision check by the value entered.

Required especially for parallel machining strategy to avoid collisions with the workpiece's yet unmachined side. In this case, the additional shaft allowance should at least equal the sum of the material removal.

Smooth overlap: above start plane / Smooth overlap: below end plane

If the Include bottom function is enabled, the Parameter → Smooth overlap: below end plane function is automatically disabled (= grayed out) so that no input is possible. The following message is also displayed: “Smooth overlap: below end plane” is not available when the bottom region is included.

Smooth overlap is supported both above the start plane and below the end plane. The function is inactive when Length = 0.

Length: Length of the additional toolpath above the start plane or below the end plane.

Start distance: Distance of the additional toolpath from the component above the Start plane. End distance: Distance of the additional toolpath from the component below the End plane.

Example of use:

Machining from above with the Ball Mill and Lollipop tool types.

(A) Smooth overlap is active.

(A) Smooth overlap is not active.

Tip

Start distance: Useful to use if, for example, due to the deflection of the tool, the machining operations between two adjacent areas on the component do not fit together seamlessly.

Technology: non-contact paths

Additional side distance: Distance to be maintained between the tool and the component (see Strategy → Parallel).

Feedrate: Feedrate for the first toolpath (see Strategy → Parallel → Oneway Inside-out).

Technology: virtual surfaces

The parameters in this area are only available for

  • parallel machining,

  • selected virtual surfaces.

Virtual surfaces are not checked against shank collisions. Extensions of the milling area therefore have no negative impact on the achievable machining depth.

In addition, for machining operations using the Parallel strategy, the path trimming function and a special feedrate are supported to optimize machining between the start plane and the first tool contact with the material.

Trim path along: (A) Trim the toolpaths along the selected virtual surfaces.

Feedrate: (B) Define a separate feedrate for the start / end of machining.

(1) Virtual surfaces, (2) Central curve, (3) Start plane, (4) End plane.

Clearance to part: The value should be greater than the thickness of the current material removal.

Safety

Clearance plane and clearance distance apply in the direction of the Z axis of the current frame.

Clearance plane (1): Plane for rapid tool movements. Specification in absolute dimensions Define the clearance plane by right-clicking on the icon. Select a point and confirm the selection.

Warning

Traversing movements on the clearance plane are not checked with regard to collisions. Therefore, this plane must be placed at a sufficient distance above the surface of the workpiece.

Clearance distance (2): Distance to the current toolpath to be milled. Above the clearance distance, infeed takes place as rapid in the Z direction (A); below the clearance distance infeed takes place at the Z feedrate (B).
