Create macro

A macro includes features and job information. It is created based on a job or a feature.

Creating job-based macros

To generate a macro based on a job:

  1. Open the hyperMILL browser and select the Jobs tab.

  2. Select the job that you want to associate with the macro and select Macros â†’ New macro from job.

The New macro window opens.

Create feature-based macro

To create a macro based on a feature:

  1. Open the hyperMILL browser and select the Feature tab.

  2. Highlight the feature whose information you wish to use.

  3. Right-click and select New macro from feature.

The New macro dialog opens.
