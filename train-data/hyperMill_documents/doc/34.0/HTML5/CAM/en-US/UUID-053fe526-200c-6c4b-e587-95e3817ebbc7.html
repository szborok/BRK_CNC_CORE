Search dialog page

Here you can determine which hole surfaces should be compared with the base hole. As the result of this comparison, holes are found which have the same parameters as the base hole.

Group selection

User: Select the required holes, whose parameters are to be compared with those of the base hole.

Note

You can also search for holes using the Window selection function in the CAD program. This is particularly useful with very complex models, for which Feature Recognition is only partly relevant.

Complete model: All holes in the model with the same parameters as the base hole are automatically recognized.

Start: Start the automatic recognition of the holes. All recognized holes are shown under Result in a list display. The numbers mean:

Example: 3 (2)

(3) Number of directions

(2) Number of holes in this direction

Recognized holes are transferred to a feature list as a Generic Hole or Compound Feature.

Holes with the same parameters (Direction, Depth) are combined into a feature. Please note that the defined base hole is also always included as a component of a feature.

Filter

Use a filter to limit the hoes to be recognized.

Same color: Only holes of the same color as the base hole are recognized.

Same layer: Only holes on the same layer as the base hole are recognized.

Same orientation: Only holes with the same orientation as the base hole are recognized.

Frame limit: Define a frame limit (angle) for the recognition process.

Frame: Define the frame to be used for the frame limit.

B axis / C axis: Use the two tool rotary axes to limit the angle for the hole recognition process. The B axis corresponds to the X/Y direction and the C axis corresponds to the Z direction of the defined frame.

The defined angle limit in each direction is shown in color on a rotary ball. Yellow = machinable area (directions from which holes are recognized). Grey = unmachinable area (directions from which holes are not recognized). Example: angle B axis: 0째 to 90째, angle C axis: 0째 to 90째.

Same depth: Only holes of the same depth as the base hole are recognized.

Depth limit:Only holes are recognized which correspond to the depth limit (from/to).

Remove hole: Mark hole and in the short-cut menu select the Remove function.

Reverse hole direction: Mark hole and in the shortcut menu select the Reverse function.

If the Reference feature is selected as the Strategy, the following functions are available:

Group selection

Points and surfaces: Select surfaces / points which should be used to create the hole features. The orientation of the newly created hole features follows the surface normals in the area of the selected points.

Use the Move to featurelist function to select an existing feature list to which the features will be assigned. Alternatively, create a new feature list (option: New).

Circles: Select circles which should be used to create the hole features. The orientation of the newly created hole features follows the surface normals in the area of the selected circles.

Reverse direction: Reverse the direction of the newly created drill feature..
