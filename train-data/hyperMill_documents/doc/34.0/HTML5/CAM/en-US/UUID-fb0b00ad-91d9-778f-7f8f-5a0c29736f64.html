3Axes

Tilt strategy

Select Tilt point (1) to specify the orientation of the tilting axis with the help of a tilt point.

Alternatively, use Synchronization lines (2) to specify the movement profile of the tilting axis.

Synchronize toolpaths

Select Synchronization lines. The number of selected lines is displayed.

Define Tilt point. The coordinate values (X / Z) of the selected tilt point are displayed and can be changed manually.

Offset angle

Offset angle: Define the angle for the tool holder. This allows the orientation of the insert be adapted to the axis of the tool holder. This ensures that the synchronization lines always refer to the axis of the tool holder.
