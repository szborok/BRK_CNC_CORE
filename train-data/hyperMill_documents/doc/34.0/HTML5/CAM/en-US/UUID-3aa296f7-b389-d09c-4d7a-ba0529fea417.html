Contours

Probing mode

Automatic (1): Select when the probing direction is to be detected automatically.

Manual (2): The probing direction can be set manually for individual probing points. To do this, activate under Contour attributes the required points and the Reverse option.

Contour selection / faces

Select Points and Surfaces. The points to be measured must be located on the selected surfaces.

Offset: If there is an allowance on the model to be measured, define a corresponding offset for the measuring points.
