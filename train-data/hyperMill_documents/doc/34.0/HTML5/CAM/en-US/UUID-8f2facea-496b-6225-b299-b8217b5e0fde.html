Calculate stock

Stock model calculations can be performed for job lists or individual jobs. The resulting stock model is calculated based on the previously defined stock model. A resulting stock model can be used as the basis for subsequent machining (stock model update).

Calculate in the job definition

On the Setup dialog page of the job definition, define the model and stock model and select the Generate resulting stock option.

Undercut trimming: Avoidance of unnecessary empty paths in undercut areas during multi-axis stock indexing.

Calculate in the browser

In the hyperMILL browser, select the required job list or job on the Jobs tab and use the Create → stockmodel function on the shortcut menu.

The Stock model calculation dialog box opens.

Click Path to select the required file in *.stl or *.vis format. The path and file name are displayed.

STL/VIS: Specify the file format and tolerance for the calculation. For medium accuracy, a default resolution of 0.5 is adequate.

Create stockmodel: Select this option to generate an entry in the hyperMILL browser for the stockmodel.

Optimized stock model generation in 5X machining

The prerequisite for optimized stock model generation is that the Cutting length parameter is defined in the tool definition of the underlying job. The following tool types are not subject to this requirement: Lollipop, Barrel cutter, T-Slot cutter and Woodruff cutter.

The following applies to tapered and non-tapered tools: If no Cutting length is defined (value is 0), then the Tip length parameter is used. The following applies for ball mills and bullnose end mills: If the Cutting length is smaller than the Corner radius, then the Corner radius is used.
