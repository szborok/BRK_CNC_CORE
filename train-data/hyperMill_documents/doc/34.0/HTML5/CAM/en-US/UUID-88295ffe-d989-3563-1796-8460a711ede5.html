Advantages and uses

  • Automation and recycling of workflows: Workflows can be automated and recycled using macros. This significantly reduces programming time, particularly when it comes to producing 2.5D parts.

  • Efficient macro management: Your macros reside in a macro database. This provides a production knowledge base that is efficiently and easily managed.

  • Updating workflows: You can modify your existing macros at any point to match user-specific new production requirements as they arise.

  • Higher degree of automation thanks to ‘intelligent’ macros: Using ‘intelligent’ macros, operations that were previously assigned to a feature manually can be automatically linked. Rules and conditions are defined to allow hyperMILL to automatically decide which machining will be applied. This method can therefore reduce the number of macros needed to machine a complete model.
