MB Flank Milling

Strategy

Circumferential, swarf cut machining of blade surfaces.

The cycle should preferably be used when the blade surfaces allow for sufficiently accurate tool fit to surface. Compared to point milling, this cycle thus achieves shorter machining times.

Requirements:

  • The blade surfaces must be regular surfaces.

  • The selected blade surfaces should not contain a transition radius to the hub.

  • You must select the trim curves to the left and right of the edge.

Milling reference

Main blade / Splitter blade: Specify the blade you want to machine. The selection of the splitter will not be available, if it has not been defined on the geometry model.

Entry position

The approach generally takes places on the edge that is not rounded (normally the trailing edge). However, the given space must also be taken into account.

Infeed strategy

This option is only important if you want to perform several infeeds in the axial and lateral direction.

Axial infeed first: All infeeds in the axial direction are carried out for each lateral layer before the next lateral infeed occurs.

Side infeed first: All lateral infeeds are carried out for each axial layer before the next axial infeed occurs. To protect the cutting tool, you can optionally define an axial retract for every lateral infeed.

Edge Milling

The following options are available for machining the blade edges:

None: There is no machining of the blade edges. No edge surface(s) is/are selected at the start position (usually the blunt edge) and the edge distance is > 0.

Enclosed edge only: Only the enclosed blade edge is machined. No edge surface(s) is/are selected at the start position (usually the blunt edge) and the edge distance is 0.

Both: Both (round) blade edges are machined.
