Tools

The following cutter types can be used: ball mill, end mill, bullnose end mill and lollipop.

Tapered tools may be used, but the conical shape is not used in the toolpath collision avoidance calculations. When using tapered tools, please ensure that you perform a visual check for collisions using a simulation.

Reference tool

The reference tool is used to calculate the grooves. It is of the same type as the machining tool. The reference tool is defined based on the diameter. As a rule of thumb: You want the diameter of the reference tool to be up to 20% greater than the diameter of the machining tool.

Note

The reference diameter should be selected so that the reference tool can no longer completely remove the stock of the groove. In other words, there should always be two contact points with the surface so that the groove can be recognized.

Frame

The frame orientation is valid for computing the grooves as well as for machining.
