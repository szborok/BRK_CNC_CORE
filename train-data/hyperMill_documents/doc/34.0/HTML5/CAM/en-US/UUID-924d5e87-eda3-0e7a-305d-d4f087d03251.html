Fit hole to stock model

Fit hole to stock model (1): Adjust a hole feature to an existing or modified stockmodel geometry. Select the hole features to be adjusted in the hyperMILL feature browser and select the Fit hole to stock model function from the shortcut menu (right-click). Spots will also be taken into account.

Selection

Automatic: Adjusts all selected hole features to the selected stock model. Note that you may have to ungroup combined features if this is necessary in order to adjust the stock model geometry (for more information, see the Tolerance option).

Manual: Manually define how to adjust the hole features to the stock model geometry. A negative stock allowance is not possible at present. The changes associated with the Manual option only concern the top side of the hole (origin).

Fit cones: Adjusts the chamfer of a Simple Hole, the sink of a Sink Hole or the cone of a Free defined Hole to the top of the stockmodel geometry, in accordance with the defined angle (1).

Top only (1): Adjusts the top side of the hole to the milling area.

Tolerance: Specify a higher tolerance value to prevent the hole features of a group from being ungrouped. If the tolerance value is higher than the difference of the distances between the smallest and largest features in the group for the stock model used, the features continue to be grouped together.

Hole features are compared on the basis of existing parameters (exception: position) and, if possible, are grouped to form new features.

Check milling area: When the hole feature is generated, the milling area is checked and the hole feature is adapted to the milling area. Function not enabled (1), function enabled (2).

Use the Fit hole to stock model function to generate a new feature in the feature browser. Enable the Remove original hole option if you want to delete the features that were originally selected for the function.

Use the Save as default function to save the value for the model tolerance as a default value.
