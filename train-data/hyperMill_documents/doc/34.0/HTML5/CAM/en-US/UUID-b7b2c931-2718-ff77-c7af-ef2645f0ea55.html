The Feature job connector

The Feature dialog page of the job definition is known as the Feature job connector. It is used to link features with the current job and comprises

  • the icons to select features (1),

  • the top area with the list of linked features (2),

  • the bottom area with representation of the feature parameters (3).

Connected features

Features linked to the job are listed in the Feature Job Connector. The number of features is displayed.

Click on the icon to open the Select feature dialog in order to link a feature with the current job.

Features which should be used in the jobs can be moved to the right window by double-clicking or clicking on the button (1).

Features which should not be used in the jobs can be moved to the left window by double-clicking or clicking on the button (2).

Feature derived: Production-related feature parameters are represented in the bottom area of the Feature Job Connector. A feature parameter, whose status is set to active, is used in the job definition for calculation purposes. If the status is inactive, the parameter is not used.

Formula: Various geometric parameters of the features: Simple Hole, Sink Hole, Free defined hole, pocket (for use in contour milling cycle) and planes can be defined using a formula. All basic calculating operations are permitted as arithmetical expressions for the purposes of defining the formula. In this case, the feature parameter can be used as a variable.

View feature: The feature linked in the job is highlighted in the graphics window.

For information on using the Feature job connector for job definitions, see section Use feature.
