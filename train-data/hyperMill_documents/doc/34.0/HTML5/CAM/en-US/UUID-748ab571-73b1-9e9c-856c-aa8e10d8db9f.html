Parameters

Vertical infeed mode

Fixed step: The value defined under infeed for the vertical infeed is retained. The value for the last infeed is automatically adjusted to the machining depth (dialog page Contours → Top/Bottom).

Fit step: The value defined under Infeed for the vertical stepdown is automatically adjusted in such a way that all of the Z distances are the same. This is done while taking account of the machining depth (dialog page Contours → Top/Bottom).

Infeed

Vertical stepdown: Z infeed to the next machining pass.

Allowance Z: Remaining material that is to be removed in subsequent fine machining. Distance from the final contour that the cutter has to keep.

Retract mode

Clearance plane (1): all vertical retract and stepdown movements to the next machining plane take place via the clearance plane.

Clearance distance / Lead-in position (2)

The stepdown movement to the next workplane is realized with Contour Milling at the clearance distance (A), with Pocket Milling at the lead-in position (start position + lead-in macro) without retraction in the Z direction (B). Approach and retract macros are executed at each machining plane.

Clearance

Select the clearance plane (1) as plane for rapid travel movements along all three axes (absolute dimensions). Below this plane, only the infeed in Z direction is in fast travel mode up to the clearance distance. For a 3D model, the clearance plane value can be taken from the drawing.

Warning

The clearance plane is always situated above the workpiece boundaries in the Z direction.

Define a clearance distance (2) as a distance from the current surface (3), for fast travel Z-stepdown infeed (relative dimensions). Below the clearance distance, Z-stepdown takes place at feedrate.
