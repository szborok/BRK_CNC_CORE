3. Turning tool: Creating an insert tool holder

To create an insert tool holder with the TOOL Builder, proceed as follows.

In the tool database, on the Tool Holders dialog page, click on the icon for  Geometry → 3D file to start the TOOL Builder.

Then use the File → Open dialog to open a 3D geometry file in *.step,*.stp, *.iges or *.hmc format.

Define body

Specify the geometry of the insert tool holder.

Select Entities. The selected geometric entities are highlighted in color and the number of entities is shown. The data is saved on a designated layer and marked with a tag.

Define axis

The geometry of the insert tool holder is recognized automatically based on the coaxial faces of the body and the program automatically suggests the axis alignment (machine side/workpiece side). A reference coordinate system is created, whose alignment corresponds to the Z axis of the insert tool holder.

Manually adjust the Direction of the axis if necessary. To do so, select entities or 2 points. The order in which the points are selected defines the direction of the axis. Invert the Direction by double-clicking the axis arrow or use the Invert command.

Define coupling

Select a face or a point. The selected entity is highlighted in color.

Specify the X direction of the reference coordinate system. To do so, select 2 points or a single point.

Click the button to rotate the X axis direction by 90°.

Use the Invert function to reverse the direction. This can also be adjusted directly for each entity in the corresponding row (red, green, blue). Use Save settings to save your changes.

Define outline profile

The program automatically generates an outline profile based on the defined body geometry and the alignment of the X axis in the reference coordinate system.

Define insert

Select the entities. The selected entities are highlighted in color.

It is not absolutely necessary to specify the geometry of the insert in order to define the tool holder.

To define the Mounting position, specify a Face or an Origin. The selected entities are highlighted in color.

Specify the alignment of the insert by defining the X direction of the reference coordinate system . To do so, select either Entities, a Point or 2 points .

Generate STL file

The program automatically generates an outline profile based on the defined body geometry and the alignment of the X axis in the reference coordinate system.

Export data

Depending on the imported 3D file, Geometry parameters of the insert tool holder may be calculated and transferred to the tool definition during data transfer (axis distance, shank distance, rake angle, inclination angle, approach angle, mounting direction, spindle orientation).

Note

The values for the Rake angle and Inclination angle parameters are calculated exclusively when defining an insert holder with the TOOL Builder software. Manual definition in the tool database is not possible. The Rake angle and Inclination angle parameters are currently not supported for recessing tools, threading tools and parting tools.
