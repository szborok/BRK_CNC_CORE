Finish clamping

Insert the correctly positioned fixture as a clamping range into the joblist. In doing so, finish the clamping.

Template commands → Fixtures → Finish clamping

Basic

Template commands → CAM → Fixtures → Finish clamping

Advanced

If all parts of the fixture are correctly positioned on the CAD side, insert the fixture as a hyperMILL® clamping range into the active joblist. The fixture layer is changed.

Afterwards, the fixture can no longer be deleted on the CAD side.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Prefix fixture layer name: Enter a prefix string for the fixture layer.

Tolerance: Maximum permissible distance between the faceted top of the polyhedron model and the real model surface.

Tool safe distance: Enter a minimum distance of the tool to the clamp.

Stop for input

Execution of the script is halted so that information can be entered.
