Read segment from XML file (position)

Read segment content from XML file.

Template commands → Files → XML Files → Read → Read segment from XML file (position)

Advanced

Read segment content from XML file. The result is not output in the dialog. The data that is read in is assigned to tool parameters.

Example 115. Determine parameters of a tool

Determine the parameters of a specific tool in an OPEN MIND tool database file [2] in order to make them available for further script steps in a loop cycle, for example.

Input: Segment: tool, Position: 3

<?xml version="1.0" encoding="UTF-8"?>
<omtdx>
	<coolants />
	<cuttingMaterials />
	<couplings/>
	<inserts />
	<toolHolders />
	<tools>
		<tools folder="Turning tool">
			<tools folder="TT1100 Axial grooving tools">
				<tool type="recessingTtool" name="318842 B2.2"/>
				<tool type="recessingTtool" name="318842 B3.1"/>
				<tool type="recessingTtool" name="318842 B3.2"/>
				<tool type="recessingTtool" name="318842 B3.3"/>
			</tools>
		</tools>
	</tools>
</omtdx>


Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Filename

Enter the file name of a file that exists in the selected folder.

Right-click the arrow at the right edge of the entry to open the application if the file extension is assigned in Microsoft Windows. (Like double-clicking a file in Microsoft Windows Explorer).

Parent: Enter the name of the required parent segment.

Segment: Enter the name of the required segment.

Position: Enter the position of the required segment resulting from the arrangement in the *.xml.

Get parameter: If the value of an attribute is required, specify the attribute here.

Data set name: Add script variable names to identify them as related data.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.



[2] This example is a highly simplified tool database file.
