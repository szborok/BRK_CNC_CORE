Manage scripts

In the main dialog, call the dialog for managing subscripts using the button. A list of all scripts is displayed . Select and apply a script .

For creating scripts and subscripts:

Create a new script.

Duplicate an existing script. Enter a new name for the duplicated script.

Rename an existing script.

Changing a script name creates a ‘new’ script. This script must be added to the toolbar and/or menu again.

Delete the selected script.

Search

Search in the list of all scripts or set a filter for the scripts that are available for selection in the main dialog. Enter a string to search for. Use the button to remove the search string.

Only available when creating scripts:

Configure defaults.

Export script.

Include used files: Include all files used, for example, *.xlsx in the export of the script.

Import script.

Copy referenced files local: Import and create and reference related data substructure.
