Predefined parameter table

Processing of entries for options in functions. Insert using arrow icon on the right in table row. Can be used for each option to carry out ‘local’ calculations, for example, to round values, read in translations, and so on.

Text editor

A text editor is located below the parameter table. Double-clicking a value inserts it into the text editor. Each inline parameter encloses the content in its brackets.

Inline functions

An example selection:

biggest

Use biggest value

biggest(10:20:30) results in 30

calc

The result of the calculation is always a whole number.

calc(1+2)

decimal

Number of decimal places

decimal()

file

Separate the pure file from a complete file name.

file()

ge

Value greater or equal to

ge(10:10 ; true:false) results in true

gt

Value greater than

gt(10:20 ; true:false) results in false

le

Value lower or equal to

le(20:20 ; true:false) results in true

lead

Pad with zeros before first number

lead(50:4) results in 0050

local

Read in translation

local([Developer_name]|#[LOC_ID])

For [Developer_name], enter a name as the work title in case there is no localization file. For [LOC_ID], enter the relevant ID of the *.loc file with the translation.

lt

Value lower than

lt(10:20 ; true:false) results in true

path

Separate the directory of a complete file name.

path()

smallest

Use smallest value

smallest(10:20:30) results in 10

condition

Check condition. A condition can also be used in a decision table.

condition(1<2) results in True
