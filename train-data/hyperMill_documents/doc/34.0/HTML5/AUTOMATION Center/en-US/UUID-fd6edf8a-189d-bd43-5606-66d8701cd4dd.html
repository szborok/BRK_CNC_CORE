Coloring arcs

Color circles and circular arcs.

Template commands → CAD → Coloring → Coloring arcs

Advanced

Circles and circular arcs are searched for as specified and colored in the selected color.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.

Color all circular arcs or limit entities to be colored using optionally selected specifications:

Min. X value, Min. Y value, Min. Z value: Optionally color entities whose coordinate values are greater than specified in the minimum values.

Example 93. 

Min. X value: 0, Min. Y value: 0, Target color: fuchsia



Max. X value. Max. Y value, Max. Z value: Optionally color entities whose coordinate values are smaller than specified in the maximum values.

Example 94. 

Max. X value: 0, Max. Y value: 0, Target color: green



Min. radius / Max. radius: Optionally color entities whose radius is between the minimum and maximum radius.

Example 95. 

Min. radius: 8 mm, Max. radius: 9 mm, Target color: red



Min. segment angle / Max. segment angle: Optionally color entities whose segment angle is between the minimum and maximum segment angle.

Example 96. 

Min. segment angle: 90°, Max. segment angle: 360° and additional Min. radius: 8 mm, Max. radius: 9 mm, Target color: blue



Min. middle angle / Max. middle angle: Optionally color entities whose curve middle is between the minimum and maximum angle in relation to the workplane.

Example 97. 

Min. middle angle = 0°, Max. middle angle = 90°, Target color: red



Example 98. 

Min. middle angle = 270°, Max. middle angle = 360°, Target color: red
