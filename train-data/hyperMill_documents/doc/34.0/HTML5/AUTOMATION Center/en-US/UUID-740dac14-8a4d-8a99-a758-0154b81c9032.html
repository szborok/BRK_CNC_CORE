Move manually

Move entities interactively in one direction or rotate around the Z axis.

Template commands → CAD → Transformation → Move manually

Advanced

Trigger a movement. To do this, select the required direction of movement in relation to the directions of the current workplane or a rotation around the Z axis of the current workplane.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Move: Enter a value for the shift in the selected direction.

Angle: Enter an angle for the rotation around the Z axis of the current workplane.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Stop layer: If faces on these layers or layer containers collide in the direction and distance of the movement, the movement is stopped and not executed for the entire required path. Enter layer names.
