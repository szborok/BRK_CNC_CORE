Blending shape between 2 faces

Create a blending shape between two faces.

Template commands → CAD → Shapes → Blending shape between 2 faces

Advanced

2025

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Face layer 1

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color 1

Enter an RGB color value for a first section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Face layer 2

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color 2

Enter an RGB color value for a second section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.
