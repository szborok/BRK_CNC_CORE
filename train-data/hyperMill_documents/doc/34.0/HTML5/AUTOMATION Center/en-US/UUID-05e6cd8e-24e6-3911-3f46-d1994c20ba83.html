2.6. AUTOMATION Center Controller

Structure and process existing scripts without calling AUTOMATION Center.

The AUTOMATION Center Controller can be found in the hyperMILL®CAD pull-down menu structure under CAM → AUTOMATION → AUTOMATION Center Controller.

Advanced

2025

The AUTOMATION Center Controller makes it possible to structure existing AUTOMATION Center scripts in a meaningful way in order to obtain an overview of their purpose. The scripts can be processed without having to activate AUTOMATION Center separately for each individual script.

Automation is considered useful even if there are no workpiece families, as almost every workpiece requires similar process steps, e.g. work preparation and the creation of workshop lists. The AUTOMATION Center Controller can be seen as a toolbox for making processes more efficient. Large scripts can be used multiple times, which is helpful to reduce complexity and allow more flexible handling of existing scripts. A process can be easily interrupted and resumed by means of suitable structuring.

UI structure

Select Script name and Script path.

In the hyperMILL® settings in Application, the following path is stored for the AUTOMATION Center by default C:\Users\Public\Documents\OPEN MIND\AutomationCenter. The data for AUTOMATION Center can be found here. If you take a closer look at this folder, you will see that it contains a folder called AC_CONTROLLER, which essentially only contains a folder structure that is created by the system.

Run – depending on the starting point, the section is run through or only the relevant script is executed. Also by double-clicking.

Set environment variable

  • AC_SCRIPT_BASE_PATH

    Defines the start of the AUTOMATION Center script base folder. The purpose of this variable is to facilitate the copying of multiple scripts between different computers.

  • ACC_BASE_PATH

    Defines where the folder tree of the AUTOMATION Center Controller should start reading the structure. If there are several user groups in a company, they can be given different scripts to choose from.

Highlight favorite scripts

In the path defined in the environment variable ACC_BASE_PATH, *.accf files can be saved, where “accf” stands for AUTOMATION Center Controller filter. By default, this is the subfolder .../AC_CONTROLLER of the default path for the AUTOMATION Center from the hyperMILL settings. The *.accf file is a simple text file.
