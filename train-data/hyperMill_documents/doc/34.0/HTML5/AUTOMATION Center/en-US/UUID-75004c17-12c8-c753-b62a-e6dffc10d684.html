Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Excel file: Select an Excel file. The file folder is transferred from Folder. The stored folder is opened if a script variable is used for Folder.

Cell address and Description: Enter cell addresses in which values are to be changed. Enter a short text in the description for displaying in the script run.

Description and Value: The first call copies the descriptions into the left column and inserts the current values from the Excel table. If no values are found, the value from the default setting is used. These values can be changed by the user during the script run and transferred to the Excel table in the specified cell address.

Condition

Check whether a script variable is <, >, = or ≠ to a value.

Info cell address: If available in the Excel worksheet, enter a cell address for checking purposes that characterises the content of the cells (of a row) to be modified.

Info value: The content of the info cell is displayed on the first call. It is possible to check whether the correct row is being processed.

Default: This value is used if there is no content in any of the respective cells.

Sheet name

Enter the name of the Excel worksheet.
