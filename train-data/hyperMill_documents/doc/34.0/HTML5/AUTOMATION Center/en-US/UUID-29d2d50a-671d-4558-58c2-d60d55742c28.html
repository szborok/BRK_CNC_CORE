Coloring lines

Color straight lines.

Template commands → CAD → Coloring → Coloring lines

Advanced

Search for straight lines as specified and color them in the selected color.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.

Color all lines or limit entities to be colored using optionally selected specifications:

Min. X value, Min. Y value, Min. Z value: Optionally color entities whose coordinate values are greater than specified in the minimum values.

Example 89. 

Min. X value: 0, Min. Y value: 0, Target color: red



Max. X value. Max. Y value, Max. Z value: Optionally color entities whose coordinate values are smaller than specified in the maximum values.

Example 90. 

Max. X value: 0, Max. Y value: 0, Target color: red



Min. length / Max. length: Optionally color entities whose length is between the minimum and maximum length.

Example 91. 

Max. length: 50 mm, Target color: red



Min. angle / Max. angle: Optionally color entities whose direction in relation to the workplane lies between the minimum and maximum angle.

Example 92. 

Min. angle: 15°, Max. angle: 35°, Target color: red
