Arc (point on curve-tangent-tangent)

Create an arc between two tangential lines at one point.

Template commands → CAD → Drafting → Arc (point on curve-tangent-tangent)

Advanced

Example 51. Arc between two intersecting lines

  • Input: Tangent 1 color (RGB)1: 73;126;253 | Tangent 2 color (RGB): 0;255;0 | Point color: 255;0;0 | Curve color: 128;0;64



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Tangent 1 color (RGB): Enter the color of the first tangent.

Tangent 1 layer: Enter the layer of the first tangent.

Tangent 2 color (RGB): Enter the color of the second tangent.

Tangent 2 layer: Enter the layer of the second tangent.

Point color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Points layer

Enter or select the points layer. Only uses points that are assigned to this layer.
