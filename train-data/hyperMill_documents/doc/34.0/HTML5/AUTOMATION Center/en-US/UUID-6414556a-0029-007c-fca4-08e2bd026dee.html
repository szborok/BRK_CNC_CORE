Workplane (dialogs)

Generate or modify a workplane with CAD commands.

Template commands → CAD → Workplanes → Workplane (dialogs)

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Command: Call a CAD command.

Select one of the following functions:

  • Through three points

  • On curve

  • On face

  • Move

Open user interface

Specify whether the user interface should be opened.

The options vary for the following functions:

On curve

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

On face

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Points layer

Enter or select the points layer. Only uses points that are assigned to this layer.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Point color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.
