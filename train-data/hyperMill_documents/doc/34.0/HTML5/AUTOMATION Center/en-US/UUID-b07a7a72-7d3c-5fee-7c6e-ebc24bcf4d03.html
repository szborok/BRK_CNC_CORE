2.5. Server controller

Software for defining a period of time for unobserved calculations in which the AUTOMATION Center server PWservere.exe should work. Start the software via the Windows Start menu or in ....\OPEN MIND\hyperMILL\[Version]\addins\hmAutoColor as PWserverStart.exe.

Server start time

Start calculations on AUTOMATION Center.

Server stop time

The AUTOMATION Center server is stopped, hyperMILL® remains open. A calculation in progress is terminated. A log is not kept.

Always on

Do not limit the time period.

Start immediately.

Stop immediately.
