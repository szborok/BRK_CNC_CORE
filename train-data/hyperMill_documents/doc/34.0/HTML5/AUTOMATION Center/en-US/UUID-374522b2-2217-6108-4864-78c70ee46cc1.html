Manual adjust fixture

Align the fixture with various functions integrated here.

Template commands → Fixtures → Manual adjust fixture

Basic

Template commands → CAM → Fixtures → Manual adjust fixture

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Rotation angle: Enter a rotation angle.

Shift: Enter a value for the shift in the X direction.

The direction for the alignment is X. To shift the fixture in the Y direction, it must be rotated first.

Shift in +Z direction: Enter a value for the shift in the Z direction.

Opening distance: Enter distance in the open state.

Function buttons in the user interface:

Apply rotation

Fit clamping to stock

Fit clamping to milling area

Shift in X direction

Shift in Z direction

Adjust fixture opening

Delete fixture

The deletion of a fixture can only be applied before setup has been completed with this function.
