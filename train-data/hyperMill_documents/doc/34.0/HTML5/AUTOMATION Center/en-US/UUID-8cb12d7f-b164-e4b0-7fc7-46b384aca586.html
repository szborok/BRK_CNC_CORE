Load features

Load Customized Process Features or features (Contour, Strategy Curve, Face Group).

Template commands → CAM → Feature → Load features

Advanced

Load Customized Process Features, abbreviated as CPF, or features (Contour, Strategy Curve, Face Group) from a *.fst file in a subfolder of the feature file folder.

Example 37. 

  1. Select a feature in the hyperMILL® browser, on the Feature tab – also called the feature browser.

  2. Right-click while holding down the CTRL key to open the context menu and select the Export features function. Save the feature, for example, in a Allfeature.fst file in the AUTOMATION Center feature folder.

    Note

    The ...\FEATURE\[FEATURECLASS] feature folders must be located in the defined default path that is specified under hyperMILL → Setup → Settings in Application under Default paths in the AUTOMATION Center branch.

  3. When compiling the script inAUTOMATION Center, make sure that theSelect feature folder function is inserted as a script step before the Load features function.

  4. During the script run, in the Select feature folder function in the Feature folder option, select the subfolder created for the feature category in ...\FEATURE\.

  5. Afterwards, in the Load features script step, the feature from the Allfeature.fst file is offered in the left column of the dialog.
