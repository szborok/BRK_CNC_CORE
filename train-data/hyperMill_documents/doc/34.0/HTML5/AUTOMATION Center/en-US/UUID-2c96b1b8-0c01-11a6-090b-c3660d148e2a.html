Select library

Select CAD data folder.

Template commands → AC general → Preparation → Select library

Advanced

Select a CAD data folder to be able to then select CAD model data. See Load model from library.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Library: From the drop-down list, select a folder in which the different CAD data of the various categories is located.

Note

For each CAD data category, a ...\LIBRARY\[CADMODELTYPE] folder must be located in the defined default path that is specified under hyperMILL → Setup → Settings in Application under Default paths in the AUTOMATION Center branch.

Stop for input

Execution of the script is halted so that information can be entered.
