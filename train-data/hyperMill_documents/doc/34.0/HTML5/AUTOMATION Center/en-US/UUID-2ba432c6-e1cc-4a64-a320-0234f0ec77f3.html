Color / layer script variable

Assign values to a color and a layer script variable.

Template commands → AC general → User interface → Color / layer script variable

Advanced

To copy the variable, click the icons . Replace the variable names with values that are to be used in the script.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Color

Enter or select a color with RGB values.

Layer

Enter a layer that is to be used several times in the script, for example.

Stop for input

Execution of the script is halted so that information can be entered.
