Clamping position from workplane

Create clamping position from workplane.

Template commands → NCS → Clamping position from workplane

Basic

Template commands → CAM → NCS → Clamping position from workplane

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Name of NCS: Enter a name.

NCS position: Select a position from various positions of the milling area bounding box.

Possible positions:

  • Bottom-Center

  • Bottom-Center-Back

  • Bottom-Center-Front

  • Bottom-Center-Left

  • Bottom-Center-Right

  • Bottom-Left-Back

  • Bottom-Left-Front

  • Bottom-Right-Back

  • Bottom-Right-Front

  • Top-Center

  • Top-Center-Back

  • Top-Center-Front

  • Top-Center-Left

  • Top-Center-Right

  • Top-Left-Back

  • Top-Left-Front

  • Top-Right-Back

  • Top-Right-Front

  • Workplane

Direction: Select a direction for the Z axis. The direction is used for the NCS in the joblist.

Keep axis: Select an axis for which a current position is to be maintained.

Fixed X Position: The X value of the position is not changed. Therefore, if the X and Y values are fixed, only the Z value will be modified.

Fixed Y position: The Y value of the position is not changed.

Fixed Z Position: The Z value of the position is not changed.

Stop for input

Execution of the script is halted so that information can be entered.
