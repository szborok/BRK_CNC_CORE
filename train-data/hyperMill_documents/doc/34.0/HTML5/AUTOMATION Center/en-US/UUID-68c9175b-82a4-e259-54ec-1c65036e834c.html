Positioning by coaxial cylinder

Position a part so that two cylinder axes are brought into alignment.

Template commands → CAD → Transformation → Positioning by coaxial cylinder

Advanced

Position a part so that two cylinder axes are brought into alignment. The positioning at height is performed automatically. The lower edge of the source cylinder is placed on the upper edge of the target cylinder. A suitable workplane must be chosen.

Also see Rotation by coaxial cylinder.

Example 62. Position clamping entity on perforated plate

Color the cylinder faces in advance, for example, using Smart interactive coloring.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.
