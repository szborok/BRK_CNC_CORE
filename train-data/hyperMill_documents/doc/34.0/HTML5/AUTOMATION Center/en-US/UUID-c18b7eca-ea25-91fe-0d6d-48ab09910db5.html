Probing points

Create probing points on contours of plane faces.

Template commands → CAD → Points → Probing points

Advanced

Create one to three points on contours, generated from a plane face. The contour may only consist of analytical entities (no NURBS). The first point is the middle point of the limit diameter. The second point is the one with the greatest distance to 1. The third point is the one with the greatest distance to the line between 1 and 2.

Example 52. 

The contour is the input information. The points are the result. The green point is the first point. It looks for the position with the largest possible face.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Points layer

Enter or select a layer for the points.

Points 2 and 3 are only generated if a point color is available:

Point color 3 (RGB): Select point color.
