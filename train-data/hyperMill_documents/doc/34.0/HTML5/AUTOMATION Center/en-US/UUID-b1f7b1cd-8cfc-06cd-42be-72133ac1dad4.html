Check color interaction

Check the correct number of entities of a desired color (after a manual interaction).

Template commands → CAD → Coloring → Check color interaction

Advanced

Check the correct number of entities of a desired color (after a manual interaction). If the number of entities of the checked color is incorrect, the function in the script jumps back to the last script step with manual interaction. This gives the user the opportunity to correct their selection.

A message is displayed if the number of entities is not correct. In the higher-level Component options, the Error level must be set to Input/Output.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Message: Enter a message text.

Color

Enter or select the color. Only entities of this color are used.

Enter the minimum and/or maximum number of entities necessary to execute the calculation correctly.

Min. count: Enter the number of entities.

Max. count: Enter the number of entities.

Filter: Select the type of entity for which to run the check. Faces, planes, cylinders, curves or all these entity types can be checked.

Reset color in case of error: Selecting Yes discards any changes to the color of entities during the script run.

Condition

Check whether a script variable is <, >, = or ≠ to a value.
