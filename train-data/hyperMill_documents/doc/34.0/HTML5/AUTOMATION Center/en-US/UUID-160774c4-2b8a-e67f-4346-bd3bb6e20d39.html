Read value from XML file

Read a value from an XML file.

Template commands → Files → XML Files → Read → Read value from XML file

Advanced

Read a value from an XML file.

Example 116. Read the value for the distance between two points

The function Dump two entities in XML was used to output information about geometric properties of two points in relation to each other to a D:\AC\Result.xml file. Now the distance between the two points is to be read in.

Input: Folder: D:\AC | File name: Result.xml | Segment: property| Search name: MinDistance | Search parameter: |

for

<PBag name="info.twoentities.description"
      version="1">
	<property name="Deltax"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">10</property>
	<property name="Deltay"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">0</property>
	<property name="Deltaz"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">0</property>
	<property name="MinDistance"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">10</property>
</PBag>

Result: Result: 10



Example 117. Narrow down the search to segment using two-stage search

The Write feature to XML file function was used to output information on CAM features to a D:\AC\Exported_features.xml file. The task now is to determine a property of a feature in this file.

01 <?xml version="1.0" encoding="UTF-8" standalone="yes"?>
02 <Features>
03   <Feature catalog="OM" type="Hole" 
              name="Drill Hole" uuid="19E92F77-3D9E">
04     <Parameter name="Hole::Diameter" value="10"/>
05     <Parameter name="Hole::Depth" value="50"/>
06     <Parameter name="Bottom_Type" value="Through"/>
07     <Parameter name="Bottom_Offset" value="0"/>
08     <Parameter name="Orientation" value="0,-1,0"/>
09     <Parameter name="Position" value="7.2,-25,16.1"/>
10   </Feature>
11   <Feature catalog="OM" type="Hole" 
              name="Drill Hole" uuid="EEDB0AC0-94E2">
12     <Parameter name="Hole::Diameter" value="10"/>
13     <Parameter name="Hole::Depth" value="50"/>
14     <Parameter name="Bottom_Type" value="Through"/>
15     <Parameter name="Bottom_Offset" value="0"/>
16     <Parameter name="Orientation" value="-1,0,0"/>
17     <Parameter name="Position" value="-25,1.7,10.0"/>
18   </Feature>
19   <Feature catalog="OM" type="Hole" 
              name="Drill Hole" uuid="1E6B8223-9108">
20     <Parameter name="Hole::Diameter" value="10"/>
21     <Parameter name="Hole::Depth" value="50"/>
22     <Parameter name="Bottom_Type" value="Through"/>
23     <Parameter name="Bottom_Offset" value="0"/>
24     <Parameter name="Orientation" value="0,0,1"/>
25     <Parameter name="Position" value="-0.1,4.4,50"/>
26   </Feature>
27 </Features>

  1. Insert the Read value from XML file function into the script and execute it.

    Input: Folder: D:\AC | File name: Exported_features.xml | Segment: Feature | Search parameter: uuid | Search name: 1E6B8223-9108

    Results: Found in line: 19

  2. Insert the Read value from XML file function into the script a second time and execute it.

    Input: Folder: D:\AC | File name: Exported_features.xml | Start search line: 19 | Segment: Parameter | Search parameter: Name | Search name: Orientation | Get parameter: value

    Results: Result: 0,0,1 | Found in line: 24



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Filename

Enter the file name of a file that exists in the selected folder.

Right-click the arrow at the right edge of the entry to open the application if the file extension is assigned in Microsoft Windows. (Like double-clicking a file in Microsoft Windows Explorer).

Start search line: In a two-stage search process, enter here the line number that was previously determined.

Segment: Enter the name of the required segment.

Search parameter: Optionally enter the value of the required type attribute.

Search name: Enter the value of the required name attribute.

Get parameter: If the value of an attribute is required, specify the attribute here.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.

Found in line: Outputs the line number in which the desired result was found.
