Using the PW Server with Excel files

Configuration file

Before starting, some parameters have to be defined. These parameters can be defined in the autoProgramConfig.pcf file. This configuration file is located at C:\Programme\OPEN MIND\hyperMILL\[Version]\addins\hmAutoColor. In this example, the version is 29.0.

Three parameters must be defined:

*EXCEL_FOLDER=c:/temp/defaultExcel.xlsx

*DEFAULT_TEMPLATE=c:/temp/HCS_template.hmc

*DEFAULT_TEMP_START_FILE=c:/temp/start.hmc

The Excel file located in the server’s internal folder is copied to the name defined in the *EXCEL_FOLDER parameter.

The file defined in *DEFAULT_TEMPLATE is copied to the file defined in *DEFAULT_TEMP_START_FILE.

hyperMILL® is started with *DEFAULT_TEMP_START_FILE.

In the script, the user should read the Excel file from *EXCEL_FOLDER. It is assumed that a script name has been defined for the process. It must be called from the start script.

A model file is to be found in Excel that should be merged with the current model.

The file should be saved in a location defined in the Excel file.

The file must be edited as administrator.

The file configured as an example:

hmct document template

It is possible to define a hyperMILL® template to be used in the process. Define layers, colors and so on in this template. The AUTOMATION Center copies the template to DEFAULT_START_FILE and starts based on this file.

Template folder

Only the file defined in DEFAULT_TEMPLATE can be saved in this folder.

Excel file

Example: Three parameters defined in the Excel file

  • Use File path to define the location of the 3D model to be merged.

  • Use File to define the 3D model to be merged.

  • Use Script to define the script to be started.

Master script to run this type of application

Create a simple script that is called when the Excel file is present in the Watch folder. In this case, read the Excel file and use the information to merge the file and call the script. This means that the file and the script based on the Excel file will be completely flexible.
