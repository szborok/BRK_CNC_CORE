Get single tag values

Retrieve the tag value for each required entity.

Template commands → CAD → Tags → Get single tag values

Advanced

Retrieve every single value of the tags analyzed in the Read all tag values function. A loop is necessary for this. The Read all tag values function must be placed in the script before the loop. Also see Read all tag values and Change values by decision table.

Example 100. Color faces differently based on tag information.

In a *.hmc there are four faces of the same color to which tags are assigned.

  1. Determine how many different values occur for the ‘T_ROU_PMETHOD’ tag. To do this, insert the Read all tag values function into a script and execute it.

  2. Insert a component into the script and configure it as a counted loop. For the number of runs, take the result from step 1.

    1. Run the Get single tag values function.

    2. Run the Change values by decision table function.

    3. Run the Change color or layer function.

      Input: Tag name: T_ROU_PMETHOD | Tag value: Retrieve the result from individual tag value, for example, M14 | Target color: Retrieve the result from Change values by decision table, for example, 128;0;128

The four faces are displayed in a different color.



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Number: Use §CURRENT_LOOP_INDEX# to control which value is to be processed.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.
