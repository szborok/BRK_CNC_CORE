2.1. Main dialog

Figure 1. Main dialog areas (schematic representation)


Script: Displays the active script. Select another script. In , you can select from the history.

Subscript: Select one of the subscripts of the active script or use the to open the dialogue for managing subscripts. As in the dialog for managing scripts, new subscripts can be created there and existing subscripts can be copied, renamed, and deleted.

To work with subscripts, activate the option Enable subscripts in Configuration.

Search

Search script. Enter a string to search for. Use the button to remove the search string.

Overview of the active script.

Commands on the context menu:

Collapse all: Collapse structure tree.

Expand all: Expand structure tree.

Double-click with the left mouse button to switch to thevariant definition mode.

Import component: Import a component or subcomponent of a script previously exported to the ...\COMPONENTS folder (default setting) in order to reuse it at the selected location. Select the required file in *.ACcomp format.

Current status or current function in the script.

Enter parameters for each of the functions currently selected in the script overview.

To find out where a parameter comes from, rest the mouse pointer on the input field. A tooltip is displayed. Optionally use the node connection editor.

Double-click with the left mouse button to switch to and from theparameter setup mode.

Create and manage bookmarks.

Bookmark script steps to jump to this script step within the script by double-clicking the bookmark. All existing bookmarks are displayed in the dialog – both from the master script and from its sub-scripts. The bookmarks make it easy to jump back and forth between the different scripts.

Create a script bookmark for a script step. Select the script step beforehand.

Delete a script bookmark. Select the bookmark beforehand.

Image field for an Instruction during the script run.

Text panel for an Instruction during the script run or the documentation of a script step while defining a script.

Node connection editor

Lists of the active parameters with their script variables and values.

Manage scripts (contains things such as Create new script, Copy script and so on)

Prepare a script.

Enable or disable editing the Parameter definition.

Control processing of the active script:

Continuous mode

Continuous mode runs through the entire script until a stop command is reached or input is necessary.

Step mode

Step mode runs only the selected steps.

Pause processing.

Processing is interrupted at the current position. Processing can usually be restarted by clicking the button again.

Stop script execution

Stopping can be switched on and off for each step in the script. Continuous mode stops at the step marked with a stop.

Show and hide parts of the main dialog:

Show the node connection editor The node connection editor is located on the left-hand side of the main dialog.
