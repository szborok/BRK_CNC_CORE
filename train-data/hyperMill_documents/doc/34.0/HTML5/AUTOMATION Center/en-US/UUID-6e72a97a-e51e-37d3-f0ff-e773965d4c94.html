Change faces color or layer by constrains

Change the color or layer of faces according to a predefined, searched-for arrangement of faces in relation to each other.

Template commands → CAD → Layer → Change faces color or layer by constrains

Advanced

Change the color or layer of faces according to a predefined, searched-for arrangement of faces in relation to each other.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Selection constrain: Select tangential, fillet, chamfer, coplanar or coaxial:

Tangential

Change faces that are ‘linked’ with a tangential transition within a solid or a face model. Specify a face within this composite as the initial information.

Angular tolerance: A value in degrees for controlling whether or not two faces are tangent to each other.

Example 72. 

Color: orange, Target color: red



Fillet

Change faces in the course of a constant fillet within a solid or a face model. Specify a face within this composite as the initial information.

Linear tolerance: Specify a permissible deviation within a fillet face.

Min. radius / Max. radius: Limit the course of the required fillet faces further by specifying a minimum and a maximum radius.

Example 73. 

Color: orange, Target color: blue



Chamfer

Change faces in the course of a constant chamfer within a solid or a face model. Specify a face within this composite as the initial information.

Linear tolerance: Specify a permissible deviation within a chamfer face.

Max. length / Min. length / 45 Degree: Limit the course of the required chamfer faces by specifying a maximum and a minimum diagonal length or using the 45° chamfer angle.

Example 74. 

Color: orange, Target color: purple



Coplanar

Change coplanar faces. Specify a face within the searched-for arrangement of faces as initial information.

Angular tolerance: Maximum permissible angle of a face to the face that is clicked first.

Linear tolerance: Maximum permissible distance to the face that is clicked first.

Example 75. 

Color: orange, Target color: green



Coaxial

Change coaxial faces. Specify a face within the searched-for arrangement of faces as initial information.

Angular tolerance: Maximum permissible deviation of the angle of the rotary axis of a face from the axis of the face that is clicked first.

Linear tolerance: Maximum permissible distance of the rotary axis of a face from the axis of the face that is clicked first.

Example 76. 

Color: orange, Target color: brown



Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target layer container

Enter or select a layer container for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.
