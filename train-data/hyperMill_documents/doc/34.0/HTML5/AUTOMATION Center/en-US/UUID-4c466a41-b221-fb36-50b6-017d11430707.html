Auto box dimension

Create a stock with predefined offset values.

Template commands → Stock → Auto box dimension

Basic

Template commands → CAM → Stock → Auto box dimension

Advanced

It is necessary to execute the Calculate stock dimensions or Calculate stock dimensions (parameter) function beforehand, otherwise the result will be wrong. Enter the size of the stock allowance there. The Auto box dimension function adopts the calculation result and displays the calculated values.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Stock name: Enter a name for the stock to be generated.

Stock position: Specify a fixed reference point for the stock.

Possible reference points:

  • Bottom-Center-Back

Stock width X: Displays the size of the stock in the X direction that was calculated using the Calculate stock dimensions function. The size can be further adjusted.

Stock width Y: Displays the size of the stock in the Y direction that was calculated using the Calculate stock dimensions function. The size can be further adjusted.

Stock height: Displays the size of the stock in the Z direction that was calculated using the Calculate stock dimensions function. The size can be further adjusted.

Stock position Z: Enter a value for the shift in the Z direction dependent on the reference point.

Bounding box layer: Enter a name for a layer on which to place the bounding box lines to be generated.

Face milling contour layer: Enter a name for a layer on which to place the Face Milling contour to be generated. The contour is located in the Z direction at the height of the upper side of the milling area and takes into account the size of the stock in the X and Y direction.

Tolerance: Enter a maximum permissible distance between the faces selected for generation as initial information and the stock facets.

Stop for input

Execution of the script is halted so that information can be entered.
