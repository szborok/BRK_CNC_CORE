Milling area multi offset

Create a milling area with multiple offsets on different face groups.

Template commands → Start → Milling area multi offset

Basic

Template commands → CAM → Start → Milling area multi offset

Advanced

The colors of the CAD model are used to specify different tolerances. A definition file can be edited in a text editor and different offsets can be assigned.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Milling area name: Enter or select a name for the milling area.

Definition files: In the drop-down menu, select a file from the list of files in the ...\MULTIOFFSET\ folder.

Note

An example is available in the ...\addins\hmAutoColor\Wizards\AutomationCenter\OMGEAPP\includeStart\... installation folder. The name of the example is ‘Multioffset.cfg’.

Save a definition file in the ...\MULTIOFFSET\ folder in order to make it available to AUTOMATION Center. The folder is located in the defined default path that is specified under hyperMILL → Setup → Settings in Application under Default paths in the AUTOMATION Center branch.

Example 27. Example of the syntax of a text file for defining multiple offsets
*START
*TARGET:="GroupSurfs0"
*ALLOWANCE:="0.0"
*ORIG_COLOR1:="95;0;0"
*ORIG_COLOR2:="255;175;175"
*ORIG_COLOR3:="0;175;175"
*ORIG_COLOR4:="255;255;0"
*ORIG_COLOR5:="255;0;255"
*ORIG_COLOR6:="239;239;239"
*ORIG_COLOR7:="255;175;0"
*END

*START
*TARGET:="GroupSurfs1"
*ALLOWANCE:="0.5" 
*ORIG_COLOR1:="225;180;180"
*ORIG_COLOR2:="255;128;0"
*ORIG_COLOR3:="80;143;12"
*ORIG_COLOR4:="0;207;111"
*ORIG_COLOR5:="210;210;255"
*ORIG_COLOR5:="168;122;168"
*END


Type: Select from Face or Solid. For example, six faces or one solid are entered in the milling area for a cube. It is useful to choose solids when associativity is required.
