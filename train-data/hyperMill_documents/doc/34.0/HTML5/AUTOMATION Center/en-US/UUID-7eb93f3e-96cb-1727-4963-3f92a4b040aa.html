Create an instruction

Create an image for an instruction during the script run and/or a text for an instruction during the script run or the documentation of a script step while defining a script. Optionally insert individual icons for options.

Figure 4. Schematic representation of the areas for creating instructions – image entities


Image field for one image per script step

Create a screenshot of the displayed CAD window. To do this, double-click the image field with the left mouse button.

Right-click to open a menu:

Import image

Insert an existing image in the format *.png, *.jpg or *.gif (also animated GIFs).

Modify image

Modify the image with Microsoft© Paint©.

Delete image

Remove the own image. The original, preset image is displayed again.

Change the icons during parameter definition. To do this, click an icon and right-click to open the context menu:

Import icon: Import an icon.

Delete icon: Remove the icon. The default icon is displayed again.

Figure 5. Schematic representation of the areas for creating instructions – text


Text field with basic Rich Text Format (RTF) functionalities and the option of transferring texts and images from a Word file via the clipboard.

Start the editor, e.g. Microsoft© Word. The editor opens with a new empty file. The file name is predefined. Enter the desired content, save and close the editor. The content is displayed in the text field.
