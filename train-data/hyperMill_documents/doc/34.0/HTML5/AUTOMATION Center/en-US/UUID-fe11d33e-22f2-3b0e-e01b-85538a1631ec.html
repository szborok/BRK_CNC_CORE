NCS best fit

Calculate an NCS based on the smallest dimensions of the CAD model.

Template commands → NCS → NCS best fit

Basic

Template commands → CAM → NCS → NCS best fit

Advanced

Calculate an NCS based on the smallest dimensions of the CAD model. The position of the CAD model to the world coordinate system does not matter.

A joblist with a milling area is required. The Z direction is based on the current view in the graphic area.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Name of NCS: Enter a name.

NCS position: Select a position from various positions of the milling area bounding box.

Possible positions:

  • Bottom-Center

  • Bottom-Center-Back

  • Bottom-Center-Front

  • Bottom-Center-Left

  • Bottom-Center-Right

  • Bottom-Left-Back

  • Bottom-Left-Front

  • Bottom-Right-Back

  • Bottom-Right-Front

  • Top-Center

  • Top-Center-Back

  • Top-Center-Front

  • Top-Center-Left

  • Top-Center-Right

  • Top-Left-Back

  • Top-Left-Front

  • Top-Right-Back

  • Top-Right-Front

  • Workplane

With a pure 3D free-form model, no precise orientation using special planes is needed, that is, no fine adjustment to planes is required. A fine adjustment is required for a prismatic CAD model. A planes search takes place in the tolerance range. The planes search can be optimized if a color and a layer are already established for a plane to be used:

Fine adjust to planes: Specify whether the NCS should be finely adjusted using a plane with a specific color. To use the fine adjustment, a suitable, colored, plane face must exist.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Fine adjust color: Specify a color for the fine adjustment using a plane. The color of the plane should only be provided for this purpose.

Stop for input

Execution of the script is halted so that information can be entered.
