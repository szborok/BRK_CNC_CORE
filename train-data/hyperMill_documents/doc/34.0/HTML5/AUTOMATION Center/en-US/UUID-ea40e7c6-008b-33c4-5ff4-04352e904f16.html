Write user variables from PMI / tags

Write user variables from PMI/tags.

Template commands → CAD → PMI → Write user variables from PMI / tags

Advanced

Generate CAM user variables from a TAG with PMI information of all filtered faces.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Tag name: Enter or select a tag name.

User variable prefix: Add a prefix to the user variable key.

Index name: Value, color, layer or tag

Stop for input

Execution of the script is halted so that information can be entered.
