Change color or layer

Change color or layer.

Template commands → CAD → Layer → Change color or layer

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Tag name: Enter or select a tag name.

Tag value: Enter a value or text based on the underlying tag type.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target layer container

Enter or select a layer container for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.

Each entity on separate layer: A separate layer is generated for each entity. The layer names are formed from the name that is specified for the target layer by appending a consecutive number to the name.

All entities from selected layer: Move all entities to a layer simply by coloring one of them.

Filter type: Process a selected entity type (for example, faces, solids, curves, point) or all these entity types.

Normal filter: Process normal to workplane origin, opposite to workplane origin or all normals.

Min. Z value: Entities below the minimum value in relation to the Z axis are not considered. Enter a value.

Max. Z value: Entities above the maximum value in relation to the Z axis are not considered. Enter a value.

Min. curve length and Max. curve length: Restrict the entities to be processed.

Max. entities in layer separation: Enter the maximum number of entities for layer separation.

2025

Must exist: The system checks whether a color or layer to be changed exists. If the color or layer is not (or no longer) available, the script run is interrupted with a message if Yes is selected. If No is selected, this status is ignored.
