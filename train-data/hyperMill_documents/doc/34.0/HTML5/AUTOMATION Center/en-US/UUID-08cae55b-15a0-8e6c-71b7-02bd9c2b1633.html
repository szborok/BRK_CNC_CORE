Load fixture to stock (Parameter)

Select CAD fixture data on the basis of a previously determined name and place on the lower edge of the stock.

Template commands → Fixtures → Load fixture to stock (Parameter)

Basic

Template commands → CAM → Fixtures → Load fixture to stock (Parameter)

Advanced

Select CAD fixture data on the basis of a previously determined name. A function such as Find best fixture (from Excel) must be executed beforehand in order to find a fixture and pass its name to this function with the help of a script variable. The fixture is placed on the lower edge of the stock.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Fixture name: Enter the name of the fixture to load or use the script variable for the result from the Find best fixture (from Excel) function.

Counter spindle (invers NCS): Choose whether the counter spindle is aligned in the direction or against the direction of the NCS.
