2. User interface

AUTOMATION Center allows commands to be called from hyperMILL® CAD and CAM. In addition, there are further additional commands that are only available in AUTOMATION Center.

Procedure

A script combines functions into a sequence, to be run from top to bottom. The script contains components, which in turn contain functions. Functions are available as template commands to be inserted into a script. Components can contain subcomponents.

  1. As a prerequisite, check under hyperMILL → Setup → Settings in Database whether the databases, in particular the tool database, are selected correctly.

    Note

    Project-specific organization of data is necessary for working with the AUTOMATION Center. For this reason, the Projekt option is enabled under Document → Directories in the hyperMILL → Setup → Settings function.

  2. Open script management . Create a new script . Enter a name. The script is still empty. Close the dialog.

  3. Prepare a script in definition mode . To do so, create a component or a subcomponent in a component in the left column. Arrange the functions in the desired sequence.

    To do so, select the function or pre-configured component in the right column and insert it in the left column using the button. This is also possible by double-clicking with the left mouse button or dragging and dropping.

    Close the dialog when the script is complete.

  4. Specify the parameters for each function in the parameters table in the main dialog. To do so, switch to parameter definition mode . Disable this mode once all parameters have been defined.

  5. Optionally, in the node connection editor, create the parameter constraints from one function to another function using leader lines.

  6. Test the script and optimize it if necessary. To do so, run the script step by step or run the entire script . Alternatively, you can double-click the command with the left mouse button to perform a single step.

  7. Once the script is ready, it can optionally be inserted in the menu and/or toolbar. Go to script management and open the toolbar definition dialog .
