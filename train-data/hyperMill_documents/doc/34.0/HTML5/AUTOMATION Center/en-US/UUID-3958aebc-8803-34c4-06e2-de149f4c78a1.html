Set workplane on cylinder

Generate a workplane on a cylinder using uniquely colored faces.

Template commands → CAD → Workplanes → Set workplane on cylinder

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Cylinder color: A uniquely colored cylinder barrel face is required. Enter or select a color with RGB values.

Z-level plane color: A uniquely colored plane front face for the location of the origin is required. Enter or select a color with RGB values.

X axis plane color: A uniquely colored planar face for the X axial direction is required. Enter or select a color with RGB values.

Z orientation: Select from which information the Z axial direction is taken.
