Calculate face angle

Calculate angle between face and reference entity in space or in plane and assign result to script variable.

Template commands → AC general → Get Variable → Calculate face angle

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Reference: NCS, WCS. Select faces or workplane as the reference. When faces are selected, the reference layer and reference color are required.

Calculate inside XY plane: Face normals are projected in the plane and the angle in the plane is determined there.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Reference face layer: Enter a layer that exists in the model whose faces should be included. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Reference face color: Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.
