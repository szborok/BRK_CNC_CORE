Fixtures

Generate a new fixture area from CAD entities of the current project.

Template commands → Fixtures → Fixtures

Basic

Template commands → CAM → Fixtures → Fixtures

Advanced

Generate a new fixture area from CAD entities of the current project. The entities are selected by assigning them before to specific unique layers and/or have a color only intended for this purpose. The fixture area is added automatically to the last available joblist.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Fixture name: Enter the name of the fixture to load or use the script variable for the result from the Find best fixture (from Excel) function.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Tolerance: Enter a maximum permissible distance between the faceted top of the polyhedron model and the real model faces. Add | METRIC to specify that this value only applies to metric models. This value is converted in the case of INCH models. The procedure is the same when | INCH is added.

Tool safe distance: Enter the minimum distance of the tool to the clamp as the clearance distance to be maintained between the tool and the clamp.

Layer invisible: Enter or select which layer should not be visible and should therefore not be evaluated. The asterisk (*) symbol can be used as a wildcard. To make a selection, first click the button in the table cell and then select a layer entity in the graphic area.

The layer visibility control is performed on reaching the function and not on executing the function.

Layer visible: Enter or select which layer should be visible and should therefore be evaluated. The asterisk (*) symbol can be used as a wildcard. To make a selection, first click the button in the table cell and then select a layer entity in the graphic area.

The layer visibility control is performed on reaching the function and not on executing the function.

Restore layer setting: Restore all layer settings to the state they were in before this function was executed.
