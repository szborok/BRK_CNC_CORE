Write image in Excel

Insert an image from an image file into an Excel file.

Template commands → Files → Excel calculation → Write → Write image in Excel

Advanced

Insert an image from an image file into an Excel file. To do this, define the size of the image precisely by length and width according to the aspect ratio of the image.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Excel file: Select an Excel file. The file folder is transferred from Folder. The stored folder is opened if a script variable is used for Folder.

Cell address for image (start): Enter the address of the Excel cell for the upper left corner of the image.

If the length and width are not specified, the image is inserted in its actual size. Otherwise, it will be resized and inserted as specified:

Length: Enter a length for the image.

Width: Enter a width for the image.

Image folder: Enter or select a folder.

Image file: Enter the file name of a file that exists in the selected folder.

Sheet name

Enter the name of the Excel worksheet.

Condition

Check whether a script variable is <, >, = or ≠ to a value.
