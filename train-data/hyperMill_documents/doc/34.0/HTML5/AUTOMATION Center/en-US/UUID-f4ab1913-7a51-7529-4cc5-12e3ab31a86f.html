Deform entities

Deform entities.

Template commands → CAD → Deformation → Deform entities

Advanced

Use the CAD command Deform entities. Deform entities.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Fix

Select boundaries, curves or points at the place where the entities to be deformed are to remain unchanged in terms of their original position and their shape.

Fixed point color: Enter the color of the points to be fixed.

Fixed point layer: Enter the layer of the points to be fixed.

Fixed contour color: Enter the color of the curves to be fixed.

Fixed contour layer: Enter the layer of the curves to be fixed.

Matching points

Select points where positions of the entities to be deformed should be brought into agreement with the target entities. Here, the number of target entities must be equal to the number of start entities:

Start point color: Enter the color of the start points.

Start point layer: Enter the layer of the start points.

Target point color: Enter the color of the target points.

Target point layer: Enter the layer of the target points.

Matching curves

Use curves and boundaries to choose where the entities to be deformed are to be brought into agreement with the target entities.

Start contour color: Enter the color of the start curves.

Start contour layer: Enter the layer of the start curves.

Target contour color: Enter the color of the target curves.

Target contour layer: Enter the layer of the target curves.

Open user interface

Specify whether the user interface should be opened.
