Create NC file

Execute the hyperMILL® Create NC file function for the last joblist or all joblists.

Template commands → Basic Programming → Create NC file

Basic

Template commands → AC general → Finish → Create NC file

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Joblist: Select a joblist for using the function.

Job range: Perform the calculation using the jobs defined in the job range of the current or last job list.

Define the job range in advance using either Define job range or Multi job selection, or define it directly. A selection made using Job range overwrites another selection, such as a job list.

  • Define an area that also contains all existing jobs within it: The following syntax is used for the start job: SID:, for example, SID:2. The following syntax is used for the end job: EID:, for example, EID:4. Insert a hyphen in between. This results, for example, in SID:2-EID:4.

  • List the jobs: The following syntax is used for the jobs: JID:. This results, for example, in JID:1; JID:4;. Separate the individual jobs with a semicolon.

Open user interface

Specify whether the user interface should be opened.

Provide information about the result of the postprocessor run:

PP return status: Information is provided on the type of message – for example, ‘Info’ or ‘Error’.

PP return message: Outputs the text of the message

PP success: Information is provided on whether the NC file was created successfully – ‘true’ for successful or ‘false’ for unsuccessful.
