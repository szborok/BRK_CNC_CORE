Read milling area property as script variable

Read a milling area property as script variable.

Template commands → AC general → Property → Read → Read milling area property as script variable

Advanced

Read a milling area property into a script variable.

Example 14. Determine the name of the joblist milling area

  1. Call the Dump internal Parameters function in the header of the hyperMILL® job list dialog by right-clicking while holding down the CTRL key. The propDump.txt file is displayed. Search the file for the name of the required property. The ModelMa parameter is used for the milling area.

  2. Determine the milling area UUID using the Read in joblist property as script variable function. To do this, place and configure the function in the script sequence beforehand:

    Joblist property

    ModelMa

    Result

    During the script run, the milling area UUID is output as the result.

  3. Place and configure the Read milling area property as script variable function explained here in the script sequence. Transfer the previously determined milling area UUID using its script variable. Enter a required milling area property, such as Name.

    Milling area UUID

    §SCRIPT_VARIABLE_#...

    Milling area property

    Name

    Result

    During the script run, the milling area property is output as the result, for example, the name ‘Model_0’ is output.



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Milling area UUID: Enter the UUID of the milling area whose property is to be determined.

Milling area property: Enter a milling area property, for example, Name or Comment.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.
