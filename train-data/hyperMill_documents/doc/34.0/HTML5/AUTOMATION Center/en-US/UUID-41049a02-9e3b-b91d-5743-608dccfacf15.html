Circular pattern (CAM)

Duplicate toolpath as a circular pattern.

Template commands → CAM → Transformation → Circular pattern (CAM)

Advanced

Duplication and positioning of toolpaths for the production of identical and/or similar components. Also see Assign transformation (CAM).

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Name: Enter a name for the transformation.

Transformation comment: Enter a comment.

Set the axis of a cylinder as the rotary axis:

Centre entity layer: Select the layer on which the entity that determines the rotation axis is located.

Centre entity colour: Select the color of the entity that determines the rotation axis.

Layout: If the Fit entities option is selected, the toolpath is duplicated as a circular pattern, taking into account the number of copies and the total angle. The Fixed distance option is used to duplicate the toolpath as a circular pattern, taking into account the step angle and the number of copies.

Angle: Enter a total angle or a step angle according to the selection in Layout.

Number of copies: Enter the number of copies.
