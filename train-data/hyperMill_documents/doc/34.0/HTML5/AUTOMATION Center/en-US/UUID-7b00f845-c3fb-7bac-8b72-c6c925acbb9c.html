Exact boundary

Generate an exact boundary from curves.

Template commands → CAD → Curves → Exact boundary

Advanced

Uniquely colored faces are required. Solids are not supported.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Curve color

Enter or select a color with RGB values.

Curve layer

Enter or select a layer for the curve to be generated.

Join curves: Connect the individual curves of the result to form a closed chain of curves.

Filter: Select which boundary to create.

  • All

    Creates all boundaries.

  • Upper

    The boundary is created on the highest Z level, depending on the workplane.

  • Lower

    The boundary is created on the lowest Z level, depending on the workplane.

  • Longest loop

    The boundary with the longest curve length is created.

  • Second longest loop

    The boundary with the second longest curve length is created.
