Calculate user variable from faces

Calculate geometry values and add to the hyperMILL® user variables.

Template commands → CAM → User variable → Calculate user variable from faces

Advanced

Calculate values from geometric information and add to the hyperMILL® user variables with a name to be specified.

Example 26. Calculate the position of a face in Z

The Z value of the red face can be determined.



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Name: Enter a name for the hyperMILL® user variable.

Description: Enter a description for the hyperMILL® user variable.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Position: Enter the position to be calculated (top, bottom, distance).

Axis: Enter the axis be calculated (X, Y, Z).

Frame: Enter the frame to be calculated with the aid of the frame name.

Round digits: Round value to the specified decimal place.

Tolerance: Specify the accuracy of the determined value.

Reset color: Restore all color settings to the state they were in before this function was executed.
