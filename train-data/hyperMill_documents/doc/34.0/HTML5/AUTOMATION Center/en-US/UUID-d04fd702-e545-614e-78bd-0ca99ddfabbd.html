Convert to NURBS

Use the Convert to NURBS CAD command. Convert one or more curves to NURBS.

Template commands → CAD → Modify → Convert to NURBS

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Open user interface

Specify whether the user interface should be opened.
