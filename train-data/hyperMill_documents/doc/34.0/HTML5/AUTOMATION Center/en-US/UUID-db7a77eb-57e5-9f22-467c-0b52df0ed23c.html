Change color or layer in boundary

Change the color or the layer for entities selected by boundary.

Template commands → CAD → Layer → Change color or layer in boundary

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Layer of outer boundary: Enter the layer on which the entities to be used are located.

Color of outer boundary: Enter or select the color. Only entities of this color are used.

Select type of outer boundary: In the case of the Completely within selection contour option, only entities that are completely within the outer selection contour are used. In the case of the Intercepted with selection contour option, the entities that ‘cross’ the selection contour are also used.

Layer of inner boundary: Enter the layer on which the entities to be used are located.

Color of inner boundary: Enter or select the color. Only entities of this color are used.

Select type of inner boundary: In the case of the Completely within selection contour option, only entities that are completely within the inner selection contour are used. In the case of the Intercepted with selection contour option, the entities that ‘cross’ the selection contour are also used.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target layer container

Enter or select a layer container for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.

Use only visible entities: Only the entities visible from the view direction are used.

Filter type: Process a selected entity type (for example, faces, solids, curves, point) or all these entity types.

The function always works from the viewing direction. If the active workplane and/or the set view direction are already correctly aligned, the following options do not need to be enabled. Otherwise, in accordance with the situation encountered, have one or both alignments performed at the position of the selected contours of the boundaries:

Adjust workplane automatically: Set the workplane to the position of the selected contours of the boundaries.

Adjust view automatically: Set the view to the position of the selected contours of the boundaries.
