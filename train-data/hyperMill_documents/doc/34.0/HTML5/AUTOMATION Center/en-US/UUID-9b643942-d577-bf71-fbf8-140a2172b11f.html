Intersection of stocks

Use the Boolean ‘Intersection’ operation to combine two stocks.

Template commands → Stock → Intersection of stocks

Basic

Template commands → CAM → Stock → Intersection of stocks

Advanced

In the current hyperMILL® project, calculate the intersection of two stocks and generate a new stock based on the result.

Also see Difference of stocks and Union of stocks.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Stock model 1: Enter the name for the first stock to be combined.

Stock model 2: Enter the name for the second stock to be combined.

New stock name: Enter a name for the resulting stock.
