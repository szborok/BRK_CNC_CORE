Close pockets with plane

Generate plane filling faces.

Template commands → CAD → Shapes → Close pockets with plane

Advanced

This function also works if the contour limit is not located within a face.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Wall color: Enter the wall color of the pocket.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Target color

Enter or select a color with RGB values.

Filter: Specify whether the plane face is located on the upper side of the top contour or on the underside of the top contour.
