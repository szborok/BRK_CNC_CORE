Start XML segment

Start XML segment.

Template commands → Files → XML Files → Write → Write manually → Start XML segment

Advanced

Example 106. Principle
<?xml version="1.0" encoding="utf-8"?>
<Segment Parameter1="Value 1" Parameter2="Value 2" 
        Parameter3="Value 3">
<!--Comment line-->
	<Function/>
</Segment>


Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Segment name: Enter a name for a parent XML element.

Write empty parameter: If there is no value for a Parameter 1 to Parameter 20, output the XML attribute anyway, for example, Parameter1="". Otherwise, the XML attribute is not output.

Parameter 1 to Parameter 20: Enter a name for an XML attribute

Value 1 to Value 20: Enter a value for this XML attribute.
