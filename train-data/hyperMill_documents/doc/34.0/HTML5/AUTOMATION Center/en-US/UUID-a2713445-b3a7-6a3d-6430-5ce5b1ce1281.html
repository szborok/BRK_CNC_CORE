Set NCS on cylinder

Generate an NCS on a cylinder using uniquely colored faces.

Name of NCS: Enter a name.

Cylinder color: A uniquely colored cylinder barrel face is required. Select the color of the face.

Z-level plane color: A uniquely colored plane front face for the location of the origin is required. Enter or select a color with RGB values.

X axis plane color: A uniquely colored planar face for the X axial direction is required. Enter or select a color with RGB values.
