Single point from points

Generate a point in the middle of a curve or on corners of an imaginary bounding box.

Template commands → CAD → Points → Single point from points

Advanced

Generate a new point on a corner of an imaginary bounding box or on the middle point, sorted by IDs. In this way, several points can first be created on a curve in order to then obtain the middle of the curve with the middle point. Only if an odd number of a minimum of three points is specified.

Example 53. 

  1. Use the Point on curve script step in advance to generate an odd number of points on a curve.

  2. Use layer and color to select these points.

  3. The ID of the middle point is used to generate a new point that lies in the middle of the contour.

  4. A bounding box is created internally around the selected points. Further points can be created at its vertices. To do this, execute this function as often as required.



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Points layer

Enter or select the points layer. Only uses points that are assigned to this layer.

Point color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Target color

Enter or select a color with RGB values.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Position: Select at which position the point is to be created. For Middle (from IDs), the position is determined using the middle point based on its entity ID from an odd number of points. For the vertices, a bounding box is created around the selected points. Its corners determine the positions of the points to be generated.
