Execute Python script

Execute a Python script.

Template commands → AC general → External automation → Execute Python script

Advanced

For complex requirements, an externally generated Python script can be executed in the Python programming language.

A PYTHON Toolkit is delivered and licensed as a separate installation. It is embedded in hyperMILL® and customized. Embedded means that only a selected set of modules is installed. The adjustments mainly concern data protection and data security. Additional modules can be installed by an administrator.

Use the Choose Python folder function to select the storage location of the Python scripts in advance.

Example 25. Determine whether a numerical value is greater than, equal to or less than another numerical value.

Input:

Description Python Parameter ‘P1’: Value1 | Description Python parameter ‘P2’: Value2

Value1: 25 | Value2: 50

Software program:

# Python script example

if P1 > P2:
	returnVal1= "The value "+ str(P1)+ " is bigger then "+ str(P2)
elif P1 == P2:
	returnVal1= "The value "+ str(P1)+ " is equal then "+ str(P2)
else:
	returnVal1= "The value "+ str(P1)+ " is less then "+ str(P2)


returnVal2="returnVal2 is currently not used"
returnVal3="returnVal3 is currently not used"
returnVal4="returnVal4 is currently not used"

Results:

Return from script: The value 25 is less then 50

Return from script: returnVal2 is currently not used

Return from script: returnVal3 is currently not used

Return from script: returnVal4 is currently not used



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Python scripts: Select a Python script from the drop-down menu.

If this option is not offered, the Python script folder has not been selected. Use the button at the end of the line to open the Python script in a text editor.

Number of return values: Select the number of returns from the script.

2025

Description Python Parameter 'P1' to Description Python Parameter 'P20': First enter a description for the parameters. A value can then be entered during the next run.

Pass up to 20 parameters to the Python script. For processing, the variables P1 to P20 must be used in the Python script, whereby the value from Description Python Parameter ‘P1’ is passed to P1 and so on.

Return from script: The results of the calculation in the function. Transfer to other functions with the help of the script variables. To do this, the results must be passed within the Python script to variables with the names returnVal1 to returnVal4.
