Dump two entities in XML

Determine and output information of two entities in relation to each other.

Template commands → Files → XML Files → Write → Dump two entities in XML

Advanced

Geometric information on entity pairs is determined and output to a *.xml file as <PBag>. For this purpose, select two entities. The information that is determined depends on the types of entities that have been selected.

Example 111. Determine the distance between two points

Determine the distance for a red point on layer ‘P1’ and a blue point on layer ‘P2’.

Input: Layer 1: P1 | Color 1: Red | Layer 2: P2 | Color 2: Blue | Folder: D:\AC | File: Result.xml

Result: In D:\AC\Result.xml

<PBag name="info.twoentities.description"
      version="1">
	<property name="Deltax"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">10</property>
	<property name="Deltay"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">0</property>
	<property name="Deltaz"
	          type="double"
	          uuid="d46df12d-5e3b-53c1-bb37-6d6b1e3223f9"
	          dimension="Length">0</property>
	<property name="FirstPoint"
	          type="bgeoPoint3d"
	          uuid="d095de30-d649-5cc7-b9f5-7ac153cba1ee"
	          dimension="Dimensionless">0|
                                            0|

                                            9.5367431640625e-07
                                            </property>
	<property name="MinDistance"
	          type="double"

	<property name="SecondPoint"

                                             0|

                                             </property>
</PBag>
