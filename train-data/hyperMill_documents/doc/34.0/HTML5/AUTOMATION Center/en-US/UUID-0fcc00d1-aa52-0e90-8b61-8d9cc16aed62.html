Client / server

Contains functions for a client/server solution included in AUTOMATION Center.

A server is understood here as a computer on which the AUTOMATION Center runs with the respective script and with the help of the two programs PWMonitor.exe and PWServer.exe that control and monitor the automatically running calculation process in hyperMILL®.

Figure 7. Overview of how the client and server interact


Server controller

Software for defining a period of time for unobserved calculations in which the AUTOMATION Center server PWservere.exe should work. Start the software via the Windows Start menu or in ....\OPEN MIND\hyperMILL\[Version]\addins\hmAutoColor as PWserverStart.exe.

Server start time

Start calculations on AUTOMATION Center.

Server stop time

The AUTOMATION Center server is stopped, hyperMILL® remains open. A calculation in progress is terminated. A log is not kept.
