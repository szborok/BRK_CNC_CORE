Turn feature recognition

Automatically recognize the rotation contour for machining a turned surface.

Template commands → Basic Programming → Turn feature recognition

Basic

Template commands → CAM → Feature → Turn feature recognition

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Type: Select Faces or Contours. If a contour from curves already exists, use the Contours option to select the desired turning profile in order to create a turning area. If there is no curve, use the Faces option to create the turning contour and the turning area.

Entities layer: Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Entities color: Enter or select the color. Only entities of this color are used.

Create turning area: Activate if a new turning area is to be created in the hyperMILL® browser under Models.

Save the created or recognized turning contour as a curve in the required color and the required layer.

Turning area layer: Enter a layer to use multiple times in the script, for example.

Turning area color: Enter or select a color with RGB values.

Activate split position: If the main spindle and the counter spindle are to be used for machining, split the machining for the two spindles.

Select a point on the contour for a position in the Z direction:

Split position - point layer: Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Split position - point color: Enter or select the color. Only entities of this color are used.

Activate inside split position: Decouple the machining of the outside profile and the inside profile.

Select a point on the contour for a position in the Z direction:

Inside split position - point layer: Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Inside split position - point color: Enter or select the color. Only entities of this color are used.

Frame: Select a frame for recognizing the turn feature.

Tolerance: Enter the fineness of the curve resolution for recognizing the turn feature.
