Bounding box

Create a cuboid or cylindrical box of curves that is large enough to allow all of the selected entities to fit inside.

Template commands → CAD → Curves → Bounding box

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Curve color

Enter or select a color with RGB values.

Curve layer

Enter or select a layer for the curve to be generated.

Tolerance: Specify the required accuracy.

Mode: Select Linear or Rotation. If Rotation is selected, curves are generated when a rotational contour created by rotating a shape is intersected by a plane.

Precise: Lines (such as arcs, straight lines, NURBS) are generated as the calculation result. If the option is disabled, curves are generated as polylines.

Entity type: Select the shape of the bounding box.

  • Lines: A bounding box made of lines.

  • Faces: A bounding box made of faces.

  • Solid: Create a solid as a bounding box.

Minimize: Minimize the volume. For this purpose, the best fit is calculated for the X/Y orientation of a cuboid. A minimum volume is achieved for the cylinder by changing the position of the center line. The Z orientation is not considered.
