Read multiple values from Excel file

Read multiple values from Excel file or cached data table (in script variable).

Template commands → Files → Excel calculation → Read → Read multiple values from Excel file

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Excel file: Select an Excel file. The file folder is transferred from Folder. The stored folder is opened if a script variable is used for Folder.

Using saved data table: Use the data saved with the Save Excel data table command.

Cell address: For each value, enter the name of the table cell (for example, A5 or A50) from which to read the value.

Value: The result of the calculation in the function. Transfer to other functions with the help of script variables.

Default: This value is used if there is no content in any of the respective cells.

Stop when the first row is empty: Stop when the entire row or only the cell in the column from which to read in is empty.

Sheet name

Enter the name of the Excel worksheet.
