Set color visible / invisible

Change the visibility of entities by means of their color.

Template commands → CAD → Coloring → Set color visible / invisible

Advanced

Use a combination of Color invisible and Color visible to enable a specific selection of entities. Use a combination of Color visible and Color invisible (second same-named option) to disable a specific selection of entities. Use the asterisk (*) symbol as a wildcard for all colors.

Example 78. Only make one color visible

  1. Use the first Color invisible option to generally disable all entities, that is, make all entities invisible. To do this, use the asterisk (*) symbol as a wildcard for all colors.

  2. In the case of the Color visible option, it is now simply necessary to enter the color that should be enabled again.



Example 79. Only make one color invisible

  1. Use the Color visible option to generally enable all colors, that is, make all entities visible. To do this, use the asterisk (*) symbol as a wildcard for all colors.

  2. For the second option Color invisible, only one color needs to be entered now that is to be switched off again.



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Color invisible: Enter or select which color should not be visible and should therefore not be evaluated. To make a selection, first click the button in the table cell and then select an entity of the color in the graphic area.

The visibility control is performed on reaching the function and not on executing the function.

Color visible: Enter or select which color should be visible and should therefore be evaluated. To make a selection, first click the button in the table cell and then select an entity of the color in the graphic area.

The visibility control is performed on reaching the function and not on executing the function.

Color invisible: Enter or select which color should not be visible and should therefore not be evaluated. To make a selection, first click the button in the table cell and then select an entity of the color in the graphic area.

The visibility control is performed on reaching the function and not on executing the function.
