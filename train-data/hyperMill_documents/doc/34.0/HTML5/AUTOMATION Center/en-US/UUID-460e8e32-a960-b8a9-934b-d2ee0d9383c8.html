Find best stock (from Excel)

Search for the stock best suited for machining in an Excel table.

Template commands → Stock → Find best stock (from Excel)

Basic

Template commands → CAM → Stock → Find best stock (from Excel)

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Stock name: Enter a name for the stock to be generated.

Tolerance: Enter a maximum permissible distance between the faces selected for generation as initial information and the stock facets.

Stock position: Relate the stock position (+Z) to top or bottom.

Stock position move Z: Shift the stock in the Z direction. Enter a value that refers to the Top-Center reference point of the stock.

Stock offset (Z): Enter a stock offset in the +Z and -Z direction of the NCS.

Offset XY: Enter a side offset in the X and Y direction of the NCS.

Contour layer: Enter a name for a layer on which to place the lines of the contour to be generated.

Face milling contour layer: Enter a name for a layer on which to place the Face Milling contour to be generated. The contour is located in the Z direction at the height of the upper side of the milling area and takes into account the size of the stock in the X and Y direction.

Stock type: Select the type of stock.

Free stock setup: Use the stock when vertical, horizontal or rotated sideways, that is, in all principal directions – suitable for manual setup. Or, if ‘No’ is selected, only as defined via the length, width and height – for setup with a robot.

Folder

Select or enter a folder.

Excel file: Select an Excel file. The file folder is transferred from Folder. The stored folder is opened if a script variable is used for Folder.

Cell address for value (start): In the Excel table, arrange the values next to each other in a table row. Use another table row for each further definition. Select the information with the address of the table cell (for example, A2 or A50) that contains the first value. Enter the name of the cell here.

Sheet name

Enter the name of the Excel worksheet.

Success: Returns whether a stock was found. If not, ‘False’ is output.

Return field value: The value in the cell of the row with the stock that has been found is output, for example, 9 for a found stock of size 100, 75, 100 . The table header is of no interest here. The fourth column after the start cell is always used – according to the sequence X, Y, Z, Return.
