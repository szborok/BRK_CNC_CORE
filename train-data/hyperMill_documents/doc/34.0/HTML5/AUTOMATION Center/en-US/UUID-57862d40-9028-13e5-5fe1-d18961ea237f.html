Dimension (face / vector)

Dimension a length.

Template commands → CAD → CAD Info → Dimension (face -> vector)

Advanced

Generate a length dimension from a reference face in a direction specified by the dimension direction and in a length provided by the vector value. The position of the dimension line is determined by Alignment dimension line and Distance dimension line. The color of the dimension lines is determined by the curve color. The dimension is placed on the curve layer.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Dimension color: Enter or select a color with RGB values.

Text color: Enter or select a color with RGB values.

Arrow color: Enter or select a color with RGB values.

Dimension layer: Enter or select a layer for the dimension to be generated.

Start face color: Enter an RGB color value for an entity in the model that is to be included. Optionally click the button in the table cell and select the entity of the respective color. The RGB color value is transferred.

Start face layer: Enter or select the layer of the start face. The face must exist.

Vector value: Enter the length of the dimension.

Text height: Enter a size for the text in points.

Axis dimension line: Select the X, Y or Z direction for the dimension line.

Axis dimension direction: Select the axial direction X, Y, Z, -X,-Y or -Z in which the dimension is aligned.

Distance dimension line: Enter the length of the reference lines.

Rotate view: Reverse the dimension in case the measured value is displayed as a mirror image from the required direction.

Text on view: If Yes is selected, always display text horizontally and parallel to the view, even if the view changes.

Reference: Specify the reference system to which the coordinates refer.
