2.4. Scripts as a toolbar and in the menu

Up to 30 scripts can be added to the menu under hyperMILL → Automation → ... and to the Automation – toolbar or to two toolbars that are made available ‘across the entire company’.

Changes made to the menu and to toolbars are visible after restarting the software.

Project: Select a script .

Run type: Specify how the script should behave after it is run .

  • SELECT: The dialog opens and the script is selected.

  • START: The dialog opens and the script is executed.

  • EXECUTE: The script is executed. The dialog remains closed.

Description: Enter a short explanation of the script .

Company toolbar

Two toolbars can be defined that are configured in the same way on all AUTOMATION Center workstations.

The icons for defining a company toolbar are only visible when the Windows HC_ADDITIONAL_HCCONFIG = [SERVER FOLDER] environment variable is set and has been started with an Advanced license. The directory on the server is created as specified. This functionality is not implemented for the BASIC license.
