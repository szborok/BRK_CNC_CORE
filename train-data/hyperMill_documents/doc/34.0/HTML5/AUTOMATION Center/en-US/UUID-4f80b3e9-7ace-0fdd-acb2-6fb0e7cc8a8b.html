Are all hole features finished?

Check whether all drilling features are programmed for complete machining.

Template commands → CAM → Feature → Advanced feature functions → Are all hole features finished?

Advanced

For this script step, a script step with the command Result stock must first be executed for a job list with drilling.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Stock name: Enter the name or the UUID for the stock to be used.

Tolerance rest volume

The system calculates whether all features have been drilled. A comparison is made with the stock. If rest volume greater than the tolerance is found, this can be displayed. Data in mm³ or cubic inch.

2025

If a final machining state other than that specified by the model geometry is to be examined for holes, define this state by entering an offset value:

Color 1 (RGB) ... Color 5 (RGB):

Select the relevant holes based on their color.

Hole offset 1 ... Hole offset 5:

Enter a value for the remaining hole offset.

The following offset values are calculated on the bounding geometries of the hole features as a kind of tolerance. Offset XY offsets the hole radius and Offset Z is added in the hole direction on both sides of the bounding geometry.

General offset XY

Enter a value.

General offset Z

Enter a value.

If a target color and a target layer are defined, the remaining material is displayed for areas that have not been fully processed:

Target color

Enter or select a color with RGB values.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

If no remaining material was determined, TRUE is output, otherwise FALSE:

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.

Rest volume

The determined rest volume is output.
