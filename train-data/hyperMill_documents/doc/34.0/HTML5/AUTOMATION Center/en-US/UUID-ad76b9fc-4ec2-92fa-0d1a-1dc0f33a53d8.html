Curves (dialogs)

Call CAD curve commands in the script.

Template commands → CAD → Curves → Curves (dialogs)

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Command: Call a CAD command.

Select one of the following functions:

  • Spline

  • Fitting

  • Boundaries

  • Project

  • Helix

  • Intersect

  • Blending

  • Join

  • Offset curve

  • Offset on face

  • Shape contour

  • Section

  • Center

  • Wrapping

  • 2D fillet

  • 2D chamfer

  • Points on face

Target color Select a color from the active CAD color library in order to make it active.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Open user interface

Specify whether the user interface should be opened.

The options vary for the following functions:

Spline, Helix

Points layer

Enter or select the points layer. Only uses points that are assigned to this layer.

Point color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Fitting

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Points layer

Enter or select the points layer. Only uses points that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Point color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Boundaries, Project, Offset on face

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Intersect, Blending

Face layer 1

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face layer 2

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color 1

Enter an RGB color value for a first section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Face color 2

Enter an RGB color value for a second section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Join, Offset curve, Section

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Shape contour, Center, Points on face

Face layer

Enter one or several layers that exist in the model whose faces should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select a face of the respective layer. The layer name is transferred.

Face color

Enter an RGB color value for the faces that exist in the model that are to be included. Optionally click the button in the table cell and select a face of the respective color. The RGB color value is transferred.

Wrapping

Entities layer: Enter one or several layers that exist in the model whose entities should be included. Separate multiple layer names with a semicolon ;, for example, Layer1; Layer2. Optionally click the button in the table cell and select an entity of the respective layer. The layer name is transferred.

Entities color: Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

2D fillet, 2D chamfer

Curve color 1: Enter an RGB color value for a first section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Curve color 2: Enter an RGB color value for a second section of entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.
