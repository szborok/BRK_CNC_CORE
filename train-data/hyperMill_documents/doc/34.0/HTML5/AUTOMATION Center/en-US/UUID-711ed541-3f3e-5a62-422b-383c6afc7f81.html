Repeat entities on points

Repeat entities on positions that are specified via points.

Template commands → CAD → Transformation → Repeat entities on points

Advanced

Repeat entities on positions that are specified via points. The face normal vector, which exists at the point, is considered for the alignment. The point must be located on the reference face.

The basic entities must be located at the origin of the world workplane. Only a normal direction is permitted.

Example 65. Distribute holes on cylinder face

Distribute a pattern hole on a cylindrical face at different positions so that the respective hole is aligned in exactly the same way as the face normal vector of the face, where the respective point is located.



Example 66. Restore holes on deformed face

If a model space, including the holes that exist there, were to be deformed, the holes would also be deformed in an adverse manner. For this reason, determine the hole start points in advance and remove the hole faces. Deform the faces along with the start points. Then insert the holes again at their start points.
