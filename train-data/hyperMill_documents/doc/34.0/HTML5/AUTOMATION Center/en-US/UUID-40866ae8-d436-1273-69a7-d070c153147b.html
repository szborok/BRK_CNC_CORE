Write tool graphic

Generate and save a tool graphic.

Template commands → Files → Excel calculation → Write → Write tool graphic

Advanced

The tool graphic is not yet inserted into an Excel file here but is saved as a file. The tool graphic can be saved in *.jpg format.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Folder

Enter or select a folder.

Filename

Select or enter a file name.

To select the tool, either specify its UUID or its tool ID:

Tool UUID: If this function is used in a loop over all tools, enter the predefined §CURRENT_TOOL# script variable here.

Tool ID: Enter the ID of the tool.

Width: Enter a width for the graphic.

Length: Enter a length for the graphic.

Axis orientation: Specify whether the tool should be displayed vertically or horizontally.

Render style: Specify whether the tool should be displayed two-dimensionally or three-dimensionally.

Show dimension: The dimensions of the tool are added to the tool graphic.
