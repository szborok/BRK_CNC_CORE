Check layer collision

Check whether faces of two layers touch or intersect.

Template commands → CAD → Layer → Check layer collision

Advanced

Check whether faces on two different layers touch or intersect.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Example 71. 

Input: Layer 1: blue face | Layer 2: purple face – result: Is collision free?: False

Input: Layer 1: blue face | Layer 2: orange face – result: Is collision free?: True



Layer 1: Enter a layer name or select an entity on the respective layer.

Layer 2: Enter a layer name or select an entity on the respective layer. Wildcards may be used.

Tolerance: The calculation is performed internally with the help of a mesh. Use this tessellation tolerance to define the maximum permissible distance (chord error) of the triangles from the given faces (face edges).

Is collision free?: ‘True’ is output if no collisions were found. ‘False’ is displayed in the event of collisions.
