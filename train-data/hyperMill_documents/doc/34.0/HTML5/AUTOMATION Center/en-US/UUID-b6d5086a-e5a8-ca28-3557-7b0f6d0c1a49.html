Get entity coordinates for minimal distance

Get entity coordinates for the minimal distance between curves.

Template commands → AC general → CAD entities → Get entity coordinates for minimal distance

Advanced

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Curve layer

Select the curve layer. Only uses curves that are assigned to this layer.

Curve color

Enter an RGB color value for the entities in the model that are to be included. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Read from cache: Enter the name of the cache from which the data is to be used.

Reference curve layer

Select the curve layer. Only curves that are assigned to this layer are used.

Reference curve color (RGB): Enter an RGB color value for entities present in the model that are to be taken into account. Optionally click the button in the table cell and select an entity of the respective color. The RGB color value is transferred.

Read from cache: Enter the name of the cache from which the data is to be used.

Reference vector: Use a search direction to restrict the determination of the minimal distance. Enter a vector for the direction.

Vector angle tolerance: Set the focus for the search in the direction specified by the reference vector as the angle value.

X->, Y->, and Z->: The X, Y, and Z coordinates of the initial point are output.

X MIN->, Y MIN->, and Z MIN->: The X, Y, and Z coordinates of the point with the minimal distance are output.

Found vector: The determined vector between the two points is output.
