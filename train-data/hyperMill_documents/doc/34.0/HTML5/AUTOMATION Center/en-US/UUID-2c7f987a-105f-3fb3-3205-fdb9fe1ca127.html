Move entities

Move entities in a direction and optionally copy (more than once).

Template commands → CAD → Transformation → Move entities

Advanced

Example 61. Create clamping situations

Distance: 2000 mm, direction vector: 1;0;0, color: blue; sequence without stop layer

Distance: 2000 mm, direction vector: 1;0;0, color: blue; on stop layer: grey faces

Distance: 2000 mm, direction vector: -1;0;0, color: yellow; on stop layer: grey faces

Result



Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Distance: Enter a value for the size of the movement.

Direction vector: Enter a vector for the direction of movement or specify by selecting an entity. The vector should be a unit vector.

Layer

Enter a layer on which the entities to be used are located. Enter a layer name or select an entity on this layer. The ID of an entity can also be entered instead. The input must start with ID:, for example, ID:1234.

Color

Enter or select the color. Only entities of this color are used.

Number of copies: Enter the number of copies.

Stop layer: If faces on these layers or layer containers collide in the direction and distance of the movement, the movement is stopped and not executed for the entire Distance. Enter layer name.

The target layer is only used when copies are enabled. Otherwise, the ‘originals’ are moved.

Target layer

Enter or select a layer for the entities that are generated or modified with this function.

Clone base contour: Duplicate the output chain of curves selected with Layer and/or Color to copy it to the target layer.

Fit view: The view is changed so that all entities are visible in the graphic area.

Tolerance: Specify how exactly the entities to be moved should be moved to the content of the stop layer.

Result

The result of the calculation in the function. Transfer to other functions with the help of the script variable.
