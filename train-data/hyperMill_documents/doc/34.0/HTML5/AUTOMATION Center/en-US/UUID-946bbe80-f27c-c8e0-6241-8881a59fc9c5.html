Load features (Parameter)

Load Customized Process Features or features (Contour, Strategy Curve, Face Group).

Template commands → CAM → Feature → Load features (Parameter)

Advanced

Transfer features from a *.fst file to the hyperMILL® browser on the Feature tab, also called the feature browser.

Folder

Enter or select a folder.

Filename: Enter an existing file name or select a *.fst file.
