2.3. Default folder for the AUTOMATION Center

Company-wide default setting

Select a folder under hyperMILL → Setup → Settings in Application under Default paths in the AUTOMATION Center branch. It is advisable to set up the default path on the AUTOMATION Center server. A search is made there for the following data:

  • ...\Apps: Scripts provided by OPEN MIND or custom scripts to be displayed in the Customer applications branch. Preset folder. Another folder can be prioritized in the configuration.

    Note

    These template commands and pre-specified script components may only be referenced or inserted once per script.

  • ...\CLAMPS: Fixtures that are stored in subfolders for each fixture system. The fixture systems can be optionally arranged according to machine.

  • ...\COMPONENTS: Component file folder for exported components or subcomponents.

  • ...\DATABASE: When working with several, different versions of AUTOMATION Center databases, for example Color_table.xml, MacroDB.db, ToolDB.db and Virtual_tool.vtx, it is advisable to store them here.

  • ...\EXPORTED: Custom, compressed scripts for data exchange or data backup

  • ...\FEATURE: Feature file folder for exported features

  • ...\LIBRARY: CAD data categories with CAD model data

  • ...\MULTIOFFSET: .cfg definition files for milling area multiple allowances

  • ...\REPORTS: *.xsl report layouts

  • ...\STOCKS: *.cfg stock definition files that are stored in subfolders for each stock category.

  • ...\variants: Global script data that is stored in subfolders for each script. Each time the AUTOMATION Center is started, the local scripts are updated if necessary. Switch off these updates using the Update runtime scripts option in Configuration → Process application.

  • ...\VB SCRIPTS: Store VBScript files categorizedin subfolders.

  • ...\PYTHON SCRIPTS: Store Python script files categorizedin subfolders.

Local default settings

In the folder C:\Users\Public\Documents\OPEN MIND\USERS\[Username]\AutomationCenter\...

  • ...\variants: Local storage location for scripts. Each time the AUTOMATION Center is started, the local scripts are updated, if necessary, from the ...\variants folder of the folder selected under hyperMILL → Setup → Settings in Application under Default paths in the AUTOMATION Center branch.

Under Configuration → Process application → Script base folder, select the folder in which scripts provided by OPEN MIND are stored to display them in theCustomer applications branch. This selection overwrites the folder that has been selected in the default path.
