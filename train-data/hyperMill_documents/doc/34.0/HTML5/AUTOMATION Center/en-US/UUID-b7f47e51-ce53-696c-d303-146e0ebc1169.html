Read translation file

Read in translations for script steps.

Template commands → AC general → Helper functions → Read translation file

Advanced

Read in translations for script steps.

Only the languages offered for hyperMILL®and their language codes are supported:

de: German, fr: French, it: Italian, nl: Dutch, es: Spanish, cs: Czech, pl: Polish, ru: Russian, sl: Slovenian, ja: Japanese, ko: Korean. zh_CN: Simplified Chinese, zh_TW: Traditional Chinese, pt_BR: Brazilian Portuguese, tr: Turkish.

Prepare the translations in a text file in *.loc format. The text file must contain the translations in the structure ID[TAB]TRANSLATION. Use a new line for each ID. There must be a tab between the ID and translation!

Example 20. Localization files for German, English and Spanish

project.de.loc

id01    Meine Elemente rot einfärben
id02    Meine Elemente blau einfärben

project.en.loc

id01    Colouring my entities in red
id02    Colouring my entities in blue

project.es.loc

id01    Colorear mis entidades en rojo
id02    Colorear mis entidades en rojo


Use the translation, for example, in Command name, for descriptions, comments, and so on by entering the call local([Developer_name]|#[LOC_ID]). A name must be entered as the work title for [Developer_name] in case there is no localization file. For [LOC_ID], enter the relevant ID of the *.loc file with the translation.

Note

Integrate the reading of the translation file into a script at the beginning of the script.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Select a folder and a translated *.loc file in this folder:

Folder

Enter or select a folder.

Filename

Enter the file name of a file that exists in the selected folder.

Right-click the arrow at the right edge of the entry to open the application if the file extension is assigned in Microsoft Windows. (Like double-clicking a file in Microsoft Windows Explorer).

The currently used language can be determined automatically and the corresponding *.loc file can then be loaded. For this purpose, instead of project.de.loc, for example, enter the call project.$LANGUAGE#.loc.

Optionally enter another, likewise translated *.loc file that will be read in if the file specified in Filename does not exist:

Default filename: Enter the file name of a file that exists in the selected folder.
