Free text or paramter

Freely design a dialog.

Template commands → AC general → User interface → Free text or parameter

Advanced

Freely design a dialog in terms of color, font and icon within a composite UI. Also see Start composite UI.

Command name

Enter a brief explanation of the purpose of the function. This explanation is used as the name in the script.

The character | is not allowed to be used. A * is replaced by the original command name.

Text 1: Enter the name of the parameter or option.

Text 2: Enter a text.

Design the appearance:

Text color 1: Select a color for text 1.

Background colour 1: Select a background color for the area of text 1.

Text color 2: Select a color for text 2.

Background colour 2: Select a background color for the area of text 2.

Text height: Enter a size for the text in points.

Font: Enter a font for the name and the optional text or value to be entered.

Bold: Select whether the name and the optional text or value to be entered are displayed in “bold” font.

Read only: Text 2 is displayed as read-only. Otherwise a text line can be entered.

Optionally display a separate icon in front of the name of the parameter or option:

Icon folder: Enter the folder in which the desired icon is stored.

Icon filename: Enter the file name of the desired icon.

If the script is processed a second or further time, the line created with the input is also displayed in parameter definition mode.
