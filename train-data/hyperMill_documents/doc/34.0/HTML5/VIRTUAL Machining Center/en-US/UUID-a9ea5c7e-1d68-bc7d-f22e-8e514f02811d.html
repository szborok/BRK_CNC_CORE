9.1. Connecting VIRTUAL Machining Center and the machine

Connect the VIRTUAL Machining Center to the Heidenhain controller of the machine.

Connect the VIRTUAL Machining Center to the Siemens controller of the machine.

Show the elements for exchanging data between the CONNECTED Machining and the machine as well as for controlling the machine.

Connect: Establish the connection to the machine. An initial consistency check takes place between the data of the NC program, which was generated with the VIRTUAL Machining Center, and the machine.

Disconnect: Disconnect the VIRTUAL Machining Center and the machine.

Configuration

(only available for Heidenhain controllers)

  1. Clicking Create Connection opens the Configure Heidenhain connection dialog.

  2. Specify the Connection name, enter the Ip address and select the Cnc type.

  3. Click Save to establish the connection.

Select the connection from the list.

Connection

(only available for Siemens controllers)

  1. Clicking the Add connection button opens the dialog for setting up the connection to the Siemens controller.

  2. Define the Name, IP, Key and Config File.

  3. If necessary, use Ping to test the connection.

  4. Click OK to save the data and close the dialog. Select Discard to enter the data again. Select Cancel to close the dialog without saving.

Click the Edit Connection button to edit the current connection to the Siemens controller.

Status

Shows the current status of the connection.

SIK: Identification number or name of the controller.

PC

NC directory: Directory on the PC (local computer) in which the NC file is saved.

NC program: Name of the generated NC file.

Heidenhain CNC

NC directory: Directory on the controller in which the NC file is saved.
