9.4. Controlling the machine

Use CONNECTED Machining to control the machine in single step run mode or program run mode, stop/continue a program, adjust the feedrate and rapid traverse speed or synchronise the current position of the machine with the VIRTUAL Machining Center.

Heidenhain CNC

NC directory: Displays the NC directory defined in the controller's memory.

Change to program run mode: Control the machine in program run mode.

Program run mode activated Program run mode not activated.

Change to single step run mode: Control the machine in single step run mode.

Single step run mode activated Single step run mode not activated.

Load NC program on machine: Load the NC program, which was transferred from CONNECTED Machining to the memory of the controller, on the machine.

Use the Select cnc folder dialog to define a folder on the machine computer for saving the generated NC program.

Create a new folder in the directory structure of the machine computer.

Current NC Program Position in CNC

Block number: Displays the current block number of the machine.

Synchronize NC program position to Virtual Machining Center

The current position of the machine (block number) is transferred to VIRTUAL Machining Center.

Program halt : Stop the program on the machine.

Rapid traverse speed and feedrate are set to 0. The control elements are greyed out. Rapid traverse speed and feedrate cannot be changed.

Double-click to activate the control elements (potentiometers) again. The potentiometers remain at 0 and can be ‘turned up’ by holding down the left mouse button (‘gentle restart’).

Program resume : Continue the program on the machine.

The values set for rapid traverse speed and feedrate are applied. The potentiometers are displayed normally. To change the feedrate or rapid traverse speed, position the mouse pointer within the potentiometer and move while holding down the mouse button.

Potentiometers for rapid traverse speed , feedrate and spindle speed of the machine as a percentage value of the defined initial value in the NC program.

Activate the value for the spindle speed by double-clicking the potentiometer. Change the value by clicking the area outside of the potentiometer (scale) or position the mouse pointer within the potentiometer and move while holding down the mouse button.
