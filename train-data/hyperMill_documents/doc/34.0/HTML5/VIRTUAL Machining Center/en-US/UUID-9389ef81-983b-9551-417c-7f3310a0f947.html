5.3. Simulating the VNC or NC program file

Set the start and end of the program run, approach a position with the machine and analyze, start and stop the simulation, move the machine axes one step at a time and regulate the speed for G0 and G1 movements.

Setting the simulation area – approach a position

Set the start and end of the program run by selecting the lower and upper limits. For this purpose, pick the required start and end points for the program run on the toolpath. You can also define the start and end points directly using the two slide controls and .

Starting and stopping the program run – regulate the speed

Move the program run one step backwards.

Start and stop the program run.

Move the program run one step forwards.

Set the speed for rapids (G0 movements).

Set the speed for the normal feedrate (G1/2/3 movements).

The speed control refers to a path unit. Minimum = 0.01 mm, Maximum = 100 mm. If necessary, interpolation is carried out between the individual blocks to provide a tool movement that is as uniform and consistent as possible. If the increment is not smooth enough, blocks may also be omitted. The increment has no influence on the collision check and material removal simulation.

Double-click one of the two speed controllers to switch the program run to whole blocks, independent of the speed, (the speed controllers are then greyed out). This always applies both to G0 and G1 movements as well as to moving forwards and backwards one step at a time. Double-click one of the two speed controllers again to restore the original behaviour.

Keyboard shortcuts

CTRL+ F: Perform a full-text search when the input focus is on the VNC/NC code display. To do so, click the display window with the right/left mouse button.

Back/Forward key: Move the program run forwards or backwards one step at a time when the input focus is on the simulation control. To do so, click the simulation control elements with the right or left mouse button.
