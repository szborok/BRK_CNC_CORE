7. Configuration

In the VIRTUAL Machining Center configuration operating mode, display the origin and tool table, edit the origin table, configure the display in the graphics window and the simulation.

Origin table: Display the origin table and adjust the origins.

Tool table: Display the tool table with the used tools.

Turret setup: Only available if the VIRTUAL Machining Center has been started in Machining planner mode and in hyperMILL (Job list â†’ Postprocessor), a corresponding Virtual Machine has been loaded.

Equip a turret with static holders and NC tools for turning operations.

Extended settings: Configure the display in the graphics window and the simulation.

Material removal options: When material removal is activated, control the display of the material removal in the simulation.
