2.1. VIRTUAL Machining dialog

Use the VIRTUAL Machining dialog to simulate and approve NC code.

The dialog consists of the following sections:

VIRTUAL Machining

Use the toolbar to select the operating modes of the VIRTUAL Machining Center.

Use the secondary navigation toolbar to control the content of the display area in the different operating modes.

The content of the display area depends on the operating mode.

  • Program run (simulation) operating mode: block display, display of VNC and NC program file and the coordinate values.

  • Project administration operating mode: For displaying the main programs and subprograms.

  • Machine configuration operating mode: Displays the machine structure and toolbars for configuring the display in the graphics area.

  • VIRTUAL Machining Center configuration operating mode: Displays the origin table and tool table.

  • Analysis operating mode: Graphic display of the movement paths of the machine axes and list of breakpoints.

Use the program run control panel to set the speed for rapids and feedrate, define the start and end points of the program run, specify positions and approach with the machine.

Reduce the control element view and reset the original view.

Reduce the control element view so that the graphics area takes up most of the screen. Only the most important control elements are available: start and stop the program run, manually move the machine one step forwards or backwards, start the collision check and simulate the removal of material.

Reset the original view. The direction of the arrow depends on the position of the docked window (left/right side of the monitor).

Two-window mode

To ensure optimal use of the simulation and analysis functions of the VIRTUAL Machining dialog, the Program run (simulation) operating mode can also be executed in a second window. At the same time, for example, the axis movements can be analyzed in the first window in the Analysis operating mode. This configuration can also be run on two monitors. To activate the configuration, proceed as follows:

  • Right-click and activate the NC Simulation and NC Simulation 2 windows.

The configuration is shown in figure of the following overview.

Configuration of the NC simulation

: All operating modes are available in the NC Simulation window:

Program run (simulation), Project administration, Machine configuration, VIRTUAL Machining Center configuration and Analysis.

The functions cannot be used simultaneously.

: All functions of the Program run (simulation) operating mode are available in the NC Simulation 2 window.

The functions of the Project administration, Machine configuration, VIRTUAL Machining Center configuration and Analysis operating modes are available in the NC Simulation window.

The functions of NC Simulation 2 and NC Simulation can be used simultaneously .
