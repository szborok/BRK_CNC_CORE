3. Operating modes

Use the main navigation toolbar to switch between the various operating modes: Project administration, Program run (simulation), Machine configuration, VIRTUAL Machining Center configuration and Analysis.

Project administration operating mode

In large NC programs, load one of several available main programs. If the project only contains one main program, this main program is loaded automatically.

Program run (simulation) operating mode

Simulate the VNC program file or NC program file. Show the control elements for regulating the speed, the block display and the coordinate values. Control how the tool, part, stock and clamp are displayed.

Machine configuration operating mode

Control the display of the machine elements. Position the machine axes manually (manual control). View the axis limits of the machine.

VIRTUAL Machining Center configuration operating mode

Display changes to the NC file, set the minimum feedrate for the rapid, show the toolpaths for linking jobs, display the tool reference point and coordinate systems of the machine in use.

Analysis operating mode

Perform a graphical analysis of the movement paths of the machine axes, define breakpoints and conditions for breakpoints, display breakpoints as a list and display breakpoint coordinates.
