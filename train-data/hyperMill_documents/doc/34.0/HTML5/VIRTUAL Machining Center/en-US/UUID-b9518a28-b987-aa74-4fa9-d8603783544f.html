7.3. Extended settings

NC data

Show line numbers for VNC data: Displays the line numbers in the VNC program file/NC file. Both numbers correlate, however, they are not identical with the line numbers of the NC program that is output.

Simulation

Minimum feedrate for rapid movements: G1 movements with a higher feedrate (than the defined value) are handled as rapid tool movements. Here, the feedrate is replaced by the default feedrate for G0 movements. The toolpath is displayed in red in the program run and the display also corresponds to G0 movements.

Number of replay steps: Specify the number of steps that the machine travels in order to approach a toolpath point again that was defined using the Pick new position function.

Simulation tolerance: Accuracy of the simulation for the collision check.

Simulate machine cycles as a single operation: Machine cycles such as Simple Drilling or Circular Pocket can be simulated separately as a single operation.

Appearance

Toolpaths

Show at simulation start: Activates the loading of the toolpaths at the start of the VIRTUAL Machining Center / SIMULATION Center, in order to show them directly. Activating this option will cause a longer starting time for the VIRTUAL Machining Center / SIMULATION Center. The setting of this option is automatically used as default.

The toolpaths can be shown after the start of the VIRTUAL Machining Center / SIMULATION Center at any time by using the Show toolpaths button on the Simulation dialog page. Showing the toolpaths for the first time results in a loading process.

Note

The status of the toolpath display is also taken into account during a program change (if, for example, several programs are loaded). If the option is activated, the toolpaths for the selected program are displayed. If the option is deactivated, the toolpaths for the selected program are not displayed.

Show linking movements: Display the toolpaths of linking jobs plus all linking movements between the jobs contained in the linking job in the program run. They are shown as blue lines independently of the feedrate.

Note

The Show linking movements function is not available for the SIMULATION Center.

Show points: Define the color and size of tool vectors.

Show tool vectors: Define the color and size of tool vectors.

Show tool reference point: Display the reference point defined in the tool definition in the graphics area. Tool reference point: Tip , Middle .

Set width of tool paths: Define the display width of the toolpaths in the graphics area.

Show only: To make things clearer when very complex toolpaths are present, limit the display to an area before and after the tool reference point.

Show coordinate systems

Display the coordinate system of the following elements: Machine, Part, Active origin, Active transformation, Origin table reference.

Set rotation target to origin of graphics system: Set the position as the center point for moving the view when rotating to the origin of the graphics system.

Activate material removal if stock is present: Material removal is automatically activated if stock is present.

Use different job colors in material removal: In a machining sequence with material removal, the material to be removed is shown in different colors.

Use job information in material removal: If this option and material removal are activated, clicking on the model displays information on the respective job.
