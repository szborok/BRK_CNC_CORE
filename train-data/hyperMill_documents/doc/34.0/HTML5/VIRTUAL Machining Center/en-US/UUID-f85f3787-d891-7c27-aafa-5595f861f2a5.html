6.3. Limits of the machine axes

Displays the limits (Minimum/Maximum) of the linear and rotary axes of the machine and the movement of the machine coordinate system. The values cannot be edited.

Use the secondary navigation toolbar to switch between machine configuration , manual control and the display of the machine axis limits .
