4.2. Move setup

Move the setup (Part / Stock / Clamp) on the machine.

Select the reference for moving the clamp. Machine axis that is moved using the buttons. Double-click a coordinate value to make it editable and adapt as required. Click OK to confirm. Show the scale for setting the step size. Select the required option to set the coordinate values of the machine axes. Restore the original setup position.

Move part, Move stock, Move fixture: Select the elements to be moved.
