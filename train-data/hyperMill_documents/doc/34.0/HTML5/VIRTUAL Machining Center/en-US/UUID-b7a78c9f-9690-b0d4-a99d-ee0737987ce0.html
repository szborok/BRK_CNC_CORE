6.2. Positioning machine axes manually

Manually position the linear and rotary axes of the machine (X, Y, Z, A, B, C) and test various machining situations on the part under real conditions to determine the optimal tool length or unclamping of the tool, for example.

Use the secondary navigation toolbar to switch between machine configuration , manual control and the display of the machine axis limits .

Enter coordinate values for linear and rotary axes of the machine coordinate system (Machine) or the NC program file (NC) directly. Values that are outside of the limits are displayed in red.

Click a button (X, Y, Z, A, B) to activate the input field (Machine/NC). Double-click the respective coordinate value to make it editable and adjust in order to move the machine to the required position.

Machine axis that is moved using the buttons or the slide control for stepwise positioning (displayed in grey).

Move the active axis to the lower/upper limit.

Reduce or increase the position of the machine axes (X, Y, Z, A, B) one step at a time.

Show the scale for setting the step size. Select the required option to set the coordinate values of the machine axes (X, Y, Z, A, B, C).

Move the active axis using the slide control.

Display of coordinate values of the active axis. The value can be changed by entering another value.

Move the machine axes to the original position.

Move along tool axis: Move the machine along the tool axis.

Note

If the Move along tool axis option is selected, the step size does not refer to an axis but instead specifies the movement in the direction of the tool axis. The allocation of this step size to the movements of all tool axes is calculated automatically. The affected tool axes are displayed in grey.

Trans / RTCP: Turn RTCP* (Rotation Tool Center Point) on or off.

* Term is controller-dependent. Also used: TCPM (Tool center Point Management), TRAORI.

Run to workplane: Move the tool or spindle (if no tool is shown) to the workplane origin. If several traversing options are possible as a result of the machine kinematics, they can be activated as solutions.

Solutions

Calculate solutions: The possible solutions are calculated and displayed based on the machine data and the machine configuration settings. = activated solution. = another solution.

Update: The data is recalculated based on the values for the displayed rotary axes.
