2. User interface

The application is adapted to the use of wide screen formats and for working with multiple screens. You can also place variable areas outside of the program window, that is, you can move them to another screen. When you exit a document the last window positions are saved.

The user interface consists of the following elements:

The title bar provides information about the name of the currently opened document. The F11 key hides the title bar.

Software functions can be accessed via the menu bar . It is static and cannot be changed.

The graphics area is used to display the machine, the part, the stock and other elements .

CONNECTED Machining dialog: The dialog used to operate the CONNECTED Machining , see CONNECTED Machining dialog.

Progress indicator: Displays the progress for computing-intensive processes, and information is provided on the current action within the process (if possible).
