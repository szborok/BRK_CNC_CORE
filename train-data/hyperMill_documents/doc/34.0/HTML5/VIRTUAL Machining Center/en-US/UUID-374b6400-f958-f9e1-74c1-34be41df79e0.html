2.2. CONNECTED Machining dialog

Use the CONNECTED Machining dialog to connect to the machine, perform consistency checks and control the machine.

The dialog consists of the following sections:

CONNECTED Machining

Connect to the machine and configure. Check the status of the machine.

Transfer data from the machine and update.

Transfer data from the PC to the machine and control the machines. Switch between single step run mode and program run mode. Load the NC program on the machines. Only available if there is a connection to the machine. Click the button to display.

Display the machine data. The following are displayed: warnings and alarms, origin table, tool table, and axes configurations. To perform a consistency check, load the machine data by clicking the button . Possible errors become visible and can be corrected.

Connect CONNECTED Machining to the machine or disconnect.

Transfer the current NC program position from the machine. Start or stop the program on the machine, control the feedrate and the rapid traverse speed, adjust the spindle speed. Only available only if VIRTUAL Machining Center and machine are connected. Click the button to display.

Reduce the control element view and reset the original view.

Reduce the control element view so that the graphics area takes up most of the screen. Only the most important control elements are available: start and stop the program run, manually move the machine one step forwards or backwards, start the collision check and simulate the removal of material.

Reset the original view. The direction of the arrow depends on the position of the docked window (left/right side of the monitor).
