7.2. Tool table

Information on the used tools is displayed in the Tool table:

ID, Nr, Register, Active, Name, L =length, R = radius, DL = additional length compensation and DR = additional radius compensation. The values for L and R are taken from the tool definition and the values for DL and DR are stored in a table.

Position the mouse pointer over the column boundary to adapt the size of a table column.

Double-click a table row to open the hyperMILL tool dialog and display all geometry and technology parameters of the tool.

The tool currently used in the simulation is marked as the active tool.

Load a tool: Load a tool file in .xml format.

Note

It is only possible to load tool files that are compatible with the neutral tool exchange format of the tool database.
