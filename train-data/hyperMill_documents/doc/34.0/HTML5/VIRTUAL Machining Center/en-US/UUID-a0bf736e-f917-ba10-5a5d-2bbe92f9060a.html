8. Analysis

In the Analysis operating mode, perform a graphical analysis of the movement paths of the linear and rotary axes, insert, activate or delete breakpoints, display a list of all breakpoints.

Graphical analysis: Perform a graphical analysis of the movement path of the machine axes and configure the display.

Display the movement path of the linear axes (X, Y, Z).

Display the movement path of the rotary axes.

Display the Feedrate and/or the Spindle speed.

Breakpoints: Insert breakpoints, define conditions for breakpoints, display breakpoints as a list.

Variables: Use control-specific default settings and parameters.
