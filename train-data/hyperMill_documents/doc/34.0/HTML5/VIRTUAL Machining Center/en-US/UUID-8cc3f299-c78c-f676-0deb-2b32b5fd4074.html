8.1. Analysing the movement path of the machine axes

Coordinates (linear axes) or degrees (rotary axes).

Collision area (highlighted in red).

Movement path of the Z axis, the Y axis, and the X axis.

Length of the toolpath.

Use the F and S buttons to show the feedrate and spindle speed.

Tooltip: Position the mouse pointer over a button (X, Y, Z, A, B) to show the minimum and maximum value of the axis.

Use the X, Y, Z, A, B buttons to show the graphical movement path of the linear axes and rotary axes.

Coordinate grid.

Absolute Display the coordinate values as absolute values. Delta: Show the coordinate values as the difference between two blocks.

Zoom: Zooming (enlarging or reducing) is performed both in the horizontal direction when displaying the coordinate values and in the vertical direction when displaying the block display.

Zoom vertical: Zooming (enlarging or reducing) is only performed in the vertical direction when displaying the coordinate values. The representation of the block display does not change.

Zoom horizontal: Zooming (enlarging or reducing) is only performed in the horizontal direction during the block display. The representation of the coordinate values does not change.

The information also applies to the analysis of the feedrate values and the spindle speed.

Visible grid: Show a coordinate grid in order to obtain a better overview.

Current simulation interval only: Display information for the current simulation interval.

Analysing the machine positions and travel range

Movement path of the Z axis, current position of the machine,

Upper limit, lower limit (travel range of the Z axis).

Keyboard shortcuts and mouse functions for controlling the analysis

CTRL key and right mouse button: Move the analysis window.

CTRL key and left mouse button: Set a position and move the machine to this position.

Left mouse button: Open a zoom window for zooming.

Right mouse button/mouse wheel: Zoom out or in. Both functions are designed as a stack function: If you zoom in three times and zoom out twice you will reach the state that existed after the first zoom in.

SHIFT key and mouse wheel/middle mouse button: After zooming out, zoom in again to a previously selected position.

SHIFT key and right mouse button: Zoom out to the original standard view again.

Double-click (right/left mouse button): Create a new breakpoint.
